## 摘要
开放环境下的桌面机器人操作要求系统具备语义理解、精确几何位姿估计和高频动作生成能力。虽然端到端的视觉-语言-动作（VLA）模型在语义泛化方面表现优异，但它们在细粒度任务中往往缺乏显式的几何约束，并且需要昂贵的训练成本。为了弥合高层语义与底层物理执行之间的鸿沟，我们提出了OpenSPM，这是一个开放环境空间持久记忆框架，由空间位姿记忆和流匹配动作生成模型组成。OpenSPM首先利用语义条件化的三维感知和卡尔曼滤波来跟踪连续的六维位姿。然后，它从人类示范中提取关键空间位姿，并将其保存为可迁移的、以物体为中心的空间持久记忆条目。在推理过程中，OpenSPM根据自然语言指令检索相关记忆条目，利用SE(3)变换将空间位姿迁移到新场景，并通过轻量级条件流匹配模型生成高频动作块。结合实时本体感受状态反馈和终端残差校正，该系统有效抑制了轨迹误差累积。在十个LIBERO-GOAL任务上的评估表明，OpenSPM达到了85.6%的成功率和1033.3 Hz的等效控制频率，同时所需的推理AI算力极低。广泛的消融实验表明，结构化的空间持久记忆和闭环残差校正在可靠的高频机器人操作中起着关键作用。

## Abstract
Open-environment tabletop robotic manipulation requires systems to possess semantic understanding, precise geometric pose estimation, and high-frequency action generation. While end-to-end vision-language-action (VLA) models excel at semantic generalization, they often lack explicit geometric constraints for fine-grained tasks and require costly training. To bridge the gap between high-level semantics and low-level physical execution, we propose OpenSPM, an open environment spatial persistent memory framework consisting of spatial pose memory and flow-matching action generation model. OpenSPM first leverages semantically conditioned 3D perception and Kalman filtering to track continuous 6D poses. It then extracts key spatial poses from human demonstrations, keeping them as transferable, object-centric spatial persistent memory entries. During inference, OpenSPM retrieves relevant memory entries in terms of natural language instructions, transfers the spatial poses to new scenes using SE(3) transformations, and generates high-frequency action chunks via a lightweight conditional flow-matching model. Combined with real-time proprioceptive state feedback and terminal residual correction, the system effectively suppresses trajectory error accumulation. Evaluated on ten LIBERO-GOAL tasks, OpenSPM achieves an 85.6% success rate and an equivalent control frequency of 1033.3 Hz, while requiring minimal inference AI computing power. Extensive ablations illustrate that structured spatial persistent memory and closed-loop residual correction play a crucial role in reliable, high-frequency robotic manipulation.

### 核心思想
将高层次语义理解与低层次几何执行解耦：高层次模型负责语义解析与经验检索，低层次模型负责在传递的几何约束下生成局部动作。

### 关键技术细节

- **语义条件化 3D 感知与卡尔曼状态估计**
  - 多视图图像 → 语义分割（SAM 3） → 3D 重建（SAM 3D） → 卡尔曼滤波预测/更新 → 连续 6D 位姿（位置 + 方向）。
  - 状态向量包含位置、速度、局部方向误差、角速度；观测协方差由分割置信度、多视图一致性等联合决定。
  - 输出经轻量平滑（位置指数平滑、方向球面插值）。

- **关键空间位姿记忆库构建**
  - 从人类演示中提取**关键位姿节点**，标记阶段边界（接近、抓取、提升、放置、释放）。
  - 存储为以物体坐标系为基准的相对位姿 $\Delta T_{o \rightarrow e}^{j} = (^w T_o)^{-1} \cdot ^w T_{e,j}$，保证了跨场景的可迁移性。
  - 记忆条目包含检索键（操作类型、目标物体、阶段结构）、语言描述、相对位姿序列、阶段标签、置信度、质量评分等。

- **在线检索与相对位姿迁移**
  - 根据语言指令解析结构化任务要素，通过语义匹配（$S_{sem}$）和几何可行性（$S_{geo}$）从记忆库中选择最佳条目。
  - 在测试场景中，利用当前目标物体位姿 $^w T_o'$ 将记忆中的相对位姿还原到世界坐标系：$^w T_{e,j}' = ^w T_o' \cdot \Delta T_{o \rightarrow e}^{j}$。
  - **定理1**证明：若目标物体位姿经历同一刚体变换，则局部相对位姿保持不变，即几何关系可无损迁移。

- **条件流匹配动作生成**
  - 输入条件：当前实际夹爪位姿、下一个目标关键位姿、相对误差和段内进度 $\tau$。
  - 训练：将真实动作块 $a_{data}$ 与高斯噪声 $a_0$ 线性插值，学习速度场 $v_\theta(a_\xi, \xi, c)$，损失为均方误差。
  - 推断：从高斯噪声开始沿 ODE 积分 $H=5$ 步，得到动作块 $\hat{a}_{t:t+H-1}$。
  - 模型仅 240K 参数，条件编码器复用隐向量，适合高频调用。

- **闭环执行与终端残差校正**
  - **段间闭环**：每个动作块开始时使用**实时本体感受位姿**作为条件，而非规划的理想起点。
  - **终端残差校正**：在关键接触节点前，若当前位置与目标位置残差 $r_t$ 超过阈值 $\epsilon_r$，则利用当前真实状态重新生成校正动作块，直至误差收敛或达到最大迭代次数。

- **误差传播分析**：闭环反馈等效于用观测值替换递归规划误差，降低累积；残差校正进一步在接触前压缩误差。

## 6. 论文的主要结论与发现

1. **结构化的空间持久记忆**（关键位姿节点）是可靠操作的基座：零样本多模态模型无法可靠预测物理接触点，记忆库提供了经过验证的几何先验。

4. 失败主要由感知误差、物体公差过小和环境几何冲突引起，而非动作模型本身。

## 7. 优点

- **显式几何约束**：以物体坐标系相对位姿为中间表示，使记忆可迁移、动作几何可控。
