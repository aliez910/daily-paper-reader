## 摘要
开放环境下的桌面机器人操作要求系统具备语义理解、精确的几何位姿估计和高频动作生成能力。虽然端到端的视觉-语言-动作（VLA）模型在语义泛化方面表现出色，但它们通常缺乏细粒度任务所需的显式几何约束，并且训练成本高昂。为了弥合高层语义与低层物理执行之间的差距，我们提出了OpenSPM，一种开放环境下的空间持久记忆框架，由空间位姿记忆和流匹配动作生成模型组成。OpenSPM首先利用语义条件化的3D感知和卡尔曼滤波来跟踪连续的6D位姿。然后，它从人类演示中提取关键空间位姿，将其保存为可迁移的、以物体为中心的空间持久记忆条目。在推理过程中，OpenSPM根据自然语言指令检索相关记忆条目，利用SE(3)变换将空间位姿迁移到新场景，并通过轻量级条件流匹配模型生成高频动作块。结合实时本体感觉状态反馈和终端残差校正，该系统有效抑制了轨迹误差累积。在十个LIBERO-GOAL任务上的评估表明，OpenSPM达到了85.6%的成功率和等效1033.3 Hz的控制频率，同时只需要极少的推理AI算力。大量消融实验表明，结构化的空间持久记忆和闭环残差校正在可靠、高频的机器人操作中起着关键作用。

## Abstract
Open-environment tabletop robotic manipulation requires systems to possess semantic understanding, precise geometric pose estimation, and high-frequency action generation. While end-to-end vision-language-action (VLA) models excel at semantic generalization, they often lack explicit geometric constraints for fine-grained tasks and require costly training. To bridge the gap between high-level semantics and low-level physical execution, we propose OpenSPM, an open environment spatial persistent memory framework consisting of spatial pose memory and flow-matching action generation model. OpenSPM first leverages semantically conditioned 3D perception and Kalman filtering to track continuous 6D poses. It then extracts key spatial poses from human demonstrations, keeping them as transferable, object-centric spatial persistent memory entries. During inference, OpenSPM retrieves relevant memory entries in terms of natural language instructions, transfers the spatial poses to new scenes using SE(3) transformations, and generates high-frequency action chunks via a lightweight conditional flow-matching model. Combined with real-time proprioceptive state feedback and terminal residual correction, the system effectively suppresses trajectory error accumulation. Evaluated on ten LIBERO-GOAL tasks, OpenSPM achieves an 85.6% success rate and an equivalent control frequency of 1033.3 Hz, while requiring minimal inference AI computing power. Extensive ablations illustrate that structured spatial persistent memory and closed-loop residual correction play a crucial role in reliable, high-frequency robotic manipulation.

## 2. 方法论

- **核心思想**：将机器人操作分解为四个阶段：语义3D感知、结构化关键位姿记忆、相对位姿迁移、局部动作生成。关键空间位姿作为中间表示，标记操作阶段边界（如接近、抓取、放置等），且以末端执行器与目标物体的相对位姿存储，利用 SE(3) 变换实现跨场景迁移。
- **关键技术细节**：
  - **语义条件3D感知**：级联结构包含多视图语义分割（基于文本提示）、3D重建（SAM 3D）和卡尔曼状态估计。卡尔曼滤波对目标物体进行连续6D姿态跟踪，观测协方差由分割置信度、掩码面积、多视图一致性等联合确定，并辅以指数平滑和球面线性插值。
  - **关键空间位姿记忆库构建**：从演示轨迹中提取关键位姿，依据是夹爪状态变化、末端速度/朝向变化以及多模态模型（Qwen3-VL）的语义判断。每个记忆条目包含操作类型、相对位姿（相对于锚定物体）、阶段标签、置信度、质量评分和版本信息。采用语义相似度和几何一致性联合评分进行在线检索。
  - **相对位姿迁移**：通过公式 \( {}^w T'_{e,j} = {}^w T'_o \cdot \Delta T^{o \to e}_j \) 将演示中的相对位姿迁移到新场景，并证明了在目标物体经历任意 SE(3) 变换时，局部交互几何保持不变（定理1）。
  - **条件流匹配动作生成**：使用轻量（240K参数）条件流匹配模型，在相邻关键位姿之间生成长度为 H=5 的动作块。条件包括当前实际位姿、目标位姿、相对误差和段内进度。训练时构造线性插值路径，学习速度场；推理时通过 ODE 积分生成动作。
  - **闭环执行与终端残差校正**：每个动作块开始时使用实时本体感受状态更新条件（段内闭环）；在抓取和放置等关键节点附近，若当前位置与目标位姿残差超过阈值，则重新生成校正动作块，直到收敛。
- **伪算法流程**：
  1. 离线：多视图图像 → 语义分割 + 3D重建 → 卡尔曼平滑 → 提取关键相对位姿 → 存入记忆库。
  2. 在线：语言指令 → 解析结构化元素 → 检索最佳记忆条目 → 感知当前物体位姿 → SE(3) 迁移关键位姿序列 → 循环：读取当前末端位姿 → 调流匹配生成 H=5 动作块 → 执行并检查残差 → 必要时校正 → 直至到达下个关键位姿。

## 6. 主要结论与发现

- OpenSPM在LIBERO-GOAL上达到85.6%成功率，等效动作频率1033.3 Hz，参数仅0.24M。

- 轻量流匹配模型只需学习低维局部动作，避免处理图像/语言token，适合高频调用。
- 显式几何与记忆是端到端VLA的重要补充，尤其适用于少量样本、高实时、细粒度操作场景。

## 7. 优点

- **显式几何约束**：通过物体中心相对位姿提供可迁移、可验证的交互几何，降低了VLA的不可解释性。
- **问题分解降低复杂度**：感知、记忆、生成、控制各模块职责清晰，使轻量模型能在有限数据下高效训练。
- **环境可迁移性**：相对位姿经SE(3)变换即可适应新场景，无需重新训练。
- **高效闭环控制**：实时状态反馈+残差校正有效抑制误差累积，且计算开销极小（每秒上千次）。
- **极低推理成本**：240K参数的流匹配模型，适合资源受限的机器人平台。
