## 摘要
用于机器人模仿学习的人类演示通常包含错误和纠正行为，例如不精确的抓取、物体错位、不稳定接触以及重复尝试。虽然这些片段通常被视为噪声或次优数据，但它们提供了关于执行何时偏离期望路径以及如何恢复任务可行性的宝贵证据。然而，现有的奖励和价值模型通常依赖单调进展假设，这些假设捕捉了粗略的任务进展，但可能忽略了不完美演示中的局部执行错误和纠正行为。在这项工作中，我们提出了ReTVL（重试监督价值学习），一种通过利用重试事件作为稀疏监督来从混合质量机器人演示中学习错误敏感价值函数的框架。ReTVL结合全局进展校准与由稀疏标注的重试关键点引发的局部成对偏好学习，捕获了错误周围的局部退化-恢复结构。然后，学习到的价值模型被用于重新加权演示片段以进行下游行为克隆，减少有害执行错误的影响，同时保留有用的纠正行为。在真实机器人操作任务上的实验表明，ReTVL产生了比基于进展的基线更细粒度的价值估计，并改善了从不完美演示中的模仿学习。

## Abstract
Human demonstrations for robot imitation learning often contain mistakes and corrective behaviors, such as imprecise grasps, object misalignment, unstable contact, and repeated attempts. While these segments are commonly treated as noisy or suboptimal data, they provide valuable evidence about when execution deviates from a desirable path and how task feasibility can be restored. However, existing reward and value models often rely on monotonic progress assumptions, which capture coarse task advancement but may overlook local execution errors and corrective behaviors in imperfect demonstrations. In this work, we propose ReTVL (ReTry-Supervised Value Learning), a framework for learning mistake-sensitive value functions from mixed-quality robot demonstrations by leveraging retry events as sparse supervision. ReTVL captures the local degradation-and-recovery structure around mistakes by combining global progress calibration with local pairwise preference learning induced by sparsely annotated retry keypoints. The learned value model is then used to reweight demonstration chunks for downstream behavior cloning, reducing the influence of harmful execution errors while preserving useful corrective behaviors. Experiments on real-robot manipulation tasks show that ReTVL produces more fine-grained value estimates than progress-based baselines and improves imitation learning from imperfect demonstrations.

## 2. 论文提出的方法论
**核心思想**：通过稀疏标注的重试关键点（retry keypoints，即开始纠正的时刻）构建局部成对偏好，使价值函数在错误附近形成“下降-反弹”的结构，同时结合全局进展校准保证整体尺度合理。

### 关键技术细节
- **价值函数定义**：  
  使用分布价值头预测K个离散进展箱的概率，以期望值作为最终价值 \(V_\theta(h_t, \ell) = \sum b_k p_\theta(k \mid h_t, \ell)\)。
- **全局进展监督**：  
  对重试区域以外的帧施加绝对进展标签（成功轨迹中 \(v^*_t = t/T\)，失败轨迹终帧为0），通过交叉熵损失 \(L_{\text{abs}}\) 约束。
- **局部成对偏好学习**：
  - 对每个重试关键点 \(r_{i,j}\)，定义三个时间窗：\(W_{\text{pre}}\)（关键点前远离区）、\(W_{\text{near}}\)（关键点附近）、\(W_{\text{post}}\)（关键点后恢复区）。
  - 构造偏好对 \((h^+, h^-)\)：预期在 \(W_{\text{near}}\) 的价值最低，\(W_{\text{pre}}\) 和 \(W_{\text{post}}\) 的价值较高。通过软窗口权重 \(w(h^+, h^-) = \exp(-d(h^+; r_{i,j}) / \tau_w)\) 降低远距离样本的贡献。
  - 偏好损失采用加权逻辑损失：  
    \[
    L_{\text{pref}} = -\mathbb{E}_{(h^+, h^-)} w(h^+, h^-) \log \sigma\left(\frac{V_\theta(h^+, \ell) - V_\theta(h^-, \ell)}{T_{\text{pref}}}\right)
    \]
- **总训练目标**：\(L_{\text{value}} = \lambda_{\text{abs}} L_{\text{abs}} + \lambda_{\text{pref}} L_{\text{pref}}\)，两种损失以固定采样比例混合。

### 算法流程（文字说明）
1. 每轮训练，从数据集中抽取轨迹。
2. 从非重试区域采样全局进展样本，计算 \(L_{\text{abs}}\)。
3. 从重试关键点构造三窗口并采样偏好对，计算软权重，计算 \(L_{\text{pref}}\)。
4. 联合更新价值模型（使用 LoRA 微调 VLM 骨干，训练价值头）。

## 7. 优点

- **细粒度的价值建模**：同时捕捉全局进展和局部错误-恢复结构，弥补了单调假设的不足。
- **实用效果**：在下游行为克隆中微妙的加权机制显著提升了成功率，且通过分析显示成功保留了恢复片段（Post-retry Weight 最高）并大幅抑制了有害片段（Strict-bad Retention 0.206，远低于其他方法）。
- **软窗口机制**：考虑标注边界不确定性，降低了训练噪声，使方法更鲁棒。
- **实验设计较完整**：包含价值评估、策略评估、消融、权重分析等多层次验证，且控制了骨干和训练条件。

## 8. 不足与局限
- **假设适用范围**：假设价值围绕重试呈“下降-反弹”模式，无法涵盖所有真实纠正行为（如探索式重试或失败的纠正）。
- **仅限离线重加权**：未将价值模型用于在线 RL 或闭环策略改进。
- **实验规模有限**：仅4个真实任务，每任务数十条轨迹，未在大规模多元化数据集上验证泛化性（如不同机器人、不同演示风格）。提出的方法可能对特定任务/数据噪声模式敏感。
- **标注依赖**：虽然标注成本低，但仍需人工标注重试关键点；对缺乏重试标签的数据集无法直接应用（将来可通过自动检测扩展）。
- **其他潜在局限**：仅使用主相机视图作为输入，忽略了机器人本体感觉和腕部信息对价值判断的贡献（策略训练使用了双视角和 proprioception，但价值模型仅用主视角）；未与更多最新的 VLA 价值学习方法（如 Robo-reward, SARM）进行对比（但实验中尽可能覆盖了代表方法）。
