## 摘要
视觉-语言-动作（VLA）模型已成为具身人工智能的重要范式。然而，现有的VLA模型通常假设光照良好且稳定的室内环境，而现实世界中的具身操作可能涉及由光照变化引起的降质RGB观测，这对鲁棒的机器人操作构成了关键挑战。为解决这一问题，我们提出了Event-VLA，一种事件增强的VLA框架，用于在不同光照条件下实现可泛化的操作。我们将降质可见性下的VLA操作形式化为以RGB为中心的策略的鲁棒性实际问题，并引入事件流作为一种对照明鲁棒、对运动敏感的补充观测，以提升跨可见性水平的鲁棒性。具体来说，与直接将事件特征合并到全局语义标记空间中的传统多模态融合不同，Event-VLA通过一个动作查询路由路径注入事件信息。它使用可学习的动作查询从VLA推理过程中提取与任务相关的语义，并通过门控交叉注意力选择性地聚合事件标记，以构建事件感知的动作表示。这种设计保留了预训练的RGB-语言语义先验，同时有效利用事件信息进行鲁棒的动作预测。在仿真和实际部署中的实验表明，Event-VLA在正常光照下保持强大的操作性能，并在低光照降级和接近黑暗的现实环境中提高了成功率。

## Abstract
Vision-Language-Action (VLA) models have become an important paradigm of embodied AI. However, existing VLA models typically assume well-lit and stable indoor settings, while real-world embodied manipulation may involve degraded RGB observations caused by illumination shifts, posing critical challenges for robust robotic manipulation. To address this gap, we propose \textbf{Event-VLA}, an event-enhanced VLA framework for generalizable manipulation across varying illumination conditions. We formulate VLA-based manipulation under degraded visibility as a practical robustness problem for RGB-centric policies, and introduce event streams as an illumination-robust, motion-sensitive complementary observation to improve robustness across visibility levels. Specifically, unlike conventional multimodal fusion that directly merges event features into the global semantic token space, Event-VLA injects event information through an action-query routing pathway. It uses learnable action queries to extract task-relevant semantics from the VLA reasoning process, and selectively aggregates event tokens via gated cross-attention to construct event-aware action representations. This design preserves the pretrained RGB-language semantic priors while effectively leveraging event information for robust action prediction. Experiments in simulation and real-world deployment show that Event-VLA maintains strong manipulation performance under normal lighting and improves success rates under low-light degradation and near-dark real-world settings.

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：事件信息应被当做动作时刻的物理残差（而非全局语义），通过一个受控的接口注入到动作表征中，而不参与 VLA 主干的全局语义自注意力。即：语义上下文由预训练 VLA 建模，事件特征以动作条件的方式路由到动作头部。
- **整体架构**：Event-VLA 包含三个主要模块：
    1. **PREI 事件表征层**：将异步事件流压缩为紧凑的物理残差图（三通道：瞬时、显著、持久）。
    2. **动作条件的事件接口**：使用可学习查询令牌（公共查询、动作查询、事件查询）通过 VLA 主干进行上下文化，然后通过门控交叉注意力将事件令牌与 VLM 隐含令牌融合，并通过查询引导路由分离出动作相关和事件相关的令牌。
    3. **辅助未来事件预测任务**：作为训练时的正则项，鼓励融合表征保留瞬态物理信息。
- **PREI（Physical Residual Event Integration）**：
    - 计算极性无关的活动图 A_τ(u) = Σ exp(-(t-τ_i)/τ)，并生成三通道残差图：
        - E_ins：瞬时通道，强调最近局部变化。
        - E_sal：显著通道，在较长尺度上突出局部活动（通过本地平滑归一化）。
        - E_per：持久通道，记录窗口内累积事件计数。
    - 使用 tanh 归一化将各通道映射到 [0,1]。
- **事件编码器**：将 PREI 图编码为 VLA 兼容的令牌（全局令牌 + 空间令牌）。通过特征蒸馏（使用冻结的 DINO 视觉教师，在 N-ImageNet 和 LIBERO 合成数据上）对齐事件特征与 VLA 隐空间。
- **动作条件融合与路由**：
    - 预训练 VLA 主干输入：RGB 视觉令牌、本体感知令牌、语言令牌、动作占位符、以及三种可学习查询令牌（公共 Q_c、动作 Q_a、事件 Q_e）。事件令牌不进入主干。
    - 主干输出上下文化隐含令牌 H_t，包括 Z_lm（视觉、语言等上下文令牌）以及上下文后的查询 Q_c^t, Q_a^t, Q_e^t。
    - 事件令牌 Y_et 通过门控交叉注意力与 Z_lm 融合：
        - Z_attn = CrossAttn(W_q Z_lm, W_k(Y_et+T_pos), W_v(Y_et+T_pos))
        - 门控 Γ = σ(f_gate(Z_lm, Q_c^t, Q_a^t, Q_e^t, Z_attn))
        - Z_fused = Norm(Z_lm + Γ ⊙ Z_attn)
    - 通过查询引导路由分离：
        - Z_act = Route_act(Z_fused, Q_c^t, Q_a^t) → 送入动作头预测动作块
        - Z_evt = Route_evt(Z_fused, Q_c^t, Q_e^t) → 送入事件头预测未来 PREI
- **训练目标**：
    - 动作损失：L_act = L1 距离（预测动作块与真实动作块）。
    - 事件辅助损失：在未来 PREI 上使用内容掩码的 L1 损失 L_evt，以及梯度一致性损失 L_deriv。
    - 总损失：L = L_act + λ_evt L_evt + λ_deriv L_deriv（λ_evt=0.1, λ_deriv=0.3）。
- **推理**：事件头部仅在训练时使用，推理时不需未来 PREI 目标；事件融合路径始终可用。

## 6. 论文的主要结论与发现

1. **Event-VLA 在正常光照下保持与最佳 RGB 基 VLA 相当的性能**（表 1：平均 96.5% vs 96.3% 等），且禁用事件路径后性能基本不变（Ours w/o event: 96.2%），表明事件接口不干扰预训练主干。

## 7. 优点：方法或实验设计上的亮点

- **实验亮点**：
    - 构建了 LIBERO-Cross 基准，系统评估多种退化等级下的鲁棒性，填补了现有 VLA 评估的空白。

## 8. 不足与局限

- **模拟器局限**：LIBERO-Cross 依赖 RGB 到事件的模拟器（v2e + 预测网络），可能无法完全捕捉真实事件相机的噪声、触发动态和校准误差。虽然真实实验部分缓解了此问题，但仿真与真实之间的差距仍然存在。
- **真实评估规模有限**：仅包含 4 个任务、单一机器人平台（Franka）、三种光照条件。更多物体、环境、长时任务和多机器人的验证仍然欠缺。
- **适用范围**：本文主要针对光照引起的视觉退化，对运动模糊、遮挡等其他退化类型的鲁棒性未充分探讨。
- **硬件依赖**：需要额外的事件相机硬件和同步设备，增加了部署复杂性和成本。
- **对比方法**：虽然对比了多种最新 RGB 基方法，但缺少与其他事件增强 VLA 方法的比较（可能因为没有已发表的工作）。事件融合基线仅来自自身消融，缺乏更广泛的文献对比。
