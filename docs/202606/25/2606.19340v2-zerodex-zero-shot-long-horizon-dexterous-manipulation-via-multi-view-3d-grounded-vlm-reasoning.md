## 摘要
我们提出了ZeroDex，一个用于长时程灵巧操作的零样本框架，它通过校准的多视图RGB图像将语言指令转化为可执行的3D任务计划。我们的系统不是训练端到端策略，而是使用视觉语言模型（VLM）生成参考帧任务接地和原始级别2D关键点，然后通过多视图融合将它们提升到3D。这种提升结合了视图级VLM接地的三角测量与参考视图射线投票，后者沿着语义相机射线搜索跨邻近视图的几何一致候选点。得到的3D关键点支持拾取放置和工具使用：对于工具使用，我们检索对应推断技能类别的以物体为中心的原子动作，并将其存储的6D工具轨迹对齐到场景；对于灵巧执行，我们将提升的抓取关键点扩展为任务条件的抓取可供性区域，并利用手臂手部运动生成器生成可行的抓取-运动对。真实世界实验显示，相比于单视图RGB-D接地和微调的VLA基线，3D接地准确性和执行可靠性有所提升。我们进一步通过闭环状态验证和重新规划展示了长时程操作，实现了在新场景中对未见物体和工具使用任务的零样本执行。

## Abstract
We present ZeroDex, a zero-shot framework for long-horizon dexterous manipulation that grounds language instructions into executable 3D task plans from calibrated multi-view RGB images. Rather than training an end-to-end policy, our system uses a vision-language model (VLM) to produce reference-frame task grounding and primitive-level 2D keypoints, then lifts them into 3D via multi-view fusion. This lifting combines triangulation of view-wise VLM groundings with reference-view ray voting, which searches along a semantic camera ray for geometrically consistent candidates across neighboring views. The resulting 3D keypoints support both pick-and-place and tool-use: for tool-use, we retrieve an object-centric atomic action corresponding to the inferred skill category and align its stored 6D tool trajectory to the scene; for dexterous execution, we expand the lifted grasp keypoint into a task-conditioned grasp affordance region and generate feasible grasp-motion pairs with an arm-hand motion generator. Real-world experiments show improved 3D grounding accuracy and execution reliability over single-view RGB-D grounding and fine-tuned VLA baselines. We further demonstrate long-horizon manipulation through closed-loop status verification and replan, enabling zero-shot execution on unseen objects and tool-use tasks in novel scenes.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **问题**：机器人灵巧操作（尤其是长时程任务）在真实世界中面临泛化挑战。现有方法（如端到端的 VLA 模型）需要大量任务特定数据或微调才能适应新物体、新场景，难以扩展到开放式环境；人类演示重定位存在 embodiment gap，往往需要额外优化。
- **目标**：提出一种**零样本**（zero-shot）框架，无需任务特定训练，仅依靠预训练视觉语言模型（VLM）和多视图 RGB 图像，就能将高层语言指令转化为可执行的 3D 任务计划，完成灵巧操作（包括抓取放置和工具使用）。
- **核心思想**：将语义推理（由 VLM 处理）与物理执行（由运动原语/控制器处理）解耦，通过多视图融合提升 VLM 的 2D 预测为一致的 3D 量，从而获得可靠的 3D 接地（grounding）。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **整体框架**（ZeroDex）：
  1. 参考帧语义接地（Reference-Frame Grounding）
  2. 基于多视图融合的 3D 提升（Multi-View Fusion-Based 3D Lifting）
  3. 以物体为中心的原子动作对齐（Object-Centric Atomic Action Alignment，用于工具使用）
  4. 灵巧抓取与运动生成（Dexterous Affordance-Guided Grasp and Motion Generation）

- **关键技术**：
  - **VLM 接地**：给定多视图图像与语言指令，VLM 选择参考视图，推理操作模式（pick 或 tool），并预测原始级别的 2D 关键点（如抓取点、功能尖端、放置点）及其语义描述。
  - **多视图融合**：
    - **三角测量（Triangulation）**：对每一对视图计算候选 3D 点，通过 RANSAC 式共识评分选出最优（重投影误差阈值 \(\epsilon_{\text{tri}}\)）。
    - **参考视图射线投票（Ray Voting）**：沿参考视图相机射线采样深度候选，将其投影到其他视图，VLM 从编号标记中选择最佳匹配，聚合后选出最一致候选。
    - **动态选择**：若三角测量共识分超过阈值 \(\tau_{\text{tri}}\) 则采用三角测量结果，否则回退到投票估计。
  - **原子动作库（Bag of Atomic Actions）**：预定义的工具使用技能类别（如倒水、扫地），每个动作存储 6D 工具轨迹和起止锚点。测试时根据 VLM 推断的技能类别检索，并通过升举的关键点对齐到当前场景。
  - **灵巧抓取生成**：
    - 将抓取关键点投影到各视图，VLM 预测 2D 抓取包围盒；通过多视角包含性评分生成 3D 可供性区域 \(R_{\text{aff}}\)。
    - 对类似手柄的可供性使用圆柱模板采样器；对一般几何使用基于优化的生成器（BODex），增加可供性引导项。
    - 生成抓取候选后，用现成的运动规划器（arm-hand motion generator）求解运动轨迹和可行性，选择可行的一对执行。

- **长时程执行**：闭环状态验证和重规划——每次子任务执行后，VLM 检查进度并返回 continue / retry / replan / done 决定，允许在预算内重试失败步骤。
