## 摘要
机器人学习中的一个核心问题是如何从人类学习所依赖的数据类型中获取技能：被动观察、具身实践以及失败经验。人类视频提供了第一种数据的丰富来源，先前的研究表明这些视频能够初始化有用的策略。但远不清楚的是，它们是否能够支持第二种和第三种数据：即从人类视频中提取的先验知识是否足以评估、修正机器人自身的尝试，并从中提升。在这项工作中，我们展示了人类视频可用于学习与具身无关的动作、动力学和价值表示，这些表示可在不同机器人形态间迁移，为机器人从自身的实际执行与失败中自主改进提供了所需的预测基础。我们提出了动力学引导的动作修正（DGAC），一种无需训练的方法，利用这些适应模型来修复失败状态：每次失败都成为一个查询，学习模型会提出并对修正动作进行排序，将失败转化为下一次策略更新的监督信号。在涵盖移动机械臂和静态机械臂的七个真实世界操作任务中，我们的方法在多个策略主干上将成功率从40%提升到81%，展示了基于人类视频先验知识的跨具身机器人自我改进。这些结果表明，人类先验知识与机器人失败经验相结合，能够实现可扩展的自主策略改进。项目页面：https://ethz-mrl.github.io/robot-self-improvement-website/。

## Abstract
A central question in robot learning is how to acquire skills from the kinds of data that humans learn from: passive observation, embodied practice, and the experience of failure. Human videos provide the first of these in abundance, and prior work has shown they can initialize useful policies. Far less clear is whether they can support the second and third: whether priors extracted from human videos can ground a robot's own attempts well enough to evaluate them, correct them, and improve from them. In this work, we show that human videos can be used to learn embodiment-agnostic action, dynamics, and value representations that transfer across robot embodiments, providing the predictive foundation required for robots to autonomously improve from their own rollouts and failures. We introduce Dynamics-Guided Action Correction (DGAC), a training-free approach that leverages these adapted models to repair failed states: each failure becomes a query for which the learned models propose and rank corrective actions, turning failures into supervision for the next policy update. Across seven real-world manipulation tasks spanning both a mobile manipulator and a static manipulator arm, our approach improves success rates from 40% to 81% across multiple policy backbones, demonstrating cross-embodiment robot self-improvement from human-video priors. These results show that human priors and robot failures can be combined to enable scalable autonomous policy improvement. Project page: https://ethz-mrl.github.io/robot-self-improvement-website/.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

**核心思想**：学习与具身无关（embodiment-agnostic）的动作、世界状态和价值表示，建立面向人类视频预训练的策略、动力学和价值模型，再通过机器人自主 rollout 对动力学和价值模型进行适应（grounding），最终利用动力学引导的动作修正（Dynamics-Guided Action Correction, DGAC）将失败状态转为监督信号，驱动迭代策略改进。

**关键技术细节**：

1. **跨具身表示设计**：
   - 动作表示：6-DoF 手腕位姿 $ \xi_t \in SE(3) $ + 手部闭合变量 $ c_t \in [0,1] $，抽象化具身关节但保留关键运动线索。
   - 世界状态表示：当前帧 DINOv3 视觉 token $ z_t $ + 短时域 3D 点轨迹 $ P_t $，语义信息变化慢，点轨迹捕捉细粒度几何和接触动态。
   - 价值表示：基于稀疏终端奖励的折现回报。

2. **策略、动力学、价值联合建模**：
   - 策略模型 $ \pi_\theta $：输入手腕姿态历史、视觉 token 历史，预测动作块（flow matching）。
   - 动力学模型 $ f_\phi $：输入世界状态历史和候选动作块，预测未来世界状态。
   - 价值模型 $ V_\psi $：输入当前世界状态，输出标量值。
   - 均使用流匹配目标（x-prediction 参数化）训练，价值使用回归损失加时间差分损失（部署时）。

3. **自主机器人适应**：通过 VLM 提出原子指令，冻结的预训练策略在机器人上自主收集约 400 个 episode（约 3 小时），用于适应动力学和价值模型。对每个新任务再收集 25 个遥操作演示初始化任务策略。

4. **动力学引导的动作修正 (DGAC)**：
   - 对失败状态 $ o_t $，在成功数据集 $ \mathcal{D}_{\text{succ}} $ 中按价值相似性和状态相似性检索参考状态 $ \tilde{o}_t $。
   - 对满足阈值条件（价值差 ≤ δ_v ，状态相似度 ≥ δ_s ）的失败状态，通过速度场组合生成候选动作：$ v_\theta^{(n)} = v_\theta(\cdots; h_t) + w_n [v_\theta(\cdots; \tilde{h}_t) - v_\theta(\cdots; h_t)] $，其中权重 $ w_n \sim U(0.1,1) $。
   - 使用动力学模型预测未来状态，价值模型对未来状态打分，选择最高分的候选作为修正动作。
   - 将修正后的转移加入修正数据集 $ \mathcal{D}_{\text{repair}} $。

5. **优势条件策略更新**：在成功和修正数据上估计块级优势 $\hat{A}$，按 CFGRL [56] 形式进行优势条件策略提取（advantage-conditioned policy extraction），更新策略。

**算法流程**（文字描述）：
- 初始化：人类视频预训练 → 动力适应（自主探索）→ 任务策略初始化。
- 迭代循环：
  1. 当前策略执行 rollout，分离 $ \mathcal{D}_{\text{succ}} $ 和 $ \mathcal{D}_{\text{fail}} $。
  2. 微调动力学和价值模型。
  3. 对 $ \mathcal{D}_{\text{fail}} $ 中每个状态执行 DGAC，生成 $ \mathcal{D}_{\text{repair}} $。
  4. 估计 $ \mathcal{D}_{\text{succ}} \cup \mathcal{D}_{\text{repair}} $ 中状态的优势。
  5. 优势条件策略更新，得到下一轮策略。

## 6. 论文的主要结论与发现

1. 人类视频先验（DGAC 结合适应的动力学和价值模型）可使机器人通过自身失败进行自主改进，无需在线人类干预。
2. DGAC 将失败状态转化为有用监督的能力突出：相比直接检索成功轨迹（V2 58.7%）或随机选择（V3 52.0%），完整 DGAC 达到 85.3%。
3. 方法策略无关：在较强基准 π0.5 上应用 DGAC 比仅 SFT 提升 25.3 个百分点（从 62.7% 到 88.0%），远超 RECAP 的提升幅度（仅 5.3 个百分点）。
4. 跨具身有效：在 Franka Panda 上平均成功率从 36.7% 提升到 70.0%。
5. 人类视频预训练规模越大，最终性能越好（从无预训练 52.0% 到全部 85.3%）；机器人适应对 grounding 是必要的。
6. 紧凑的异质状态表示（语义 token + 点轨迹）比单一模态更有效。
7. 整个改进管线是学习型而非简单重加权失败数据——显式修正失败状态是关键。

## 7. 优点

- **自主性高**：不需要操作者在线监控或遥操作纠正，完全基于模型自主纠错，大幅减少人力成本。
- **跨具身迁移**：动作、动力学、价值表示的设计使其天然适用于不同机器人形体，实验验证了从 Stretch 3 到 Franka 的迁移能力。
- **策略主干无关**：DGAC 可作为即插即用模块，对自带策略（如 π0.5）都能提供显著增益。
- **数据效率**：只需收集较少的遥操作演示（25 个）和数百个自主 rollout 即可启动改进，且人类视频预训练是离线的、可扩展的。
- **修正可解释性**：通过检索参考和速度场组合，修正动作具有物理意义，可视化表明修正后状态向成功区域移动。
- **算法设计巧妙**：将失败视为查询，利用检索、组合生成、模型评分三步骤，避免了直接复制失败或依赖价值函数过度乐观。

## 8. 不足与局限

- **仍需少量人工标注**：当前依赖人工判断 rollout 成功/失败（文中提到"determine rollout outcomes"）。若能用基础模型自动判别，可进一步扩展。
- **短期修正限制**：DGAC 仅作用于短时域动作块（chunk size 30），对复杂长时序连锁失败缺乏规划级恢复能力。
- **人类视频数据的多样性有限**：预训练仅使用了 HOI4D、Arti4D、EgoDex，可能未覆盖所有操作模式，扩展至更大数据集（如 EPIC-Kitchens）可能带来更丰富的物理先验。
- **动力学模型精度瓶颈**：在接触密集或高动态场景下，动力学预测准确性可能下降，导致修正策略失效。
- **实验可重复性**：真实机器人实验场景难以精确相同，基线复现可能因随机种子、硬件差异等存在偏差。
- **消融的地域限制**：所有实验在 ETH Zurich/TUM 场景下，物体和夹具布局有限，未在其他国家实验室验证。
