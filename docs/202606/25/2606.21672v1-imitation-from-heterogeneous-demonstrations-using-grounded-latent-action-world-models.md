## 摘要
模仿学习已成为学习视觉运动策略的强大范式，但其泛化能力和稳定性受限于所需演示数据的规模和质量。一个有前景的方向是利用更丰富但异构的数据源，这些数据源在动作空间上不同，且通常完全没有动作标签。现有的结合异构数据源的协同训练方法依赖于启发式和手动设计的对齐技术。相比之下，我们认为动作表示应基于预测：对环境产生相同效果的动作应共享相同的表示，无论其来源如何。为此，我们通过使用基于潜在动作世界模型（GLAM）来实例化这一原则，该模型是一对生成模型，具有跨数据源的共享潜在动作空间，该空间通过在数据源之间一致地预测未来观察结果来接地。这个潜在动作空间用于训练下游的行为克隆（BC）策略，将观察映射到潜在动作，并将它们解码回机器人动作，从而提供了一种从异构数据中学习的范式。实验上，我们证明了GLAM成功学习了一个对齐的潜在动作空间，促进了在带有和不带有动作标签的数据源之间的动作转移。在模拟和现实世界的五个操作任务中，GLAM对齐策略显著优于BC基线和先前的潜在动作方法，在相同数据稀缺设置下，任务成功率平均提高了48%。视频和代码可在https://viccccciv.github.io/glam/获取。

## Abstract
Imitation learning has emerged as a powerful paradigm for learning visuomotor policies, but its generalisation and stability are limited by the scale and quality of demonstration data needed. A promising direction is to leverage more abundant but heterogeneous data sources, which differ in action space and often lack action labels altogether. Existing co-training approaches that combine heterogeneous data sources rely on heuristic and hand-engineered alignment techniques. In contrast, we argue that action representations should be grounded in prediction: actions that produce the same effect on the environment should share the same representation, regardless of their sources. To this end, we instantiate this principle by using a grounded latent-action world model (GLAM), a pair of generative models with a shared latent action space across data sources that is grounded by predicting future observations consistently across sources. This latent action space is used to train downstream behavioural cloning (BC) policies which map observations to latent actions and decode them back to robot actions, providing a paradigm for learning from heterogeneous data. Empirically, we demonstrate that GLAM successfully learns an aligned latent action space that facilitates action transfer across data sources with and without action labels. Across five manipulation tasks in simulation and in the real world, GLAM-aligned policies significantly outperform BC baselines and prior latent-action methods, achieving an average of +48% improvement in task success rate with the same data-scarce setting. Videos and code are available at https://viccccciv.github.io/glam/.

## 1. 论文的核心问题与整体含义（研究动机和背景）
现代模仿学习（尤其是行为克隆）在视觉运动操作任务中表现出色，但其泛化能力和稳定性严重依赖大规模、高质量的**同质演示数据**（即目标机器人上的动作标签轨迹）。然而，收集这类数据成本高昂且耗时，成为拓展到实际应用的主要瓶颈。一个自然且有效的缓解思路是利用更丰富但**异构的数据源**，例如便携设备采集的数据、仿真数据、人类视频等。这些数据通常具有不同的动作空间、不同的视觉外观，并且普遍缺乏动作标签，直接混合训练往往失败。现有方法（如 Sim-and-Real Co-training）依赖手工设计的对齐策略（如最优传输损失、DTW 时序对齐、经验性混合比例），缺乏通用性和原理性。论文提出一种更根本的原则：**动作表示应基于其对环境产生的影响**——只要两个动作在环境中产生相同的状态转移效果，就应该共享相同的表示，无论来源如何。基于此，论文设计了 **Grounded Latent-Action World Model (GLAM)**，通过生成模型自动学习跨数据源对齐的潜在动作空间，从而无需手工对齐即可将异构数据整合到下游模仿学习中。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 2.1 核心思想
- 构建一对生成模型（世界模型）：一个用于**异构数据（目标+辅助）**，另一个用于**目标数据（带动作标签）**。
- 异构部分通过**逆动力学模型（IDM）**直接从观测转移 `(xt, xt+1)` 推断潜在动作 `zt`，不依赖动作标签，因此可推广到所有数据源。
- 目标部分通过**动作编码器**从状态和动作 `(st, at)` 推断 `zt`，利用真值动作标签注入控制语义。
- 通过**共享的前向动力学模型**（预测下一观测/状态）将两个后验的潜在空间接地于相同的物理动态。
- 通过**不对称 KL 散度**（stop-gradient）让 IDM 后验向动作编码器后验对齐，使 IDM 获得可执行的控制信息，同时保持动作编码器不被扭曲。
- 预训练后，用 IDM 对所有数据重新标注潜在动作，训练下游 BC 策略预测这些潜在动作，再经解码器映射为实际机器人动作。

### 2.2 关键技术细节（流程文字说明）
1. **Stage 1：GLAM 预训练**
   - 异构生成模型：`p(X,O,Z) = p(x1) ∏ p(xt+1|xt, zt) p(ot|xt) p(zt)`，后验 `qψ(xt|ot)`（视觉编码器）、`qϕ(zt|xt, xt+1)`（IDM）。
   - 目标生成模型：`p(S,A,Z) = p(s1) ∏ p(st+1|st, zt) p(at|zt) p(zt)`，后验 `qη(zt|st, at)`（动作编码器）。
   - 损失函数：`L = -JH - JT + λ CKL`。`JH` 为异构 ELBO（含图像重建、KL、前向预测）；`JT` 为目标 ELBO（含状态预测、动作重建）；`CKL` 为不对称 KL 对齐（仅目标数据上计算，IDM 后验向固定的动作编码器后验对齐）。
   - 共享前向模型 `pθ(xt+1|xt, zt)` 同时被两个后验驱动，确保动作空间一致性。

2. **Stage 2：世界模型对齐的行为克隆**
   - 使用预训练的 IDM 均值重新标注所有数据（目标+辅助）：`zt ← μIDM(xt, xt+1)`。
   - 下游 BC 策略（本文采用 MIP 架构）输入视觉特征 + 本体状态，输出潜在动作块，再经动作解码器 `haω` 得到实际动作。
   - 损失：`Lπ = ||ˆz - z||²`（所有数据） + `||ˆa - a||²`（仅目标数据，可选）。

3. **可选的观测形式**：使用预训练分割模型提取操作对象的二值掩码替代 RGB，进一步强化基于物体运动的效果接地。

## 3. 实验设计
### 3.1 任务与场景
- **真实世界（3 个任务）**：lifting（抓取立方体）、pick-and-place（将立方体放入碗中）、knock-down（打倒芥末瓶）。使用 7-DoF Kinova Gen3 臂 + 两个 RealSense 相机。
- **仿真（2 个任务，MuJoCo）**：stack-two（单臂堆叠两个立方体）、stack-three（双臂堆叠三个立方体）。

### 3.2 数据集
- **目标数据集 Dtar**：每个任务 100 个遥操作轨迹（含动作标签、本体状态）。
- **辅助数据集 Daux**：每个任务 400 个**无动作标签**的仿真 UMI 夹爪轨迹（与环境交互的帧序列）。用于测试跨实体、仿真→真实的迁移。
- 额外可选：100 个 Kinova-sim 轨迹用于多源实验。

### 3.3 对比方法（baselines）
所有基线使用相同的网络架构、下游 MIP 策略和训练数据规模，仅世界模型组件不同：
- **MIP**：原始回归 BC，仅用目标数据训练。
- **CLAM**：连续潜在动作模型，单实体设置，仅用目标数据。
- **LAPA**：VQ-VAE 离散潜在动作，可训练于双源（RGB 重建）。
- **CLAM-O / LAPA-O**：将输入替换为对象二值掩码的变体。
- **GLAM / GLAM-O**：本文方法，分别使用 RGB 或对象掩码。

### 3.4 评估协议
- 真实世界：每任务 20 次试验，报告成功率及 95% Wilson 置信区间。
- 仿真：3 个训练种子，每种子 50 次试验（5 个测试初始条件×10），报告均值与标准差。

## 7. 优点（方法或实验设计亮点）
- **原理驱动**：基于“环境效果决定动作表示”的物理原则，而非启发式对齐，思想清晰且具通用性。
- **双模型不对称对齐设计巧妙**：IDM 负责跨源统一，动作编码器负责注入控制语义，不对称 KL 防止模式崩塌，且 stop-gradient 保护编码器不被劣化。

- **涵盖真实与仿真，任务多样性好**：从简单抓取到双臂堆叠，从 RGB 到掩码输入，验证了方法的鲁棒性。
- **数据效率分析深入**：通过缩放曲线和平滑性分析，量化了辅助数据对目标数据的替代能力，支撑核心主张。

## 8. 不足与局限
- **辅助数据场景受限**：辅助数据与目标数据共享相同的相机视角和桌面场景，未在真正“野外”数据（如网络视频）上测试，后者存在视角/场景漂移，需要额外的不变性压力。
