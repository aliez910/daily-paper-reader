## 摘要
如今最广泛采用的机器人学习流程是从机器人演示或结构化人类数据中学习技能，这些数据收集成本高且与特定具身形式绑定。相比之下，非结构化人类视频提供了一种可扩展的替代方案。它们包含跨物体、场景和策略的多样化操作演示，但并未直接与机器人动作关联。我们提出LUCID，一个两阶段框架，从互联网规模数据集中抽取的非结构化人类视频中学习任务意图，并在大规模并行仿真中学习机器人控制。意图模型以闭环方式根据当前观测预测短时意图（场景中接下来应该发生什么）。一个具身特定的感觉运动策略将此意图转换为机器人动作。意图接口在控制器之间共享，因此同一意图模型可应用于不同的具身形式，从我们主要使用的灵巧手到平行爪夹持器。我们在五个真实世界操作任务上评估LUCID：仅由互联网视频监督的搅拌、擦拭和分拣，实现对新型场景和物体实例的零样本迁移；以及各由1小时自采集智能手机视频监督的推杆T和电缆布线。项目页面：https://lucid-robot.github.io/。

## Abstract
The most widely-adopted robot learning pipelines today learn skills from robot demonstrations or structured human data, which are expensive to collect and tied to specific embodiments. In contrast, unstructured human videos provide a scalable alternative. They contain diverse manipulation demonstrations across objects, scenes, and strategies, but are not directly connected to robot action. We propose LUCID, a two-stage framework that learns task intent from unstructured human videos drawn from internet-scale datasets and learns robot control in massively-parallel simulation. The intent model predicts short-horizon intent (what should happen next in the scene) from the current observation in closed loop. An embodiment-specific sensorimotor policy converts this intent into robot actions. The intent interface is shared across controllers, so the same intent model can be applied to different embodiments, from our primary dexterous hand to a parallel-jaw gripper. We evaluate LUCID on five real-world manipulation tasks: stirring, wiping, and binning supervised by only internet video, with zero-shot transfer to novel scenes and object instances; and push-T and cable routing supervised by 1 hr each of self-collected smartphone video. Project page: https://lucid-robot.github.io/.

## 1. 核心问题与整体含义（研究动机和背景）

- **现有瓶颈**：当前机器人技能学习主要依赖**机器人演示**或**结构化人类数据**（动作捕捉等），数据收集成本高、扩展性差，且学习到的策略与特定**机器人本体（embodiment）**绑定。
- **可扩展但难用的数据源**：互联网上存在海量**非结构化人类视频**，包含丰富的操作演示，但缺少机器人可直接使用的动作标注（actionless）；另一方面，**物理仿真**可生成任意数量的动作标注数据，但需要为每个任务手工设计奖励函数，尤其难定义高层意图。
- **核心思路**：这两个数据源天然互补，应通过**意图-控制解耦**加以结合：人类视频提供**与本体无关的意图**，大规模仿真提供**稳健的感觉运动控制**。
- **研究问题**：如何从非结构化人类视频中学习可迁移的任务意图，并利用仿真训练本体相关的控制器，最终实现**零样本跨本体技能迁移**和**可扩展的灵巧操作学习**。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 训练一个**意图模型** \(f_\theta\) 从当前 RGB-D 观测中**闭环预测**短时对象运动（object flow）和手掌姿态参考；同时训练一个**感觉运动策略** \(\pi\) 在仿真中跟踪这些参考。
- 意图模型只关注“场景中应该发生什么”，不涉及具体机器人关节命令；控制器只负责“如何实现意图”，两者通过**短时参考** \(R\)（对象点轨迹 + 手掌姿态序列）通信。

### 关键技术细节

#### 意图模型 \(f_\theta\)
- **输入**：最近 \(F\) 帧 RGB-D 图像、当前对象查询点的 3D 位置、当前手掌姿态 (palm pose)。
- **输出**：未来 \(T\) 步的对象点位移和手掌姿态位移。
- **架构**：基于 **CoTracker3**[55] 的点-令牌（point-token）Transformer，做三点修改：
  1. 条件化在冻结的 **DINOv3** 视觉骨干上，深度信息通过**残差适配器**融入；
  2. 预测**未来时间步**（而非在已观测帧之间跟踪）；
  3. 增加一个手掌姿态令牌，与对象令牌**联合预测**。
- **训练**：使用 MSE 损失（跟踪、手掌位置、手掌旋转），训练数据来自非结构人类视频经过**四阶段提取流程**：
  - **ViPE** 恢复相机参数和度量深度；
  - **SAM 3.1** 获取物体和人手掩码；
  - **DenseTrack3Dv2** 跟踪 3D 点轨迹；
  - **WiLoR** 重建手网格并刚性对齐到深度，得到手掌姿态。
- **增强**：对人手区域进行强增强（颜色抖动、随机填充等），防止意图模型依赖人手外观（部署时可见的是机器人手）。

#### 感觉运动策略 \(\pi\)
- **训练方式**：在 **Isaac Lab** 中大规模并行执行**目标条件化强化学习**，使用 **PPO**。教师 \(\pi_T\) 在特权状态上训练，学生 \(\pi_S\) 蒸馏得到。
- **参考轨迹生成**：过程化生成四段链（approach→in-hand motion→goal→disengage），包含不同物体形状（blob、tool、plate）、手-物耦合模式和抓取方式。避免过拟合到人类视频的特定模式。
- **动作空间**：包含手臂关节增量、**eigen-grasp基**系数（拟合人手抓取的主成分）和每关节残差，使探索偏向自然抓取。
- **观测空间**：教师可访问完整表面点云和物体位姿；学生只接收外部相机可见的点、腕部深度图像和本体感知。
- **奖励**：单一任务无关奖励，平衡对象跟踪、手掌跟随、手指接触和正则化。
- **域随机化和课程学习**：重力从近零增加到全量，成功容差和干扰力逐渐加强；匹配真实传感器噪声，包括腕部深度噪声[72]。
- **蒸馏**：混合 PPO + 行为克隆（BC）损失，权重从大逐渐减小，模拟主导起初，PPO 最终接管。

#### 部署
- **闭环执行**：每次意图循环（1 Hz）使用 SAM 3.1 刷新物体掩码，DenseTrack3Dv2 保持点跟踪，重新调用意图模型生成新参考，然后策略以 50 Hz 跟踪。
- **跨本体**：同一意图模型的手掌参考可映射到平行夹爪（闭合轴对齐到人手拇指对掌方向），只需重新训练夹爪策略。

## 6. 主要结论与发现

1. **端到端可行**：LUCID 在仅凭互联网视频监督下即完成搅拌、擦拭、分拣任务，零样本泛化到新场景、物体和视角。平均成功率 73%（闭环比开环 28%）。
2. **闭环意图是关键**：当抓取失败或物体移位时，闭环重新查询意图模型指导策略修正；开环计划一旦偏离无法恢复。
3. **数据规模正相关**：分拣成功率随训练剪辑数从 1k 到 20k 单调上升，同时验证损失下降（幂律拟合）。
4. **跨本体可转移**：同一意图模型驱动的灵巧手和夹爪在推 T 和电缆布线上成功率相当（各 19/30），说明意图接口通用。
5. **消融证实设计**：令牌化编码器、eigen-grasp 基、混合蒸馏、腕部相机均关键。
6. **涌现行为**：擦除任务中，意图模型自发将污渍纸巾放入附近垃圾桶，虽未获得显式分箱监督（归因于训练数据包含类似动作）。

## 7. 优点：方法或实验设计亮点

- **意图-控制解耦**：创新地分离“做什么”（意图）与“怎么做”（控制），使两种大数据源（视频、仿真）自然互补，避免直接模仿轨迹导致的泛化困难。
- **闭环意图**：随观测持续更新，增加对不确定性（抓取失败、物体滑动）的鲁棒性。
- **跨本体泛化**：同一意图模型无需任何调整即可用于灵巧手和夹爪，展示了原型作为“任务层”具身无关性。
- **可扩展数据利用**：仅需互联网视频或少量自拍视频（1 小时），不依赖遥操作或多视角捕获，大幅降低数据收集成本。
- **仿真训练的任务无关性**：过程化生成轨迹和物体，策略可跟踪任意意图模型输出的参考，不需针对每个任务重写奖励。
- **对透明/变形物体的处理**：虽然策略仅在刚性物体上训练，但在真实软物体（毛巾、纸巾）上表现良好，体现了一定泛化能力。
- **实验揭示数据效应**：通过不同大小训练集测试，验证意图模型性能随数据量提高的可预测性。

## 8. 不足与局限

1. **管道脆弱**：系统过度模块化，依赖 SAM 3.1、DenseTrack3Dv2、ViPE、WiLoR 等多个第三方模型，每个环节都可能失败（如遮挡导致掩码漂移、纹理缺失影响跟踪），影响下游策略。端到端训练单模型取代该链可提高稳健性。
2. **任务条件困难**：对无明确终止条件的任务，意图模型会无限循环（需外部停止）；训练新任务需手动筛选视频（按动作标签剪辑），难以扩展至数千任务。文本条件化（如“搅拌”然后“放下勺子”）可解决。
3. **接口信息损失**：显式 3D 流+手掌姿态接口丢弃了手指配置、接触细节等信号，即使意图预测准确，策略无法完全复现人类演示。隐式潜在接口或许更高效。
4. **实验对比不足**：仅与开环生成执行对比，未与同类方法（如 DexMV、Track2Act、DexTrack 等）在相同任务上直接比较。自拍任务无基线。实验场景规模相对有限（5 种任务）。
