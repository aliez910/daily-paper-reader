## 摘要
在双臂机器人操作中，任务相关的视觉信息随任务阶段和上下文而变化，同时双臂之间的交互在独立模式和协调模式之间切换，这使得策略学习具有挑战性。然而，现有的整体型视觉-语言-动作（VLA）策略通过单一的共享表示和动作生成路径处理多样的视觉输入和交互模式，通常无法分别考虑视觉相关性和双臂交互结构。为了解决这个问题，我们提出了一种基于双层结构分解的双臂操作VLA框架。视图选择视觉路由器动态调整腕部视图的贡献以强调相关视觉线索，而交互感知动作混合专家（MoE）将动作生成分解为协调路径和单臂路径，以适应不同的双臂交互模式。我们在RoboTwin 2.0中的六个模拟双臂操作任务和三个长时域真实世界任务上评估了该方法。相比整体型基线，我们的模型在模拟评估中提高了27.7%的总体平均成功率，在真实世界评估中提高了43.3%，并且在两种设置下始终优于单模块变体。这些结果表明，联合考虑选择性视觉处理和双臂交互结构的显式分解，为鲁棒的双臂操作提供了有效的归纳偏差。

## Abstract
In bimanual robotic manipulation, task-relevant visual information varies with the task stage and context, while the interaction of the two arms shifts between independent and coordinated modes, making policy learning challenging. However, existing monolithic Vision-Language-Action (VLA) policies process diverse visual inputs and interaction patterns through a single shared representation and action generation pathway, often failing to separately account for visual relevance and bimanual interaction structure. To address this issue, we propose a bimanual manipulation VLA framework based on Dual-Level Structural Decomposition. The View-Selective Visual Router dynamically adjusts wrist-view contributions to emphasize relevant visual cues, while the Interaction-Aware Action Mixture-of-Experts (MoE) decomposes action generation into coordinated and arm-wise pathways to adapt to varying bimanual interaction modes. We evaluate the proposed method on six simulated bimanual manipulation tasks in RoboTwin 2.0 and three long-horizon real-world tasks. Our model improves the overall average success rate over a monolithic baseline by 27.7% in simulation and 43.3% in real-world evaluation, while consistently outperforming single-module variants across both settings. These results demonstrate that jointly considering selective visual processing and explicit decomposition of bimanual interaction structures provides an effective inductive bias for robust bimanual manipulation.

## 2. 方法论

### 核心思想
提出一种 **Dual-Level Structural Decomposition** 框架，在感知层面和动作层面分别分解异构性：
- **View-Selective Visual Router (VSR)**：根据任务上下文动态调节左右腕部视图的权重，抑制无关视觉信息，强调相关线索。
- **Interaction-Aware Action Mixture-of-Experts (IAMoE)**：将动作生成分解为协调专家和单臂专家，根据交互模式选择相应的专家路径，避免不同交互模式间的干扰。

### 关键技术细节
- 基座模型：使用预训练 VLA 模型 π0.5，通过 LoRA 微调适配特定任务。
- 输入：三视角图像（外部 + 左腕 + 右腕）、语言指令、机器人状态。
- **VSR**：
  - 从外部视觉 tokens 和语言-状态 tokens 构建路由上下文，通过 MLP + sigmoid 预测左右腕部权重 w_t ∈ [0,1]²。
  - 权重逐 token 乘到对应腕部视觉 tokens 上，再与外部视觉 tokens 和语言-状态 tokens 一起送入 VLM 得到共享条件表征 G_t。
  - 权重预测的监督标签通过半自动 KNN 标签传播生成。
- **IAMoE**：
  - 三个 LoRA 专家：协调专家（处理全动作空间）、左臂专家、右臂专家（各处理对应单臂动作子空间，通过动作掩码实现）。
  - 每个专家使用独立的 LoRA adapter 和视角级注意力掩码（协调专家可访问所有视角，单臂专家只能访问外部和自身腕部视角）。
  - **Action Router**：从外部视觉 tokens 和语言-状态 tokens 预测 hard 交互模式（协调/独立），使用 Gumbel-Softmax (训练时过 soft 梯度，推理时 argmax)。
  - 最终速度场 = 协调专家的输出（若选中协调模式）或左右单臂专家输出之和（若选中独立模式）。
- **训练损失**：
  - 主动作损失 (conditional flow matching)
  - VSR 二分类交叉熵损失 (L_vis)
  - Action Router 交叉熵损失 (L_act)
  - 辅助专家损失 (L_aux)：确保每个专家都收到足够学习信号。
  - 最终总损失：L_total = L_main + λ_vis L_vis + λ_act L_act + λ_aux L_aux，其中 λ_aux 随训练余弦衰减。

### 公式/算法流程（文字描述）
1. 编码多视角图像和语言-状态 tokens。
2. VSR 根据外部视觉 tokens 和语言-状态 tokens 计算左右腕部权重，对腕部 tokens 进行加权。
3. 将加权后的 tokens 输入 VLM 得到共享条件表征 G_t。
4. Action Router 根据同样路由上下文预测交互模式（协调或独立）。
5. IAMoE 三个专家并行预测速度场：
   - 协调专家输入全动作空间
   - 左/右臂专家输入各自掩码动作
6. 根据预测的交互模式，选择协调专家输出或左右臂专家输出之和作为最终速度。
7. 通过 CFM 目标训练，并加入路由监督损失和辅助专家损失。
