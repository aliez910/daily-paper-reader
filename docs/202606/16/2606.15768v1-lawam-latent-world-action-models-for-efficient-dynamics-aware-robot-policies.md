## 摘要
视觉-语言-动作模型（VLA）利用大规模视觉-语言预训练进行语义机器人控制，但通常缺乏对机器人动作如何改变场景的显式预见。世界动作模型（WAM）通过将策略条件建立在预测的未来上解决了这一限制，然而现有方法通常依赖计算成本高昂的视频生成，存在大量像素级冗余。我们提出LaWAM，一种潜在世界动作模型，它通过紧凑的潜在视觉子目标而非重建的未来视频，将预测动力学暴露给机器人策略。LaWAM的核心是一个由潜在动作条件化的潜在世界模型（LaWM）。我们通过在预训练视觉基础模型的潜在空间中训练潜在动作模型，并重新利用其前向解码器预测未来观测特征以进行场景演化，从而获得LaWM。然后，LaWAM将动作生成条件建立在这些预测的潜在视觉子目标上，以实现动力学感知的机器人控制。LaWAM在LIBERO（98.6%成功率）、RoboTwin（91.22%成功率）以及真实世界操作任务中取得了最先进或具有竞争力的成功率，同时保持了低延迟推理。LaWAM每次动作块预测运行时间187毫秒，相比像素空间WAM，实现了高达24倍的挂钟延迟降低。

## Abstract
Vision-Language-Action models (VLAs) leverage large-scale vision-language pretraining for semantic robot control, but often lack explicit foresight into how robot actions change the scene. World-Action Models (WAMs) address this limitation by conditioning policies on predicted futures, yet existing approaches typically rely on computationally expensive video generation with substantial pixel-level redundancy. We present LaWAM, a Latent World Action Model that exposes predictive dynamics to robot policies through compact latent visual subgoals instead of reconstructed future video. At the core of LaWAM is a latent-action-conditioned Latent World Model (LaWM). We obtain LaWM by training a latent action model in the latent space of a pretrained vision foundation model and repurposing its forward decoder to predict future observation features for scene evolution. LaWAM then conditions action generation on these predicted latent visual subgoals to enable dynamics-aware robot control. LaWAM achieves state-of-the-art or competitive success rates (SRs) across LIBERO (98.6% SR), RoboTwin (91.22% SR), and real-world manipulation tasks while retaining low-latency inference. LaWAM runs in 187 ms per action-chunk prediction and achieves up to 24x lower wall-clock latency than pixel-space WAMs.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有视觉-语言-动作模型（VLA）能实现语义级操控，但缺乏对动作如何改变场景的显式因果建模；世界动作模型（WAM）通过预测未来视频来感知动态，但计算开销极大（需迭代生成像素），存在大量与动作无关的冗余信息。
- **动机**：能否在保持动态感知优势的同时，避免像素级的昂贵生成，实现高效、低延迟的动力学感知机器人控制？
- **整体贡献**：提出 LaWAM（Latent World Action Model），利用预训练视觉基础模型的潜空间，训练一个潜在世界模型（LaWM）直接预测未来的潜视觉特征作为子目标，以紧凑的潜表示替代像素级视频，实现既具备动态感知又高效（187 ms/动作块，延迟比像素WAM低 24 倍）的机器人策略。

## 2. 方法论
- **核心思想**：将动作生成条件建立在由潜在世界模型预测的紧凑潜视觉子目标上，而非重建的未来像素帧。整个框架分为两阶段预训练。
- **关键技术细节**：
    - **第一阶段：训练潜在世界模型（LaWM）**
        - 在冻结的 DINOv3 ViT-B/16 视觉编码器潜空间上训练一个潜在动作模型（LAM），包含：
            - **逆动力学编码器**（IDM）：从当前观测 \(u = f_\psi(o)\) 与视界观测 \(u_T = f_\psi(o_T)\) 推断潜在动作 \(z \sim q_\phi(z|u,u_T)\)。
            - **前向解码器**（即 LaWM）：以当前潜特征 \(u\) 和潜在动作 \(z\) 预测未来潜特征 \(\tilde{u}_T = \text{LaWM}_\omega(u, z)\)。
        - 训练损失：\(\mathcal{L}_{\text{LAM}} = \mathcal{L}_{\text{wm}} + \mathcal{L}_{\text{aux}} + \beta D_{\text{KL}}(q_\phi \| \mathcal{N}(0,I))\)，其中 \(\mathcal{L}_{\text{wm}}\) 为潜特征预测 L2 损失，\(\mathcal{L}_{\text{aux}}\) 为辅助末端执行器状态预测。训练后保留 LaWM 作为世界模型，逆动力学编码器仅用于师生蒸馏。
    - **第二阶段：集成到策略（LaWAM）**
        - 预训练 VLM 骨干（Qwen3-VL 前 16 层）提取语义上下文，额外加入**潜在动作查询令牌**（LA Query），策略先验 \(p_\theta(\hat{z}|o,l)\) 预测潜在动作。
        - 预测的 \(\hat{z}\) 输入冻结的 LaWM 得到潜子目标 \(\hat{u}_T\)。
        - **动作专家**（Alternate-DiT 架构）在语义流和动力学流（当前潜特征 + 预测子目标）之间交替，生成动作块 \(a_{1:T}\)。
        - 策略训练损失：\(\mathcal{L}_{\text{LaWAM}} = \lambda_{\text{distill}} \mathcal{L}_{\text{distill}} + \lambda_{\text{wm}} \mathcal{L}_{\text{wm}} + \mathcal{L}_{\text{act}}\)，其中 \(\mathcal{L}_{\text{distill}}\) 为潜在动作蒸馏，\(\mathcal{L}_{\text{wm}}\) 为子目标监督，\(\mathcal{L}_{\text{act}}\) 为条件流匹配损失。
    - **知识绝缘**（Knowledge Insulation, KI）：防止动作专家梯度覆盖 LaWM 参数。
    - **物理时间对齐**：不同频率数据集使用固定物理时间间隔 \(\tau\)，动作令牌附加物理时间正弦编码，避免离散索引歧义。
- **公式流程**（用文字说明）：
    - 训练时：观测对 → DINO → IDM 推断 \(z\) → LaWM 预测 \(\tilde{u}_T\)，并用未来真值监督。
    - 推理时：当前观测+指令 → VLM → 先验给出 \(\hat{z}\) → LaWM 解码 \(\hat{u}_T\) → 结合 \(\hat{u}_T\) 和上下文 → 动作专家去噪生成动作块。
