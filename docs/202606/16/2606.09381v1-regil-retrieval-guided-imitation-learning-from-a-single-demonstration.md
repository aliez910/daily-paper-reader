## 摘要
利用深度神经网络从单一示范中学习机器人操作策略仍然极具挑战性，因为即使与示范轨迹有微小偏差也会迅速累积导致失败，而收集大量在线交互数据的成本高昂。我们提出ReGIL，一种将单一示范视为外部存储的检索引导模仿学习框架。ReGIL在训练过程中反复查询这个静态存储，以同时指导探索、生成正则化缓冲区和构建奖励。具体来说，它通过当前轨迹与检索到的片段之间的局部时间对齐来计算奖励，为策略改进提供逐步且信息丰富的反馈。我们在LIBERO和Meta-World基准测试的机器人操作任务上，在单一示范设置下评估ReGIL。ReGIL在成功率和训练效率方面均优于先前的基线方法。在真实机器人实验中，仅使用一个示范和不到一小时的在线训练，ReGIL在三个操作任务上实现了超过75%的成功率，这些任务在初始机器人姿态和目标位置上均具有随机性。这些结果表明，将单一示范作为可重用记忆，可以为高效机器人学习提供不止于静态监督的作用。更多详情请访问我们的网站：https://regil2026.github.io/

## Abstract
Learning robot manipulation policies with deep neural networks from a single demonstration remains highly challenging, as even small deviations from the demonstrated trajectory can quickly compound into failure, while collecting substantial online interaction data is costly. We propose ReGIL, a retrieval-guided imitation learning framework that treats a single demonstration as an external memory. ReGIL repeatedly queries this static memory throughout training to simultaneously guide exploration, generate the regularization buffer, and construct rewards. Specifically, it computes rewards through local temporal alignment between the current trajectory and the retrieved segment, providing step-wise and informative feedback for policy improvement. We evaluate ReGIL on robotic manipulation tasks from the LIBERO and Meta-World benchmarks under the single demonstration setting. ReGIL outperforms prior baselines in both success rate and training efficiency. In real-robot experiments, using only one demonstration and less than one hour of online training, ReGIL achieves over 75% success rate across three manipulation tasks with randomness in both initial robot pose and target position. These results demonstrate that leveraging the single demonstration as reusable memory can provide more than static supervision for efficient robot learning. ## 2. 论文提出的方法论

*   **核心思想**：ReGIL 将单条专家示范轨迹视为持久的外部记忆，在在线交互的每一时刻，根据当前观测和历史轨迹检索相关专家片段，并用该片段**同时**引导探索、构建稠密奖励、扩充正则化缓冲区，最终通过 TD3 风格 actor-critic 进行策略优化。
*   **关键技术与流程**：
    1.  **专家片段检索（粗到精）**：
        *   候选选择：基于冻结视觉基础模型（ViT编码器 DINOv3）的嵌入计算视觉相似度，取 top‑k 近邻中时间最靠后的索引作为候选终点，保证视觉匹配且任务进展。
        *   局部时间对齐：将最近 H 步观测历史与专家轨迹候选段通过 **Subsequence Dynamic Time Warping (S‑DTW)** 对齐，得到最低对齐代价对应的专家片段及匹配终点。
    2.  **检索引导探索**：
        *   训练初期（warm‑up 阶段），直接执行对齐后检索出的专家动作，使 agent 快速进入成功区域，收集成功轨迹存入“成功缓冲区” Bsuc。
        *   Bsuc 初始包含原始演示，后续不断用在线成功经验扩充，增加监督多样性，缓解分布偏移。
    3.  **检索引导策略优化**：
        *   **奖励构造**：每一步的奖励定义为归一化的负 S‑DTW 距离（公式3）。该奖励基于历史轨迹与检索专家片段，不需要完整未来轨迹，提供逐步、密集且保持时间一致性的反馈。
        *   **优化目标**：采用 TD3 框架，actor 最大化 Q 值的同时添加对 Bsuc 中数据的行为克隆（BC）正则项，且正则项系数 λ(t) 随时间线性衰减，使策略从模仿逐渐过渡到强化学习，最终可能超越演示性能。

*   **算法概要**（伪代码见附录 A.1）：每与环境交互一步，执行检索→执行动作（探索期用专家动作，后期用当前策略）→计算奖励→存储经验→若成功且处于探索期则加入 Bsuc →从 replay buffer 和 Bsuc 采样更新 critic 和 actor。

## 8. 不足与局限

*   **视觉模糊受限**：在 Insert 等任务中，相似视觉模式导致检索错误（4/4 失败），且光照变化影响性能，表明依赖单一的 RGB 表示不够鲁棒。
*   **非刚性物体操作困难**：Open 任务中盖子变形导致夹持器滑脱，说明纯视觉无法提供接触力反馈，需要多模态（如触觉）辅助。
*   **探索阶段瓶颈**：对于极复杂任务，检索引导探索可能无法收集足够多的成功轨迹，稀释了成功缓冲区的好处，进而降低正则化效果（如在 Metaworld Hammer 任务上缓冲区规模小、性能略低）。
*   **实验覆盖有限**：真实世界仅 3 个任务，且每条件 20 次试验；模拟任务未涉及高度动态场景或接触丰富的组装任务。未与其他大规模模仿学习或离线 RL 方法对比（如 implicit BC, diffuser-based 方法），但这点受限于单演示设定。
*   **计算过度依赖 DINOv3**：编码器占 93%+ 延迟，虽然整体仍可用，但在更低算力设备上可能成为瓶颈。
*   **通用性前提假设**：方法假设初始模型为随机，若任务需要精细预训练表示则不一定适用。未来方向如结合预训练策略以缓解。
