## 摘要
视觉-语言-动作（VLA）模型越来越多地依赖历史、深度或4D特征来对机器人策略进行条件化，以解决长程操作中的歧义性。然而，更多的时空证据并不一定更好：当注入的证据缺乏运动一致性时，会引入几何漂移、碎片化的时间线索以及不稳定的动作生成。这引出一个简单问题：VLA应该记住过去的帧，还是记住连接它们的运动？我们提出MotionVLA，一种运动历史接口，将短暂的仅过去视频窗口转换为紧凑、时间连续的轨迹场标记。MotionVLA不再将历史视为一组稀疏的独立提升帧，而是将最近的观测表示为物理一致的运动证据。当前的视觉标记查询该历史以检索与任务相关的运动信息，随后在基于轨迹的监督下重新耦合到VLA流中。在模拟基准和初步真实机器人部署上的实验表明，MotionVLA不仅改进了长程操作，还产生了更平滑、更直接的执行。这些结果表明，有效的VLA记忆不仅在于提供更多的4D上下文，更在于暴露可用于控制的运动一致性证据。

## Abstract
Vision-language-action (VLA) models increasingly condition robot policies on history, depth, or 4D features to resolve ambiguity in long-horizon manipulation. However, more spatiotemporal evidence is not necessarily better: when the injected evidence is not motion-consistent, it can introduce geometric drift, fragmented temporal cues, and unstable action generation. This raises a simple question: should a VLA remember past frames, or remember the motion that connects them? We introduce MotionVLA, a motion-history interface that converts a short past-only video window into compact, time-continuous trajectory-field tokens. Instead of treating history as a sparse set of ndependently lifted frames, MotionVLA represents recent observations as physically coherent motion evidence. Current visual tokens query this history to retrieve task-relevant motion information, which is then recoupled into the VLA stream under trajectory-grounded supervision. Experiments across simulation benchmarks and preliminary real-robot rollouts show that MotionVLA improves long-horizon manipulation while producing smoother and more direct executions. These results suggest that effective VLA memory is not just about providing more 4D context, but about exposing motion-consistent evidence that is usable for control.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：当前视觉-语言-动作（VLA）模型在长程操作中常注入历史帧、深度或4D特征以缓解状态歧义，但注入的证据若缺乏运动一致性，反而会引入几何漂移（geometric drift）、时间碎片化和动作抖动。作者认为，VLA记忆的核心不是“记住更多过去帧”，而是“记住连接这些帧的运动”。

- **背景问题**：在典型4D注入流程中，独立帧提升（per-frame lifting）导致同一物理点在不同帧中映射到漂移的3D位置；稀疏/非均匀历史采样又造成时序碎片化。这些效应使动作出现迂回（detour）和不稳定优化，即使最终任务成功，执行的路径效率也较低。因此，有效VLA记忆应表示一致的运动证据，而非仅堆叠时空上下文。

## 2. 论文提出的方法论

### 核心思想
设计一个**运动历史接口**（Motion-History Interface），将最近一段仅过去（past-only）的头部相机帧窗口转换为紧凑、时间连续的**轨迹场令牌（trajectory-field tokens）**。当前视觉令牌使用交叉注意力从该历史中**查询**与任务相关的运动证据，再通过轻量融合模块**重新耦合**（recouple）回VLA流中，并施加**轨迹接地监督**（trajectory-grounded supervision）。

### 关键技术细节
1. **轨迹场4D表示**：使用冻结的预训练轨迹提取器（TraceAnything）处理短时头部相机序列，输出连续时间3D轨迹函数。取最终解码器层令牌作为紧凑4D特征。
2. **仅过去运动历史构建（Algorithm 1）**：
   - 在时刻t，选取历史窗口W_t（严格不包括当前帧），包含Thist帧，步长Δ；训练/推理均按相同定义（训练从episode索引，推理用FIFO缓冲）。
   - 左填充重复最早帧直到满足长度。
   - 经归一化、缩放后送入轨迹提取器得到令牌，再线性投影到VLA隐藏维度，得到可查询的运动历史M_kv。
3. **解耦-再耦合（Decouple-then-Recouple）**：
   - **查询**：当前头部相机视觉令牌V_head作为Query，对M_kv执行多头交叉注意力（MHA），获得运动条件令牌M，长度与当前令牌一致。
   - **轨迹接地监督**：附加轻量化预测头g_ϕ，从M重建历史窗口对应的动作序列a_hist，优化L_traj = MSE（公式4）。主要动作损失L_action沿用π0的流匹配行为克隆（公式3）。总损失L = L_action + α L_traj。
   - **再耦合**：将M与当前多视角视觉令牌拼接，经MLP融合为F，与文本/状态令牌一同送入VLA主干进行自注意力。
4. **两阶段训练**：
   - Stage I：在异构操作混合数据（~100小时）上用L_action+L_traj对齐运动历史接口。
   - Stage II：移除辅助头，在目标任务上微调。
