## 摘要
视觉-语言-动作（VLA）模型在基准测试中表现强劲，但在现实世界部署中仍难以应对未见过的物体、背景变化以及不同的机器人本体。我们认为，这源于缺乏统一的几何感知操作表征，导致现有VLA容易受到低级轨迹监督、未对齐的3D特征以及本体差异的影响。为解决这一问题，我们提出GEAR-VLA，这是一个用于学习统一几何感知动作表征以实现可泛化机器人操作的VLA框架。GEAR-VLA采用从粗到细的动作学习，多源具身预训练使VLM具备具身推理和离散动作理解能力，随后潜在动作标记将动作语义与梯度解耦的DiT连续动作专家连接起来。它还通过将可训练的3D空间骨干与VLA表示对齐，同时冻结原始VLM对齐的视觉通路，执行语义对齐的3D集成。为了在机器人之间共享该表示，GEAR-VLA使用本体规范化，其中本体感知状态和本体不变动作将机器人差异限制在低级接口。大量的仿真和真实世界实验证明了其强大的泛化能力：GEAR-VLA在LIBERO、零样本LIBERO-Plus和RoboTwin 2.0上取得了最先进的性能，在AgileX上达到85.9%的成功率，在预训练未见过的LDT-01本体上达到81.0%，并在包含212个未见物体的6360次试验的通用抓取基准上获得90.1%的成功率。代码和模型将在https://github.com/babynabeauty/GEAR-VLA发布。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

## 1. 核心问题与研究动机

现有的视觉-语言-动作（VLA）模型在标准评测中表现优异，但在真实世界部署中仍面临三个关键泛化瓶颈：
- **物体与场景泛化**：对未见过的物体、背景变化鲁棒性差。
- **3D 空间理解缺失**：现有 VLA 依赖 2D 语义，缺乏几何感知，难以精准操作。
- **跨本体转移困难**：不同机器人本体（构型、自由度、标定）差异大，现有方法常将本体身份注入高层次语义表示，导致迁移效率低、数据不平衡时性能下降。

作者认为，根本原因在于**缺乏一个统一的、几何感知的动作表征**，使得模型难以在低级轨迹模仿、未对齐的 3D 特征和本体差异之间取得平衡。

## 2. 方法（GEAR-VLA 框架）

### 2.1 核心思想
构建一个**语义驱动的、几何感知的、跨本体可共享的动作表征**，通过三个设计实现：  
**粗到细动作学习**、**语义对齐 3D 集成**、**本体规范化**。

### 2.2 关键技术细节

#### ① 粗到细动作学习
- **阶段一：具身 VLM 预训练**  
  使用多源数据（VQA、空间定位、轨迹推理、操作视频、FAST 风格的离散动作 token）对 VLM 进行自回归 token 预测训练，让模型掌握任务语义、离散动作模式和视觉动态，**避免直接对低级轨迹建模**。  
  同时训练一个**因果 VQ‑VAE 潜在动作 tokenizer**，从无动作标签的视频中提取潜在动作 ID，补充丰富的操作动态学习。
- **阶段二：连续动作生成**  
  将 VLM 输出的**潜在动作 token** 作为条件，通过**梯度解耦的 DiT 动作专家**生成连续动作。  
  流匹配损失（公式 2）不反向传播至 VLM 骨干，保护已学语义不被破坏。

#### ② 语义对齐 3D 集成
- 在原有 2D 视觉通路旁新增一个可训练的 **3D 空间编码器（VGGT）**，利用多视图一致性提取几何结构。
- 2D 视觉编码器**冻结**以保持与 VLM 语义空间的对齐。
- 2D 和 3D 特征拼接后通过**零初始化扩展投影器**映射到 LLM 的视觉 token 空间（公式 3），初始阶段不扰动原始分布，训练中 3D 编码器逐步适应 VLA 表示。

#### ③ 本体规范化
- **动作输出**：采用**相对末端执行器动作**（公式 4、5），以当前位姿为基准预测未来位姿的相对变换，避免累积误差且减少对机器人基坐标系依赖。
- **状态输入**：使用**末端执行器位姿 + 关节角度**作为配对状态，隐式表征本体信息；每个机器人仅使用一个轻量状态投影器映射到共享表示。
- **迁移策略**：两阶段轻量适应：先冻结 VLA 骨干训练新投影器，再轻量端到端微调，使新本体迅速融入共享表示。

## 3. 实验设计

### 3.1 仿真基准
| 基准 | 内容 | 对比方法 |
|------|-----|----------|
| **LIBERO**（10 任务，4 个族） | 标准表格操作 | OpenVLA, WorldVLA, π0, UniVLA, π0.5, X‑VLA, ACoT 等 |
| **LIBERO‑Plus**（7 种分布偏移） | 零样本 OOD 泛化（相机、机器人、语言、光照、背景、噪声、布局） | 同上 |

### 3.2 真实世界实验
- **双机械臂任务**（T‑shirt 折叠、shorts 折叠、包裹翻转）：
  - **AgileX**（14 DoF）：每任务 200 条演示（固定物体颜色），测试三种未见颜色/大小变种。
  - **LDT‑01**（16 DoF，预训练中无类似本体）：同样每任务 200 条演示，测试同色变种，评估**跨本体迁移**。
- **大规模通用抓取基准**：
  - 训练集：35 个物体（各 100 条演示，含视觉/几何相似干扰）。
  - 测试集：**212 个未见物体**，分三个场景（稀疏、密集、背景/光照变化），每物体 10 次 = **共 2,120 次/设置 × 3 = 6,360 次真实机器人试验**。
  - 使用第一帧 mask 引导目标，评估泛化到任意物体的能力。
  - 对比方法：π0.5、DexGraspVLA。

### 3.3 消融实验
在 **LIBERO‑Plus** 上系统消融每个关键设计：
- **离散动作学习**：移除潜在 action IDs / FAST tokens → 性能降低 1.6% / 2.5%。
- **3D 几何集成**：移除 VGGT（↓ 3.6%）、冻结 VGGT（↓ 3.5%）、无零初始投影器（↓ 6.8%）、训练 2D ViT（↓ 2.1%）。
- **本体规范化**：一步适应（↓ 9.7%）、X‑VLA 软提示（↓ 3.7%）、无本体特定投影器（↓ 2.0%）。
- 另外还消融了注意力级模态丢弃、潜在动作建模方式、数据效率等。

## 7. 优点

- **统一几何感知表征**：将动作语义、3D 几何、本体信息融合在共享表示中，而非分模块拼接。
- **粗到细学习范式**：先让 VLM 理解任务和动作语义，再用解耦的扩散专家生成连续动作，**防止低级动作信号破坏高层语义**。
- **语义对齐的 3D 集成**：通过零初始化投影器和冻结 2D 通路，**稳定注入几何特征而不扰乱原有视觉‑语言对齐**。
- **高效跨本体迁移**：仅通过状态投影器和相对动作空间区分本体，避免在高层表示中引入本体区分，**迁移速度快、对数据不平衡容忍度高**。
- **实验覆盖广**：从仿真标准、分布外泛化到真实多任务和大规模抓取，验证充分，且结果一致。

## 8. 不足与局限

- **计算开销大**：预训练需 240 张 H200，对实验室可及性要求高；轻量适配仍需多张 GPU。
- **视觉输入依赖**：依赖多视角 RGB 图像和第一帧 mask 引导，可能增加感知系统复杂度；低纹理或极端光照下 VGGT 效果未知。
- **动作空间受限**：相对末端执行器动作虽然适合刚性抓取，但对软体、多指灵巧手等高维或连续变形任务可能不够灵活。
- **本体多样性有限**：真实迁移仅测试了两个平台（AgileX、LDT‑01），缺少对人形机器人、移动操作平台等更多异构本体的验证。
- **未讨论失败模式与安全性**：论文未分析错误案例、碰撞风险或违反安全约束的情境。
- **语言指令假设**：实验中均使用明确语言指令，未评估无语言场景（如纯视觉或零指令）下的表现。
- **基准物体类别局限**：通用抓取基准中物体虽然数量多，但集中在几个类别，真实场景中的极端变形或动态物体未覆盖。
