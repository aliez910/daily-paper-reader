## 摘要
视觉-语言-动作（VLA）模型利用预训练视觉-语言模型（VLM）丰富的世界知识，实现遵循指令的机器人操作。然而，VLM语义空间与具身控制策略之间的结构性差异常常阻碍精确感知-动作映射的学习。为了解决这一挑战，我们提出了AffordanceVLA，这是一个统一的框架，它引入结构化的可供性预测作为面向任务的中间表示，以建立更精确和鲁棒的感知-动作映射。具体地，我们通过三个互补组件逐步建模操作先验：1）Which2Act，通过视觉潜在预测进行以物体为中心的定位，以抑制干扰；2）Where2Act，通过可供性图估计进行二维交互定位；3）How2Act，用于指导操作策略的三维几何推理。这些可供性线索提供了空间基础化、语义条件化和动作耦合的中间表示，从而自然桥接了视觉、语言和动作。我们将这些模块集成到具有专用专家的混合变换器（MoT）架构中，并使用三阶段训练策略配合渐进式数据课程训练模型。为了克服机器人数据集中密集可供性标签的稀缺性，我们还开发了一个鲁棒的自动数据增强管道。在仿真和真实世界上的大量实验表明，AffordanceVLA在多种操作场景中实现了强大的性能。

## Abstract
Vision-Language-Action (VLA) models leverage the rich world knowledge of pretrained vision-language models (VLMs) to enable instruction-following robotic manipulation. However, the structural mismatch between VLM semantic spaces and embodied control policies often hinders the learning of precise perception--action mappings. To address this challenge, we propose \textbf{AffordanceVLA}, a unified framework that introduces structured affordance forecasting as a task-oriented intermediate representation to establish a more precise and robust perception--action mapping. Specifically, we progressively model manipulation priors through three complementary components: 1) \textbf{Which2Act} for object-centric grounding via visual latent prediction to suppress distractions; 2) \textbf{Where2Act} for 2D interaction localization via affordance map estimation; and 3) \textbf{How2Act} for 3D geometric reasoning to guide manipulation policies. These affordance cues provide spatially grounded, semantically conditioned, and action-coupled intermediate representations, thereby naturally bridging vision, language and action. We integrate these modules into a Mixture-of-Transformer (MoT) architecture with specialized experts and train the model using a three-stage training strategy with a progressive data curriculum. To overcome the scarcity of dense affordance labels in robotic datasets, we also develop a robust automated data augmentation pipeline. Extensive experiments on simulation and real-world demonstrate that AffordanceVLA achieves strong performance across diverse manipulation scenarios.

- **整体含义**：AffordanceVLA 提出将**可供性预测**（affordance forecasting）结构化为 **Which2Act、Where2Act、How2Act** 三个互补子任务，作为感知与动作之间的自然桥梁，从而提升指令跟随机器人操作的精确性与泛化能力，为 VLA 模型提供新的表示学习范式。

## 2. 论文提出的方法论

### 核心思想
- 构建 **Mixture-of-Transformer (MoT)** 架构，包含三个专门专家：**Understanding Expert**（语义对齐）、**Affordance Generation Expert**（结构化可供性预测）、**Action Expert**（动作生成）。
- 采用 **Understanding-Affordance-Action (UAA) 渐进式注意力机制**：Affordance 专家只注意 Understanding 的输出，Action 专家同时注意前两者，防止信息泄露。
- 通过**三阶段渐进训练课程**（VQA预训练→大规模合成机器人数据联合训练→目标任务微调）逐步引导模型从语义理解过渡到动作执行。

### 关键技术细节
- **Which2Act（物体级定位）**：预测目标物体的潜在视觉表示（使用冻结 Flux VAE 编码的连续 latent），采用 MSE 损失：

  `L_which = 1/(C·H·W) Σ ||z_hat - z_gt||²`

- **Where2Act（2D交互定位）**：预测2D可供性热力图，采用逐像素 BCE 损失：

  `L_where = -1/(H_tW_t) Σ [M log σ(y_hat) + (1-M) log(1-σ(y_hat))]`

- **How2Act（3D几何推理）**：分为两个分支：
  - **形状生成**：条件扩散过程预测3D体素 latent（噪声预测损失 L_shape）。
  - **布局回归**：MLP 输出10-DoF空间布局向量（旋转+尺度+平移），Smooth-L1 损失 L_layout。

- **总损失**：各阶段结合 action 损失（flow-matching），affordance 权重逐步衰减（Stage II λ_afd=0.5，Stage III 降至 0.15）。
- **自动数据标注流水线**：关键帧提取 → LLM（Claude Opus 4.5）指令分解 → VLM（Qwen3-VL）生成检测类别和可供性指令 → RexOmni + SAM + SAM-3D 生成 bbox、热力图、3D标签。

1. **结构化可供性预测**（Which2Act + Where2Act + How2Act）是比视频预测、稠密光流更有效的 VLA 中间表示，能过滤背景噪声、提供明确空间指导。

3. **三阶段渐进训练**（VQA → 大规模合成数据 → 任务微调）是有序桥接语义与动作控制的关键，Stage II 对于 OOD 泛化尤其重要。

- **表示效率**：连续 Flux VAE 替代离散 VQ-VAE，避免量化误差且最大化 token 表征容量。
- **算法系统性**：MoT 架构、渐进式注意力、三阶段课程、自动标注流水线，各组件相互配合、理论一致。
- **实验严谨**：多个控制实验排除“数据量/监督密度”的竞争解释，证据链清晰；数据效率曲线揭示架构-表示-数据之间的正反馈。

- **泛化迁移**：从仿真到真实通过 DROID 子集预训练+少量 in-house fine-tune 即可有效，real-world 任务多样。
