## 摘要
大型视觉-语言-动作策略越来越多地被训练为作用于动作片段的条件生成模型。然而部署过程中会产生质量参差不齐的经验——成功示范、部分完成、可恢复的错误和失败——这些经验难以用于标准模仿学习。完整行为克隆模仿失败，过滤后的行为克隆丢弃有用的子轨迹，而离线强化学习则增加了一个大规模的评价器。我们提出了ForesightFlow，一种自引导的流匹配策略，通过学习的成功势轨迹来增强每个生成的动作片段。同一个流同时提出并评分候选动作，从而无需外部评价器即可实现最佳K推理。关键问题在于策略改进和价值校准需要不同的监督：优势加权应强调高质量动作，但将相同的权重应用于势坐标会抑制失败梯度并产生过于自信的评分。我们通过解耦的优势加权流匹配来解决这个问题，仅在动作速度上应用指数化优势权重，同时均匀训练势速度。我们进一步推导了条件流匹配的单步边界估计器，允许通过单次停止梯度前向传播来计算优势。在五个BEHAVIOR-1K仿真任务和五个真实世界双手任务上，ForesightFlow优于模仿基线，在仿真成功率上达到最强分离评价器基线水平，提升了真实世界成功率，并将训练计算量减少了38%。消融实验表明，解耦防止了价值幻觉，单步估计器保持了候选排名保真度，自引导采样改进了长周期执行。

## Abstract
Large vision-language-action (VLA) policies are increasingly trained as conditional generative models over action chunks. Yet deployment produces mixed-quality experience-successful demonstrations, partial completions, recoverable mistakes, and failures-that is difficult to use with standard imitation. Full behavior cloning (BC) imitates failures, filtered BC discards useful sub-trajectories, and offline reinforcement learning adds a large critic. We introduce ForesightFlow, a self-guided flow-matching policy that augments each generated action chunk with a learned success-potential trajectory. The same flow proposes and scores candidate actions, enabling best-of-$K$ inference without an external critic. The key issue is that policy improvement and value calibration require different supervision: advantage weighting should emphasize high-quality actions, but applying the same weights to potential coordinates suppresses failure gradients and creates overconfident scores. We address this with decoupled advantage-weighted flow matching, applying exponentiated advantage weights only to action velocities while training potential velocities uniformly. We further derive a one-step boundary estimator for conditional flow matching, allowing advantage computation with a single stop-gradient forward pass. Across five BEHAVIOR-1K simulation tasks and five real-world bimanual tasks, ForesightFlow improves over imitation baselines, matches the strongest separate-critic baseline in simulation success, improves real-world success, and reduces training compute by $38\%$. Ablations show that decoupling prevents value hallucination, the one-step estimator preserves candidate-ranking fidelity, and self-guided sampling improves long-horizon execution.

- **整体含义**：通过将成功势（success-potential）坐标嵌入流匹配端点，实现自引导的best-of-K推理，有望降低训练成本并更充分地利用混合质量数据。

## 2. 方法：ForesightFlow
### 核心思想
- 在流匹配框架中，将每个端点（endpoint）从纯动作块 `a` 扩展为 `x = [a; s]`，其中 `s ∈ [0,1]^H` 是与动作块时间对齐的成功势向量。
- 通过解耦优势加权流匹配：动作速度使用指数化优势权重，势速度使用均匀权重，从而分别优化策略改进和势校准。
- 利用条件流匹配的边界恒等式推导单步前沿估计器，高效计算优势基线，无需对每个训练样本进行完整ODE积分。

### 关键技术细节
| 组件 | 描述 |
|------|------|
| **动作-势流端点** | `x = [a; s]`, `s∈[0,1]^H`。推理时对`s`时间平均得到块级评分 `Q̂(c,a) = mean(s)`，用于候选排序。 |
| **阶段级势目标** | 将稀疏阶段完成标签 `y_{i,t} ∈ {0,1}` 广播为`s`的目标，使失败的子段也可贡献校准信号。优点 `A = y - V̂`。 |
| **单步前沿估计** | 由于端点独立采样，最优边界速度 `v*(x0,0,c) = E[x1|c] - x0`，因此单步预测 `x̂1 = x0 + vθ(x0,0,c)` 的势部分提供了上下文基线 `V̂`，且一次停止梯度即可。 |
| **解耦损失** | 动作损失：`w ∥v^(a) - u^(a)∥²`，权重 `w = min(M, exp(A/τ))`；势损失：`∥v^(s) - u^(s)∥²`，权重恒为1。该设计防止失败样本的势校正梯度被压制，避免价值幻觉。 |
| **自引导推理** | 采样 `K` 个噪声，流积分得 `K` 个 `[a^(k); s^(k)]`，选择 `mean(s^(k))` 最高的动作块执行。 |

### 算法流程
- 训练：对每个minibatch，采样轨迹和时间戳，构建端点，采样噪声和插值时间，用停止梯度单步计算 `V̂` 和优势权重，然后计算解耦损失，更新全模型（包括动作专家头与势投影头）。
- 推理：给定观察，多次重复噪声采样→流积分→选择最高势动作块→执行。

- **潜在偏差**：未提供统计显著性（如置信区间或p值），可能为点估计；真实世界试验次数未明确，但小规模可接受；阶段标签依赖人工设计，可能引入主观性。

## 6. 主要结论与发现
- **性能优势**：ForesightFlow在仿真中平均分数最高（0.46），平均成功率（39.6%）与IDQL（39.0%）持平；在真实世界中平均成功率（35.4%）和分数（0.62）均最佳。
- **解耦防止价值幻觉**：联合加权会导致失败样本被高估且校正被压制；解耦后失败样本的势校准梯度保留，势预测更准确。
- **单步前沿排序保真度高**：与高NFE（100步）参考相比，Kendall τ均值0.83，Top-1一致率87.2%，MC增益率96.8%，且势子空间轨迹接近线性（κ中位数≤0.039）。
- **自引导采样改善长周期**：K=5相对K=1，ForesightFlow成功率提升5个百分点，而IDQL基本无提升，表明耦合的势信号有效指导了候选选择。

## 7. 优点
- **自引导性**：无需外部评论家，同一流同时扮演生成器和评分器，节省参数、存储和计算。
- **解耦设计巧妙**：分别处理动作优化和势校准，统一了策略改进与价值学习范式。
- **高效前沿估计**：基于CFM边界恒等式的单步前沿将优势计算开销降至最低，并在实践中保持了高排序保真度。
- **数据利用充分**：通过阶段势目标，即使是失败轨迹中的有用子段也被保留用于校准，避免了过滤丢弃。
- **实验验证充分**：融合仿真和真实、多个任务、多种消融，覆盖了核心方法改进点。
- **计算效率**：训练时间减少38%，推理速度更快。

## 8. 不足与局限
- **阶段标签依赖**：方法需要人工定义的阶段稀疏标签，在多阶段任务中获取虽较易，但在缺少自然阶段分解的任务中可能需要额外标注工作。
- **稀疏结果限制**：对于非常长周期且回报极其稀疏的任务，阶段级信用分配仍然困难，可能导致势信号噪声增大。
- **单步估计近似性**：单步边界估计基于最优CFM求解器而非有限网络，论文虽通过实证验证了排序保持，但理论保证有限，且要求势子空间轨迹接近线性（实际中κ很小，但非零）。
- **实验范围有限**：仅评估了π0.5/π0架构在特定任务上的表现，未测试其他VLA骨干（如RT-2）或不同机器人形态。真实世界任务数较少（5个），可能无法代表所有真实场景。
- **未与在线RL对比**：方法属于离线改进，未讨论在线微调场景或更先进的离线策略（如IQL、CQL变体）的对比。
- **安全与泛化风险**：论文提及需要失败检测和人类监督，但未系统分析势模型可能产生的失败模式或潜在安全后果。
