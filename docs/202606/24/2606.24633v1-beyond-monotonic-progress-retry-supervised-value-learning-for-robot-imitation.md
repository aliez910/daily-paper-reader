## 摘要
用于机器人模仿学习的人类演示通常包含错误和纠正行为，例如不精确的抓取、物体错位、不稳定的接触以及重复尝试。虽然这些片段通常被视为噪声或次优数据，但它们提供了关于执行何时偏离期望路径以及如何恢复任务可行性的宝贵证据。然而，现有的奖励和价值模型通常依赖于单调进展假设，这捕捉了粗略的任务进展，但可能忽视不完美演示中的局部执行错误和纠正行为。在这项工作中，我们提出了ReTVL（基于重试监督的价值学习），一个通过利用重试事件作为稀疏监督从混合质量的机器人演示中学习对错误敏感的价值函数的框架。ReTVL通过将全局进展校准与由稀疏标注的重试关键点引发的局部成对偏好学习相结合，捕捉错误周围的局部退化与恢复结构。然后，学习到的价值模型用于为下游的行为克隆重新加权演示片段，减少有害执行错误的影响，同时保留有用的纠正行为。在实际机器人操作任务上的实验表明，ReTVL提供了比基于进展的基线更细粒度的价值估计，并改进了从不完美演示中的模仿学习。

## Abstract
Human demonstrations for robot imitation learning often contain mistakes and corrective behaviors, such as imprecise grasps, object misalignment, unstable contact, and repeated attempts. While these segments are commonly treated as noisy or suboptimal data, they provide valuable evidence about when execution deviates from a desirable path and how task feasibility can be restored. However, existing reward and value models often rely on monotonic progress assumptions, which capture coarse task advancement but may overlook local execution errors and corrective behaviors in imperfect demonstrations. In this work, we propose ReTVL (ReTry-Supervised Value Learning), a framework for learning mistake-sensitive value functions from mixed-quality robot demonstrations by leveraging retry events as sparse supervision. ReTVL captures the local degradation-and-recovery structure around mistakes by combining global progress calibration with local pairwise preference learning induced by sparsely annotated retry keypoints. The learned value model is then used to reweight demonstration chunks for downstream behavior cloning, reducing the influence of harmful execution errors while preserving useful corrective behaviors. Experiments on real-robot manipulation tasks show that ReTVL produces more fine-grained value estimates than progress-based baselines and improves imitation learning from imperfect demonstrations.

## 1. 核心问题与研究动机

- **背景**：机器人模仿学习常使用人类演示数据，但演示中常含有错误、重复尝试、纠正行为等“不完美”部分。传统方法要么丢弃这些数据，要么依赖“单调进展假设”（即任务进展沿轨迹单调增加），无法反映局部错误和纠正的复杂动态。
- **问题**：如何从不完美演示中学习能感知局部错误的价值函数，并用于提升下游策略学习？
- **核心动机**：重试事件（retry events）虽然通常被视为噪声，但它们提供了关于“执行何时偏离期望路径”以及“如何恢复任务可行性”的宝贵信息。论文希望利用重试事件作为稀疏监督信号，学习能捕捉局部退化与恢复结构（value drop and rebound）的价值模型。

## 2. 方法论：ReTVL

- **核心思想**：将全局进度校准与基于重试事件的局部成对偏好学习结合，训练价值函数。价值函数输入为历史观测窗口和任务指令，输出为离散化的进展值（0~1）。
- **关键技术细节**：
    1. **全局进度监督（绝对损失）**：对无重试区域的帧，使用线性进度标签（成功轨迹中t/T，失败轨迹终端为0）进行交叉熵训练，提供全局进展基准。
    2. **局部偏好学习（基于重试关键点）**：
        - 重试关键点仅标注在纠正行为开始时刻，标注成本低且一致性好。
        - 围绕每个重试关键点定义三个窗口：预重试窗口（\(W_{\text{pre}}\)）、近重试窗口（\(W_{\text{near}}\)）、后重试窗口（\(W_{\text{post}}\)）。
        - 构建偏好对：假设近重试窗口价值最低，预重试窗口价值递减，后重试窗口价值递增。从中采样正负样本（\(h^+, h^-\)），并应用**软窗口权重**（基于与关键点距离的指数衰减）降低边界不确定性。
        - 损失函数为带权的成对逻辑损失（Bradley-Terry）。总体目标：\(L_{\text{value}} = \lambda_{\text{abs}} L_{\text{abs}} + \lambda_{\text{pref}} L_{\text{pref}}\)。
    3. **价值引导的行为克隆（Weighted BC）**：
        - 将轨迹切分为动作块（action chunk），计算每个块的预测价值改进（\(V_{t+\Delta} - V_t\)），并通过离线统计标准化为权重\(\alpha_t\)。
        - 使用归一化的加权BC损失，对低价值改进或负改进的块降权，强调推动进展的有用动作。

## 3. 实验设计

## 6. 主要结论与发现

- ReTVL在局部错误敏感性上**显著优于所有基线**（例如Drop AUC从0.372→0.797，Pre > Retry从0.329→0.740），同时保持与最好基线相当的全局进度感知（VOC 0.987，S/F检测1.000）。
- 在下游策略上，ReTVL-BC平均成功率80%，对比Standard BC（41.25%）和RECAP-BC（62.5%），提升显著。
- 价值引导的权重更加准确：坏动作平均权重从0.54（RECAP-BC）降至0.11，有效抑制有害片段，同时保留后重试恢复片段的高权重。
- 消融表明：偏好损失是获得局部敏感性的关键；软窗口平衡全局与局部；绝对校准维持全局尺度。

## 7. 优点

- **方法创新**：首次将重试事件作为有价值的监督信号用于价值学习，赋予价值模型对局部错误和纠正的感知能力。
- **标注友好**：仅需标注纠正行为开始时刻，成本低且跨标注者一致性高（平均误差0.19秒）。
- **鲁棒设计**：软窗口权重减少边界模糊带来的噪声；偏好学习直接比较相对排序，不依赖精确数值目标。
- **实用有效**：在真实机器人任务上验证，下游策略提升显著，且权重分析显示细粒度过滤能力。
- **可扩展性**：价值模型与下游策略解耦，可适配不同行为克隆或强化学习方法。
