## 摘要
人类自然利用多种感知模态与物理世界交互，而大多数用于机器人的视觉-语言-动作（VLA）模型仅依赖RGB观测。这限制了它们感知难以或无法从RGB相机推断的物理属性（如温度、声音或雷达响应）的能力。我们提出了MuseVLA，一种适应性多模态感知VLA模型，它将新颖的传感器集成作为机器人操作的按需工具。给定任务指令和视觉上下文，MuseVLA首先生成一个传感器令牌和目标描述，选择要调用的感知模态和关注对象，类似于带参数的工具调用。然后，它将选定的传感器测量转换为接地传感器图像，这是一种统一的中间表示，编码异构读数以进行多模态融合和动作生成。这种设计将传感器特定处理与VLA主干解耦，实现了多种模态的高效集成。为了减少对昂贵多感官机器人数据集的需求，我们进一步引入了一个数据合成流水线，用接地传感器图像增强现有RGB视频数据集，从而实现对未见过的传感器引导任务的泛化。我们在真实机器人上评估了MuseVLA，涉及需要多模态感知输入的挑战性灵巧手操作任务，包括温度引导的拾取与放置、音频驱动的物体搜索和雷达辅助的隐藏物体检索。MuseVLA平均成功率达到80.6%，显著优于仅RGB和多感官VLA基线，并在未见任务上展现出强大的零样本能力。

## Abstract
Humans naturally leverage diverse sensing modalities to interact with the physical world, while most Vision-Language-Action (VLA) models for robotics rely solely on RGB observations. This limits their ability to perceive physical properties that are difficult or impossible to infer from RGB cameras, such as temperature, sound, or radar response. We present MuseVLA, an adaptive multimodal sensing VLA model that integrates novel sensors as on-demand tools for robotic manipulation. Given a task instruction and visual context, MuseVLA first generates a sensor token and target description that select the sensing modality to invoke and what to attend to, analogous to a tool call with arguments. It then converts the selected sensor measurement into a grounded sensor image, a unified intermediate representation that encodes heterogeneous readings for multimodal fusion and action generation. This design decouples sensor-specific processing from the VLA backbone, enabling efficient integration of diverse modalities. To reduce the need for expensive multisensory robot datasets, we further introduce a data synthesis pipeline that augments existing RGB video datasets with grounded sensor images, enabling generalization to unseen sensor-guided tasks. We evaluate MuseVLA on a real-world robot across challenging dexterous hand manipulation tasks that require multimodal sensing inputs, including temperature-guided pick-and-place, audio-driven object search, and radar-assisted hidden object retrieval. MuseVLA achieves 80.6% success rate on average, outperforming RGB-only and multisensory VLA baselines significantly, and exhibits strong zero-shot capabilities on unseen tasks.

## 2. 方法论

### 核心思想
MuseVLA 将机器人操作任务分解为三个紧密耦合的阶段：
1. **传感器与目标生成**：基于任务指令和 RGB 图像，模型生成一个传感器令牌（如 `<thermal>`）和一个目标描述（如“the mugs”），决定调用哪种传感器以及关注哪个物体。
2. **传感器数据转换**：调用所选传感器获取物理测量值，并将其处理为统一格式的**接地传感器图像**（Grounded Sensor Image），即在该物体的 RGB 区域上叠加传感器热图。
3. **动作生成**：将接地传感器图像与原始 RGB、任务指令等一起输入 VLA 模型，生成机器人动作序列。

### 关键技术细节

- **传感器令牌**：预定义的类别性令牌（`<None>`, `<thermal>`, `<acoustic>`, `<mmWave>`），与语言 token 一样从 VLM 中生成，实现自适应选择。
- **接地传感器图像**：
  - 先用预训练分割模型 SAM3 对目标描述所指物体进行语义分割，得到掩膜。
  - 将传感器 2D 热图（如温度图、雷达反射强度图、声音强度图）通过颜色映射（如 inferno、viridis、plasma）后，仅覆盖在掩膜区域，其余保持 RGB 原图。
  - 这一表示将所有传感器读入统一 RGB 空间，无需为每个传感器设计专用编码器，且便于数据合成。
- **模型架构**：
  - **VLM 主干**：基于 PaliGemma-2（Gemma-2 语言模型 + SigLIP 视觉编码器），使用 VITRA 预训练权重初始化。
  - **动作专家**：扩散 Transformer（DiT），以 VLM 输出的视觉-语言特征、机器人状态和噪声动作为条件，预测去噪后的动作片段。
  - **联合训练**：VLM 生成传感器令牌和目标描述（交叉熵损失），与扩散 MSE 损失联合优化，保留 VLM 随动作生成能力。
- **数据合成流水线**：
  - 构造**传感器字典**，每个模态对应一组物理属性关键词（如 thermal → “hot”, “cold”等）。
  - 从现有 RGB 机器人数据集（MolmoAct、AgiBotWorld-Alpha、VITRA）中，随机选择模态和关键词，注入任务指令（如“Pick up the hot mug”）。
  - 利用 VLM 生成目标描述，SAM3 分割目标物体，叠加对应颜色（如黄色代表热）模拟传感器读数。
  - 增加数据增强：在错误区域克隆颜色掩膜，迫使模型依靠传感器信息而非空间先验。
  - 最终合成 9600 个片段、105 万帧，覆盖 1000 多个物体。

## 3. 实验设计

### 数据集与场景
- **真实机器人数据集**：在桌面实验台上使用 12-DoF 灵巧手（Robotera XHand）和传感器套件（RGB-D 相机、两个热像仪、60GHz mmWave 雷达、6+1 麦克风阵列）。通过遥操作手套收集 720 个示范轨迹，涵盖：
  - **热引导的拾取与放置**（热/冷/常温饮品的区分与抓取）
  - **音频引导的物体搜索**（通过声音定位被衣物覆盖的手机，并移除覆盖物）
  - **毫米波雷达引导的隐藏物体检索**（透视闭盒、打开有物体的盒子）
  - **多阶段多传感器任务**：如 Radar → RGB、Radar → Thermal 的长链条操作。
- **合成数据集**：在上述真实数据之外，将合成数据（9600 个片段）与大容量 RGB 数据集结合，用于零样本泛化测试。

### 评测指标
- **任务成功率**：端到端完成整个指令的比例。
- **任务得分**：分解为感知阶段（0.5 分，准确选择并定位正确目标）和操作阶段（0.5 分，正确完成操作）计算。
- **自适应选择准确率**：分别评估传感器令牌和目标描述的生成准确性（通过 GPT-5.2 判断语义正确性）。

### 对比方法
- **π0-RGB / π0.5-RGB**：仅输入 RGB，无传感器数据。
- **π0-Raw / π0.5-Raw**：将全部传感器热图（无接地处理）直接拼接作为额外图像输入。
- **MuseVLA-RGB**：MuseVLA 架构但仅用 RGB，不调用传感器。
- **MuseVLA-Raw**：MuseVLA 架构但输入全部传感器原始热图，无自适应选择。
- **MuseVLA-RawAdapt**：MuseVLA 架构但仅使用自适应选择的传感器原始热图（无接地图像处理）。
- **MuseVLA（完整版）**：包括接地传感器图像与自适应选择。

所有方法均在相同真实数据集上微调，并以 Fisher 精确检验报告显著性。

## 6. 论文的主要结论与发现

- **显著提升多感官操作成功率**：完整 MuseVLA 在训练任务上平均成功率达 76.4%（表 1），引入合成数据预训练后达 80.6%（表 3），远超 RGB-only 基线（20-22%）和原始传感器热图基线（27-33%）。
- **自适应选择至关重要**：MuseVLA-RawAdapt（选择但无接地图像）仅 59.7%，而完整版达 76.4% 或 80.6%，说明统一接地表示的必要性。
- **零样本泛化能力**：仅靠真实数据训练的 MuseVLA 零样本成功率仅 27.1%，加入合成数据预训练后跃升至 66.7%（各模态提升显著）。
- **传感器选择准确率高**：在训练任务上传感器选型 100%，目标描述 93.5%；在未完成任务上分别达 100% 和 82.0%。
- **多阶段任务能力**：Radar→RGB 与 Radar→Thermal 链条的成功率分别为 66.7% 和 75.0%，证明模型可在同一任务内自适应切换传感器。

## 7. 优点

- **新颖的自适应传感器选择机制**：将传感器视为按需工具，只有任务所需的传感器才被激活，既降低计算开销又提升性能，符合人类按需感知的直觉。
- **统一的接地传感器图像表示**：将所有传感器读数转换为 RGB 空间中的局部叠加热图，绕过了对不同传感器设计专用编码器的需要，且天然支持基于现有 RGB 数据集的数据合成。
- **高效的数据合成流水线**：无需昂贵且难以扩展的多感官数据收集，仅利用已有的 RGB 视频数据集即可生成海量高质量多感官训练数据，极大提升了零样本泛化能力。
- **端到端联合训练**：VLM 损失与扩散损失共同优化，既维持了精准的传感器/目标选择能力，又有效指挥了动作生成。
- **在灵巧手这一困难基底上取得领先成果**：之前多感官 VLA 研究很少关注灵巧手操作，MuseVLA 在此场景下展示了扎实的物理感知和精细操控能力。

## 8. 不足与局限

- **真实数据集规模有限**：仅包含 720 个示范轨迹，且任务物体和场景多样性较低，可能限制模型对更广泛任务和场景的泛化。
- **依赖高性能分割模块**：接地传感器图像中依赖 SAM3 进行目标分割；若分割失败或对未见物体分割不准，会直接影响后续操作质量。
- **传感器标定一次性且简化**：采用单次离线 2D 同形映射，假设工作距离基本恒定。对于深度变化大或传感器与 RGB 视角差异大的场景，对齐误差可能增大。
- **仅在单一平台验证**：所有实验在单一种类的灵巧手和同一传感器套件上进行，未评测跨硬件、跨平台的可迁移性。
- **声学任务成功率偏低**（58.3%），部分原因是接触柔软布料时抓取困难，表明模型在精细抓取策略上仍有改进空间。
- **缺少与更强 VLA 基线的对比**：大部分对比基于 π0 和 π0.5 在自己的数据集上微调，未与近年发布的其他多模态 VLA（如 OmnIVLA、Touch-VLA 等）进行直接比较，但部分原因可能是它们未在灵巧手场景下公开评估。
