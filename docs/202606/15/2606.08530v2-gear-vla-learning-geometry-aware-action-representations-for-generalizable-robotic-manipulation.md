## 摘要
视觉-语言-动作（VLA）模型在基准测试上取得了强劲性能，但在真实世界部署中仍难以应对未见物体、背景变化和不同机器人本体。我们认为，这源于缺乏统一的几何感知操作表征，导致现有VLA模型易受低级轨迹监督、3D特征错位以及本体差异的影响。为了解决这一问题，我们提出GEAR-VLA，一种用于学习统一几何感知动作表征以实现可泛化机器人操作的VLA框架。GEAR-VLA采用从粗到细的动作学习，首先通过多源具身预训练使视觉语言模型具备具身推理和离散动作理解能力，随后潜在动作标记将动作语义与梯度解耦的DiT连续动作专家相连接。它还通过将可训练的3D空间骨干网络与VLA表征对齐，同时冻结原始与VLM对齐的视觉通路，执行语义对齐的3D集成。为了在机器人之间共享这一表征，GEAR-VLA使用本体标准化，其中本体感知状态和本体不变动作将机器人差异限制在低级接口层面。大量的仿真和真实世界实验证明了其强大的泛化能力：GEAR-VLA在LIBERO、零样本LIBERO-Plus和RoboTwin 2.0上实现了最先进的性能，在AgileX上达到85.9%的成功率，在预训练未见过的LDT-01本体上达到81.0%，并在包含212个未见物体的6,360次试验的通用抓取基准上获得了90.1%的成功率。代码和模型将在https://github.com/babynabeauty/GEAR-VLA 发布。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

## 1. 核心问题与整体含义

- **根本原因**：缺乏**统一的几何感知动作表征**，导致模型容易陷入低级轨迹拟合、3D特征错位以及本体差异的干扰。
- **整体含义**：GEAR-VLA 通过构建一种**语义与几何融合、共享于不同机器人的动作表征**，显著提升VLA的泛化性和可迁移性，为通用机器人操作提供了新范式。

## 2. 方法论（核心思想与技术细节）
GEAR-VLA 由**三个核心设计**构成：

### 2.1 粗到细动作学习（Coarse‑to‑Fine Action Learning）
- **具身VLM预训练**：混合视觉‑语言理解、空间定位、轨迹推理、指向、掩码跟踪、操作视频等多源数据，使用**FAST动作令牌**（离散化机器人轨迹）和**潜动作ID**（通过因果VQ‑VAE从无动作标签的视频中提取）作为自回归预测目标，使VLM掌握离散动作语义和具身推理。
- **连续动作生成**：VLM输出的**潜动作令牌**作为连接，输入到**梯度解耦的DiT（扩散变压器）动作专家**中，使用**流匹配损失**（flow matching loss）预测连续动作。通过stop‑gradient防止连续动作损失回传影响VLM主干，保持语义稳定性。
- **公式**：  
  - 预训练目标：\(L_{\text{VLM}} = -\sum_i \log p_\theta(y_i \mid y_{<i}, O, l, s)\)  
  - 动作专家目标：\(L_{\text{FM}} = \mathbb{E}[ \| v_\phi(A_\tau, \tau, \text{sg}(h_{\text{la}})) - (A - \epsilon) \|_2^2 ]\)，其中 \(\text{sg}\) 表示stop‑gradient。

### 2.2 语义对齐3D集成（Semantic‑Aligned 3D Integration）
- **冻结原始2D视觉通路**（已在VLM预训练中对齐语义），保持2D语义特征稳定。
- **添加可训练的3D空间编码器**（选用VGGT），从多视图图像中提取**几何结构特征**（场景布局、物体形状、空间关系）。
- **零初始化扩展视觉投影器**：将2D与3D特征在特征维度拼接，通过扩展的视觉投影器映射到VLM标记空间。投影器中新增的3D部分**初始化为零**，不破坏原有分布，随后逐步学习几何信息。
- **公式**：\(Z_{\text{vis}} = [H_{\text{2D}} ; H_{\text{3D}}] W_{\text{vis}} + b\)，其中\(W_{\text{vis}}^{(0)} = [W_{\text{Qwen}} ; 0]\)。

### 2.3 本体规范化（Embodiment Canonicalization）
- **动作输出侧**：使用**相对末端执行器动作**，将未来位姿表示为相对于当前末端位姿的SE(3)变换，避免累积误差且弱化机器人基坐标系差异。
- **状态输入侧**：采用**末端位姿 + 关节角度**作为本体感知状态，隐含反映自由度、运动学约束等本体信息；为每个机器人使用**轻量状态投影器**将状态映射到共享VLA表征空间。
- **跨本体迁移**：两阶段轻量适应（先冻结VLA主干仅训练新状态投影器，再端到端微调），不向上层表征注入机器人身份软提示。
