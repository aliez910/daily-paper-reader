## 摘要
时序建模对于机器人操作至关重要，因为有效控制既需要记忆过去的交互，也需要想象未来的状态。然而，大多数VLA模型主要依赖当前观测，因此在长时域和时域依赖的任务上表现不足。认知科学表明，人类依赖工作记忆缓冲短期背景，依赖海马系统保存过去经历的情景记忆，并依赖内部模型想象可能的未来状态演化。受这些机制的启发，我们提出了MemoryVLA++，一个完整的时序建模框架，为VLA模型配备了用于机器人操作的记忆与想象。一个预训练的VLM将当前观测编码为感知和认知token，形成工作记忆。这些token查询感知-认知记忆库以检索相关历史背景。该记忆库存储过去交互中的低级细节和高级语义，并通过冗余感知整合进行更新。一个世界模型在去噪潜在空间中想象未来状态，并在记忆引导下整合想象的潜在表示以形成完整的时域感知token。得到的token条件化一个扩散动作专家，以预测时序一致的动作序列。我们在5个模拟基准和3个类别的真实机器人任务上进行了广泛实验，涵盖3台机器人，涉及通用操作、长时域时序任务、鲁棒性和泛化性。我们的方法在Libero、SimplerEnv、Mikasa-Robo、Calvin、Libero-Plus以及多样化的真实机器人任务上取得了强劲性能，验证了具有记忆与想象的完整时序建模的有效性。例如，在真实机器人上，我们在通用任务、记忆依赖任务和想象依赖任务上分别获得了+9%、+26%、+28%的提升。项目页面：https://shihao1895.github.io/MemoryVLA-PP-Web

## Abstract
Temporal modeling is essential for robotic manipulation, as effective control requires both memory of past interactions and imagination of future states. However, most VLA models rely primarily on the current observation and therefore struggle with long-horizon, temporally dependent tasks. Cognitive science suggests that humans rely on working memory to buffer short-lived context, the hippocampal system to preserve episodic memory of past experience, and internal models to imagine possible future state evolution. Inspired by these mechanisms, we propose MemoryVLA++, a full temporal modeling framework that equips VLA models with memory and imagination for robotic manipulation. A pretrained VLM encodes the current observation into perceptual and cognitive tokens, forming working memory. These tokens query a Perceptual-Cognitive Memory Bank to retrieve relevant historical context. This bank stores low-level details and high-level semantics from past interactions, and is updated through redundancy-aware consolidation. A world model imagines future states in a denoising latent space, and the imagined latents are integrated under memory guidance to form full temporal-aware tokens. The resulting tokens condition a diffusion action expert to predict temporally consistent action sequences. We conduct extensive experiments on 5 simulation benchmarks and 3 categories of real-robot tasks across 3 robots, covering general manipulation, long-horizon temporal tasks, robustness, and generalization. Our method achieves strong performance across Libero, SimplerEnv, Mikasa-Robo, Calvin, Libero-Plus, and diverse real-robot tasks, validating the effectiveness of full temporal modeling with memory and imagination. ## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的 Vision-Language-Action (VLA) 模型（如 OpenVLA、π0）大多仅依赖当前单一观测帧进行动作预测，缺乏对过去交互和未来状态演化的显式建模，导致在需要记忆（如按钮是否已按）或需预判动态（如传送带上抓取）的长时域任务中表现不佳。
- **认知科学启发**：人类通过**工作记忆**缓冲短期上下文，**海马系统**保存过去经验的情景记忆（包含低层细节和高层语义的“逐字”与“要点”表示），并利用**内部模型**模拟未来状态演化。受此启发，作者提出将记忆与想象机制融入 VLA 模型，实现包含过去、现在、未来的完整时序建模。
- **整体意义**：填补现有 VLA 在时序建模上的空白，提升长时域和时域依赖机器人操作任务的性能，向着更通用、更智能的机器人策略迈出一步。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
构建一个端到端框架，使 VLA 模型能同时利用**当前感知**、**过去记忆**和**未来想象**，生成时序一致的动作序列。

### 关键技术细节

- **工作记忆与感知-认知记忆库**：
  - 使用预训练的 Prismatic VLM（7B 参数）将当前 RGB 观测和语言指令编码为**感知 token**（低层视觉细节）和**认知 token**（高层语义），形成工作记忆。
  - 设计**感知-认知记忆库（PCMB）**，按时间顺序存储历史交互中的感知和认知表示（各最多 L 条记录）。
  - **记忆检索**：当前工作记忆 token 作为 query，通过带时间步位置编码的交叉注意力从记忆库中检索相关历史上下文。
  - **门控融合**：检索到的历史表示与当前表示通过可学习门控进行自适应融合（`g = σ(MLP(concat[x, H]))`，输出 `x̃ = g ⊙ H + (1−g) ⊙ x`）。
  - **记忆整合**：当记忆条目超过容量 L 时，合并余弦相似度最高的相邻条目（取平均并删除被合并的条目），保持记忆紧凑且无冗余。

- **世界模型与想象建模**：
  - 采用 Stable Video Diffusion（SVD，1.5B 参数）作为世界模型，用操作视频微调。
  - **想象生成**：推理时世界模型冻结，对随机噪声进行**部分去噪**（仅若干步），从 UNet 中间层提取多尺度特征，经 FPN 聚合和想象压缩器（含空间和时间注意力）得到**想象潜在 token**，编码未来状态演化的紧凑信息。
  - **记忆引导的想象整合**：记忆增强后的感知 token 通过交叉注意力关注想象潜在，再经门控融合，保留决策相关的未来信息，抑制不相关或噪声预测。

- **全时间感知动作专家**：
  - 使用 DiT 扩散模型（DDIM，10 步推理）进行动作序列预测。
  - 噪声动作 token 与添加了时间步嵌入的认知 token 拼接后执行**认知注意力**（提供高层语义指导），然后通过**感知注意力**关注感知 token（注入细粒度视觉细节），最后经 MLP 输出连续 7-DoF 动作。

  - 与最新公开方法比较，并重现基线（如 CogACT 官方检查点）保证公平。
  - 消融实验控制了变量，验证各组件贡献。
  - 但真实任务样本量相对较小（10–25 次/任务），且未做跨机器人泛化测试。

## 6. 论文的主要结论与发现

- **全时间建模显著提升性能**：
  - Libero 平均成功率 98.4%（超 CogACT +5.2%），SimplerEnv 73.9%（+16.6%），Mikasa-Robo 44.4%（+15.0%），Calvin 平均任务长度 4.29（+1.04）。
  - 真实机器人通用任务 +9%，记忆依赖 +26%，想象依赖 +28%。
- **记忆与想象组件各自有效**：
  - 记忆建模中，门控融合、时序感知检索、冗余整合均优于朴素替代。
  - 想象建模中，记忆引导整合显著优于直接相加；部分去噪（1–3 步）已捕获足够未来动态。
- **鲁棒性与泛化性良好**：在 Libero-Plus 多种偏移下保持高成功率，真实 OOD 下性能下降很小。
- **效率与性能平衡**：MemoryVLA++ 的推理延迟仍在可接受范围（0.24–0.30s），适合实际部署。

## 7. 优点

- ​**理论创新强**：从认知科学汲取灵感，统一记忆与想象于 VLA 框架，概念清晰完整。
- ​**设计实用**：
  - 记忆库采用双流（感知+认知），兼顾细节与语义。
  - 想象在潜空间部分去噪，避免像素级预测开销并过滤噪声。
  - 门控融合实现自适应整合。
  - 扩散动作专家适合连续多模态动作。
- ​**实验全面**：横跨 5 模拟基准 + 3 类真实任务、3 种机器人，覆盖通用、长时域记忆/想象、鲁棒性等多个维度，并与 20+ 方法比较。
- ​**消融与分析详尽**：对记忆长度、融合方式、整合策略、去噪步数等均做单独消融；可视化记忆检索注意力、想象结果，论证机制有效性。
- ​**开源与可复现**：提供项目页面，代码与模型将公开。

## 8. 不足与局限

- ​**计算开销**：
  - MemoryVLA++ 推理延迟比基线增加约 29%（0.187s → 0.241s），虽可接受但在要求更高实时性场景中需优化。
  - 训练成本较高（8 GPU，7B+1.5B+300M 模型），限制了低资源团队复现。
- ​**泛化边界**：
  - 真实任务集中于桌面操作，未测试复杂非结构化环境（如移动操作、多机器人协作）。
  - 世界模型仅依赖 SVD + 操作视频微调，在极端动态场景下可能想象不准确。
  - 记忆库容量 L 为固定超参数，需针对不同任务调优。
- ​**实验限制**：
  - 真实任务每任务仅 10–25 轮评估，统计不确定性可能较大。
  - 未与其他显式帧序列方法（如 TTF-VLA）直接对比推理效率。
  - 缺乏对失败模式（如记忆检索错误、想象引导错误）的深入分析。
  - 未讨论安全性和可解释性（为何选择某个历史帧？）。
- ​**本体依赖**：方法设计假定固定多视角 RGB 输入，对单目或深度传感器方案需调整。
