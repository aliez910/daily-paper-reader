## 摘要
视觉-语言-动作（VLA）模型在基准测试上取得了强劲性能，但在真实世界部署中仍面临挑战，如处理未见过的物体、背景变化以及不同的机器人形态。我们认为，这源于缺乏统一的几何感知操作表示，导致现有VLA模型易受低级轨迹监督、三维特征不对齐以及形态差异的影响。为了解决这个问题，我们提出了GEAR-VLA，一个用于学习统一几何感知动作表示以实现通用机器人操作的VLA框架。GEAR-VLA采用从粗到细的动作学习，首先通过多源具身预训练使VLM具备具身推理和离散动作理解能力，随后通过潜在动作标记将动作语义与梯度解耦的DiT连续动作专家模块连接。此外，它还通过将可训练的三维空间主干网络与VLA表示对齐，同时冻结原始VLM对齐的视觉通路，实现语义对齐的三维集成。为了在不同机器人之间共享该表示，GEAR-VLA使用了形态规范化，通过感知形态的状态和形态不变的动作将机器人差异限制在低级接口层面。大量的仿真和真实世界实验证明了其强大的泛化能力：GEAR-VLA在LIBERO、零样本LIBERO-Plus和RoboTwin 2.0上取得了最先进性能，在AgileX上达到85.9%的成功率，在预训练未见过的LDT-01形态上达到81.0%，并在包含212个未见物体的6,360次试验的通用抓取基准上获得90.1%的成功率。代码和模型将在https://github.com/babynabeauty/GEAR-VLA发布。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

- **整体含义**：提出 GEAR-VLA 旨在学习一种同时具备语义基础、几何感知且跨形态可共享的统一动作表示，从而提升 VLA 在未知物体、场景和机器人形态上的泛化能力，推动通用机器人操作的实际部署。

## 2. 方法论：核心思想与关键技术

GEAR-VLA 框架围绕三个关键设计展开，通过**粗到细动作学习**、**语义对齐的 3D 集成** 和**形态规范化** 实现统一的几何感知动作表示。

### 2.1 粗到细动作学习（Coarse-to-Fine Action Learning）

- **具身 VLM 预训练**：
  - 使用多源数据（视觉语言理解、空间定位、轨迹推理、指向、掩码追踪、操作视频等）进行自回归 token 预测训练，使 VLM 获得具身推理和离散动作语义理解。
  - 对于带动作标签的数据，采用 **FAST 风格动作 Token** 将连续动作序列离散化为监督信号。
  - 对于无动作标签的操控视频，训练一个**因果 VQ-VAE 潜动作 Tokenizer**，从连续视频片段中提取离散的潜动作 ID，作为预训练目标，学习动作相关的视觉动态。
  - 所有任务均统一为自回归 token 预测损失：\( L_{VLM} = -\sum_i \log p_\theta(y_i | y_{<i}, O, l, s) \)。

- **连续动作生成**：
  - 在 VLM 后端附加一个 **DiT 动作专家**，但只使用 VLM 产生的潜动作 token 的 K/V 缓存作为输入，**梯度解耦**（stop-gradient），防止连续动作损失回传到 VLM 主干。
  - DiT 动作专家使用 flow matching 损失训练：\( L_{FM} = \mathbb{E}_{A,\epsilon,\tau} [ \| v_\phi(A_\tau, \tau, sg(h_{la})) - (A - \epsilon) \|_2^2 ] \)，其中 \( sg \) 为停止梯度。
  - 潜动作 token 作为语义桥接，将离散动作语义稳定映射到可执行的连续控制。

### 2.2 语义对齐的 3D 几何集成（Semantic-Aligned 3D Geometry Integration）

- 在原有 2D 视觉通路旁增加一个**可训练的 3D 空间编码器**（采用 VGGT，利用多视图一致性建模场景布局和空间关系）。
- **冻结**原始的 2D 视觉编码器（已与 LLM 语义空间对齐），避免干扰原有视觉 token 分布。
- 通过一个**扩展的视觉投影器**将 2D 和 3D 特征在特征维度拼接后映射到 LLM token 空间：
  \[ Z_{vis} = [H_{2D}; H_{3D}] W_{vis} + b, \quad W_{vis}^{(0)} = [W_{Qwen}; 0] \]
  其中新添加的 3D 块采用**零初始化**，保持初始阶段 VLM 原有分布，随训练逐渐注入有效的几何结构。

### 2.3 形态规范化（Embodiment Canonicalization）

- **动作输出端**：使用**相对末端执行器动作**（Relative End-effector Action）。将未来目标姿态表示为相对于当前末端姿态的 SE(3) 变换：
  \[ \Delta T_{t+i} = (T_t^{ee})^{-1} T_{t+i}^{ee} \]
  避免累积误差并减少对基座标系的依赖。
- **状态输入端**：使用末端执行器姿态和关节角度作为配对状态输入 \( s_t = \{ T_t^{ee}, q_t \} \)，通过轻量级状态投影器 \( f_{s,\psi_e} \) 映射到共享表示空间。不同机器人使用独立的投影器（参数极少）。
- **跨形态迁移**：采用两阶段微调——先冻结 VLM 主干仅训练新状态投影器，再进行轻量端到端微调，将新机器人状态分布引入已有共享表示。

- **消融与分析结论**：三大组件（粗到细动作学习、语义对齐3D几何、形态规范化）对最终性能均有重要贡献，其中3D几何集成和形态规范化的提升最为突出，去除任一组件均在LIBERO-Plus上导致至少4-5%的性能下降；注意力模态丢弃实验表明，丢弃状态输入（关节角/EE位姿）影响最大（-12%），其次为腕部图像（-5%），验证了多模态输入的必要性；数据效率分析显示，仅使用25%的训练数据，GEAR-VLA仍能达到约90%的完整性能，远优于基线（通常需50%以上数据才能接近完整性能），说明统一表示具有更强的数据高效性；潜动作建模方式的选择上，本文采用的连续潜动作（通过VQ-VAE且保留连续性）优于LAPA式离散化或无潜动作的直连，证明语义桥接与梯度解耦的设计有效。

## 7. 核心创新点与贡献

- **首次提出几何感知的统一动作表示**，将离散动作语义（来自VLM预训练）、连续动作生成（DiT专家）、3D几何感知（VGGT）和形态规范化四个维度的设计系统性地整合到一个VLA框架中。
- **粗到细动作学习**：通过潜动作token桥接离散语义与连续控制，同时利用stop-gradient防止梯度耦合，兼得VLM的语义泛化能力与DiT的精度。
- **语义对齐的3D集成策略**：冻结预训练的2D视觉编码器，仅通过零初始化的扩展投影器引入3D特征，在不破坏原有语义分布的基础上补充几何结构，实现知识高效融合。
- **形态规范化机制**：通过相对末端执行器动作和独立轻量状态投影器，将不同机器人实例映射到共享表示空间，首次在VLA中实现了对训练时未见机器人形态的零样本迁移（如LDT-01）。
- **大规模真实世界验证**：在超过6000次真实机器人试验上建立了通用抓取评估基准，并公开了详细的OOD测试协议，为后续研究提供了可比参考。

## 8. 局限性

- **算力需求大**：预训练需要240块H200持续运行数十万次迭代，对资源有限的研究组复制困难。
- **依赖基座VLM规模**：当前以Qwen2.5-1.5B为语言骨干，更大规模模型（如7B/72B）的扩展性未验证，可能存在性能收益但会进一步放大算力瓶颈。
- **形态规范化的覆盖有限**：当前仅支持带末端执行器的操作臂（一个夹爪），尚未考虑灵巧手、移动底盘或非刚体末端；对自由度的变化（如5-DoF vs 7-DoF）假设了基于EE空间的动作表示，在欠驱动或过驱动系统上的表现未知。
- **状态输入依赖**：需要实时获取关节角度和末端位姿，在硬件反馈延迟大或状态噪声大的场景中可能影响稳定性；论文未讨论状态丢失或延迟下的鲁棒性。
- **3D编码器的局限**：采用VGGT（基于视频的深度估计），对于高速运动或透明/高反光物体的深度质量可能下降；且仅在静态场景或准静态假设下验证，动态干扰环境下表现待检验。
- **语言指令限制**：仅支持单步任务语言指令，缺乏长期任务分解与记忆机制；目前所有实验均基于短时程操控任务。

## 9. 总结与展望

GEAR-VLA通过粗到细动作学习、语义对齐3D集成和形态规范化三大设计，在多个仿真和真实机器人基准上实现了SOTA操控能力，尤其在通用抓取和跨形态迁移上显著超越现有VLA方法。论文设计的系统级方案和大量消融实验为后续研究提供了坚实基线。未来方向包括：扩展至更丰富的机器人形态（灵巧手、移动双臂）、引入基于3D的闭锁推理（如抓取规划）、利用更高效的LLM骨干降低训练成本、以及结合扩散或世界模型进行动作闭环修正。总体而言，GEAR-VLA代表了VLA类模型向通用、可泛化机器人操作迈出的重要一步。
