## 摘要
当前最广泛采用的机器人学习流程是从机器人演示或结构化人类数据中学习技能，这些数据收集成本高昂且与特定具身绑定。相比之下，非结构化人类视频提供了一种可扩展的替代方案。它们包含跨物体、场景和策略的多样化操作演示，但并未直接连接到机器人动作。我们提出了LUCID，这是一个两阶段框架，从互联网规模的数据集中抽取非结构化人类视频学习任务意图，并在大规模并行仿真中学习机器人控制。意图模型从当前闭环观测中预测短时域意图（场景中下一步应该发生什么）。一个具身特定的感觉运动策略将该意图转换为机器人动作。意图接口在控制器之间共享，因此相同的意图模型可以应用于不同的具身，从我们的主要灵巧手到平行爪夹持器。我们在五个真实世界操作任务上评估了LUCID：仅由互联网视频监督的搅拌、擦拭和分拣，零样本迁移到新场景和物体实例；以及分别由1小时自收集智能手机视频监督的推T和电缆布线。项目页面：https://lucid-robot.github.io/。

## Abstract
The most widely-adopted robot learning pipelines today learn skills from robot demonstrations or structured human data, which are expensive to collect and tied to specific embodiments. In contrast, unstructured human videos provide a scalable alternative. They contain diverse manipulation demonstrations across objects, scenes, and strategies, but are not directly connected to robot action. We propose LUCID, a two-stage framework that learns task intent from unstructured human videos drawn from internet-scale datasets and learns robot control in massively-parallel simulation. The intent model predicts short-horizon intent (what should happen next in the scene) from the current observation in closed loop. An embodiment-specific sensorimotor policy converts this intent into robot actions. The intent interface is shared across controllers, so the same intent model can be applied to different embodiments, from our primary dexterous hand to a parallel-jaw gripper. We evaluate LUCID on five real-world manipulation tasks: stirring, wiping, and binning supervised by only internet video, with zero-shot transfer to novel scenes and object instances; and push-T and cable routing supervised by 1 hr each of self-collected smartphone video. ## 2. 论文提出的方法论

### 核心思想
- **意图与控制的分离**：将任务意图（预测场景中接下来应发生什么）与具体动作执行（如何通过机器人关节实现该意图）分为两个独立模块，通过一个短时域参考接口（物体流 + 手掌姿态）连接。
- **闭环意图预测**：部署时，意图模型持续从当前实时观测（RGB-D）重新查询并更新未来参考，而非一次性生成固定计划，从而提升应对执行偏差的鲁棒性。
- **利用两种可扩展数据源**：意图模型从大量无结构人类视频（互联网/自拍视频）中学习，无需任何机器人数据；控制策略从并行仿真中学习，利用程序化生成的目标和参考轨迹进行任务无关训练。

### 关键技术细节

1.  **意图模型 \( f_\theta \)**
    - **输入**：当前观测（叠加 F 帧 RGB-D）、当前查询点在目标物体上的 3D 位置 \(\{x_{n,0}^{trk}\}\)、当前手掌姿态 \((p_0^{palm}, R_0^{palm})\)。
    - **输出**：短时域（T 步，约 1 秒）的物体轨迹 \(\{\hat{x}_{n,\tau}^{trk}\}\) 和手掌姿态参考 \(\{(p_\tau^{palm}, R_\tau^{palm})\}\)。
    - **架构**：基于 CoTracker3 的点 token Transformer。使用冻结的 DINOv3 提取 RGB 特征，通过残差适配器融合深度特征。在点 token 基础上增加一个手掌姿态 token，实现联合预测。预测方向为前向（从当前到未来），不同于常规点追踪器的过去帧追踪。
    - **训练**：使用从视频中提取的监督信号（通过 ViPE、SAM 3.1、DenseTrack3Dv2、WiLoR 流水线），损失函数为物体流和手掌姿态的均方误差组合。关键是对视频中的人类像素进行强数据增强（颜色抖动、填充、网格等），迫使模型专注于物体运动而非示教者手部外观。
    - **监督数据提取**：对每个视频片段进行四阶段处理：① ViPE 恢复相机参数和深度；② SAM 3.1 分割物体和人类掩码；③ DenseTrack3Dv2 采样并追踪物体点云；④ WiLoR 重建手部网格并通过刚性拟合获得手掌姿态。

2.  **感觉运动策略 \( \pi \)**
    - **核心功能**：将意图参考映射为具体机器人的关节动作。为每种形态（灵巧手/平行爪）单独训练一次，训练配方相同。
    - **训练数据**：在仿真中程序化生成目标物体（简单几何体组合）和参考轨迹（包含接近、手中运动、目标、脱离四个阶段），覆盖抓取、握内操作、放置、释放等行为。每种轨迹包含随机 grasping 方式和手掌-物体耦合模式。
    - **动作空间**：输出手臂关节增量 + 特征抓取基系数 + 手指残余关节指令。特征抓取基基于人类抓取姿态的 PCA，引导探索偏向自然抓取，同时允许残余项补偿。
    - **观测与目标**：教师策略使用特权状态（物体位姿、全表面点、力等）。学生策略使用外部 RGB-D 相机可见的物体点云、腕部深度图像、本体感知。两者都接收同一条参考轨迹（物体流 + 手掌姿态）作为目标。
    - **奖励**：单一任务无关奖励，平衡物体跟踪、手掌姿态跟随、手指接触正则化和动作惩罚。
    - **架构**：基于 AME（交叉注意力策略）。学生策略将腕部深度图编码为 patch token，与点轨迹 token 一起通过自注意力和交叉注意力，最终生成动作分布和状态值。
    - **训练与 Sim-to-Real**：教师通过 PPO 训练，伴随难度自动调整（课程学习）。通过系统辨识和领域随机化（静态 + 自适应）匹配真实环境。教师策略通过**混合蒸馏**（PPO 损失 + 教师动作回归损失）训练学生策略，早期以模仿为主，后期 PPO 主导，减少部署部署模态差距。

3.  **真实部署**
    - **慢意图循环**（1 Hz）：SAM 刷新物体掩码 → 采样并反投影查询点 → 查询意图模型得到新参考。
    - **快控制循环**（50 Hz）：学生策略接收当前点云、参考、腕部深度、本体感知，输出动作。
    - **帧间对齐**：使用滑动窗口点追踪器保持查询点在实时场景中的一致性。

2. **意图模型可零样本迁移到新场景和物体实例**：

2. **意图模型可零样本迁移到新场景和物体实例**：在三个互联网视频任务（搅拌、擦拭、分拣）中，LUCID 直接部署于包含全新物体实例（不同颜色、形状、材质）、桌面背景以及外置摄像头视角的多样化现实场景，无需任何微调或域适应。例如，搅拌任务中模型泛化到不同杯子和搅拌棒，擦拭任务泛化到不同桌布和海绵，分拣任务泛化到玩具类型和容器排列的变化。这表明从海量人类视频中学习的意图模型捕捉的是任务本质的物体流动模式，而非特定视觉外观，因此对视觉变化具有鲁棒性。

3. **跨形态迁移成为现实**：在自收集任务（推 T 形块和电缆布线）中，同一个意图模型与两种不同机器人形态的感觉运动策略（LEAP 灵巧手与平行爪夹持器）组合后，在真实世界取得了相近的成功率（推 T：灵巧手 86.7%，平行爪 80.0%；电缆布线：灵巧手 73.3%，平行爪 66.7%）。这证明意图模型编码的是形态无关的物体‑手掌空间关系，实现了训练时未见形态的零样本迁移。

4. **数据规模化显著提升性能**：通过构造不同规模的训练集（1k、2k、5k、10k、20k 视频片段）训练意图模型，并在真实世界评估分拣任务，发现成功率从约 30%（1k）单调提升至约 73%（20k）。曲线斜率随数据量增大而减小，提示进一步增加数据仍可能带来收益，但需配合更高效的模型容量。

5. **关键设计选择的价值**：
   - **闭环更新**：在 web 任务上将平均成功率从开环基线的 28% 提升至 73%，是性能提升的根本原因。
   - **特征抓取基**：去除后策略学习曲线明显下降，且真实执行时手指抖动增加，说明该先验有效引导探索并限制不自然抓取。
   - **腕部深度图像**：消融导致成功率大幅降低，证明其为学生策略提供了关键的近场几何信息以克服外部相机遮挡。
   - **混合蒸馏策略**（PPO+BC 权重逐渐转换）比纯 DAgger 更鲁棒，能在模态差距下稳定部署策略。
   - **查询点数量**：在仿真中发现性能随点数（2→16）提升后饱和，为计算‑精度权衡提供了参考。

6. **局限性与发展方向**：当前方法对透明/反光物体、快速连续操作场景仍不稳定；意图模型依赖点追踪器，严重遮挡下可能漂移；感觉运动策略仍需针对每种形态在仿真中单独训练，未能完全实现零样本迁移到全新机器人硬件。未来工作可探索通用策略架构、更强大的视觉基础模型或更直接的跨形态意图映射。
