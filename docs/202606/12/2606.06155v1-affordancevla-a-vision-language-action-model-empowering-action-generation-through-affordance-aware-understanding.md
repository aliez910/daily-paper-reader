## 摘要
视觉-语言-动作（VLA）模型利用预训练的视觉-语言模型（VLM）丰富的世界知识来实现遵循指令的机器人操作。然而，VLM语义空间与具身控制策略之间的结构不匹配往往阻碍了精确的感知-动作映射的学习。为了解决这一挑战，我们提出了AffordanceVLA，这是一个统一的框架，通过引入结构化的可供性预测作为面向任务的中间表示，以建立更精确和鲁棒的感知-动作映射。具体来说，我们通过三个互补组件逐步建模操作先验：1）Which2Act，通过视觉潜在预测实现以物体为中心的定位，以抑制干扰；2）Where2Act，通过可供性图估计实现二维交互定位；3）How2Act，通过三维几何推理引导操作策略。这些可供性线索提供了空间基础、语义条件和动作耦合的中间表示，从而自然地连接了视觉、语言和动作。我们将这些模块集成到包含专门专家的混合Transformer（MoT）架构中，并使用渐进式数据课程的三阶段训练策略来训练模型。为了克服机器人数据集中密集可供性标签的稀缺性，我们还开发了一个鲁棒的自动化数据增强流水线。在仿真和真实世界中的大量实验表明，AffordanceVLA在各种操作场景中均取得了强劲的性能。

## Abstract
Vision-Language-Action (VLA) models leverage the rich world knowledge of pretrained vision-language models (VLMs) to enable instruction-following robotic manipulation. However, the structural mismatch between VLM semantic spaces and embodied control policies often hinders the learning of precise perception--action mappings. To address this challenge, we propose \textbf{AffordanceVLA}, a unified framework that introduces structured affordance forecasting as a task-oriented intermediate representation to establish a more precise and robust perception--action mapping. Specifically, we progressively model manipulation priors through three complementary components: 1) \textbf{Which2Act} for object-centric grounding via visual latent prediction to suppress distractions; 2) \textbf{Where2Act} for 2D interaction localization via affordance map estimation; and 3) \textbf{How2Act} for 3D geometric reasoning to guide manipulation policies. These affordance cues provide spatially grounded, semantically conditioned, and action-coupled intermediate representations, thereby naturally bridging vision, language and action. We integrate these modules into a Mixture-of-Transformer (MoT) architecture with specialized experts and train the model using a three-stage training strategy with a progressive data curriculum. To overcome the scarcity of dense affordance labels in robotic datasets, we also develop a robust automated data augmentation pipeline. Extensive experiments on simulation and real-world demonstrate that AffordanceVLA achieves strong performance across diverse manipulation scenarios.

## 1. 核心问题与整体含义

**研究动机：**  
现有 Vision-Language-Action (VLA) 模型利用预训练 VLM 的丰富世界知识来实现指令跟随的机器人操作。但 VLM 语义空间与具身控制策略（3D 物理空间）之间存在**结构性不匹配**，直接端到端映射难以学到精确的感知-动作映射。

**核心洞察：**  
- 传统方法要么依赖视频预测（冗余、计算重），要么使用粗略的文本推理，缺乏对操作关键要素（操作什么、在哪操作、如何操作）的显式建模。
- 受人类感知启发（看见杯子把手就知道可抓取），**可供性（Affordance）** 是一种天然桥接：空间接地于视觉、语义条件于语言、动作耦合于执行。
- 本文主张：**数据规模扩大不是瓶颈，表示学习才是关键**。引入结构化可供性作为中间表示，可以更鲁棒地桥接感知与动作。

## 2. 方法论

### 核心思想
采用**可供性预测作为面向任务的中间监督**，将抽象语义转化为可操作的空间几何信息，从而建立更精确的感知-动作映射。

### 关键技术细节

**（1）三个互补的可供性预测模块：**

| 模块 | 功能 | 技术实现 | 损失函数 |
|------|------|----------|----------|
| **Which2Act** | 物体级视觉定位，抑制背景干扰 | 预测目标 bounding box 的连续视觉潜在（使用 Frozen Flux VAE 编码），通过 MSE 重构 | MSE |
| **Where2Act** | 2D 交互热点定位 | Transformer 解码器将 1D 查询映射为 2D 空间 logits，生成可供性热力图 | 逐像素 BCE |
| **How2Act** | 3D 几何与空间布局推理 | 双分支：形状分支用条件扩散生成 3D 体素潜在；布局分支用 MLP 回归 10-DoF 向量（旋转+缩放+平移） | 噪声预测 + Smooth-L1 |

**（2）Mixture-of-Transformer (MoT) 架构：**
- **Understanding Expert**：基于预训练 VLM（SigLIP + PaliGemma），融合视觉观察与语言指令为语义特征。
- **Affordance Generation Expert**：接收语义特征，并行解码三个可供性查询（Which2Act / Where2Act / How2Act）。
- **Action Expert**：基于语义特征和可供性特征，通过扩散/流匹配生成连续动作块。
- **UAA 渐进注意力机制**：专家间严格的单向因果注意力（Affordance 只查 Understanding，Action 查两者），防止动作信息泄露到预测阶段；专家内部双向注意力促进上下文融合。

**（3）三阶段渐进训练策略：**

| 阶段 | 数据 | 可训练参数 | 目标 |
|------|------|------------|------|
| **Stage I** 通用可供性接地预训练 | VQA 数据（AGD20K, RefSpatial, PRISM）+ SAM-3D 增广 | 仅 Affordance Generation Expert + 查询 + 解码器 | 多任务可供性损失（仅预测） |
| **Stage II** 可供性增强的机器人数据联合训练 | 大规模合成机器人数据（InternData-A1）+ 自动化标注流水线生成的标签 | 全部解冻（视觉编码器低 LR） | 动作损失 + 可供性损失（权重 1:0.5） |
| **Stage III** 目标任务后训练 | 目标数据集（LIBERO / CALVIN / DROID） | 全部，可供性损失权重降至 0.15 | 侧重动作精调 |

**（4）自动化数据标注流水线（克服标签稀缺）：**
1. 规则化关键帧提取
2. LLM（Claude Opus 4.5）将全局指令分解为逐关键帧子指令
3. VLM（Qwen3-VL）为每个关键帧生成检测类别和可供性查询
4. 精调后的 RexOmni + SAM + SAM-3D 生成 bbox、热力图、形状/布局潜在
5. 严格质量验证（点-框一致性 + 人工抽检 3000 样本 100% 通过）

## 6. 主要结论与发现

1. **结构化可供性表示是更有效的中间表示(回答 Q1)**：三模块（Which/Where/How）渐进协同，从粗到细、从 2D 到 3D，显著优于冗余视频预测或单一视觉特征。
2. **MoT 解耦架构防止表示坍缩(回答 Q2)**：冻结式可供性或统一网络结构均导致性能严重下降，解耦且联合优化是核心增益来源。
3. **三阶段渐进训练桥接泛化与特化(回答 Q3)**：Stage II 大规模机器人数据联合训练是 OOD 泛化的关键桥梁，跳过则 CALVIN 平均链长从 4.33 降至 3.81。
4. **数据效率显著提升**：仅用 40% 微调数据即达到全量 Pi0 上限，证明架构-表示-数据三者相互放大，而非可替代。
5. **真实世界多任务优异**：基本任务平均成功率 88.3%，复杂任务 82.9%，在指令敏感性任务（Drawer / Toaster）中显著优于 Pi0（44.8%）。
6. **可供性作为语义锚点**：假设可供性损失保护 VLM 的指令跟随能力不被动作损失侵蚀（类似 π0.5/π0.7 策略），虽缺乏直接机制证据，但实验行为一致。

## 7. 优点（方法 / 实验设计亮点）

**方法方面：**
- **结构化表示设计**：将可供性分解为 Which / Where / How 三个正交维度，覆盖物体、位置、几何三重先验，自然桥接 VLA。
