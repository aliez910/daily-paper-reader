## 摘要
机器人技术面临数据稀缺的根本挑战。与语言或视觉研究不同，机器人操作领域没有互联网规模的数据集。一个有前景的前进方向是利用第一人称人类数据，这类数据更容易收集，范围更广，规模更大。为此，我们研究了跨人类和配备灵巧五指人形本体的学习关键设计选择，以π₀.₅模型为基础。我们的结果表明，人类数据使机器人能够学习新的任务语义，并将现有技能组合成新颖行为，而无需相应的机器人数据。论文网站：https://egopipaper.github.io/

## Abstract
Robotics faces a fundamental challenge of data scarcity. Unlike language or vision research, there is no internet-scale dataset for robotic manipulation. A promising path forward is to leverage egocentric human data, which can be collected more easily, with greater breadth, and at a larger scale. Towards this end, we investigate key design choices for learning across human and humanoid embodiments equipped with dexterous five-finger hands, using the $π_{0.5}$ model as a foundation. Our results show that human data enables robots to learn new task semantics and compose existing skills into novel behaviors without corresponding robot data. ### 关键技术细节
1. **动作交错表示（Action Interleaving）**  
   - π₀.₅ 原始动作头最大输出维度为 32，而灵巧双手各需 29 维（单手：位置3+姿态6+关节20=29），总 58 维超出容量。  
   - 将左右手动作分布到两个动作 token 中（先左后右），有效 horizon 减半，但不修改预训练参数，实现良好收敛。  
   - 公式：  
     \( a_{t:t+H/2} = \{ a_t^L, a_t^R, ..., a_{t+H/2}^L, a_{t+H/2}^R \} \)

2. **机器人中心动作对齐**  
   - 传统方法先预测指尖位置再逆运动学求解，易导致高自由度手自碰撞。  
   - 改为将人类手部 20 个关键点（MANO 约定）先转换为每连杆关节角，再通过带偏置和缩放因子的线性映射到机器人关节空间（Tesollo 20 关节 / Inspire 6 关节）。

3. **视觉对齐技术**

4. **辅助损失：子任务预测**

## 6. 主要结论与发现
- **人类数据可有效传递高级任务语义**：在番茄分拣（92%）、包装（90%）任务中，仅通过联合训练即可学会排序和顺序规则，无需机器人演示。
- **子任务预测是复杂任务的关键**：对于需要认知理解（先开箱后放物）的装箱任务，简单联合训练失败（27%），引入子任务预测后提升至 93%。
- **骨架线叠加效果有限**：视觉对齐帮助不明显，模型可能更依赖抽象特征而非显式对应。
- **手型选择影响迁移效果**：更接近人手的 Inspire（小型）比 Tesollo（大型）在装箱任务中表现更好，可能因视觉差异更小。
- **手腕相机在测试中不可或缺**：即使训练时丢弃 40%，测试期去掉手腕相机仍导致抓取成功率明显下降，说明策略实际依赖该模态。
