## 摘要
视觉-语言-动作（VLA）模型在机器人操作中展现出了强大的能力，但由于累积误差传播，其性能在长程任务中显著下降。这一限制很大程度上源于静态特征融合机制，该机制依赖固定权重来组合视觉、语言和动作表示，阻碍了模型适应任务执行的不同阶段。为了克服这一限制，我们提出了S$^2$-VLA，该框架引入了状态空间引导的自适应注意力（SSGAA）机制。SSGAA维护一个跟踪任务进度信念状态，并生成动态门控权重，以自适应地融合来自三个互补源的信息：用于空间感知的视觉特征、用于高级任务规划的任务意图以及用于执行一致性的时序动作序列。这种自适应融合允许模型在任务执行过程中动态调整关注重点，以满足不同任务阶段不断演变的需求。尽管其参数规模仅为2B，S$^2$-VLA始终优于更大的7B规模模型，并在LIBERO和SimplerEnv等长程操作基准上取得了最先进的性能，突显了自适应特征融合对于长程机器人操作的关键作用。

## Abstract
Vision-Language-Action (VLA) models have demonstrated strong capabilities in robotic manipulation, but their performance degrades significantly in long-horizon tasks due to cumulative error propagation. This limitation largely arises from static feature fusion mechanisms that rely on fixed weights to combine visual, language, and action representations, preventing the model from adapting to different phases of task execution. To address this limitation, we propose S$^2$-VLA, a framework that introduces a State-Space Guided Adaptive Attention (SSGAA) mechanism. SSGAA maintains a belief state that tracks task progression and generates dynamic gating weights to adaptively fuse information from three complementary sources visual features for spatial perception, task intents for high-level task planning, and temporal action sequences for execution consistency. This adaptive fusion allows the model to shift its focus throughout task execution, aligning with the evolving requirements of different task stages. Despite its compact 2B parameter size, S$^2$-VLA consistently outperforms larger 7B-scale models and achieves state-of-the-art performance on long-horizon manipulation benchmarks, including LIBERO and SimplerEnv. highlighting the importance of adaptive feature fusion for long-horizon robotic manipulation.

## 2. 方法论
- **核心思想**：提出S²-VLA框架，引入**状态空间引导的自适应注意力（SSGAA）**机制。通过维护一个紧凑的信念状态（belief state）动态跟踪任务进度，并据此生成门控权重，自适应地融合三个互补信息源：低层视觉特征（空间感知）、高层任务意图（语义规划）和动作序列（时序一致性）。
- **关键技术细节**：
  - **信念状态更新**：使用轻量级GRU，以近期动作序列和本体感受为输入，递归更新隐藏状态，再通过线性层输出信念状态 \(b_t\)。该过程编码了任务阶段信息和执行质量。
  - **统一输入构建**：将视觉特征（ViT提取）、语言指令tokens、可学习意图tokens拼接为序列，送入VLM Transformer（Qwen3-VL）得到层级表示。
  - **SSGAA注意力的三个分支**：
    - 动作序列自注意力：建模K步预测间的时序依赖。
    - 低层视觉交叉注意力：查询动作表示对视觉tokens做交叉注意，保留精细空间细节。
    - 高层意图交叉注意力：查询动作表示对意图tokens做交叉注意，提取任务语义。
  - **门控融合**：信念状态经MLP和Softmax生成三层动态权重，对各分支输出加权求和。融合后的表示经前馈网络处理后既作为下一层查询输入，也回馈至当前层动作自注意力分支，实现迭代优化。
  - **并行动作解码**：最后SSGAA层输出经层归一化和线性投影，直接得到K步连续动作预测。
- **训练与推理**：端到端训练，损失为均方根误差（动作预测误差）。推理时每步先更新信念状态，再计算门控权重并预测动作。

## 6. 主要结论与发现
- S²-VLA在LIBERO上平均成功率98.2%（Spatial 98.4%, Object 99.6%, Goal 98.4%, Long 96.4%），在SimplerEnv上平均78.1%，均超越所有对比方法（包括多个7B参数模型）。
- 自适应门控在**中间层**（如第12层）效果最佳；在浅层或深层设置门控反而无增益。
- 信念状态引导是性能提升的关键：替换为静态向量后性能下降至95.8%（原为96.4%），说明动态调制不可或缺。
- 门控权重可视化显示：模型在需要精细定位时提升视觉分支权重，在任务切换时提升意图分支权重，在稳态运动时动作分支主导，证实了阶段自适应能力。
- 紧凑的2B参数架构比大规模静态融合模型更有效，自适应融合优于单纯扩大模型规模。
