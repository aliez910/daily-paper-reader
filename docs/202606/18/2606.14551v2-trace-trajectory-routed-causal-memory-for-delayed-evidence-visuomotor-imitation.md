## 摘要
自主运行的机器人可能需要基于不再可见的证据做出决策。我们研究延迟证据任务，其中早期线索在后续决策点之前消失，因此视觉上相似的观察结果可能需要不同的动作。在这些场景中，当前的观察结果不足以作为控制状态。我们引入基于轨迹路由的因果证据（TRACE），这是一种用于视觉运动模仿策略的记忆框架。TRACE将任务相关的视觉和机器人状态证据（如物体身份、目标选择或路线依赖状态）存储在一个固定大小的潜在记忆中，该记忆在长时间段内保持有界。TRACE并非通过原始时间或手动提供的任务标签来索引记忆，而是使用路径签名：执行机器人状态轨迹的紧凑、顺序敏感特征。这些签名不存储视觉线索本身，而是为写入和检索线索可见时存储的证据提供基于轨迹的键。当机器人随后遇到模糊观察时，策略以TRACE记忆为条件，以恢复缺失的上下文并选择正确的分支。TRACE通过轻量级适配器附加到策略上，而不改变策略主干、动作头或模仿目标。在具有视觉模糊分支点的真实世界长期操作任务中，TRACE在分支选择和任务成功率上优于替代基线，包括短期记忆和循环记忆。项目页面：https://jeong-zju.github.io/trace

## Abstract
Robots under autonomous operation may require decisions based on evidence that is no longer visible. We study delayed-evidence tasks, where an early cue disappears before a later decision point, so visually similar observations can require different actions. In these settings, the current observation is not a sufficient state for control. We introduce TRAjectory-routed Causal Evidence (TRACE), a memory framework for visuomotor imitation policies. TRACE stores task-relevant visual and robot-state evidence, such as object identity, target choice, or route-dependent state, in a fixed-size latent memory that remains bounded over long episodes. Instead of indexing memory by raw time or manually provided task labels, TRACE uses path signatures: compact, order-sensitive features of the executed robot-state trajectory. These signatures do not store the visual cue itself; rather, they provide trajectory-conditioned keys for writing and retrieving the evidence stored when the cue was visible. When the robot later reaches an ambiguous observation, the policy conditions on TRACE memory to recover the missing context and choose the correct branch. TRACE attaches through lightweight adapters to policies, without changing the policy backbone, action head, or imitation objective. Across real-world long-horizon manipulation tasks with visually ambiguous branch points, TRACE improves branch selection and task success over alternative baselines, including short-history and recurrent memory. ## 1. 论文的核心问题与整体含义（研究动机和背景）
- 研究 **延迟证据（delayed-evidence）任务**：机器人需要在早期线索已经消失的决策点做出正确选择，而当前观测的视觉信息在多个历史轨迹下可能非常相似，无法区分。
- 现有视觉运动模仿策略大多基于马尔可夫假设（当前观测足够用于决策），在延迟证据场景下失效。
- 短期历史窗口无法保留已消失的线索；循环记忆或通用记忆容易导致相关线索被覆盖、稀释或与任务无关信号混合。
- 因此需要一种**因果性、固定预算**的记忆机制，能够在线更新、保持早期任务证据直到决策时被使用。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：将记忆的内容（视觉/机器人状态证据）与记忆的访问（由执行轨迹决定）分离。利用**轨迹路径签名（path signatures）**作为紧凑、顺序敏感的轨迹描述符，用于对固定大小的潜在记忆槽进行写入和读取索引。
- **关键技术**：
  - **路径签名**：对机器人状态轨迹（分段线性插值）计算到一定深度（p=3）的签名，得到固定长度的向量，对采样率、速度不敏感，但对轨迹顺序敏感。同时计算增量签名（signature 导数）。
  - **签名路由插槽记忆（Signature-Routed Slot Memory）**：
    - 使用学习投影将签名转化为路由特征 `q_t`，用于计算软路由权重 `ω_t`，确定对 K 个潜在内存槽的写入程度。
    - 当前观测编码为内容 `e_t`，与路由特征结合产生候选内容，通过门控更新被选中的槽。
    - 读取时，用基于当前观测和路由特征形成的查询注意力从更新后的槽中提取记忆读出 `z_mem`。
  - **策略适配器（Policy Adapter）**：TRACE 提供统一的记忆状态，通过轻量适配器接入不同策略家族（如回归动作块策略或扩散策略），不修改策略主干、动作头或模仿损失。
  - **训练**：使用固定预算的历史序列训练因果扫描，并加入辅助损失（平衡槽使用、熵正则化、读写一致性）防止模式坍塌，保持模仿损失不变。

## 6. 论文的主要结论与发现
- **TRACE 显著提升延迟证据操作**：在 5 个任务的平均阶段进度上，TRACE 回归版本 69.23%，扩散版本 59.53%，远超无记忆基线（25.00–25.50%）和最强 VLA 基线（π0.5：50.47%）。
- **增益跨策略家族传递**：TRACE 使用相同的记忆更新器，仅更换适配器即可同时提升回归和扩散策略。
- **优于通用历史记忆**：TRACE 相比 GRU、Transformer 上下文、LRU、检索提示等均有明显优势，说明将记忆绑定到因果轨迹顺序的重要性。
- **诊断证实记忆使用历史证据而非决策点观测**：顺序保持变换下路径相似性和分支一致性高，而顺序反转大幅下降，证明 TRACE 确实依赖有序历史。

## 7. 优点
- **模块化设计**：TRACE 通过轻量适配器附加到现有策略，不改变策略主干、动作头或模仿损失，便于集成。
- **使用路径签名索引**：签名是紧凑、顺序敏感、对时间参数化不敏感的描述符，天然适合作为轨迹路由的键。
- **固定预算记忆**：K 个潜在槽（≤6），在整个长回合中保持有界，避免无限增长。
- **路由分离内容与地址**：视觉证据存储为内容，轨迹签名决定写/读位置，避免信号混合。
- **丰富的诊断实验**：通过历史变换对比，直接证明机制有效；同时也分析了任务级方差和失败模式。

## 8. 不足与局限
- **签名维度爆炸**：机器人状态维度高或签名深度增加时，原始签名维度快速增长（17维状态在p=3时达5,219），增加标准化和投影成本；更高深度可能不实用。
- **任务覆盖有限**：仅在 5 个桌面操作任务上评估，未在更复杂环境（动态场景、多机器人、室外）测试。
- **理论保障缺失**：固定槽记忆在极长或信息极丰富的回合中仍可能发生覆盖或模糊，论文仅通过诊断实验表明其稳定性，缺乏理论保证。
- **未公开完整训练资源细节**：未明确训练所用 GPU 数量、型号和实际时间，可能影响可重复性。
- **高方差任务分析有限**：Tool 任务高方差的根本原因（延迟记忆 vs 接触执行）未完全分离，只能通过 leave-one-out 间接说明。
