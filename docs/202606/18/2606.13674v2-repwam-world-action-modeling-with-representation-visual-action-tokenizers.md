## 摘要
该工作提出了RepWAM，一种以表征为中心的世界动作模型（WAM），其基于表征视觉-动作标记器构建。现有的WAM通常从预训练的视频生成模型中继承面向重建的视频标记器。尽管这些标记器保留了视觉保真度，但仅靠像素重建对于学习连接未来预测与机器人控制的指令遵循动态提供的指导有限。为解决这一问题，我们探索了一种语义视觉-动作潜在空间，用于以表征为中心的世界动作建模。具体而言，我们训练了一个表征视觉-动作标记器，将视觉输入映射为对齐的视觉和潜在动作标记。然后，我们预训练WAM以联合建模未来视觉状态和连接它们的潜在动作，同时遵循语言指令，之后适应真实机器人轨迹以实现闭环操作。在真实世界操作任务和模拟基准上的实验表明，RepWAM在各种操作设置下均表现出色，而消融实验则突出了语义视觉-动作标记化相较于面向重建替代方案的价值。这些结果确立了表征视觉-动作标记化作为世界动作模型的一个有前途的基础，并朝着通用机器人策略迈出了一步。代码和权重将在https://github.com/wdrink/RepWAM提供。

## Abstract
This work presents RepWAM, a representation-centric world action model (WAM) built on representation visual-action tokenizers. Existing WAMs typically inherit reconstruction-oriented video tokenizers from pretrained video generation models. Although these tokenizers preserve visual fidelity, pixel reconstruction alone provides limited guidance for learning instruction-following dynamics that connect future prediction with robot control. To address this, we explore a semantic visual-action latent space for representation-centric world action modeling. Specifically, we train a representation visual-action tokenizer that maps visual inputs into aligned visual and latent action tokens. We then pretrain our WAM to jointly model future visual states and the latent actions that connect them under language instructions, followed by adaptation to real robot trajectories for closed-loop manipulation. Experiments on real-world manipulation tasks and simulation benchmarks show that RepWAM delivers strong performance across diverse manipulation settings, while ablations highlight the value of semantic visual-action tokenization over reconstruction-oriented alternatives. These results establish representation visual-action tokenization as a promising foundation for world action models and a step toward generalist robot policies. Code and weights will be available at https://github.com/wdrink/RepWAM.

## 1. 核心问题与整体含义（研究动机和背景）

- **现有世界动作模型（WAM）的瓶颈**：现有 WAM 通常从预训练视频生成模型中继承面向重建的视频标记器（如 WAN2.2 VAE）。这类标记器优化像素重建，导致潜空间被低级外观（如背景纹理）占用，缺乏语义结构（如物体身份、空间关系），使得语言指令难以有效引导未来预测与机器人动作生成。
- **关键问题**：视觉潜变量与机器人动作处于不相交的空间，每步都需要逆动力学模型（IDM）跨越模态缝隙，限制了指令跟随动态的学习效率和最终控制性能。
- **本文动机**：重新设计 WAM 的潜空间，建立一个语义丰富的视觉-动作联合表征，使视觉状态及其动作引起的转变在同一空间中表示，从而提升世界模型预测与动作推理的质量。

## 2. 论文提出的方法论

### 核心思想
- 构建**表征视觉-动作标记器（RepViTok）**，将视频观测压缩为语义对齐的视觉标记和揭示状态间转变的潜在动作标记；在此语义潜空间上训练因果世界动作模型，联合预测未来视觉状态与对应的潜在动作，最后通过微调适应真实机器人控制。

### 关键技术细节
- **视觉标记化**：使用 ViT 自编码器（12 层 Transformer，隐维度 768）对视频帧/时空块编码，输出视觉潜变量 \(z\)。除像素级重建损失（L1、感知、对抗）外，额外引入**特征对齐损失**：将 \(z\) 线性投影后与冻结的视觉基础模型（Perception Encoder）输出进行均方误差匹配，迫使潜空间蕴含语义信息。
- **潜在动作标记化**：固定视觉标记器后，训练潜在动作标记器（LAT），包含逆动力学模型（IDM）和正向动力学模型（FDM）：
  - IDM 将连续视觉状态 \((z_t, z_{t+1})\) 压缩为低维潜在动作 \(\ell_t\)（维度 \(d_\ell=4\)）。
  - FDM 以 \((z_t, \ell_t)\) 生成传输矩阵 \(K_t\) 和残差 \(\delta_t\)，然后重建下一状态 \(\hat{z}_{t+1}=K_t z_t + \delta_t\)（\(K_t\) 类似语义空间的软传输算子，\(\delta_t\) 捕捉残余变化）。
  - 训练目标：前向预测损失 + 反向一致性损失（对逆序状态对同样预测）。
- **因果世界动作模型**：
  - 将视觉标记与潜在动作标记组织成块（chunk size 从 [1,4] 采样），每个块包含连续的视觉标记和它们之间的潜在动作。
  - 采用**因果扩散 Transformer**（30 层），语言指令经预训练文本编码器通过交叉注意力注入；相同注意力权重视觉和动作共享，但使用独立的 FFN。
  - 训练采用**条件流匹配（Flow Matching）**：对每个块加入高斯噪声，学习从噪声到真实数据的线性插值速度，分别计算视觉和潜在动作部分的均方误差，并用权重 \(\lambda_a\) 平衡。
- **两阶段训练**：第一阶段在大量机器人视频数据（AgiBot）上预训练，第二阶段在混合真实操作数据（AgiBot、RoboMIND、RoboCOIN、InternA1）上微调以适应具体机器人动作。

## 6. 论文的主要结论与发现

- **表征视觉-动作标记化显著优于重建导向方案**：RepWAM 在真实和模拟环境中均取得更高成功率，视觉生成和动作预测指标也更优。
- **语义对齐带来多重收益**：强语义潜空间减少对语言-视觉对齐的依赖，使世界模型能有效遵循指令；潜在动作作为中间表征提升迁移性，两阶段训练（潜动作预训练 + 机器人动作微调）优于直接联合预测或无潜动作训练。
- **CFG 依赖性降低**：RepViTok 使世界模型在 CFG scale=1.0 即取得最优效果，无需额外无条件分支外推，可降低推理延迟和内存。
- **结论**：世界动作模型的性能瓶颈在于潜空间设计，语义导向的视觉-动作标记化是一个有前景的基础，有助于构建通用机器人策略。

## 7. 优点

- **问题定义清晰**：精准指出现有 WAM 潜空间的两大不足（语义浅层、动作分离），并提出针对性解决方案。
- **创新方法设计**：将传输算子（K_t, δ_t）引入潜空间正向动力学，用软注意力模拟语义状态转移；两阶段解耦潜动作学习与机器人动作适应，避免相互干扰。
- **实验设计全面**：同时包含真实和模拟评估，多种指标覆盖生成、开环、闭环各层面，消融实验控制变量充分，对比方法覆盖主流 VLA 和 WAM。
- **开源承诺**：代码和权重将公开（GitHub），促进复现和后续研究。

## 8. 不足与局限
