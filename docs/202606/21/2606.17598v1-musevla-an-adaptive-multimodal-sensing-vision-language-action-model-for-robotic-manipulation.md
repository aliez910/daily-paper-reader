## 摘要
人类自然地利用多种感知模态与物理世界交互，而大多数用于机器人的视觉-语言-动作（VLA）模型仅依赖RGB观测数据。这限制了它们感知难以或无法从RGB相机推断的物理属性的能力，例如温度、声音或雷达响应。我们提出MuseVLA，一种适应性多模态感知VLA模型，它将新型传感器作为按需工具集成到机器人操作中。给定任务指令和视觉上下文，MuseVLA首先生成一个传感器令牌和目标描述，用于选择要调用的感知模态以及关注的内容，类似于带参数的函数调用。然后，它将选定的传感器测量转换为具象化传感器图像，这是一种统一的中间表示，编码异质读数以进行多模态融合和动作生成。这种设计将传感器特定处理与VLA主干解耦，从而能够高效集成多种模态。为了减少对昂贵的多传感器机器人数据集的需求，我们进一步引入了一个数据合成流水线，用具象化传感器图像增强现有的RGB视频数据集，从而实现对未见过的传感器引导任务的泛化。我们在真实机器人上评估了MuseVLA，这些机器人执行需要多模态感知输入的挑战性灵巧手操作任务，包括温度引导的拾取与放置、音频驱动的物体搜索以及雷达辅助的隐藏物体检索。MuseVLA平均成功率达到80.6%，显著优于仅RGB和多传感器VLA基线，并在未见任务上表现出强大的零样本能力。

## Abstract
Humans naturally leverage diverse sensing modalities to interact with the physical world, while most Vision-Language-Action (VLA) models for robotics rely solely on RGB observations. This limits their ability to perceive physical properties that are difficult or impossible to infer from RGB cameras, such as temperature, sound, or radar response. We present MuseVLA, an adaptive multimodal sensing VLA model that integrates novel sensors as on-demand tools for robotic manipulation. Given a task instruction and visual context, MuseVLA first generates a sensor token and target description that select the sensing modality to invoke and what to attend to, analogous to a tool call with arguments. It then converts the selected sensor measurement into a grounded sensor image, a unified intermediate representation that encodes heterogeneous readings for multimodal fusion and action generation. This design decouples sensor-specific processing from the VLA backbone, enabling efficient integration of diverse modalities. To reduce the need for expensive multisensory robot datasets, we further introduce a data synthesis pipeline that augments existing RGB video datasets with grounded sensor images, enabling generalization to unseen sensor-guided tasks. We evaluate MuseVLA on a real-world robot across challenging dexterous hand manipulation tasks that require multimodal sensing inputs, including temperature-guided pick-and-place, audio-driven object search, and radar-assisted hidden object retrieval. MuseVLA achieves 80.6% success rate on average, outperforming RGB-only and multisensory VLA baselines significantly, and exhibits strong zero-shot capabilities on unseen tasks.

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：将多传感器集成解耦为三个阶段：  
  1. **传感器选择与目标描述生成**：VLM根据任务指令和RGB图像输出一个传感器token（如 `<thermal>`）和一个描述性目标描述（如 “the hot mug”），相当于工具调用中的“工具名”和“参数”。  
  2. **接地传感器图像（Grounded Sensor Image）构建**：  
     - 首先用SAM3模型对目标描述进行语义分割，得到RGB图像上的二值掩码 `M`。  
     - 将选定传感器的原始观测（如热图、声强图、雷达反射图）通过固定的单应矩阵投影到RGB像素空间，然后用掩码 `M` 将传感器热图叠加到对应区域，其他区域保持原始RGB，形成一张统一的RGB格式图像 `m`。  
     - 数学上：`m = M ⊙ s + (1-M) ⊙ o_RGB`，其中 `s` 是传感器热图，`o_RGB` 是原图。  
  3. **动作生成**：将接地传感器图像与RGB图像、任务指令、传感器token、目标描述一起送入VLM视觉编码器，提取条件特征，再由扩散Transformer（DiT）动作专家预测未来动作序列。
- **模型架构**：采用PaliGemma-2作为VLM骨干，包含Gemma-2语言模型和SigLIP视觉编码器；动作专家使用DiT。从VITRA预训练权重初始化。
- **训练**：联合优化VLM的交叉熵损失（传感器token预测 + 目标描述生成）和动作专家的扩散MSE损失，端到端训练。
- **数据合成流水线**：为了降低多传感器数据集采集成本，设计了一个数据合成方法：  
  - 从已有RGB机器人数据集（MolmoAct、AgiBotWorld-Alpha、VITRA）中选取片段。  
  - 构造传感器词典（如thermal→“hot/cold/warm”），随机选择一个模态和物理关键字，将其注入任务指令，并用VLM（GPT-5.2）生成对应目标描述。  
  - 用SAM3分割出目标物体，用颜色编码（如黄色代表热）覆盖该区域，模拟传感器观测。  
  - 最终合成9.6K episode、105万帧数据，覆盖1000+物体和所有模态。  
- **自适应传感器选择机制**：通过可学习的传感器token实现，模型生成后只处理选定传感器的接地图像，避免了所有传感器同时输入带来的计算冗余和噪声。

## 6. 主要结论与发现
- **整体性能**：MuseVLA在三个模态任务上平均成功率76.4%（训练任务），显著优于RGB-only基线（20-22%）和直接输入原始热图的基线（23-33%）。  
- **自适应选择价值**：自适应传感器选择使成功率比无自适应版本提升34个百分点（MuseVLA-RawAdapt 33.3%→MuseVLA 76.4%），且在推理时降低一半显存消耗。  
- **接地传感器图像价值**：对比MuseVLA-RawAdapt（有自适应但无接地图像）和MuseVLA，接地表示带来约17个百分点的提升（59.7%→76.4%），并统一了多模态输入。  
- **数据合成预训练**：合成数据预训练大幅提升零样本泛化能力（从27.1%到66.7%），同时不影响训练任务性能。  
- **传感器选择准确性**：MuseVLA在训练和未见任务上的传感器token准确率均接近100%，目标描述准确率分别达93.5%和82.0%（有预训练）。  
- **多阶段任务**：链式调用不同传感器（如Radar→Thermal）可完成复杂长时操作，端到端成功率75.0%。

## 7. 优点
- **解耦与统一**：用接地传感器图像将异质传感器表示统一为RGB空间，无需专用编码器，方便新传感器扩展。  
- **自适应按需选择**：类似人类主动选择最相关感官，提升性能并降低计算成本。  
- **高效数据利用**：数据合成流水线显著减少昂贵多传感器数据采集需求，同时增强零样本能力。  
- **端到端训练**：VLM和动作专家联合优化，整个流程在单一模型中完成。  
- **强泛化性**：合成数据预训练使模型能适应未见过的任务指令和物体组合。  
- **模块化设计**：分割模块（SAM3）与VLA骨干分离，可异步更新，减少延迟。

## 8. 不足与局限
- **真实数据集规模有限**：仅720个演示片段，物体和任务种类不充分，可能限制对更复杂场景的泛化。  
- **对分割模块的依赖**：接地传感器图像的生成依赖SAM3的分割质量，在强遮挡、透明物体或细长物体上可能不准确。  
- **传感器硬件依赖**：传感器标定和热图生成需离线处理，且每个传感器需一次单应矩阵标定，对动态环境或剧烈视角变化鲁棒性未知。  
- **任务范围限制**：仅在桌面灵巧手操作上评估，未覆盖移动操作、人机协作等；仅有三个模态，但框架可扩展。  
- **零样本实验规模**：每项任务仅16次试验，统计效力有限；且未见任务仍集中在已知模态组合，未涉及新传感器类型。  
- **消融分析中个别变体性能反常**：如MuseVLA-RGB的感知成功率低于π0-RGB，作者未充分解释原因。  
- **统计检验**：仅报告了总体平均的p值，缺乏各模态单独检验，且对比方法数量较多，多重比较校正未提及。
