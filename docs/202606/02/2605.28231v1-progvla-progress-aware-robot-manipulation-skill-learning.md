## 摘要
我们提出了ProgVLA，一种紧凑的视觉-语言-动作（VLA）模型，专为在严格的计算和内存预算下实现可靠的机器人操作而设计。该模型特别关注高效处理长序列多模态信息，通过在长时间范围内维护任务进度的显式表示来实现。为此，ProgVLA集成了两个关键组件。首先，一个具有两阶段Perceiver重采样方案的多模态编码器，将可变长度的视觉、语言和本体感受流压缩成一组固定大小的控制就绪上下文令牌，在保持跨模态对齐的同时显著减少序列长度。其次，一组辅助进度头通过离线强化学习（RL）目标进行训练，以联合学习针对归一化剩余时间目标的评论家。这为策略提供了任务进度的内部估计，并实现了基于优势和成功加权的流匹配模仿学习。在两个公认的多任务机器人操作基准测试中，一个0.1B参数的ProgVLA模型达到了与显著更大的预训练基线相当的成功率，并且在长时域和更困难的任务层次上甚至超过了它们。消融实验表明，学习到的上下文重采样器和任务自适应视觉微调是最大的单一贡献因素，而进度感知训练提供了持续的额外增益，主要集中在长时域和多目标任务上。我们还在现实世界的玩具厨房环境中进一步验证了该方法。

## Abstract
We present ProgVLA, a compact vision-language-action (VLA) model designed for reliable robot manipulation under tight compute and memory budgets. The model specifically focuses on efficiently processing long multi-modal sequences by maintaining an explicit representation of task progress over extended horizons. To this end, ProgVLA integrates two key components. First, a multi-modal encoder with a two-stage Perceiver resampling scheme compresses variable-length visual, language, and proprioceptive streams into a fixed set of control-ready context tokens, substantially reducing sequence length while preserving cross-modal grounding. Second, an auxiliary set of progress heads is trained with offline reinforcement learning (RL) objectives to jointly learn critics over normalized remaining-horizon targets. This provides the policy with an internal estimate of task progress and enables advantage- and success-weighted flow-matching imitation learning. On two well-established multi-task robot manipulation benchmarks, a 0.1B-parameter ProgVLA model reaches success rates that are competitive with, and on long-horizon and harder task tiers exceed, substantially larger pretrained baselines. Ablations indicate that the learned context resampler and task-adaptive visual fine-tuning are the largest single contributors, while progress-aware training provides a consistent additional gain that is concentrated on long-horizon and multi-object tasks. We further validate the approach in real-world toy-kitchen environments.

## 论文核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的视觉-语言-动作（VLA）模型在机器人操作任务中，尤其是长时域（long-horizon）任务上表现不佳。大型VLA模型（如OpenVLA、π₀）依赖大规模跨本体机器人预训练，参数量巨大（数B至7B），限制了部署场景；而紧凑型VLA（如SmolVLA）虽可在低算力硬件上运行，但仍需跨本体预训练，且在长时域任务中差距明显。
- **研究动机**：能否设计一个完全从基准演示数据训练、不使用大规模机器人预训练的紧凑模型（约0.1B参数），在长时域任务中匹配甚至超越更大规模的预训练基线？
- **整体含义**：ProgVLA通过高效的多模态序列压缩（两阶段Perceiver重采样）、通用视觉骨干（DUNE）的微调以及内置的进度感知训练（auxiliary progress heads），在极低参数和计算开销下实现了与大规模模型竞争甚至超越的性能，显示压缩和进度信号可替代昂贵的大规模预训练。

## 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：在紧凑模型中显式引入任务进度估计，将进度信号作为模仿学习的样本加权因子，同时通过高效的多模态编码器减少序列长度，从而提升长时域操作的可靠性。
- **关键技术细节**：
  1. **多模态编码器**（两阶段Perceiver重采样）：
     - 视觉：使用DUNE（ViT-Small，多任务蒸馏通用编码器）提取特征；语言：冻结T5文本编码器；本体感受：MLP映射。
     - 第一阶段：每个模态分别通过Perceiver Resampler压缩为固定数量令牌（视觉每图像8令牌，语言16令牌）。
     - 第二阶段：所有模态令牌经共享Transformer融合后，再通过一个后融合Perceiver Resampler进一步压缩为少量“上下文令牌”（context tokens，4个）供下游使用。
  2. **动作专家**（Flow-Matching）：
     - 基于SmolVLA的流匹配动作生成，条件于上下文令牌cₜ，预测动作块（chunk）。训练时从噪声到数据的线性插值，学习速度场；推理时用10步Heun法积分。
  3. **进度头（Progress Heads）**：
     - 目标：定义一个归一化剩余时间信号 rₜ = max(0, 1 - (T - t)/T_M)，其中T_M=500。
     - 两个头：Q头（状态-动作值）和V头（状态值），均回归到rₜ；V头采用IQL式expectile回归（ρ=0.8）得到乐观基线；置信头S（二分类，阈值r_succ≈0.966）。
     - 损失：Q与V之差作为优势Aₜ，经温度β=0.5裁剪后得到权重w_A；S头的sigmoid概率权重w_S=0.5+0.5p_succ。模仿损失乘以乘积w_A·w_S。
     - 总损失：L_total = 加权流匹配损失 + λ_V L_V + λ_Q L_Q + λ_S L_S。
  4. **训练**：只使用成功演示，所有头共享上下文令牌cₜ，梯度回传时优势权重被detach（不作为策略梯度信号）。

- **公式/算法流程**（文字说明）：
  - 输入：多摄像头图像Iₜ、本体感受qₜ、语言指令lₙ。
  - 编码器：DUNE→per-image resampler（8 tok） + T5→language resampler（16 tok） + MLP投影→融合Transformer→后融合resampler→4个上下文cₜ。
  - 动作专家：采样噪声y₀~N(0,I)，时间τ~U[0,1]，插值y_τ，预测速度→L_FM。
  - 进度头：Q(cₜ, aₜ)→回归rₜ（Huber）；V(cₜ)→expectile回归rₜ；S(cₜ)→BCE对y_succ。
  - 优势A = Q - V（detach）→w_A；S→p_succ→w_S。
  - 最终模仿损失：w_A * w_S * ℓ_FM。

## 论文的主要结论与发现

1. **ProgVLA（0.1B参数）在LIBERO和Meta-World上的平均成功率超过或持平于显著更大的预训练模型**，尤其在长时域和高难度任务上提升明显（LIBERO Long分类88.6% vs SmolVLA 2.25B 77%；Meta-World非常难分类79.6% vs SmolVLA 2.25B 64%）。
2. **两阶段Perceiver重采样是最大的单因素贡献**：去掉后融合resampler导致LIBERO平均成功率下降16%，长时域任务从88.6%骤降至51.2%。
3. **微调视觉主干（DUNE）是第二大贡献**：冻结DUNE导致平均成功率下降13.5%。
4. **进度感知训练提供一致但较小的额外增益**：去掉进度头平均下降2.3%，主要集中在多目标（Object）和长时域（Long）任务。
5. 真实世界实验验证了方案可直接迁移，10任务平均成功率68%，主要失败原因为场景遮挡、夹爪超时、抓错物体。

## 优点
