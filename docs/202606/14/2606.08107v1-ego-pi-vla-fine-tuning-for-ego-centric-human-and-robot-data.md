## Abstract
Robotics faces a fundamental challenge of data scarcity. Unlike language or vision research, there is no internet-scale dataset for robotic manipulation. A promising path forward is to leverage egocentric human data, which can be collected more easily, with greater breadth, and at a larger scale. Towards this end, we investigate key design choices for learning across human and humanoid embodiments equipped with dexterous five-finger hands, using the $π_{0.5}$ model as a foundation. Our results show that human data enables robots to learn new task semantics and compose existing skills into novel behaviors without corresponding robot data. ## 2. 方法论

- **基础模型**：采用基于流匹配（flow-matching）的视觉-语言-动作（VLA）模型 π0.5 作为预训练基础。
- **动作空间对齐**：
  - 人类手部动作（Manus 手套采集 20 个关键点）通过逐关节偏移与缩放映射到机器人关节空间（Tesollo 手 20 关节，Inspire 手 6 关节）。
  - 机器人动作表示为 `a = {p, r, q}`，其中 `p` 为手腕位置（3维），`r` 为 6D 旋转表示（6维），`q` 为手指关节角度（20或6维）。每个手共 29 维，双手共 58 维。
- **适应高维灵巧控制的行动交错（Interleaved Action Sequence）**：
  - 原始 π0.5 支持最大 32 维动作，无法直接输出双手 58 维动作。
  - 将左右手动作分别放入两个 action token，按时间步交错排列，有效 horizon 从 H 减半为 H/2，从而在不改变预训练投影层参数的前提下扩展动作容量。
- **视觉对齐**：
  - 在人类与机器人图像上叠加骨架线（每根手指不同颜色，深度感知遮挡处理），使模型能更直观理解不同本体之间的手指对应关系。
- **辅助任务：子任务预测（Subtask Generation）**：
  - 在 VLM 部分额外训练模型输出子任务文本（如“pick tomato”），作为动作预测的中间步骤，增强任务语义理解。
- **训练策略**：
  - 联合训练时，每批次人类与机器人数据各占 50%；人类数据缺少腕部相机，因此机器人腕部图像在训练时以 40% 概率丢弃，使模型适应无腕部图像的情况。

## 6. 主要结论与发现

- **高级任务语义转移可行**：通过联合训练人类数据，机器人能学会颜色分拣、顺序拆箱&放置、按规则重物优先包装，成功率可达 90% 以上。
- **简单联合训练对简单任务有效**：番茄分拣（92%）和包装（90%）直接受益，但装箱任务仅 27%。
- **子任务预测是关键**：装箱任务中，加入子任务预测后成功率从 27% 提升至 93%，表明复杂顺序任务需要显式中间推理。
- **骨架线叠加效果有限**：在实验中未带来显著的定量提升，仅在视觉对齐上有定性帮助。
- **手型选择影响性能**：更接近人手大小的 Inspire 手（6 关节）在装箱任务中优于大型的 Tesollo 手（20 关节，仅达 67%），暗示视觉相似性对跨本体迁移的重要影响。
- **机器人仍依赖腕部相机**：训练时即使部分丢弃腕部图像，测试时去掉腕部相机仍导致抓取不稳定（如番茄抓取失败），说明策略实际依靠腕部观测。

## 7. 优点

- **系统而全面的消融实验**：从数据源、辅助损失、视觉对齐、硬件选择到感知输入，覆盖了跨本体学习的主要设计维度，结论可信。
- **跨本体视觉对齐**：通过骨架线深度绘制和颜色编码，将视觉外观差异极大的灵巧手映射为统一结构，简洁有效。

- **实际硬件部署验证**：在真实人形机器人上完成多任务，表明方法具备实用潜力。
