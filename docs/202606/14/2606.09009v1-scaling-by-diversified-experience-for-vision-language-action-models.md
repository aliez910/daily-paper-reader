## 摘要
视觉-语言-动作模型在实际部署中面临重大挑战，原因在于高层推理与低层控制的纠缠以及策略优化的不稳定性。在本文中，我们介绍了SyVLA，一个通过多样化经验训练的鲁棒VLA模型。我们提出了一种意图解耦算法，将控制相关特征从推理上下文中分离出来，并设计了一个基于相似样本引导的强化学习管道来稳定策略更新并缓解分布偏移。在真实世界机器人任务和多模态基准上的大量实验表明，与现有方法相比，SyVLA实现了更高的任务成功率和更强的分布外泛化能力，同时有效保留了核心的视觉-语言能力。代码和数据集已在项目页面发布。

## Abstract
Vision-Language-Action models face significant challenges in real-world deployment due to the entanglement of high-level reasoning with low-level control, and the instability of policy optimization. In this paper, we introduce SyVLA, a robust VLA model trained with diversified experiences. We propose an Intention Decoupling algorithm to isolate control-relevant features from reasoning contexts and a similar-sample guided RL pipeline to stabilize policy updates and mitigate distribution shift. Extensive experiments on real-world robotic tasks and multi-modal benchmarks demonstrate that SyVLA achieves superior task success rates and stronger out-of-distribution generalization compared to existing methods, while effectively preserving core vision-language capabilities. Codes and Datasets is released on \href{https://sy-vla.github.io/}{project page}.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- 当前视觉-语言-动作（VLA）模型在实际部署中面临两大瓶颈：  
  - **高层推理与低层控制的纠缠**：VLA 模型在训练中往往过度强调动作学习，导致底层 VLM 的视觉-语言理解与推理能力大幅退化甚至灾难性遗忘，无法像人类一样通过知识迁移实现广泛任务能力。  
  - **策略优化不稳定**：模仿学习（IL）的训练目标（逐帧拟合专家动作）与实际任务成功（闭环执行上百步动作）之间存在错配，导致误差累积和分布外（OoD）漂移。强化学习（RL）虽可缓解，但 VLA 模型参数规模巨大、动作空间高维，RL 极易不稳定、策略漂移甚至能力崩溃。  
- 本文提出 **SyVLA**，通过**多样化经验**（混合多模态数据、IL 数据与在线 RL 数据）训练，旨在**同时提升机器人任务执行能力与视觉-语言理解能力**，并实现稳定的真实世界 RL 优化。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 2.1 总体架构
- **双系统架构**：  
  - **VLM（Qwen2.5VL-3B）**：负责高层感知、语言推理与控制意图生成，采用“Think-Before-Act”方式输出显式推理链（CoT）和 Feature Query Tokens。  
  - **Action Expert（基于 Transformer 的 Flow Matching 模型，0.69B 参数）**：以 Feature Query Tokens 的最后隐藏状态（Feature Query States）作为条件，生成连续动作。  
- 连接方式：一组可学习的 **Feature Query Tokens** 附加在 VLM 的 CoT 输出之后，其隐藏状态经 MLP 适配器映射为控制条件，输入 Action Expert。

### 2.2 意图解耦（Intention Decoupling）算法
- **问题**：推理链的语义信息泄漏到 Feature Query States，导致动作犹豫不决或精度下降。  
- **方法**：  
  1. 前向一次计算动作损失 \(L_{\text{action}}\)，并求其对各 Feature Query State \(h_i\) 的梯度 \(g_i = \partial L_{\text{action}} / \partial h_i\)。  
  2. 计算每个梯度向量的 L2 范数 \(\|g_i\|_2\)，屏蔽低于阈值 \(\tau\)（经验设为 5% 分位数）的 token 隐藏状态（置零）：  
     \[
     h_i^{\text{masked}} = h_i^{\text{raw}} \cdot \mathbb{I}[\|g_i\|_2 \ge \tau]
     \]  
  3. 使用屏蔽后的 H_masked 进行第二次前向，计算实际梯度并更新模型。  
- **理论支撑**：简化 Action Expert 为单层 Transformer，证明梯度范数大小由注意力得分和值向量与输出距离决定，范数小的 token 属于控制无关或冗余信息，屏蔽后可避免短视决策，提升 OoD 泛化。

### 2.3 相似样本引导的强化学习（Similar-Sample Guided RL）
- **目的**：解决 IL 训练的 VLA 在真实世界 RL 中的不稳定性和性能崩溃。  
- **方法**：  
  1. 对每个时间步的观测用 CLIP 编码为特征向量，基于多视角余弦相似度从 IL 数据集中检索语义相似的样本。  
  2. 将检索到的专家样本与当前 rollout 样本合并成训练 batch。  
  3. 为避免价值模型对专家样本的错误优势估计导致梯度爆炸，**将专家样本的优势固定为 1.0**（即最大化专家动作似然），同时对 rollout 样本的 GAE 截断至 \([-0.8, +0.8]\)。  
- **效果**：在稀疏奖励的长时域任务（如折衬衫）上，RL 阶段带来最高 15% 的绝对成功率提升。

## 6. 论文的主要结论与发现
- **SyVLA 在真实任务上取得最高平均成功率**（In Domain 0.73，OoD 0.64），显著优于所有基线，特别在需要精细操作和推理的任务（计算并包裹）上领先。  
- **多模态基准上 SyVLA 平衡最佳**：在 AI2D、MME、HallBench 上最优，在 DocVQA、MMMU 上略低于 ChatVLA，但远好于仅关注动作的模型（如 Pi0）。  
- **意图解耦算法有效**：消融显示不使用解耦则成功率大幅下降至 0.43（全模型 0.86）。  
- **相似样本引导 RL 稳定有效**：不使用专家数据导致崩溃（0.21），使用但不用相似检索或标准 GAE 均效果差（0.79 或 0.00），而全方案最优。  
- **关键发现**：推理与控制分离是维持高动作精度的必要条件；将专家优势固定为 1.0 是稳定 VLA 规模 RL 的关键设计。

## 7. 优点：方法或实验设计亮点
- **创新性融合**：将意图解耦（梯度掩码）与相似样本引导 RL 结合，分别解决推理-控制纠缠和 RL 不稳定，实用且有效。  
- **轻量架构**：Feature Query Tokens 比 KV Cache 更轻，支持异步推理，降低延迟。  
- **理论分析**：在单层 Transformer 下证明梯度掩码的合理性，为算法提供直观解释。  
- **开源贡献**：承诺完全开源代码、数据集和模型权重，是首批具备视觉-语言推理能力的完整开源 VLA 之一。  
- **对比公平**：为排除大规模预训练影响，手动训练了 Pi0(from scratch)，并明确数据量差异，验证自身方法的效率。
