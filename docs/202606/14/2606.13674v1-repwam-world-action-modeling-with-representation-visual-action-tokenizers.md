## 摘要
本文提出了RepWAM，一种基于表征视觉-动作分词器的以表征为中心的世界动作模型。现有的世界动作模型通常从预训练的视频生成模型中继承面向重建的视频分词器。尽管这些分词器保持了视觉保真度，但仅凭像素重建对学习将未来预测与机器人控制联系起来的指令遵循动态的指导有限。为了解决这个问题，我们探索了一个用于以表征为中心的世界动作建模的语义视觉-动作潜在空间。具体来说，我们训练了一个表征视觉-动作分词器，将视觉输入映射到对齐的视觉和潜在动作标记。然后，我们预训练我们的世界动作模型，以联合建模未来的视觉状态以及在语言指令下连接这些状态的潜在动作，随后适应于真实机器人轨迹以进行闭环操作。在真实世界操作任务和模拟基准上的实验表明，RepWAM在多种操作设置中表现出强大的性能，而消融实验突出了语义视觉-动作分词化相对于面向重建的替代方案的价值。这些结果确立了表征视觉-动作分词化作为世界动作模型的有前途的基础，以及向通用机器人策略迈出的一步。代码和权重将在https://github.com/wdrink/RepWAM提供。

## Abstract
This work presents RepWAM, a representation-centric world action model (WAM) built on representation visual-action tokenizers. Existing WAMs typically inherit reconstruction-oriented video tokenizers from pretrained video generation models. Although these tokenizers preserve visual fidelity, pixel reconstruction alone provides limited guidance for learning instruction-following dynamics that connect future prediction with robot control. To address this, we explore a semantic visual-action latent space for representation-centric world action modeling. Specifically, we train a representation visual-action tokenizer that maps visual inputs into aligned visual and latent action tokens. We then pretrain our WAM to jointly model future visual states and the latent actions that connect them under language instructions, followed by adaptation to real robot trajectories for closed-loop manipulation. Experiments on real-world manipulation tasks and simulation benchmarks show that RepWAM delivers strong performance across diverse manipulation settings, while ablations highlight the value of semantic visual-action tokenization over reconstruction-oriented alternatives. These results establish representation visual-action tokenization as a promising foundation for world action models and a step toward generalist robot policies. Code and weights will be available at https://github.com/wdrink/RepWAM.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：世界动作模型（World Action Models, WAM）将经典世界模型由被动未来预测扩展至具身控制，需要在单一潜在空间中同时预测视觉动态并推断实现这些动态的机器人动作。
- **现有问题**：当前WAM通常直接继承自预训练视频生成模型（如WAN）的重建导向视频分词器。这些分词器优化像素重建，偏重背景纹理等低级外观，对物体身份、空间关系、交互线索等操作相关语义表征不足；同时，视觉潜在表示与机器人动作处于**分离空间**，迫使逆动力学模型（IDM）每一步都需弥合模态差距。
- **核心问题**：WAM实际需要什么样的表示？现有表示在语义上浅薄且在结构上与动作解耦，限制了指令遵循动态的学习。
- **论文目标**：提出 RepWAM，一个**以表示为中心**的WAM，通过构建**表示视觉-动作分词器**（RepViTok），使视觉潜在与潜在动作在统一语义空间中联合建模，弥补上述缺陷，提升下游机器人操作的性能。

## 2. 方法论：核心思想、关键技术细节

- **总体框架**：训练一个表示视觉-动作分词器（RepViTok）将视觉观测映射为对齐的视觉token和潜在动作token；在此基础上预训练一个因果扩散Transformer作为WAM，在语言指令条件下联合建模未来视觉状态及连接它们的潜在动作；最后通过少量真实机器人演示进行适配，实现闭环操作。
- **关键技术细节**：
  1. **视觉分词器**：使用Transformer VAE，将帧和后续帧管（tubelets）编码为视觉潜在 \(z\)。训练目标除重建损失（L1、感知、对抗）外，加入**特征对齐损失** \(\mathcal{L}_{\text{align}}\)，将视觉潜在向冻结的视觉基础模型（如Perception Encoder）的输出做平均池化对齐，引导语义丰富性。
  2. **潜在动作分词器（LAT）**：冻结视觉分词器后，训练一个逆动态模型（IDM）和一个前向动态模型（FDM）。IDM将连续视觉潜在对 \((z_t, z_{t+1})\) 压缩为低维潜在动作 \(\ell_t\)，FDM利用 \(\ell_t\) 预测传输矩阵 \(K_t\) 和残差 \(\delta_t\)，重构 \(z_{t+1} = K_t z_t + \delta_t\)。损失包括前向预测损失和反向一致性损失。该设计使潜在动作在语义视觉空间内描述运动转换，而非像素空间。
  3. **因果世界动作模型**：基于因果扩散Transformer，将语言指令嵌入为条件token，视觉token与潜在动作token按时间窗口分组形成块（chunk），采用块因果掩码。训练使用**条件流匹配**目标，联合回归视觉和动作分量的速度场。流匹配损失分视觉项和动作项，用 \(\lambda_a\) 平衡。
  4. **两阶段训练**：第一阶段在无动作标签的视频上预训练WAM预测视觉和潜在动作token；第二阶段在含真实机器人动作（如末端执行器运动）的演示上微调，使动作专家能将潜在动作映射为实际电机命令。

## 6. 主要结论与发现

- **语义视觉-动作分词优于重建导向分词**：RepViTok在gFVD、PSNR、SSIM和OLS上全面超过WAN2.2 VAE和ViTok，并带来真实世界PickFruit成功率从10-20%提升至30%。
- **两阶段训练（先潜在动作预训练，后机器人动作适配）最佳**：相比无潜在动作或联合预测，Two Stages显著提升视觉动态质量（gFVD降幅大）和开环/闭环动作表现（OLS从18.82→19.87，PickFruit成功率从30%→50%）。
- **潜在动作能有效转译实际机器人动作**：RepViTok的潜在动作比LAPA的潜在动作在IDM解码中损失更低，可视化更聚焦于操作相关变化。
- **语义潜在空间可降低视频CFG依赖**：RepWAM在CFG=1.0时达到最高成功率（Easy 87.4，Hard 88.4），无需额外CFG外推，从而降低推理延迟。
- **表示设计对于WAM至关重要**：统一语义视觉-动作空间是世界动作建模的有效基础，即使不依赖大规模视频预训练也能取得有竞争力性能。

## 7. 优点

- **方法创新**：引入特征对齐损失将视觉分词器语义化，并提出在语义视觉空间内学习潜在动作，使动作token成为状态转换的语义表示，弥合视觉与动作模态差距。
- **训练效率与可扩展性**：世界动作模型从头训练，无需继承预训练视频模型，通过两阶段设计先在无动作视频上预训练潜在动作，再用少量演示适配，降低对标注动作数据的依赖。
- **实验扎实**：多组消融从不同层面验证每个设计组件的贡献，真实世界+仿真+标准数据集全面评估。
- **实际价值**：在Franka平台上三个困难操作均优于或持平基线，仿真50任务平均成功率在Easy/Hard达到89.3/88.4（5B模型），显示了实用性。
- **代码与权重开源**：承诺提供，有利于复现和后续研究。

## 8. 不足与局限

- **与强基线相比的差距**：在仿真RoboTwin上，RepWAM 1.3B/5B的平均成功率低于使用WAN视频生成预训练的Lingbot-VA（易86.6vs87.4，Hard83.1vs84.0）。作者承认主要差距源于缺乏大规模视频预训练，表明表示分词本身虽好但仍需配合足够规模的预训练数据与计算。
- **实验场景有限**：真实世界仅用Franka双臂平台操作三种任务，未涵盖其他机器人形态（单臂、轮式、灵巧手）或更复杂的环境（杂乱场景、动态物体）。
- **潜在动作的可解释性与抽象层次**：潜在动作维度设为4，虽压缩强，但动作空间解释性较弱，能否泛化到全新运动类别尚未验证。
- **缺少零样本/跨本体泛化的显式测试**：虽仿真有域随机化，但未评估在不同机器人本体或视觉域（如真实→仿真）上的零样本迁移能力。
- **资源消耗**：训练5B模型需128块H20，成本较高，且文中未报告完整预训练步数或能耗，可重复性有一定门槛。
- **未与其他闭源WAM直接对比**：除π0.5、Motus、Lingbot-VA外，与最新WAM方法（如Fast-WAM、Being-H0.7等）的比较缺乏。
