## Abstract
Egocentric RGB-D videos offer a natural source of human dexterous manipulation demonstrations, but existing data is difficult to use for robot learning because object pose, geometry, and contact information are often missing or require pre-scanned object assets. We present EgoAERO, the first framework that learns dexterous manipulation from a single egocentric RGB-D human demonstration without object assets. EgoAERO reconstructs contact-consistent hand-object trajectories through asset-free object tracking and reconstruction, ego motion compensation, and adaptive contact optimization, then converts them into robot policies using two-stage residual learning. We further introduce an online quality assessment mechanism and construct EgoDex-R, a large-scale egocentric dataset with 4.3M RGB-D frames for dexterous policy learning. Simulation and real-world experiments show that EgoAERO enables single-demonstration dexterous manipulation and achieves downstream performance close to CAD-based reconstructions on HOI4D.

- **整体含义**：EgoAERO 是首个无资产学习框架，实现了从自然人类演示到机器人灵巧操作的单次示范学习，为大规模、低成本、通用化的机器人数据收集与策略学习提供了新路径。

## 2. 方法论

### 核心思想
- 通过组合 **语义预处理、无资产物体追踪与重建、自我运动补偿、自适应接触优化** 四个模块，从原始自我中心 RGB-D 视频中重建结构化的、接触一致的手物轨迹；再通过 **两阶段残差策略学习**，将重建轨迹转换为闭环机器人灵巧操作策略。

### 关键技术细节
- **MLLM 引导语义预处理**：
  - 从输入视频采样关键帧，结合任务描述输入 MLLM，识别目标操作物体及相关辅助物体（支撑物、容器等）。
  - 利用 MLLM 生成 SAM3 分割提示，获得物体遮罩，避免后续追踪中的语义歧义。
- **无资产物体追踪与重建**（核心）：
  1. **粗物体姿态初始化**：在当前帧 RGB-D 和物体遮罩上，通过 RANSAC 提升 2D-3D 对应，估计粗姿态 $\tilde{T}_t$。
  2. **关键帧记忆池姿态优化**：维护历史高质量帧的记忆池 $\mathcal{P}$；为当前帧选择相关关键帧构建局部姿态图，联合优化节点姿态，目标函数包括特征一致项 $E_{\text{feat}}$、几何项 $E_{\text{geo}}$、SDF 项 $E_{\text{sdf}}$、遮罩项 $E_{\text{mask}}$ 和姿态正则项 $E_{\text{pose}}$；通过记忆池的迭代优化获得稳定 6-DoF 轨迹。
  3. **从粗到细网格重建**：先用在线神经物体场（Neural Object Field）基于关键帧获得时间一致但偏粗的网格 $\mathbf{M}_{\text{coarse}}$；再用 SAM3D 结合原始 RGB-D 数据恢复精细细节 $\mathbf{M}_{\text{sam}}$；通过刚性和尺度对齐获得最终网格 $\mathbf{M}_O$。
- **自我中心手部姿态估计**：
  - 使用 HaWoR 估计 MANO 参数（关节、形状、根姿态）。
  - 利用 RGB-D 深度图对全局手部平移进行轻量校正，减少单目估计的深度偏差。
- **自我运动补偿**：
  - 使用 RGB-D SLAM（ORB-SLAM3）估计相机轨迹，转换到手部固定桌框架（table frame），消除头部运动引起的物体漂移。
  - 手部遮罩用于降低动态手区域对 SLAM 的影响。
- **自适应接触优化**：
  - 在主动操作窗口（抓取、移动、放置阶段）内，保持物体轨迹、网格和 MANO 关节角度不变，仅对全局手部平移和局部指尖位置进行有界修正。
  - 构造拇指指腹、动态选择的非拇指指尖区域、掌丘区域三个候选接触区域；通过吸引方向、全局平移、局部指尖偏移和穿透推回逐步优化接触一致性。
  - 默认参数：接触间隙 0.5mm，全局平移上限 34mm，局部指尖偏移上限 15mm，穿透推回上限 8mm，并施加时间平滑和边界过渡。
- **两阶段残差策略学习**：
  - **阶段 I（手部轨迹跟踪）**：学习策略 $\pi_I$ 以跟踪重建的人类手部参考轨迹（腕部和手指），不依赖物体动态；提供稳定的手部控制先验。
  - **阶段 II（物体-接触残差策略）**：在阶段 I 基础上，引入物体参考轨迹、当前物体姿态/速度、几何编码、手物距离和模拟接触力；学习残差策略 $\pi_R$ 在基础动作上施加小修正，同时优化手部跟踪、物体轨迹跟踪和接触奖励。
  - 奖励函数包括腕部跟踪、手指模仿、平滑度、物体跟踪、接触和残差正则项。
