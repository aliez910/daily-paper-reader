## 摘要
时序建模对于机器人操作至关重要，因为有效的控制需要记忆过去的交互和想象未来的状态。然而，大多数VLA模型主要依赖当前观测，因此难以处理长时域、时间依赖的任务。认知科学表明，人类依赖工作记忆来缓冲短暂上下文，依赖海马系统保存过去经验的情景记忆，以及依赖内部模型想象可能的未来状态演变。受这些机制启发，我们提出MemoryVLA++，一个完整的时序建模框架，为VLA模型配备记忆和想象能力用于机器人操作。预训练的VLM将当前观测编码为感知和认知token，形成工作记忆。这些token查询感知-认知记忆库以检索相关历史上下文。该记忆库存储来自过去交互的底层细节和高层语义，并通过冗余感知合并进行更新。一个世界模型在去噪潜在空间中想象未来状态，并在记忆引导下整合想象潜在表示，形成全时序感知token。得到的token条件化一个扩散动作专家来预测时间一致的动作序列。我们在5个仿真基准和3个机器人上的3类真实机器人任务上进行了广泛实验，涵盖通用操作、长时域任务、鲁棒性和泛化性。我们的方法在Libero、SimplerEnv、Mikasa-Robo、Calvin、Libero-Plus以及多样化的真实机器人任务上取得了强劲性能，验证了结合记忆与想象的完整时序建模的有效性。例如，在真实机器人上，在通用、记忆依赖和想象依赖任务上分别提升了+9%、+26%、+28%。项目页面：https://shihao1895.github.io/MemoryVLA-PP-Web

## Abstract
Temporal modeling is essential for robotic manipulation, as effective control requires both memory of past interactions and imagination of future states. However, most VLA models rely primarily on the current observation and therefore struggle with long-horizon, temporally dependent tasks. Cognitive science suggests that humans rely on working memory to buffer short-lived context, the hippocampal system to preserve episodic memory of past experience, and internal models to imagine possible future state evolution. Inspired by these mechanisms, we propose MemoryVLA++, a full temporal modeling framework that equips VLA models with memory and imagination for robotic manipulation. A pretrained VLM encodes the current observation into perceptual and cognitive tokens, forming working memory. These tokens query a Perceptual-Cognitive Memory Bank to retrieve relevant historical context. This bank stores low-level details and high-level semantics from past interactions, and is updated through redundancy-aware consolidation. A world model imagines future states in a denoising latent space, and the imagined latents are integrated under memory guidance to form full temporal-aware tokens. The resulting tokens condition a diffusion action expert to predict temporally consistent action sequences. We conduct extensive experiments on 5 simulation benchmarks and 3 categories of real-robot tasks across 3 robots, covering general manipulation, long-horizon temporal tasks, robustness, and generalization. Our method achieves strong performance across Libero, SimplerEnv, Mikasa-Robo, Calvin, Libero-Plus, and diverse real-robot tasks, validating the effectiveness of full temporal modeling with memory and imagination. For example, on real robots, it achieves +9%, +26%, +28% gains on general, memory-dependent, and imagination-dependent tasks. Project Page: https://shihao1895.github.io/MemoryVLA-PP-Web

## 2. 论文提出的方法论：核心思想与技术细节

### 核心思想
- 使用预训练 VLM 编码当前观测得到 **感知 token**（细节）和 **认知 token**（语义），构成**工作记忆**。
- 建立**感知–认知记忆库（PCMB）**，存储过去交互的底层细节与高层语义，通过冗余感知合并保持紧凑。
- 工作时，工作记忆作为查询检索记忆库，并通过门控机制自适应融合当前与历史信息。
- 利用**视频生成世界模型**（基于 Stable Video Diffusion）在潜在空间中进行部分去噪，得到**未来想象 latent tokens**。
- 记忆增强后的 token 通过**记忆引导的集成模块**筛选并融合未来想象 token，形成**全时序感知 token**（整合过去、现在、未来）。
- 最终用这些条件 token 驱动**扩散动作专家**（DiT 架构），输出时间一致的动作序列。

### 关键技术细节（流程）
1. **视觉–语言–认知模块**：Prismatic-7B VLM，使用 DINOv2 + SigLIP 提取视觉特征，经双分支生成感知 token（保留细节）和认知 token（LLM EOS 位置输出）。
2. **感知–认知记忆库（PCMB）**：
   - 存储条目：每个条目包括对应时刻的感知记忆（低层细节）和认知记忆（高层语义），附带时间戳位置编码。
   - **检索**：当前工作记忆（感知、认知 token）通过交叉注意力查询所有记忆条目，得到检索结果 Hp、Hc。
   - **门控融合**：用 Sigmoid 门控自适应结合当前 token 和检索结果：`~x = g ⊙ Hx + (1−g) ⊙ x`。
   - **合并（Consolidation）**：当条目数超过容量 L 时，计算相邻条目余弦相似度，合并最相似的一对（取平均），移除冗余。
3. **世界模型想象模块**：
   - 世界模型：SVD（1.5B 参数）在操作视频上微调，学习机器人相关的未来动态。
   - 想象生成：在 VLA 训练中冻结世界模型，对当前观测 + 指令进行**部分去噪**（仅 1 步或少量步），提取多尺度 UNet 特征，经 FPN + Imagination Former（可学习 query）得到未来 latent tokens。
   - **记忆–引导集成**：记忆增强的感知 token 作为 query，交叉注意力未来 latent tokens，再门控融合，抑制不可靠想象并保留决策相关信息。
4. **全时序感知动作专家**：
   - 使用 DiT + DDIM 扩散。认知 token 加时间嵌入后与噪声动作序列做自注意力（认知注意力），再与感知 token 做交叉注意力（感知注意力），通过 MLP 投影输出 7-DoF 动作序列。

- **实验数量**：极为丰富。总共涵盖 **5 个仿真 benchmark、3 类真实任务（17 子任务）、3 种机器人**，涉及近 200 个任务变种。仿真 benchmark 共包含超过 150 个具体任务。消融实验覆盖记忆建模（长度、融合、合并、检索、类型）和想象建模（去噪步数、想象范围、更新策略、集成方式）的所有关键设计。此外还有效率对比、世界模型质量评估、更强预训练分析、可视化案例。
- **充分性与公平性**：
  - 对比方法覆盖面广，且为官方/复现最优结果。
  - 仿真和真实场景均采用标准评估协议（多次试验、最佳检查点）。
  - 在真实机器人上使用 step-wise 评分以更好反映长程任务进展。
  - 消融实验分别基于 MemoryVLA 和 MemoryVLA++ 进行以避免混淆。
  - 在 SimplerEnv 和 Libero 上额外测试了更强的 VLM 骨干（Qwen2.5 + Dexbotic），显示可扩展性。
- **评价**：总体实验设计全面、系统、重复性好，结论有说服力。唯一的不足是真实机器人任务未与其他使用视频预测的方法（如 ForeAct、VISTA）直接对比，但已比较了 π0、CogACT 等既有 VLA。

## 6. 论文的主要结论与发现

- MemoryVLA++ 在所有评估设置上**一致显著优于**仅依赖当前帧或简单历史帧堆叠的方法。
- **仿真结果亮点**：
  - Libero 平均成功率 **98.4%**，比 CogACT 基线提高 +5.2 百分点。
  - SimplerEnv 平均 **73.9%**，提高 +16.6 百分点。
  - Mikasa-Robo **44.4%**（+15.0），Shell Game Touch 项目提升高达 +50。
  - Calvin 平均完成任务数 **4.29**（+1.04），在所有步骤上提升。
  - Libero-Plus 零样本和微调分别达 **73.1%** 和 **82.7%**，优于基线。
- **真实机器人亮点**：
  - 通用操控 +9%，长时记忆依赖 +26%，长时想象依赖 +28%（MemoryVLA++ 对 MemoryVLA 再提升 +12 点）。
  - 在分布外变化下表现稳健（仅轻微下降）。
- **消融与分析**：
  - 门控融合、冗余感知合并、双重记忆类型、时间戳位置编码均不可或缺。
  - 想象模块中，1 步去噪即可获得有效未来信息；冻结世界模型优于更新；记忆引导集成优于直接相加。
  - 推理额外开销可接受（MemoryVLA++ 在 RTX 4090 上 66 Hz，在可接受范围内）。

## 7. 优点：方法或实验设计上的亮点

- **认知科学启发**：紧密结合人脑工作机制（工作记忆、情景记忆、内部模型），为 VLA 时序建模提供新视角。
- **全时序建模**：同时处理过去（记忆）、现在（工作记忆）和未来（世界模型想象），实现完整的时间维度覆盖。
- **潜在空间想象**：避免了显式视频预测的高计算开销和像素级噪声，通过部分去噪提取决策相关的紧凑特征。
- **记忆与想象的协同**：通过记忆引导集成机制，利用历史上下文筛选未来预测中的噪声，实现信息互补。
- **模块化可迁移**：各模块（VLM 编码、记忆、世界模型、动作专家）可独立替换或升级（如用到更强 VLM 或世界模型）。
- **实验极其彻底**：模拟 5 基准 + 真实 3 平台，覆盖最大范围的任务变种与分布外场景；消融与分析全面。
- **开源与可复现**：代码和项目页面公开，促进后续研究。

## 8. 不足与局限

- **世界模型依赖外部视频数据**：需要预训练并微调视频生成模型，这部分数据获取与训练开销不低，且泛化到全新场景可能受限。
- **仅使用 2D 观测**：尽管多个视角，但未显式引入 3D 结构（如点云、位姿），在处理复杂 3D 操作（如精密装配）时可能不够。
- **想象模块可靠性**：虽然通过记忆引导降低了噪声影响，但仍可能存在对未来动态预测不准确的情况（尤其在高度随机环境），文中仅部分定量评估（表 IX），未见对失败案例的深入分析。
- **真实机器人对比不够全面**

- **真实机器人对比不够全面**：尽管 MemoryVLA++ 在三种真实机器人平台上进行了测试，但对比基线主要聚焦于 OpenVLA、π0、CogACT 等经典 VLA 模型，缺少与同样显式利用视频预测或想象机制的方法（如 ForeAct、VISTA）在真实场景中的直接比较，这使得想象模块的独立贡献难以在同一对比框架下精确量化。此外，真实任务的评估次数有限（每任务 10～20 次），且以二值成功率为主要指标，对于长程任务中部分步骤的进展仅用 step-wise 评分粗略衡量，未能像仿真实验那样提供细粒度统计。环境设置上侧重桌面操控，未覆盖动态人机交互、非结构化家庭场景或高度随机扰动，限制了结论的泛化范围。
- **训练与部署复杂性较高**：框架包含 VLM 骨干、扩散动作专家、记忆库、世界模型四个主要组件，其中世界模型需要额外收集视频数据预训练并微调（20k–40k 步），记忆库的合并与检索机制增加了系统代码复杂度和调参难度。尽管推理速度尚可（66 Hz），但训练时需同时协调多个模块的梯度流，对硬件和工程实现要求较高，可能影响可复现性和推广性。
- **对任务指令的依赖与语义理解仍有限**：认知 token 从 LLM 获取，但当前 VLM 对复杂、多步骤指令的语义分解能力有限，例如在“先按红色按钮再拿蓝色方块”这类指令中，记忆模块需依赖正确的语义索引，错误的分步理解可能导致检索偏差。文中未深入分析指令误解对记忆检索和想象引导的影响。

综上，MemoryVLA++ 在赋予 VLA 记忆与想象能力上取得了显著成效，实验覆盖面广、消融充分，但真实对比的完善度、系统复杂度和对复杂指令的鲁棒性仍有提升空间。未来可探索更轻量的世界模型、跨模态记忆压缩以及端到端集成，以推动其在更广泛机器人任务中的实用化。
