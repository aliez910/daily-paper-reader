## 摘要
导航与操作是具身智能的基础能力，使机器人能够理解自然语言指令并与物理环境进行交互。然而，当前的视觉-语言-动作（VLA）模型仍受限于特定任务的架构，专门处理导航或操作，这阻碍了通用型机器人智能体的发展。为弥合这一差距，我们提出了OneVLA，一种统一的架构，将这些不同任务整合到一个连贯的框架中。具体而言，我们设计了一个统一的动作头，能够生成导航和操作动作，无需特定任务变体。此外，我们提出了一种多阶段渐进训练策略，结合精心策划的数据构建和思维链（CoT）微调，促进了两个领域之间的强正迁移和相互增强。在模拟和真实环境中的大量实验表明，OneVLA达到了最先进的性能，显著优于专门的单任务模型和现有跨任务模型。通过统一这些核心能力，OneVLA为真正的通用机器人系统铺平了道路。模型和源代码将公开发布。

## Abstract
Navigation and manipulation are fundamental capabilities of embodied intelligence, enabling robots to interpret natural language commands and interact physically with their surroundings. However, current Vision-Language-Action (VLA) models remain constrained by task-specific architectures, specializing in either navigation or manipulation, which hinders the development of general-purpose robotic agents. To bridge this gap, we introduce OneVLA, a unified architecture that integrates these distinct tasks into a single, cohesive framework. Specifically, we design a unified action head capable of generating both navigation and manipulation actions without requiring task-specific variants. Furthermore, we propose a multi stage progressive training strategy-incorporating curated data construction and Chain-of-Thought (CoT) fine-tuning that facilitates strong positive transfer and mutual reinforcement between the two domains. Extensive experiments in both simulated and real-world environments demonstrate that OneVLA achieves state-of-the-art performance, significantly outperforming both specialized single-task and existing cross-task models. By unifying these core capabilities, OneVLA paves the way for truly general-purpose robotic systems. The model and source code will be publicly released.

#### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：构建一个统一的VLA架构（OneVLA），能够在一个模型中处理多种具身任务（导航和操作）的输入和输出，通过创新的动作头和训练策略实现跨任务的知识迁移和性能增强。
- **关键技术细节**：
    - **统一架构**：模型接收多模态输入（多视角RGB图像\(o_t\)、自然语言指令\(l_t\)、可选的机器人状态\(r_t\)），并产生双重输出：文本解释\(y_t\)和未来动作序列\(a_{t:t+T}\)。输出通过特殊标记`<text>...</text><action>...</action>`在单次前向传播中区分语言生成和动作生成。
    - **统一动作头（Unified Action Head）**：这是核心的架构创新。它将导航和操作的动作空间“拼接”成一个**11维的统一动作空间\(a_{unified}\)**，包含4维导航离散动作（前进概率、左转概率、右转概率、停止概率）和7维操作连续动作（末端执行器的6维位姿变化和夹爪状态）。
        - 动作头采用**流匹配（Flow-Matching）**扩散过程来预测动作，通过一个DiT骨干网络处理带有噪声的动作轨迹，并与视觉-语言特征进行交叉注意力融合。
        - **任务特定权重掩码（Task-Specific Weight Mask）**：为了解决不同任务动作空间的语义差异，在训练时引入了权重掩码\(w_{\tau_i}\)。对于属于导航任务的样本，只计算导航动作维度（4维）的损失；对于操作任务样本，只计算操作动作维度（7维）的损失。这使得模型可以在不改变架构的情况下，同时学习两个任务。
    - **多阶段渐进训练策略**：采用三阶段训练，逐步引入任务复杂度：
        1.  **阶段1（建立操作能力）**：仅使用操作数据集和通用VQA数据集训练，让模型掌握基本的操作技能。
        2.  **阶段2（发展导航能力）**：引入导航数据集，与操作、VQA数据一起进行联合训练。此阶段的关键是让统一动作头通过权重掩码同时学习两个任务，从而实现跨任务知识迁移。
        3.  **阶段3（增强跨任务性能）**：引入思维链（Chain-of-Thought, CoT）数据，增强模型的推理能力和决策质量，进一步优化两个任务的性能。
    - **损失函数设计**：总损失函数\(L_{total} = \lambda_{vlm} L_{vlm} + \lambda_{text} L_{text} + \lambda_{action} L_{action}\)，包含通用视觉-语言理解损失、文本推理损失和动作损失，权重系数分别为0.1，0.1，1.0。

#### 6. 论文的主要结论与发现

- **主要结论**：OneVLA成功构建了一个统一的VLA框架，能够在一个模型中无需任务特定变体即可完成导航和操作任务，并在多个基准测试中取得了最先进的性能。
- **关键发现**：
    1.  **统一架构的有效性**：一个单一的VLA模型可以高效地完成导航和操作任务。
    2.  **联合训练的互惠性**：将导航和操作数据进行联合训练，可以显著提升各自任务的性能。例如，联合训练比单任务训练在R2R上提升7.3%，在SimperEnv上提升5.5%。
    3.  **多阶段训练的优势**：渐进式引入任务复杂度（先操作，再导航，最后CoT推理）比一次性同时学习所有任务效果更好，在多阶段训练下R2R提升12.5%，SimperEnv提升18.3%。
    4.  **小模型也能取得大成就**：仅3B参数的OneVLA在性能上超越了多个7B参数的任务特定模型，证明了高效架构和训练策略的重要性。

#### 8. 不足与局限

- **任务复杂性有限**：当前实验覆盖的任务（室内导航、桌面操作）相对基础和结构化，对于需要复杂长序列规划、技能组合或常识推理的复杂任务的泛化能力尚未验证。
- **Sim-to-Real Gap的验证**：虽然做了真实实验，但仅在有限的场景和任务上进行，与现实世界中复杂、动态、非结构化的环境仍有差距。
- **对零样本泛化的探索有限**：方法主要关注如何在一个统一的框架下同时学习多个任务，但在模型从未见过的全新任务（零样本场景）上的泛化能力未做深入探索。
- **计算资源未公开**：未能提供训练OneVLA所需的计算资源和时间，使得复现和评估其训练成本变得困难。
