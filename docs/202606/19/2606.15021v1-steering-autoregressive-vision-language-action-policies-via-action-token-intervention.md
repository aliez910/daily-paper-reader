## 摘要
我们提出令牌引导（Token Steering, TS），一种通过在动作令牌空间中进行直接干预来动态引导由自回归视觉-语言-动作（VLA）模型生成的轨迹的方法。TS将低维用户输入注入到模型原生的动作令牌表示中，使用户能够影响轨迹生成，而无需修改底层的视觉-语言模型（VLM）架构。由于TS完全在推理时运行，因此不需要额外的训练或微调。用户输入引导而非覆盖预训练策略，使用户能够在影响机器人动作的同时，保留VLA学习到的灵巧性、平滑性和任务先验知识。我们在两项家庭操作任务上评估了TS——物体放置后关闭抽屉和状态感知物体交换——并将成功率分别从10.0%提高到72.5%，从16.7%提高到93.8%。通过实现对机器人基础模型的轻量级、直观引导，我们的接口有潜力改善消费环境中的人机交互，并拓宽对于身体控制能力受限个体的可访问性。项目网站：https://jasontchan.github.io/token-steering/ 。

## Abstract
We present Token Steering (TS), a method for dynamically steering trajectories generated by an autoregressive vision-language-action (VLA) model through direct intervention in the action-token space. TS injects low-dimensional user inputs into the model's native action-token representation, allowing users to influence trajectory generation without modifying the underlying vision-language model (VLM) architecture. Because TS operates entirely at inference time, it requires no additional training or finetuning. User inputs guide rather than override the pretrained policy, allowing users to influence robot actions while preserving the dexterity, smoothness, and task priors learned by the VLA. We evaluate TS on two household manipulation tasks -- drawer closing after object placement and state-aware object swapping -- and improve success rates from 10.0% to 72.5% and from 16.7% to 93.8%, respectively. By enabling lightweight, intuitive steering over robot foundation models, our interface has the potential to improve human-robot interaction in consumer environments and broaden accessibility for individuals with limited physical control. ## 1. 核心问题与整体含义（研究动机和背景）
- **背景**：自回归视觉-语言-动作（VLA）模型（如RT-2、OpenVLA、π0）在机器人操作任务中表现出强大的泛化能力，但在细粒度、实时的轨迹修正方面存在困难。现有交互方式（语言指令、完整遥操作）各有局限：语言不适合实时精调，完整遥操作会覆盖策略的自主能力。
- **研究动机**：探索一种**轻量级的、推理时的引导机制**，让用户能够在保持策略已有能力的前提下，对轨迹方向进行实时干预。
- **整体含义**：提出 **Token Steering (TS)** —— 一种通过直接修改动作令牌（action tokens）来引导自回归VLA模型生成轨迹的方法，无需训练或微调，仅需低维用户输入（如键盘方向）。

## 2. 方法论
- **核心思想**：利用自回归VLA模型的离散令牌生成过程，将用户意图编码为与模型原生令牌空间一致的“引导令牌”，并替换生成序列中的部分令牌，让策略根据修改后的前缀自动补全剩余轨迹。
- **关键技术细节**：
  - 基于 **π0-FAST** 模型，该模型采用 **FAST 令牌化**：将连续动作轨迹（H×D）沿时间维度做离散余弦变换（DCT），得到低频到高频的系数，再经字节对编码（BPE）压缩为离散令牌。因此令牌序列具有频率语义：低频令牌对应全局运动，高频令牌对应局部精细调整。
  - **TS 流程**：
    1. 用户低维输入（例如键盘方向） → 转换为笛卡尔速度向量 → 通过逆运动学（IK）转换为关节速度 → 采用与模型相同的FAST令牌化得到引导令牌序列。
    2. 在模型自回归生成新动作令牌时，选择一段窗口（起始索引 \(b\)，窗口大小 \(w\)）将对应位置的令牌强制替换为引导令牌，其余位置仍由模型正常采样。
    3. 引导令牌注入后，模型继续自回归生成剩余令牌，结合视觉、语言和状态条件，完成轨迹。
  - 该方法完全在推理时进行，不需要任何训练或微调。

- **平台与环境**：
  - 机器人硬件：DROID平台（Franka Panda臂 + Robotiq 2F-85夹爪），配备一个场景相机和一个腕部相机。
  - 模型：π0-FAST（在DROID数据集上预训练），动作推理时长10，动作维度8。所有实验零样本进行。
- **基准与对比**：
  - 未与其他引导方法（如扩散策略引导）进行横向对比，主要将**自主策略（无引导）**作为基线。
  - 通过消融实验自对比不同注入参数（窗口大小、注入起始位置/令牌频率）。
- **实验类型与场景**：
  1. **注入窗口大小实验**：改变引导令牌数量（1/2/4/6），评估引导意图成功率（SIR）与路径效率（MPE）。
  2. **令牌频率实验**：固定窗口大小2，改变注入起始索引（b=0~3，对应低频→高频），评估引导成功率。
  3. **策略多样性实验**：20种随机场景×4个方向，注入2个令牌后测量后续令牌熵，检验是否导致动作多样性丧失。
  4. **语言纠正实验**：设置语言指令与目标不一致的场景（如“拿起绿色方块”但策略总是选错），观察TS能否纠正。
  5. **用户研究**：7名用户在两项家庭任务上评估：
     - 任务1：把玩具香蕉放入抽屉并关闭抽屉（精度+力控制）。
     - 任务2：交换两个海绵的位置（需记忆与多步操作）。
- **评估指标**：SIR、MPE、任务成功率、中位完成时间、熵。用户研究使用Wilcoxon秩和检验。

## 6. 主要结论与发现
- 部分注入引导令牌可以在不消除策略自主能力的前提下，有效控制轨迹方向。
- **低频令牌**对全局语义影响更大，高频令牌主要影响局部细节；引导成功主要依赖低频注入。
- 注入少量（<25%）令牌即可纠正错误语言跟随，说明动作令牌空间可作为可控语义接口。
- 用户使用键盘方向引导，在两项任务上将成功率分别从 **10.0% → 72.5%**（关门）和 **16.7% → 93.8%**（交换），且完成时间显著缩短。
- 引导后的动作仍然保持平滑与安全（用户提供粗方向，策略自动生成精细轨迹，如温柔关门）。

## 7. 优点
- **轻量无训练**：推理时直接干预，无需任何训练/微调，即插即用。
- **保留策略能力**：不覆盖策略，而让策略基于引导前缀自主完成，保留灵巧性与任务先验。
- **通用输入映射**：低维意图（键盘方向）经简单变换即可生成引导令牌，理论上可扩展到任何低维信号（如脑机接口）。
- **视角新颖**：将动作令牌空间视为可控语义接口，而不是单纯的压缩表示，为VLA交互提供新思路。
- **实验设计清晰**：包含系统化的消融分析、熵分析、语言纠正验证和真实用户测试。

## 8. 不足与局限
- **生成速度**：自回归逐令牌生成速度慢于流/扩散模型，在需要快速响应的场景中可能受限（文中提到可通过异步块生成改进，但未实现）。
- **实验覆盖**：仅在一个机器人平台（DROID）和一个模型（π0-FAST）上测试，未验证在其他VLA（如OpenVLA）或不同令牌化方案下的效果。
- **输入类型有限**：仅验证了六方向笛卡尔速度输入，对更复杂的引导信号（如连续方向、力/力矩）未做探索。
- **用户研究规模小**：7位用户、两项固定任务，可能无法完全体现方法在不同用户、不同任务上的稳定性。
- **缺乏与现有方法的直接对比**：未与扩散策略引导（DSRL、DiSCo）、共享自主方法进行定量比较，方法的相对优势不完全明确。
- **依赖IK与FAST**：需要逆运动学与FAST令牌化流程，对无关节模型或不同令牌格式可能不适用。
