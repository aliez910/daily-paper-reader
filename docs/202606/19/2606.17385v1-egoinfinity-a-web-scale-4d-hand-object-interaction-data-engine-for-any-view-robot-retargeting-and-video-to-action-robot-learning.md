## 摘要
互联网视频构成了人类具身操作知识的最大宝库，但将任意RGB视频转化为可操作的机器人训练数据仍然是一个主要瓶颈。现有的实验室或工厂收集的数据集在规模和多样性上很狭窄，限制了开放世界机器人学习。我们并未提出一个静态数据集，而是引入了EgoInfinity，一个通用的4D手物交互数据引擎，能够实现网络规模的机器人重定向和学习数据生成。EgoInfinity是一个模块化引擎，集成了感知、分割、重建、交互感知精炼和重定向，以自动化这一传统上不可扩展的视频到动作问题，无需人工在环标注。其模块化设计使得引擎能够持续受益于任何集成组件的进步。借助EgoInfinity，野外的人类操作视频被提升为与智能体无关的、公制的4D手物表示，包括手部轨迹、物体6自由度姿态和接触相关状态。EgoInfinity并非简单连接独立组件，而是结合跨模块公制校准与交互感知精炼来提高物理可靠性，减少纯视觉重建中常见的漂移和接触不一致。我们进一步提出了一种新颖的运动重定向器，将恢复的3D手部运动编译为适用于各种机器人形态的可执行关节轨迹，从而实现从任意视角和拍摄尺寸（例如，人体仅部分可见）在任何机器人上的视频到动作重定向。我们通过感知保真度、运动学可行性、接触一致性、跨实体泛化以及真实机器人技能获取（例如抓取、切割、擦拭和倾倒）对EgoInfinity进行了验证，展示了从互联网视频到可执行机器人行为的一个可扩展桥梁，用于开放世界机器人学习。

## Abstract
Internet videos constitute the largest reservoir of embodied human manipulation knowledge, yet converting arbitrary RGB footage into actionable robot training data remains a major bottleneck. Existing lab- or factory-collected datasets are narrow in scale and diversity, limiting open-world robot learning. Instead of proposing a static dataset, we introduce EgoInfinity, a universal 4D hand-object interaction data engine that enables web-scale data generation for robot retargeting and learning. EgoInfinity is a modular engine integrating perception, segmentation, reconstruction, interaction-aware refinement, and retargeting to automate this traditionally unscalable video-to-action problem without human-in-the-loop annotation. Its modular design lets the engine continuously benefit from advances in any incorporated component. With EgoInfinity, in-the-wild human manipulation videos are lifted into agent-agnostic, metric 4D hand-object representations, including hand trajectories, 6-DoF object poses, and contact-relevant states. Rather than naively connecting standalone components, EgoInfinity combines cross-module metric calibration with interaction-aware refinement to improve physical reliability, reducing drift and contact inconsistencies common in pure visual reconstruction. We further propose a novel motion retargeter that compiles the recovered 3D hand motions into executable joint trajectories for diverse robot morphologies, enabling video-to-action retargeting on any robot from arbitrary viewpoints and shot sizes (e.g., the human body is only partially visible). We validate EgoInfinity across perception fidelity, kinematic feasibility, contact consistency, cross-embodiment generalization, and real-robot skill acquisition (e.g., grasping, cutting, wiping, and pouring), demonstrating a scalable bridge from internet videos to executable robot behavior for open-world robot learning.

## 2. 论文提出的方法论

### 核心思想
- **模块化数据引擎**：集成感知、分割、重建、交互相应优化、重定向等组件，自动将RGB视频提升为公制4D手物交互表示，无需人工在环标注。引擎采用**两阶段策略**：Pass 1轻量扫描过滤有效操作片段，Pass 2对活动片段运行完整重建栈。
- **跨模块公制校准**：不同模块（手部姿态、物体位姿）输出在尺度、参考坐标系上并不一致。引擎首先使用MoGe-2估计相机焦距和全局公制尺度、Flow3r预测密集深度、GeoCalib估计重力方向，将所有预测对齐到统一的公制相机坐标系。
- **交互感知精炼**：利用检测到的交互状态（static/grasped/moving）优化物体轨迹，减少纯视觉跟踪的漂移。例如，抓取状态下将物体刚性附着于手部，静态状态下锁定其质心，仅对有可观运动的物体保留原始视觉提议。
- **跨实体运动重定向**：提出一种基于SE(3)等变神经网络和流匹配的根框架估计器Φ，从恢复的3D手部轨迹和可选重力方向，预测机器人特定的运动学根框架（如人形机器人的躯干框架）。然后将手部运动转换到根框架下，通过逆运动学（IK）生成关节轨迹。重定向器通过仿真训练，无需真实世界数据。

### 关键技术细节
- **手部重建**：WilOR估计MANO参数（姿态θ、形状β），生成手部网格{M_h^t}、3D关键点{K_h^t}、全局姿态cp_h^t ∈ SE(3)。后接补全模块处理缺失帧，Savitzky–Golay滤波去除抖动。
- **目标发现与重建**：利用视频文本描述驱动SAM-3检测物体区域，SAM-2跟踪传播掩码；结合深度背投得到每帧点云{P_o^t}；SAM-3D重建物体网格M_o；FoundationPose++跟踪物体6-DoF位姿轨迹{cp_o^t}。
- **交互状态定义**：细粒度六状态Σ = {STATIC_GLOBAL, STATIC, GRASPED_L, GRASPED_R, GRASPED_BOTH, MOVING}，通过全局静态门限、每帧运动施密特触发器、每手抓取信号（2D掩码重叠+3D指尖/手腕距离+时间形态学平滑）分层分配。
- **状态依赖位姿源**（Table 3）：静态状态使用稳健质心和平方向；抓取状态使用刚性手部绑定（ĉp_o^t = cp_h^t·T_cano）；移动状态使用掩码包围盒中心高斯平滑。每个抓取段内估计手部到物体的固定规范变换T_cano（旋转取SO(3)弦平均，平移由几何放置将物体贴到手心）。
- **重定向网络架构**：Vector Neuron (VN) 层实现SE(3)等变。输入为左右手位置、旋转轴、重力方向编码成5个3D向量通道；VN-Transformer编码时间序列；流匹配条件生成（prior均匀SO(3)上采样，20步Euler积分）输出根框架。**等变性质**：若输入轨迹经SE(3)变换，预测根框架相应变换（G·Φ(x) = Φ(G·x)）。

### 算法流程（文字说明）
1. 对每段原始视频，通过摄像机运动估计和手部运动统计过滤出可能包含操作的手部可见片段。
2. 对活动片段同时处理：
   - 手部：WilOR估计MANO参数 → 手网格、关键点、全局姿态（相机坐标系）；补全中断帧并平滑。
   - 物体：文本提示驱动SAM-3检测 → SAM-2跟踪 → 深度背投得点云 → SAM-3D重建物体网格 → FoundationPose++跟踪位姿。
   - 公制校准：MoGe-2提供焦距和尺度；Flow3r提供深度；GeoCalib提供重力方向。
3. 精炼阶段：利用MEMFOF光流和手部关键点分类每帧状态（static/grasped/moving）。抓取状态将物体刚性绑到手上；静态状态锁定稳健质心；移动状态保持原始提议。同时进行尺度检查和虚假背景剔除。
4. 数据输出：公制手部轨迹、物体点云/网格、6-DoF位姿轨迹、接触状态。可选进行外到自我视角重投影（刚性坐标变换）。
5. 跨实体重定向：输入手部轨迹+重力，经过仿真训练的等变网络Φ（VN+流匹配）产生候选根框架；聚类选择最佳候选（通过IK收敛率、可操作性、关节限位裕度等评分）。然后IK跟踪，并对手指关节用几何映射单独处理。

## 6. 论文的主要结论与发现

- EgoInfinity能够**完全自动化**地将任意视角互联网RGB视频转换为包含公制手部轨迹、物体6-DoF姿态、接触状态的4D表示，无需穿戴设备、深度传感器、CAD模型或人工标注。

- **功能化跨实体重定向器**能在多种机器人（人形、空间机器人、双臂平台）上产生可行性高、平滑的运动，即使人类身体仅部分可见。
- 真实机器人实验验证了直接重放和作为策略先验训练的可行性，展示了抓取、切割、倾倒、擦拭等技能的可用性。
- 整体上，EgoInfinity为从互联网视频到可执行机器人行为提供了可扩展的基础设施，有助于开放世界机器人学习、多模态理解和人机交互。

## 7. 优点

- **完全自动化**：无需人工在环标注或手动指定物体/分割。
- **模块化设计**：每个组件可独立升级，引擎可持续受益于感知、重建等领域的进步。
- **跨模态统一**：通过公制校准将所有结果对齐到统一坐标系，支持手-物联合推理。
- **交互感知精炼**：利用时间状态（抓取/静止/移动）优化物体轨迹，减少纯视觉漂移，提升物理合理性和接触一致性。
- **功能性重定向**：不依赖完整人体姿态恢复，仅利用手部轨迹和重力方向；SE(3)等变网络+流匹配提供多假设选择，提高鲁棒性；支持任意视角和部分可见场景。
- **端到端验证**：不仅产生数据，还直接通过IK重放和策略学习验证真实机器人执行，覆盖从感知到执行的全链条。
- **交互式数据服务器**：提供在线浏览器，可搜索、可视化、下载处理结果，方便诊断和进一步使用。

## 8. 不足与局限

- **静态相机假设**：目前仅处理近似静态摄像机视频（教程、操作展示类），不处理身体佩戴或手持相机视频。虽然允许通过升级深度模块放宽此假设，但缺少SLAM使其无法适应运动相机内容，限制了语料多样性。
- **接触级精度不足**：交互精炼提供的是粗粒度的抓取检测和空间关联，不能保证精确接触一致性（如指尖位置、力一致性、无滑移约束），不适合需要精细力控的任务。
- **缺乏触觉信息**：不提供触觉观测，对需要力反馈的精细操作受限。
- **重定向器机器人特定**：对于新型机器人需要重新训练（约1.5-2h训练时间，可接受但非零开销）。更严重的是，它专注于功能性动作转移，对于要求精确手部姿态、接触时间或极度灵巧操作的任务可能不足。
- **缺乏与基线方法的定量比较**：没有与其他自动视频处理管线（如Track2Act、Hawor、其他手物重建方法）比较重建精度或重定向成功率。实验设计以自评估为主，说服力有限。
- **数据集规模有限**：目前只处理了106个Action100M视频，虽然声称引擎与语料无关，但实际效果和扩展性尚未在大规模数据上充分验证。

- **量化指标缺失**：对手部重投影误差、物体位姿误差、接触精确度等缺乏系统的量化指标，仅以定性结果和IK成功率为评价标准，难以客观衡量数据引擎的重建质量与基线方法的差距。
- **场景与物体通用性受限**：引擎依赖SAM系列的检测与分割，对于透明、高反光、非刚性或极小物体可能失效；静态相机假设进一步限制了摄像机运动较大视频的处理，从互联网视频中筛选满足条件（摄像机近似静止、手部可见）的片段实际占比未知，影响数据扩展效率。
- **时间上下文与长期任务建模不足**：当前处理以片段为单位，对于需要长期规划或复杂任务切换的视频，缺乏语义分割和任务层次建模，仅通过简单状态机合并动作，可能无法充分利用视频中的任务结构信息。
- **真实机器人评估不充分**：重定向器仅在仿真中训练和评估IK指标，真实机器人执行时未系统报告成功率和鲁棒性，演示样本数量较少且成功标准不明，限制了结论的可靠性和泛化性论证。

## 9. 总体总结

EgoInfinity提出了一个从互联网视频到可执行机器人行为的全自动数据引擎，在模块化设计、跨模态校准、交互感知精炼和功能性重定向方面有显著创新，为大规模机器人学习数据生成提供了新范式。然而，当前工作在定量对比、重建精度评估、真实机器人验证的严谨性以及场景通用性上仍存在明显不足。未来可进一步放宽静态相机假设、融合SLAM处理运动视角、引入细粒度接触优化、扩大处理视频规模并建立标准化的重定向与重建测评基准，以推动其在实际机器人学习中的应用。
