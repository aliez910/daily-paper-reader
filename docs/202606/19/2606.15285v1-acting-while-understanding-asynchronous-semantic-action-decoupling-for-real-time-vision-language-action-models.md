## 摘要
视觉-语言-动作模型（VLA）在机器人操作中展现了强大的任务理解与泛化能力，但全模型推理的高计算成本限制了其在低延迟、高频闭环控制中的部署。我们提出一种异步语义-动作解耦框架，该框架沿现有VLA的内部语义-动作接口将语义理解与动作生成分离，无需重新设计视觉-语言主干或引入外部规划器。低频理解模块异步更新可复用的语义条件，而高频动作模块持续输出控制动作，无需重复调用完整模型。为缓解过期语义与当前执行状态之间的时间错配，我们进一步引入历史动作条件化与时序失配训练，提供短时执行上下文并提升在陈旧语义条件下的反馈控制鲁棒性。在LIBERO上使用π0.5和UniVLA的实验，以及使用UniVLA的真实机器人部署表明，所提出的框架实现了高达35.6 Hz的服务器端动作模块推理吞吐量，并提供了低侵入式的高频闭环控制路径，而无需以控制速率运行完整VLA推理。

## Abstract
Vision-Language-Action models (VLAs) have demonstrated strong task understanding and generalization in robotic manipulation, yet the high computational cost of full-model inference limits their deployment in low-latency, high-frequency closed-loop control. We propose an asynchronous semantic-action decoupling framework that separates semantic understanding from action generation along the internal semantic-action interface of existing VLAs, without redesigning the vision-language backbone or introducing an external planner. A low-frequency understanding module asynchronously updates reusable semantic conditions, while a high-frequency action module continuously outputs control actions without repeatedly invoking the full model. To mitigate the temporal mismatch between stale semantics and the current execution state, we further introduce historical action conditioning and time-misalignment training, which provide short-horizon execution context and improve feedback control robustness under stale semantic conditions. Experiments on LIBERO with $π_{0.5}$ and UniVLA, together with real-robot deployment using UniVLA, show that the proposed framework achieves up to 35.6 Hz server-side action-module inference throughput and offers a low-intrusion path to high-frequency closed-loop control without running full VLA inference at control rate.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：Vision-Language-Action 模型（VLA）虽然具备强大的语义理解与跨任务泛化能力，但其全模型推理计算开销极大，难以在低延迟、高频（如 20 Hz）的机器人闭环控制中直接部署。现有方法（如动作分块）仅缓解了推理频率与控制频率的冲突，并未从根本上解除语义理解与动作生成之间的同步耦合，导致在执行过程中语义判断可能过时，影响控制质量。
- **整体含义**：本文旨在以一种低侵入、无需修改 VLA 主干架构或引入外部规划器的方式，实现高频闭环控制。通过将 VLA 内部的语义理解与动作生成沿内部接口解耦，让理解模块低频运行，动作模块高频运行，并专门应对因语义缓存过时带来的控制稳定性挑战。

## 2. 论文提出的方法论

- **核心思想**：利用 VLA 中语义理解变化较慢、动作生成需要高频更新的内在多时间尺度特性，将完整 VLA 推理过程拆分为两个异步执行的角色：
  - **低频理解模块**：处理视觉观测和语言指令，以较低频率（约 3–5 Hz）更新一个可复用的**中间语义条件**（如 UniVLA 的最后一层隐藏状态 / π0.5 的全 KV 缓存），将其写入语义缓存。
  - **高频动作模块**：在每个控制步（如 20 Hz）从语义缓存读取最新的语义条件，结合当前机器人状态与近期动作历史，快速生成动作，避免重复运行完整模型。
- **关键技术细节**：
  1. **历史动作条件化（Historical Action Conditioning）**：为动作模块提供过去 m 步的执行动作序列 \(h_t\)，帮助其理解当前执行进度和局部运动趋势，以弥补陈旧语义缺乏的时序信息。
  2. **时间错位训练（Time-Misalignment Training）**：在训练时故意向动作模块提供延迟的语义条件 \(z_{t-\Delta}\)（\(\Delta\) 在 0 到动作块长度间均匀采样），模拟部署时语义过时的情况，使模型学会在陈旧条件下仍能输出合理动作。
  3. **异步部署流程**：理解模块与动作模块作为独立进程异步运行；每个控制步动作模块利用最新的已提交语义缓存、当前状态和历史动作进行动作块重规划，并通过 ACT 风格的时间集成（指数加权）合并重叠预测，输出最终控制动作。

## 7. 优点

- **低侵入性**：沿 VLA 内部的语义-动作接口解耦，无需重新设计视觉-语言主干，也不依赖外部规划器或复杂多阶段训练管道。
- **通用性**：可适配不同内部接口形式的 VLA（UniVLA 的隐藏状态、π0.5 的 KV-cache），在两种骨干上均验证有效。
- **针对性强**：明确识别出“陈旧语义”是异步复用的主要挑战，并设计了两个互补的机制（历史动作条件化 + 延迟训练）来缓解，消融实验证实两者缺一不可。
- **实际部署价值**：在真实机器人上实现了更高的成功率与更快的任务执行速度，吞吐量提升显著，证明了缓存复用加高频重规划在闭环控制中的优势。
- **实验规范**：所有训练/评估遵循官方配置，控制变量严格，基线选择合理，消融设计清晰。

## 8. 不足与局限

- **依赖内部接口**：框架要求 VLA 存在一个可复用的中间语义条件，因此不适用于内部结构不满足此条件或有明显接口差异的模型，限制了通用性。
- **陈旧语义的边界**：当语义缓存年龄过大、场景突变或执行阶段快速切换时，动作模块可能仍受误导；当前固定刷新策略不一定最优，缺乏自适应调度。
- **任务覆盖有限**：实验仅涉及单臂操作任务，未验证在快速动态操作、移动操作、高度接触交互等场景下的表现；真实机器人仅部署了 UniVLA，π0.5 异步变体未在真实平台上验证闭环控制。
- **实际管道开销**：尽管动作模块推理吞吐量高达 35.6 Hz，但通信延迟（10–15 ms）与调度开销使实际控制步的端到端延迟仍有一定标准差，尤其 π0.5 异步后在考虑通信时难以稳定跟上 20 Hz 控制循环（19.7 Hz 未留余量）。
- **未深入探讨自适应调度**：当前框架未研究根据状态不确定性或执行阶段动态调整语义刷新频率的方法。
