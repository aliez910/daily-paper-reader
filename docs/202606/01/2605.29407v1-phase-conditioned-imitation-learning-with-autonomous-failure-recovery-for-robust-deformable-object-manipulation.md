## 摘要
本文提出了一种相位条件、力感知的鲁棒可变形物体操作框架。标准的模仿学习策略，如基于Transformer的动作分块（ACT），在推理时依赖马尔可夫假设，当视觉上相似的观测需要矛盾动作时会导致状态混淆，并阻碍从执行失败中自主恢复。我们通过闭环分层架构解决了这一问题。FiLM条件的ACT编码器根据当前任务相位调制特征提取，使得单一统一策略能够在各相位间共享动作动态的同时产生相位特定行为。融合视觉、力和位姿反馈的多模态相位预测器实时估计相位，检测仅靠视觉无法发现的接触故障，并自主触发恢复轨迹。该系统由混合阻抗控制器（用于柔顺执行）和力觉遥操作接口（用于力感知数据收集）完成。消融研究表明，基于FiLM的调制显著优于无条件基线和Token级条件基线，t-SNE分析证实FiLM产生分离良好的相位特定特征表示。通过在双臂挂取和脱下T恤上的验证，闭环系统通过自主错误恢复将挂取成功率从56%提升至87%。代码和视频：https://leledeyuan00.github.io/phaser/

## Abstract
This paper presents a phase-conditioned, force-aware framework for robust deformable object manipulation. Standard imitation learning policies such as Action Chunking with Transformers (ACT) rely on a Markovian assumption at inference, causing state aliasing when visually similar observations require contradictory actions and preventing autonomous recovery from execution failures. We address this with a closed-loop hierarchical architecture. A FiLM-conditioned ACT encoder modulates feature extraction based on the current task phase, enabling a single unified policy to produce phase-specific behaviors while sharing action dynamics across phases. A multi-modal phase predictor fusing visual, force, and pose feedback estimates the phase in real time, detecting contact failures that are invisible to vision alone and autonomously triggering recovery trajectories. The system is completed by a hybrid impedance controller for compliant execution and a haptic teleoperation interface for force-aware data collection. Ablation studies show that FiLM-based modulation significantly outperforms both unconditioned and token-level conditioned baselines, and t-SNE analysis confirms that FiLM induces well-separated, phase-specific feature representations. Validated on hanging and removing a T-shirt with dual arms, the closed-loop system improves the hanging success rate from 56\% to 87\% through autonomous error recovery. # 基于相位条件模仿学习与自主故障恢复的鲁棒可变形物体操作

## 1. 核心问题与研究动机

- **背景**：可变形物体操作（如衣物、线缆）因无限自由度、严重自遮挡和非线性接触动力学，难以建立精确分析模型，传统控制方法效果有限。数据驱动方法（强化学习、模仿学习）成为有前景的替代方案，其中模仿学习（IL）因直接从人类示教中学习而具有样本效率优势。
- **核心问题**：标准模仿学习策略（如ACT、Diffusion Policy）在推理时依赖马尔可夫假设（将当前观测直接映射到未来动作），在长时域任务中会导致**状态混叠（state aliasing）**：不同任务阶段可能呈现视觉上相似的状态，但需要相反的动作轨迹（如正常插入与故障恢复中的推进vs.后退）。当训练数据包含此类混叠状态时，策略会学习到矛盾动作的混合，产生平均或振荡行为。这并非数据不足，而是需要显式条件化处理的结构性歧义。
- **解决思路**：注入相位信息作为显式动态先验，并引入故障恢复机制，实现闭环鲁棒操作。

## 2. 方法论

### 2.1 整体架构：闭环分层控制系统

论文提出三层分层架构：
- **顶层（决策层）**：多模态相位预测器，融合视觉、力和位姿反馈实时估计当前任务相位，检测接触故障并触发恢复相位。
- **中间层（策略层）**：基于FiLM调制的ACT策略，根据相位生成特定行为，同时共享跨相位的动作动态。
- **底层（交互层）**：混合阻抗控制器，保证柔顺执行和安全。

### 2.2 关键技术细节

#### 中间层：相位条件分层ACT

- 将相位标签通过**Feature-wise Linear Modulation (FiLM)** 注入ACT编码器的每一层，直接调制层归一化和MLP输出，迫使编码器产生相位特定的特征表示。这解决了Token级条件化（将相位作为额外输入Token）被大量视觉Token稀释的问题。
- FiLM调制应用于编码器而非解码器理由：状态混叠源于感知层；保持解码器相位无关，使其学习跨相位共享的低级动作动态（如到达、插入、后退），提高数据效率。
- 输入：4个RGB图像（经ResNet-18提取特征） + 双臂末端执行器位姿（6D旋转） + 力/扭矩数据 + 相位标签嵌入；所有Token拼接后送入Transformer编码器。
- CVAE编码器对动作序列进行变分编码，推理时使用先验均值进行确定性解码。
- 输出：动作块（horizon Nh=50），包含双臂位置、6D方向和夹爪命令。

#### 顶层：相位预测与故障恢复

- 相位预测器共享已训练ACT策略的冻结图像骨干和位姿/力投影层，对每个相机特征图进行全局平均池化，将所有信息拼接为特征向量，经MLP输出相位logits。额外注入任务ID嵌入以约束预测空间。
- 恢复机制：一旦预测到故障相位（如夹持、错位、滑动），策略执行预定义的恢复轨迹（如抖动、后退、空中抓取）；恢复成功后自动切回正常相位。
- 对相位切换应用1秒低通滤波以抑制虚假切换，并在每个相位过渡时重置ACT的时间集成器。

#### 底层：混合阻抗控制器（Inc-IC）

- 将策略预测的末端位姿通过插值生成高频平滑轨迹（x_intp）和增量命令（x_inc = x_pred - x_obs），通过参数α和β混合，实现类似隐式力控制的柔顺行为。
- 根据夹爪状态切换控制模式：夹爪闭合时（与物体接触）启用力导向模式；夹爪打开时恢复纯阻抗控制。

### 2.3 数据收集

- 使用双侧触觉遥操作接口，操作员通过Meta Quest Pro控制器感知力传感器读数（振动反馈），在视觉模糊的接触状态中也能做出精准操控。

## 6. 主要结论与发现

1. **状态混叠的解决**：FiLM调制在编码器级别注入相位条件，产生分离良好的相位特定特征表示（t-SNE可视化证实），显著优于Token级条件化。
2. **闭环恢复的有效性**：闭环系统将挂取成功率从开环的56%提升至87%（32%绝对提升），脱下成功率从88%提升至92%（4%绝对提升）。故障检测率：挂取中90.91%（40/44）故障被检测到，恢复率78.9%（31/40）；脱下降66.67%（8/12）被检测到，恢复率50%（4/8）。
3. **恢复数据有用但不充分**：仅加入恢复演示（Model B）可提升非扰动性能，但在严重错位下因状态混叠完全失效（0%）。
4. **多任务数据加剧混叠**：未显式条件化时，多任务训练（Model C）反而降低性能（45% vs 40%）。
5. **Token级条件作用有限**：相位作为Token（Model D）对编码器影响微弱，恢复能力差（错位0%、夹持20%）。
6. **力反馈关键**：相位预测器融合力感知是检测视觉模糊接触故障（如夹持）的关键。
