## 摘要
我们提出了统一运动-动作（UMA）模型，该方法利用3D物体运动轨迹作为共享接口，以桥接视觉运动控制与动力学建模。UMA将物体运动和机器人动作视为共同演化的变量，在掩码生成目标下进行建模，其中掩码模式决定了预训练期间的监督方式以及部署时的推理模式。利用后见之明重新标记的运动上下文和对比目标，将任务意图与场景几何解耦，UMA能够在无需手动标注任务指令的情况下，跨异构数据源进行多任务预训练。在部署时，相同的预训练参数支持基于运动条件的视觉运动控制、基于运动的动力学建模以及通过少量演示进行任务适应。通过在机器人演示、人类视频和模拟数据的混合数据上进行预训练，UMA在每种推理模式下始终优于专门的最先进基线方法。

## Abstract
We present Unified Motion-Action (UMA) Model, an approach that uses 3D object motion trajectories as a shared interface to bridge visuomotor control and dynamics modeling. UMA treats object motion and robot actions as co-evolving variables under a masked generative objective, in which the mask pattern determines both the supervision regime during pretraining and the inference mode at deployment. Using hindsight-relabeled motion contexts and a contrastive objective that disentangles task intent from scene geometry, UMA enables multi-task pretraining across heterogeneous data sources without requiring manually annotated task instructions. At deployment, the same pretrained parameters support motion-conditioned visuomotor control, motion-based dynamics modeling, and task adaptation from few-shot demonstrations. Pretrained on a mixture of robot demonstrations, human videos, and simulated data, UMA consistently outperforms state-of-the-art baselines specialized for each inference mode.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有机器人基础模型在处理异构数据（真实机器人数据、人类视频、模拟数据）时难以统一，主要障碍在于：（1）基于运动条件的方法需要成对的运动-动作标签，无法利用无动作视频；（2）运动预测模型需额外控制器，无法直接利用动作标签。因此缺乏一种能同时从无动作视频学习运动、并从有标签机器人数据中利用动作标签的统一框架。
- **整体含义**：本文提出以**3D物体运动轨迹**作为共享接口，将视觉运动控制与动力学建模统一在一个框架内。通过将运动与动作视为共同演化的变量，采用掩码生成式训练，使异构数据源无需人工标注任务描述即可贡献监督，从而提升机器人学习的泛化性和数据效率。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将物体运动 (x) 和机器人动作 (a) 作为共同演化的序列变量，通过一个基于掩码扩散变换器（Masked DiT）的联合生成模型建模。在训练时随机掩码运动或动作token，不同掩码模式对应不同监督信号；在推理时通过指定哪些token作为条件（给定）哪些作为预测，支持多种模式（运动条件控制、动力学预测、少样本任务适应）。
- **关键技术细节**：
  - **任务编码器（Task Encoder）**：基于初始观测（点云+图像特征）和稀疏参考运动（参考轨迹片段），通过PointNet++和交叉注意力提取任务潜在变量 c，并通过对比学习（SimCLR）使其对关键点子采样、时间子采样、SE(3)变换具有不变性，从而解耦任务意图与场景几何。
  - **掩码扩散变换器（Masked DiT）**：使用流匹配（Flow Matching）损失对掩码的运动和动作token进行去噪。设计双自适应层归一化（dual adaLN）——掩码token（目标）与给定token（条件）使用不同缩放偏移参数，分别适应去噪与保护干净条件。注意力采用结构化时空分解：空间注意力（同时间步耦合运动与动作）、时间注意力（同关键点跨时间步）、上下文注意力（连接到观测和任务token）。
  - **异构监督**：真实机器人数据同时监督 Lx 和 La；无动作视频、跨平台数据仅监督 Lx；模拟数据监督两者。对比损失 Lc 作用在任务编码器上。

## 6. 论文的主要结论与发现

1. **统一框架有效性**：单个UMA模型在零样本运动控制、动力学预测、少样本适应三种模式下均显著优于专门化基线（成功率高20-25个百分点）。
2. **异构数据互补**：模拟数据主要贡献运动-动作耦合（使动力学预测MSE降低5倍）；人类视频贡献任务多样性（对变形操作性能提升重要）。
3. **对比学习关键**：去除对比损失（w/o CL）后少样本成功率下降30-60个百分点，说明任务意图解耦是少样本迁移的核心。
4. **模型设计验证**：联合预测运动+动作优于仅预测动作；结构化注意力优于密集注意力（在同样预算下）。
5. **失败主要源于执行精度**：82%错误是执行失败，指出现有瓶颈在于精细动作跟踪而非任务意图识别。
6. **规模带来提升**：在模拟任务中，数据量和参数量扩大都带来稳定性能提升。

## 7. 优点

- **统一的生成式框架**：将控制与动力学有机融合在单一模型、单一预训练目标中，无需切换模型或架构即可切换推理模式，理论优雅且实用。
- **高效利用异构数据**：通过掩码机制完美兼容无动作视频、模拟数据、真实机器人数据，极大扩展可用的预训练数据池，无需人工任务标注。
- **任务意图解耦创新**：引入后见之明运动重标注与对比学习，使任务潜在变量对参考运动的稀疏性、时间子采样及场景刚性变换具有不变性，从而支持跨场景迁移和软提示微调。
- **结构注意力设计节省计算**：Masked DiT的双调制层归一化避免了在全图上应用去噪声调制，保持条件信息完整，且注意力因子化降低复杂度。
- **丰富消融与分析**：从数据构成、模型设计、对比学习、失败模式到缩放行为均有系统实验，结论有据可依。

## 8. 不足与局限

- **动作头依赖新平台**：若要在新机器人上部署，仍需监督训练与该平台兼容的动作头（action head），跨实体泛化尚未完全解决。
- **观测空间限制**：目前仅使用单帧视觉观测作为条件，未利用多帧历史观测和帧间密集对应，可能限制时序理解能力。
- **外参依赖**：所有轨迹在单个相机的摄像机坐标系下表示，部署动作时需标定相机到机器人基座的刚性变换，且无动作视频数据不需标定，但机器人数据动作需要在机器人坐标系下表达，实际操作流程较复杂。
- **实验环境相对聚焦**：仅在桌面操作场景（DROID配置）评估，未涉及移动操作、长时域任务或高度变形场景（除简化折叠任务外），泛化到更多元环境的能力有待验证。
- **参考运动获取**：测试时需用户通过交互界面指定参考运动或提供目标图像，部分场景下自动化任务指定可能仍有困难。
- **计算资源**：600M参数模型在8张A100上训练，虽不如当前最大模型，但仍需一定算力开销，小规模团队的完整复现可能存在门槛。
