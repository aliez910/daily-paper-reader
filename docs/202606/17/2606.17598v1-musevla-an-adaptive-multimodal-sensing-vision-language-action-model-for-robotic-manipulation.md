## 摘要
人类自然地利用多种感知模态与物理世界交互，而大多数用于机器人的视觉-语言-动作（VLA）模型仅依赖RGB观测。这限制了它们感知难以或无法从RGB相机推断的物理属性的能力，例如温度、声音或雷达响应。我们提出MuseVLA，一种适应性多模态感知VLA模型，它将新型传感器集成为机器人操作的按需工具。给定任务指令和视觉上下文，MuseVLA首先生成一个传感器令牌和目标描述，选择要调用的感知模态以及需要注意的对象，类似于带参数的工具调用。然后，它将选定的传感器测量转换为一个接地传感器图像，这是一种统一的中间表示，编码异构读数用于多模态融合和动作生成。这种设计将传感器特定处理与VLA主干解耦，实现了多样模态的高效集成。为了减少对昂贵的多感官机器人数据集的需求，我们进一步引入了一个数据合成流程，用接地传感器图像增强现有的RGB视频数据集，从而实现对未见过的传感器引导任务的泛化。我们在真实机器人上评估MuseVLA，针对需要多模态感知输入的高难度灵巧手操作任务，包括温度引导的拾取和放置、音频驱动的物体搜索和雷达辅助的隐藏物体检索。MuseVLA平均成功率达到80.6%，显著优于纯RGB和多感官VLA基线，并且在未见任务上展现出强大的零样本能力。

## Abstract
Humans naturally leverage diverse sensing modalities to interact with the physical world, while most Vision-Language-Action (VLA) models for robotics rely solely on RGB observations. This limits their ability to perceive physical properties that are difficult or impossible to infer from RGB cameras, such as temperature, sound, or radar response. We present MuseVLA, an adaptive multimodal sensing VLA model that integrates novel sensors as on-demand tools for robotic manipulation. Given a task instruction and visual context, MuseVLA first generates a sensor token and target description that select the sensing modality to invoke and what to attend to, analogous to a tool call with arguments. It then converts the selected sensor measurement into a grounded sensor image, a unified intermediate representation that encodes heterogeneous readings for multimodal fusion and action generation. This design decouples sensor-specific processing from the VLA backbone, enabling efficient integration of diverse modalities. To reduce the need for expensive multisensory robot datasets, we further introduce a data synthesis pipeline that augments existing RGB video datasets with grounded sensor images, enabling generalization to unseen sensor-guided tasks. We evaluate MuseVLA on a real-world robot across challenging dexterous hand manipulation tasks that require multimodal sensing inputs, including temperature-guided pick-and-place, audio-driven object search, and radar-assisted hidden object retrieval. MuseVLA achieves 80.6% success rate on average, outperforming RGB-only and multisensory VLA baselines significantly, and exhibits strong zero-shot capabilities on unseen tasks.

## 2. 论文提出的方法论
- **核心思想**：将传感器视为可动态调用的工具，任务执行分为三个阶段：
  1. **传感器选择与目标描述**：根据任务指令和 RGB 图像，VLM 生成一个传感器令牌（`<None>/<Thermal>/<Acoustic>/<mmWave>`）和一个语言目标描述（如“the hot drink”），相当于“工具调用+参数”。
  2. **接地传感器图像构建**：调用被选中的传感器，将其原始测量（热力图、功率热图等）通过传感器专用颜色映射转换成 RGB 空间的热力图；再利用一个预训练的分割模型（SAM3）根据目标描述分割出目标区域，将热力图叠加到对应 RGB 区域上，生成一张统一的**接地传感器图像**（Grounded Sensor Image）。
  3. **动作生成**：将接地传感器图像作为额外输入（与 RGB 图像共用同一视觉编码器），结合指令、传感器令牌、目标描述，送入扩散 Transformer 动作专家以生成机器人动作序列。
- **关键技术细节**：
  - **传感器令牌**：可学习的四个离散令牌，分别对应无传感器、热成像、声学、毫米波雷达。VLM 以自回归方式预测正确令牌。
  - **接地传感器图像**：数学上表示为 `m = M ⊙ s + (1−M) ⊙ o_RGB`，其中 `M` 是 SAM3 生成的目标区域二值掩码，`s` 是传感器热力图，`o_RGB` 是原始 RGB 图像。这种设计将异构传感数据统一到 RGB 空间，无需专用传感器编码器。
  - **数据合成流水线**：构建“传感器词典”（如 thermal →“hot”“cold”），将关键词注入已有 RGB 操作数据集的指令中，用 VLM 生成目标描述，再用 SAM3 分割目标并覆盖对应颜色的掩码来模拟传感器观测。合成数据用于预训练，有效缓解真实多模态数据稀缺问题。
  - **训练损失**：联合优化 VLM 生成损失（传感器令牌交叉熵 + 目标描述交叉熵）和扩散动作预测的 MSE 损失，系数 λ 设为 1e-2。

- **任务场景**：设计三大类需要超越 RGB 感知的任务（每个包含多个子指令和物体）：
  - **热引导抓取**：指定温度（hot/cold/room‑temperature）的饮料拾取和放置。
  - **音频驱动搜索**：通过声音定位被覆盖的物体（如响铃的手机）并移除覆盖物。

- **基线方法**：

## 7. 优点
- **方法创新性**：将传感器视为“按需工具”，通过传感器令牌实现自适应选择，避免对所有模态进行固定融合，更接近人类感知策略，且计算高效。
- **统一表示设计**：接地传感器图像利用 RGB 空间编码异构传感数据，无需专用编码器，使新传感器的集成变得简便，数据合成也成为可能。
- **数据合成流水线**：无需真实多传感器数据进行大规模训练，利用已有 RGB 数据集即可生成多样化多模态训练数据，降低了成本并提升了零样本泛化。
- **实验评估全面**：在真实灵巧手平台上进行多类任务测试，覆盖热、声、雷达三种模态，并包含多阶段任务、零样本泛化，消融分析完整。
- **开源承诺**：将开源数据和代码，有利于社区进一步研究。

## 8. 不足与局限
- **真实数据集规模有限**：仅 720 个演示，物体种类和场景变化不够丰富，可能无法展现方法的全部潜力。
- **依赖分割模型**：接地传感器图像的构建依赖 SAM3 的分割质量，在目标描述语义复杂或物体极小时可能产生错误掩码，影响后续动作。
- **传感器校准**：需要离线单应性对齐和颜色映射预设，对传感器安装和场景光照有要求，实际部署时校准误差可能导致性能下降。
- **任务覆盖范围**：实验集中于桌面灵巧手操作，未见在移动操作、人形机器人或户外环境中的评估；模态种类也局限于四种，扩展更多传感器（如触觉、力）未实验。
- **合成数据真实性**：基于颜色编码的模拟传感器数据与真实物理传感器的输出分布存在差距，对零样本泛化的进一步提升可能有瓶颈。
- **零样本泛化仍有限**：虽然在未见任务上获得 66.7% 成功率，但离完全通用仍有差距，特别是在复杂指令或全新物体类别上表现可能下降。
- **计算资源需求较高**：64 A100 的训练配置对一般研究团队来说门槛较高。
