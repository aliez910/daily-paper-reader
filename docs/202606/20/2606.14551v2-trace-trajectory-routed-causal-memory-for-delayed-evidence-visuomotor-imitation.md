## 摘要
自主运行的机器人可能需要基于不再可见的证据做出决策。我们研究延迟证据任务，其中早期线索在后续决策点之前消失，因此视觉上相似的观察可能需要不同的动作。在这些设置中，当前观察不足以作为控制的状态。我们提出了轨迹路由因果证据（TRACE），一种用于视觉运动模仿策略的记忆框架。TRACE将任务相关的视觉和机器人状态证据（如物体身份、目标选择或依赖路径的状态）存储在一个固定大小的潜在记忆中，该记忆在长时间片段内保持有界。TRACE不通过原始时间或手动提供的任务标签索引记忆，而是使用路径签名：对已执行的机器人状态轨迹的紧凑、顺序敏感特征。这些签名不存储视觉线索本身；而是提供了基于轨迹的键，用于写入和检索当线索可见时存储的证据。当机器人后来遇到模糊的观察时，策略基于TRACE记忆来恢复缺失的上下文并选择正确的分支。TRACE通过轻量级适配器附加到策略上，而不改变策略主干、动作头或模仿目标。在具有视觉模糊分支点的实际长时域操作任务中，TRACE在分支选择和任务成功率上优于替代基线（包括短历史和循环记忆）。项目页面：https://jeong-zju.github.io/trace

## Abstract
Robots under autonomous operation may require decisions based on evidence that is no longer visible. We study delayed-evidence tasks, where an early cue disappears before a later decision point, so visually similar observations can require different actions. In these settings, the current observation is not a sufficient state for control. We introduce TRAjectory-routed Causal Evidence (TRACE), a memory framework for visuomotor imitation policies. TRACE stores task-relevant visual and robot-state evidence, such as object identity, target choice, or route-dependent state, in a fixed-size latent memory that remains bounded over long episodes. Instead of indexing memory by raw time or manually provided task labels, TRACE uses path signatures: compact, order-sensitive features of the executed robot-state trajectory. These signatures do not store the visual cue itself; rather, they provide trajectory-conditioned keys for writing and retrieving the evidence stored when the cue was visible. When the robot later reaches an ambiguous observation, the policy conditions on TRACE memory to recover the missing context and choose the correct branch. TRACE attaches through lightweight adapters to policies, without changing the policy backbone, action head, or imitation objective. Across real-world long-horizon manipulation tasks with visually ambiguous branch points, TRACE improves branch selection and task success over alternative baselines, including short-history and recurrent memory. ## 1. 核心问题与整体含义（研究动机与背景）
- **核心问题**：在延迟证据（delayed-evidence）的视觉模仿任务中，早期关键线索（如物体身份、起始位置）在后续决策点之前消失，导致视觉上相似的观测需要不同的动作，违反了隐式马尔可夫假设（当前观测足以决策）。
- **意义**：机器人自主操作需要能够记住历史线索来消除歧义，但现有方法（短窗口、循环网络、通用记忆）无法在固定预算内有效保留和检索分支相关的早期证据。
- **本文贡献**：提出TRACE，一种基于路径签名（path signature）路由的因果记忆框架，能够在线存储并恢复早期任务证据，且作为轻量模块附加到任意视觉模仿策略上，不改变其主干、动作头或损失函数。

## 2. 方法论：核心思想、关键技术细节与算法流程
- **核心思想**：
  - 将**记忆内容**（当前视觉-状态证据）与**记忆访问**分离，使用机器人状态轨迹的路径签名作为顺序敏感的键，在固定数量潜在槽（latent slots）中写入和读取证据。
  - 轨迹签名不存储视觉线索本身，仅提供基于轨迹的地址，使得在不同历史下到达相同视觉分支点时能够检索到不同的存储证据。
- **关键技术细节**：
  - **路径签名**：对机器人状态轨迹的分段线性插值计算截至当前时刻的深度‑p签名 ξₜ = Sig≤p(Sₜ) 和增量 δₜ = ξₜ - ξₜ₋₁，作为紧凑且顺序敏感的轨迹描述符，支持流式更新。
  - **签名路由槽记忆**：
    - 轨迹键 qₜ = φq([gₜ, Δgₜ])，其中 gₜ=φg(ξₜ), Δgₜ=φΔ(δₜ)。
    - 路由权重 ωₜ,ₖ 基于 qₜ 与上一时刻槽内容的注意力计算，决定当前写入哪些槽。
    - 门控写入：βₜ,ₖ 控制更新强度，使选中的槽吸收当前证据，未选中的几乎不变。
    - 读取：使用当前视觉证据和轨迹状态形成的查询对更新后的槽做注意力读取，得到记忆读出 zᵐᵉᵐₜ。
  - **策略适配器**：TRACE状态 Rₜ = (Mₜ, zᵐᵉᵐₜ, gₜ, Δgₜ) 通过轻量适配器转化为策略本征条件。对回归策略（如ACT）拼接为记忆token；对扩散策略（如Diffusion Policy）通过零初始化MLP生成全局条件向量。
  - **辅助损失**：防止槽崩溃、扩散写入和读写不匹配的平衡、熵、一致性损失，保持记忆稳定。

## 6. 论文的主要结论与发现
- TRACE 显著提升延迟证据操纵的成功率：回归策略平均进度从 25.50 提升至 69.23，扩散策略从 25.00 提升至 59.53，优于所有非 TRACE 基线（包括 π₀.5 的 50.47）。

- 诊断证明 TRACE 使用历史有序证据而非仅决策点观测：顺序反转导致路由相似度和分支一致性大幅下降（37.80% 和 41.60%），而顺序保持的变化保持较高水平（约 90% 和 95%）。
- 长期记忆稳定性诊断（附录表12）表明固定槽能维持长时间历史而不会严重覆盖或发散，分支决策一致性保持在 0.87 以上（2倍历史长度扩展时）。

## 7. 优点：方法与实验设计亮点
- **模块化设计**：TRACE 通过轻量适配器与现有策略结合，不改变策略主干、动作头或损失函数，易于集成。
- **基于路径签名的访问机制**：利用签名的时间重参数化不变性，使得访问键仅依赖轨迹几何，与执行速度或坐标偏移无关，且支持流式更新。
- **固定预算记忆**：槽数量固定，不会随时间无限增长，适合长时域在线执行。
- **路由写作和读取分离**：通过轨迹键决定写入槽，通过视觉证据查询读出，使存储和检索解耦，提高隔离性。
- **丰富的实验诊断**：除了任务成功率，还提供了路由相似度、分支一致性、长期稳定性等内部诊断，从机制上验证记忆行为。
- **公平比较**：所有基线使用相同数据集、观测接口和评估协议，微调 VLA 模型时不额外泄漏延迟线索。

## 8. 不足与局限
