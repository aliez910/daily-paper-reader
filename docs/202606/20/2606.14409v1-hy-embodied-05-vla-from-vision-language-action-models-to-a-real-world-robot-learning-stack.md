## Abstract
In this report, we present Hy-Embodied-0.5-VLA, abbreviated as HyVLA-0.5, an end-to-end system that spans the full robot learning stack: data collection, model design, continued pre-training and supervised fine-tuning, RL post-training, and real-world deployment. Each component serves a distinct role in this stack.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：当前 Vision-Language-Action（VLA）模型发展迅速，但多数工作聚焦于单个环节（如模型架构或策略学习），缺乏一个从**数据收集 → 模型设计 → 训练 → 后训练 → 真实部署**的完整机器人学习栈。一个真正通用的机器人不可能仅凭单一模型实现，必须建立在系统性集成的全栈框架之上。
- **核心问题**：现有 VLA 系统在以下方面存在脱节：
  - **数据采集**：传统遥操作缺乏触觉反馈，难以获取高精度演示；手持 UMI 方法虽降低了数据获取门槛，但精度受限（依赖 SLAM），且缺乏指尖级力觉。
  - **模型设计**：通用视觉 - 语言骨干缺乏针对机器人控制的显式设计，离散动作预测限制了控制频率和精度。
  - **策略优化**：标准模仿学习难以处理长尾问题，奖励驱动强化学习在接触丰富的操作中依赖脆弱的奖励模型。
  - **部署执行**：高容量 VLA 模型推理延迟高，难以满足实时闭环控制需求。
- **总体含义**：本文提出 HyVLA-0.5，一个 0.5B 参数的端到端 VLA 系统，从**数据、表示、训练、后训练、部署**全链路协同设计，目标是为可部署的通用机器人提供实用框架。

### 2. 论文提出的方法论：核心思想、关键技术细节

**核心思想**：以 **Hy-Embodied-0.5** 骨干为基础，扩展流匹配动作专家、紧凑记忆编码器和增量式动作表示，通过大规模 UMI 预训练、双轨监督微调、基于偏好优化的 RL 后训练，以及异步推理与平滑部署，实现从数据到真实机器人的完整闭环。

**关键技术细节**：

- **数据收集**：定制指尖 UMI 设备，配合光学动捕系统（非 SLAM），实现亚毫米级 6DoF 轨迹记录；采集场景包括 6 类任务族，总计超 10,000 小时、100 万条演示。
- **模型架构**：
  - **骨干**：HY-Embodied-0.5-MoT（Mixture-of-Transformers，4B 参数），独立视觉 / 语言参数，原生分辨率 ViT。
  - **动作专家**：基于流匹配的连续动作生成，采用双塔路由（理解 VLM + 动作生成），块状因果注意力。损失为流匹配回归损失 Lfm。
  - **紧凑记忆编码器**：在 ViT 中插入时空注意力，压缩多帧历史（K=6 时使用），参数量不变（重用 QKV 投影），为当前帧注入历史上下文。
  - **动作表示**：Delta-chunk（增量动作），相对于当前末端执行器位置的相对运动，解耦策略与具体运动学。
- **训练流程**：
  - **持续预训练**：在 Hy-UMI-10K 上以 Lfm 为主损失（λntp=0 时可关闭辅助损失），训练 200K 步，batch=1024。
  - **监督微调（SFT）**：双轨设计：
    - Track A（内载体适应）：在同一机器人平台收集演示并微调（如 Dobot X-Trainer）。
    - Track B（跨载体迁移）：仅用 UMI 演示微调，部署到形态不同的机器人（如 JAKA K1、Astribot S1），无需目标机器人遥操作数据。
  - **RL 后训练（FlowPRO）**：提出 Proximalized Preference Optimization（PRO）的流匹配适配版本，免奖励模型、免价值网络。通过“干预 - 回滚”管道收集偏好对，用 RPRO 损失（对比项 + 近端正则化 + SFT 项）优化，实现从失败案例中快速改进。
- **部署**：
  - 异步推理 - 执行流水线：生产者 - 消费者模式，重叠推理和执行。
  - 延迟感知的 Bézier 块拼接：丢弃过期前缀，用三次贝塞尔曲线保证 C1 连续过渡（位置用 R³ 贝塞尔，方向用 SLERP，夹爪线性插值），无需额外学习模块。

### 3. 实验设计

- **数据集**：
  - **预训练**：自收集 Hy-UMI-10K（70+ 任务，10K+ 小时，亚毫米精度）。
  - **模拟微调**：RoboTwin 2.0（Aloha-AgileX 双臂设置，50 任务，每任务 50 / 500 演示，共 2.75K 条，600 万帧）。
  - **真实世界微调**：
    - Track A：Dobot X-Trainer，4 任务，每任务 300 演示（共 18 小时）。
    - Track B：JAKA K1（1 任务，300 UMI 演示），Astribot S1（1 任务，200 UMI 演示），Unitree G1（1 任务力验证，400 UMI 演示）。
  - **FlowPRO 后训练**：Dobot X-Trainer 上 4 任务（Bottle, Cap, USB, Zip），每轮采集 ≤ O(10²) 偏好对。
- **Benchmark**：
  - **模拟**：RoboTwin 2.0 的 Clean 和 Randomized 设置，每任务 100 次随机 rollout，取 50 任务平均成功率。
  - **真实**：每任务 100 次随机 rollout，报告成功率和完成时间。
- **对比方法**：
  - **模拟**：π0, π0.5, ABot-M0, LingBot-VLA, starVLA, Motus, JoyAI-RA, Qwen-VLA。
  - **真实**：π0, π0.5, DAgger, π0.6*（优势条件回归）。
  - **消融**：移除记忆编码器、移除 UMI 预训练。

### 6. 论文的主要结论与发现

- HyVLA-0.5 在 RoboTwin 2.0 模拟上达到 90.9% / 90.1%（Clean / Randomized），优于所有对比方法（如 JoyAI-RA 90.5/89.3%）。
- 真实世界 Track A（内载体）任务成功率显著优于 π0、π0.5，尤其在高精度瓶颈步骤。
- Track B（跨载体转移）仅用 UMI 演示微调即可部署到形态不同的机器人，成功率明显优于同配置基线。
- UMI 预训练的主要价值在于锐化精密操作关键步骤的动作分布。
- FlowPRO 后训练在 4 个真实任务上超越 DAgger 和 π0.6*，达到接近天花板成功率（98-99%），同时缩短完成时间。
- 紧凑记忆编码器和 UMI 预训练均带来一致增益（模拟消融：分别 +2.1/1.5 和 +0.7/0.7 绝对百分点）。
- 力觉验证：通过指尖力传感器，Unitree G1 能利用触觉信号区分物体重量，完成力引导任务。

### 7. 优点

- **系统完整性**：真正打通了从数据采集到真实部署的全链路，每个组件都有明确的设计动机。
- **数据质量高**：光学动捕 + 指尖 UMI 提供亚毫米级精度，且自然包含力觉，可复用为 RL 后训练数据。
- **动作表示巧妙**：Delta-chunk + EEF 表示解耦策略与具体机器人运动学，便于跨载体迁移。
- **免奖励 RL 后训练**：FlowPRO 无需训练奖励 / 价值模型，利用失败案例直接优化，且对比梯度抵消特性安全混合 SFT 数据。
- **部署实用性强**：异步推理 + Bézier 拼接保证高频实时控制，不引入额外学习模块。
- **双轨微调设计**：Track A/B 分离了内载体适应与跨载体转移，揭示不同场景下数据的价值。
- **实验规范**：模拟和真实均采用 100 次随机 rollout，FlowPRO 报告 3 种子均值标准差。

### 8. 不足与局限
