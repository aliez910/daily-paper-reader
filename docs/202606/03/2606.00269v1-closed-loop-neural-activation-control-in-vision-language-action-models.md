## 摘要
视觉-语言-动作（VLA）模型可以通过干预语义上有意义的内部方向在测试时进行引导，但现有方法使用固定的引导系数，实际上是在开环中运行。这不利于具身控制，其中任务状态和概念误差随时间演变，常常导致过度校正、振荡和任务成功率降低，特别是对于速度和平滑度等时间行为。我们提出CTRL-STEER，一个闭环框架，用自适应、时变控制信号替代静态干预强度。关键思想是将表示与调节解耦：我们不假设时间概念由单个神经元直接控制，而是沿着与运动对齐的残差方向进行引导，同时反馈控制器在线调整干预幅度。我们使用基于PID和强化学习的控制器实例化该框架。在四个LIBERO任务套件上使用微调OpenVLA策略进行的实验表明，CTRL-STEER比固定系数基线实现了更稳定的概念调节和更好的引导-任务成功权衡，且无需修改或重新训练基础模型。

## Abstract
Vision-Language-Action (VLA) models can be steered at test time by intervening on semantically meaningful internal directions, but existing methods use a fixed steering coefficient, effectively operating in open loop. This is poorly suited to embodied control, where task state and concept error evolve over time, often causing overcorrection, oscillation, and reduced task success, especially for temporal behaviors such as speed and smoothness. We propose CTRL-STEER, a closed-loop framework that replaces static intervention strength with adaptive, time-varying control signals. The key idea is to decouple representation from regulation: rather than assuming temporal concepts are directly controlled by individual neurons, we steer along motion-aligned residual directions while a feedback controller adjusts intervention magnitude online. We instantiate this framework with both PID and reinforcement learning based controllers. Experiments with a fine-tuned OpenVLA policy on four LIBERO task suites show that CTRL-STEER achieves more stable concept regulation and a better steering-task success trade-off than fixed-coefficient baselines, without modifying or retraining the base model.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的视觉-语言-动作（VLA）模型在测试时可以通过干预内部神经元的语义方向来引导行为（steering），但现有方法使用固定的干预系数，属于**开环控制**。开环控制在具身任务中表现不佳，因为任务状态和概念误差随环境动态演变，固定系数会导致**过校正、振荡以及任务成功率显著下降**，对于速度、平滑度等时间依赖性概念尤为严重。
- **研究背景**：VLA模型（如OpenVLA）结合了视觉-语言骨干和动作解码器，具备语义泛化能力，但难以推广到速度、高度等连续控制属性。现有机械可解释性方法虽能定位与概念相关的神经元，但采用静态干预，没有考虑对任务完成度的反馈，导致改善概念引导的同时常常破坏任务成功。作者旨在**提出闭环框架，动态调整干预强度，兼顾概念引导与任务成功率**。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- **解耦表示与调控**：不是假设时间概念由单个神经元直接控制，而是沿着**与运动对齐的残差方向**进行引导，同时使用**反馈控制器**在线调节干预幅度，形成闭环控制。
- 提出**CTRL-STEER**框架，包含两个关键组件：①通过机械可解释性识别与概念相关的神经元集合；②使用**PID控制器**或**强化学习（RL）控制器**动态调整这些神经元的激活系数。

### 关键技术细节
1. **神经元识别（机械可解释性）**：
   - 利用FFN分解：每个神经元对应一个值向量\( v_{\ell i} \)，可投影到词汇空间得到概率分布。通过语义嵌入（概率加权词嵌入平均）表征神经元语义。选取与目标概念（如up, down等）最相似的k个神经元（k=5），经人工过滤后保留10个神经元作为干预集合\( S \)。
   - 特别之处：针对**连续控制概念**，不是只选一个特征，而是选取一组相关特征（如运动方向），并处理了神经元多义性问题。

2. **PID闭环控制器**：
   - 定义误差\( e_t = c^* - c_t \)，其中\( c_t \)为当前概念值（如高度、速度），\( c^* \)为目标值。
   - 控制器输出时变系数\( \alpha_t^{PID} = K_P e_t + K_I \sum_{\tau=0}^t e_\tau + K_D (e_t - e_{t-1}) \)，其中\( K_P=4.0, K_I=0.5, K_D=1.0 \)，限制\( \alpha_t \in [0,20] \)。
   - 该标量信号施加到所有选中的神经元上。

3. **RL闭环控制器**：
   - 状态\( s_t = [a_t, \Delta a_t, \alpha_{t-1}, t/T] \)，其中\( a_t \)为选中的k个神经元的激活值。
   - 策略\( \pi_\theta \)输出k维向量\( \alpha_t^{RL} \)，每个神经元单独调制。
   - 使用PPO优化，奖励函数\( r_t = r_{\text{steer}}(t) + \lambda \cdot r_{\text{task}} \)，其中\( r_{\text{steer}} \)为即时概念指标（高度或速度），\( r_{\text{task}} \)为任务完成奖励，\( \lambda \)权衡系数。
   - 初始化：RL策略首先用PID生成的轨迹进行预热，然后与环境交互优化。

4. **实现细节**：使用OpenVLA（基于LLaMA-2的7B模型），每任务独立训练RL策略，控制总步数920（20步热身+900步执行）。

## 6. 论文的主要结论与发现

- **开环静态引导的局限性**：高静态干预系数（C=20）虽能提升概念指标，但导致任务成功率大幅下降（如速度引导成功率从~78%降至1.8%）；而CTRL-STEER的PID控制器保持成功率接近无引导基线（71.37%），RL控制器甚至超过基线（73.88%高度、76.12%速度）。
- **闭环框架的有效性**：动态调节干预强度能更好平衡概念引导与任务成功，避免过度校正和振荡。
- **PID初始化对RL训练很重要**：没有PID初始化的RL策略成功率降低（如LIBERO-Long从66.5%降至57%），概念引导效果也略差。
- **计算开销低**：CTRL-STEER相比无引导仅增加少量推理时间（0.1869→0.2094秒/步），峰值GPU内存不变（~14.26 GB）。
- **迁移性**：在X-VLA和BridgeData V2上也验证了有效性，表明框架可适用于不同VLA架构。

## 7. 优点

- **创新性**：首次将闭环控制思想引入VLA模型的激活引导领域，克服了开环静态方法的根本缺陷。
- **技术互补性**：PID控制器简单有效，RL控制器能进一步优化长期任务成功，两者结合（PID初始化RL）提供了稳定起点。
- **实验扎实**：在多个标准基准、多个任务、多个指标上系统对比，提供了丰富的定量结果及定性可视化（图1、2、4、5）。

## 8. 不足与局限

- **RL控制器需任务级训练**：每个任务都需要单独训练一个RL策略，无法直接零样本迁移到新任务，限制了实用性。

- **仅评估两种概念**：高度和速度虽具代表性，但未覆盖其他连续控制概念（如加速度、转动角度、力度等），也未考虑多概念同时引导的场景。
- **实验规模有限**：仅在模拟器（LIBERO, BridgeData V2）上评估，未在真实机器人上验证，可能存在仿真到现实的差距。
