## 摘要
人类为机器人模仿学习提供的演示通常包含错误和纠正行为，例如不精确的抓取、物体未对齐、不稳定接触和重复尝试。虽然这些片段通常被视为噪声或次优数据，但它们提供了关于执行何时偏离理想路径以及如何恢复任务可行性的宝贵证据。然而，现有的奖励和值模型通常依赖于单调进展假设，这些假设捕捉粗略的任务进展，但可能忽略不完美演示中的局部执行错误和纠正行为。在这项工作中，我们提出了ReTVL（重试监督的值学习），一个通过利用重试事件作为稀疏监督从混合质量机器人演示中学习对错误敏感的值函数的框架。ReTVL通过结合全局进展校准和由稀疏标注的重试关键点诱导的局部成对偏好学习，捕捉错误周围的局部退化与恢复结构。学习到的值模型随后用于重新加权演示片段以进行下游行为克隆，减少有害执行错误的影响，同时保留有用的纠正行为。在真实机器人操作任务上的实验表明，ReTVL比基于进展的基线产生更细粒度的值估计，并改进了从不完美演示中的模仿学习。

## Abstract
Human demonstrations for robot imitation learning often contain mistakes and corrective behaviors, such as imprecise grasps, object misalignment, unstable contact, and repeated attempts. While these segments are commonly treated as noisy or suboptimal data, they provide valuable evidence about when execution deviates from a desirable path and how task feasibility can be restored. However, existing reward and value models often rely on monotonic progress assumptions, which capture coarse task advancement but may overlook local execution errors and corrective behaviors in imperfect demonstrations. In this work, we propose ReTVL (ReTry-Supervised Value Learning), a framework for learning mistake-sensitive value functions from mixed-quality robot demonstrations by leveraging retry events as sparse supervision. ReTVL captures the local degradation-and-recovery structure around mistakes by combining global progress calibration with local pairwise preference learning induced by sparsely annotated retry keypoints. The learned value model is then used to reweight demonstration chunks for downstream behavior cloning, reducing the influence of harmful execution errors while preserving useful corrective behaviors. Experiments on real-robot manipulation tasks show that ReTVL produces more fine-grained value estimates than progress-based baselines and improves imitation learning from imperfect demonstrations.

### 方法论

#### 核心思想
- 利用**稀疏标注的重试关键点**（retry keypoint）构造时序偏好对：重试附近的值应形成“先降后升”的凹形曲线。
- 结合**全局进度校准**（绝对进度损失）与**局部偏好学习**（重试驱动的成对排序损失），训练一个对**错误敏感**的价值函数 $V_\theta(h_t, \ell)$。
- 学习到的价值用于**重加权**下游行为克隆中的动作片段，抑制有害片段，保留有用纠正行为。

#### 关键技术细节

1. **价值模型结构**  
   - 基于 VLM 骨干（如 Robometer‑4B） + 离散值头，输出分布化价值 $V_\theta = \sum b_k p_\theta(k|h_t,\ell)$，增强稳定性。

2. **全局进度监督**  
   - 对非重试区域的帧施加绝对进度损失 $\mathcal{L}_{\text{abs}}$，使用截止帧的目标进度 $v^*_t = t/T$（成功轨迹）或 $v^*_T=0$（失败轨迹），通过交叉熵训练离散值头。

3. **重试驱动的局部偏好学习**  
   - **区域划分**：对每个重试关键点 $r_{i,j}$，定义三个窗口：
     - $W_{\text{pre}}$（退化段）、$W_{\text{near}}$（最坏段）、$W_{\text{post}}$（恢复段）。
   - **偏好构造**：$W_{\text{pre}}$ 和 $W_{\text{post}}$ 中的状态值应高于 $W_{\text{near}}$（即 $V_{\text{pre}} > V_{\text{near}}$, $V_{\text{post}} > V_{\text{near}}$），且随靠近关键点值递减。
   - **软窗口加权**：对每个偏好对 $(h^+, h^-)$，按锚点距重关键点的距离 $\exp(-d/\tau_w)$ 加权，减少边界模糊噪声。
   - **偏好损失**：  
     $$ \mathcal{L}_{\text{pref}} = -\mathbb{E}[w \log \sigma((V_\theta(h^+)-V_\theta(h^-))/T_{\text{pref}})] $$  
     强制局部值排序，可同时构造“pre vs near”“near vs post”“pre vs pre”“post vs post”多种对。

4. **联合训练**  
   - 总损失 $\mathcal{L}_{\text{value}} = \lambda_{\text{abs}} \mathcal{L}_{\text{abs}} + \lambda_{\text{pref}} \mathcal{L}_{\text{pref}}$，用固定比例 $\rho_{\text{pref}}$ 混合。

5. **价值引导行为克隆（Value‑Weighted BC）**  
   - 对动作片段 $t$，计算预测进度提升 $r_t = V_\theta(h_{t+\Delta_a}) - V_\theta(h_t)$，然后归一化到 $[0,1]$ 权重 $\alpha_t = \text{clip}((r_t - (\mu-2\sigma))/(4\sigma+\epsilon), 0, 1)$。
   - 加权的 BC 目标：$\mathcal{L}_{\text{wBC}} = \sum_t \alpha_t \ell_{\text{BC}} / (\sum_t \alpha_t + \epsilon)$，自动降低有害片段权重。

### 实验设计

#### 数据集 / 场景
- **任务**：4 个真实机器人操作任务（Piper 机械臂）

  - Stack Blocks（长时序堆叠）

- **训练数据**：每个任务 30 条标注轨迹（含成功、失败、重试）用于价值模型训练；另外 200 条轨迹（Pick up Spoon 为 80 条）用于下游策略学习。对象位置随机化。

- **Robometer**：在大规模机器人数据上训练的通用奖励模型，使用轨迹回退和偏好排序。
- **RECAP‑Value**：实现 RECAP 中的值组件，使用蒙特卡洛步进目标学习进度值函数。

- **价值评估**：20 条保留轨迹，分三个全局指标（VOC、Success/Failure Detection）和四个局部指标（Drop AUC、Drop Probability、Pre>Retry、Post>Retry）。

### 主要结论与发现

1. **价值评估**：ReTVL 在全局指标上表现与最强基线相当（S/F Detection 1.000），在局部错误敏感指标上大幅超越所有基线（如 Pre>Retry 0.740 vs 基线最高 0.329）。证明重试监督可以产生精细的局部退化‑恢复感知。
2. **策略学习**：ReTVL‑BC 平均成功率 80% vs Standard BC 41% vs RECAP‑BC 63%，在四个任务上均最优。原因在于其有效抑制有害动作片段（坏动作权重仅 0.11，RECAP‑BC 为 0.54），同时保留有用纠正片段（Post‑retry Weight 0.636，最高）。
3. **消融**：偏好损失是捕捉局部错误的关键；软窗口稳定全局‑局部平衡；绝对校准提供全局尺度。三者缺一不可。
4. **标注成本**：稀疏重试关键点标注非常高效（每轨迹 < 1 分钟），且人工一致性好（0.19 秒误差），说明方法具有实用潜力。

### 优点

1. **新颖的监督信号**：首次将“重试事件”作为有结构的稀疏监督利用，而非视为噪声，揭示了局部退化‑恢复的价值模式。
2. **灵活的组合框架**：集成了全局进度校准（利用来自广泛数据集的粗糙标签）与局部偏好学习（利用稀疏标注），兼顾粗糙与精细估计。
3. **软窗口设计**：减轻了硬窗口边界带来的噪声，使偏好损失对标注误差更鲁棒。
4. **实用导向的下游应用**：用学习到的价值直接加权 BC，避免了复杂任务奖励设计，且显著提升混合质量演示下的策略性能。
