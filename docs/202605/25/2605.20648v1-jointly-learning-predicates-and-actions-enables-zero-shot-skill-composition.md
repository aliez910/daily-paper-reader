## 摘要
从演示中学习（LfD）使机器人能够从专家示例中学习复杂行为，然而现有方法往往无法泛化到已知技能的新组合，除非重新训练。现代生成式策略仅对动作轨迹进行分布建模，因此无法推理出稳健组合所需的符号化结果。我们提出技能应同时建模动作轨迹及其引发的符号化结果。为了解决这一差距，我们引入了谓词动作技能（PACTS），这是一类闭环视觉运动策略，将技能建模为动作和谓词信念轨迹上的联合生成过程，在单一模型中生成连贯的动作-结果展开。联合生成动作和谓词使PACTS能够学习改进动作生成和谓词分类的内部表示。此外，我们通过利用PACTS的在线谓词预测作为排序和监控执行的符号化接口，展示了通过规划实现已学技能的零样本组合。项目网站：https://planpacts.github.io/

## Abstract
Learning from Demonstration (LfD) enables robots to learn complex behaviors from expert examples, yet existing approaches often fail to generalize to new compositions of known skills without retraining. Modern generative policies model distributions over action trajectories alone, thus are unable to reason about the symbolic outcomes required for robust composition. We propose that skills should jointly model action trajectories and the symbolic outcomes they induce. To address this gap, we introduce Predicate Action Skills (PACTS), a class of closed-loop visuomotor policies that model skills as a joint generative process over action and predicate belief trajectories, producing coherent action-outcome rollouts within a single model. Jointly generating actions and predicates enables PACTS to learn internal representations that improve both action generation and predicate classification. Furthermore, we demonstrate zero-shot composition of learned skills via planning by leveraging online predicate predictions from PACTS as a symbolic interface for sequencing and monitoring execution. Project website: https://planpacts.github.io/

## 2. 方法论

### 核心思想
- 将技能建模为 **联合生成过程**：给定当前观测 \(o_t\)，学习动作轨迹 \(\mathbf{x}\) 和谓词信念轨迹 \(\mathbf{z}\) 的耦合分布 \(p(\mathbf{x}, \mathbf{z} \mid o_t)\)。
- 定义联合变量 \(\mathbf{y} = [\mathbf{x} \mid \mathbf{z}] \in \mathbb{R}^{H \times (d_a + J)}\)，其中 \(H\) 为规划长度，\(d_a\) 为动作维度，\(J\) 为谓词数量。
- 通过单一条件生成模型 \(p_\theta(\mathbf{y} \mid o_t)\) 采样得到一致的动作-结果展开（action-outcome rollout）。

### 关键技术细节
- **谓词信念轨迹**：\(\mathbf{z} \in [0,1]^{H \times J}\) 表示对每个谓词在未来时刻成立的置信度。训练时使用二进制标签（归一化至 \([-1,1]\)），推理时通过阈值（如 \(0.5\)）获得离散谓词状态。
- **生成框架**：支持 **DDPM**（去噪扩散概率模型）和 **条件流匹配（CFM）** 两种目标。
    - **DDPM**：训练噪声预测网络 \(\epsilon_\theta(o_t, \mathbf{y}_k, k)\)，最小化 MSE 损失。
    - **CFM**：训练向量场 \(v_\theta(o_t, \mathbf{y}_\tau, \tau)\)，回归常数速度 \(\mathbf{y}_1 - \mathbf{y}_0\)，通过积分从噪声到数据。
- **架构**：由观测编码器（如视觉编码器）和条件轨迹生成器（**UNet** 或 **Transformer**）组成。生成器接收观测嵌入和时间步条件。
- **推理与执行**：采用滚动时域控制（receding horizon），每次采样长度 \(H\) 的动作块，执行前几步后重新观测并重新采样。

### 规划组合算法（Algorithm 1）
1. 维护当前预测谓词状态 \(\hat{p}_t\)（从信念轨迹阈值化得到）。
2. 调用离线规划器（如PDDL求解器）根据当前状态与目标生成符号计划 \(\pi\)。
3. 取出下一个技能 \(s\)，执行滚动：
    - 采样联合展开 \((\mathbf{x}, \mathbf{z}) \sim p_\theta(\mathbf{x}, \mathbf{z} \mid o_t, s)\)。
    - 执行动作前缀，更新观测。
    - 用 \(\mathbf{z}_{t+1}\) 更新预测状态。
4. 检查技能效果是否满足，若未满足或发生意外（如扰动），触发重规划。
