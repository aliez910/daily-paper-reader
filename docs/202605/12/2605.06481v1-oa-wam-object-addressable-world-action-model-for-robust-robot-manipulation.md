## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记（tokens）或全局潜变量。当指令涉及特定对象时，动作解码器很难寻址这些表示，特别是在场景偏移导致对象身份与上下文纠缠的情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并以块因果序列的形式与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配（flow-matching）动作头在同一次前向传播中解码 16 步连续动作块。通过仅地址键（address-only keys）路由跨槽位注意力并在每个 Transformer 层重置地址切片，强制实现了可寻址性，从而在不增加额外标记的情况下，将“操作哪个对象”与“该对象当前的状态”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试得出的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

#### 1. 核心问题与整体含义（研究动机和背景）
*   **研究动机**：现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）通常将场景表示为整体图像、视频标记或全局潜变量。这种“整体性”表示在面对场景偏移（如相机视角变化、物体布局改变）时，容易将目标物体的身份与周围环境上下文纠缠在一起，导致动作解码器无法准确定位指令中的特定物体，从而造成策略失效。
*   **核心问题**：如何赋予模型“对象可寻址性”（Object Addressability），即在复杂的场景扰动下，依然能稳定地根据语言指令锁定目标物体，并将“操作哪个物体”与“该物体当前的状态”在架构层面进行解耦。

#### 2. 方法论
*   **核心思想**：将每一帧分解为 $N+1$ 个对象槽位（Slot），并引入一种架构约束，使得跨槽位的注意力机制仅通过物体的“身份地址”进行路由，而不受其时变状态的影响。
*   **关键技术细节**：
    *   **槽位分解**：每个槽位向量由两部分组成：**地址向量 ($addr_k$)**（由语言标签和初始特征计算，每集固定，代表身份）和**内容向量 ($cnt_t^k$)**（每帧更新，代表姿态和外观）。
    *   **对象可寻址注意力**：在 Transformer 的 32 个层中，跨槽位注意力的“键”（Key）投影仅读取 $addr$ 子向量。
    *   **残差重置（Residual Reset）**：在每一层之后，强制将残差流中的地址切片重置为初始地址，防止身份信息在深层网络中发生漂移。
    *   **多头架构**：
        *   **世界头（World Head）**：回归预测下一帧各槽位的内容和姿态。
        *   **动作头（Action Head）**：采用流匹配（Flow-matching）解码 16 步连续动作块。
    *   **统一序列**：将文本、图像 VQ、槽位、本体感受和过去动作标记组合成块因果序列，输入 7B 参数的 Chameleon 风格骨干网络。

#### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准操作基准（Spatial, Object, Goal, Long 四个套件）。
    *   **SimplerEnv**：用于评估模拟到真实的视觉匹配（WidowX/Bridge 任务）。
    *   **LIBERO-Plus**：鲁棒性压力测试，包含相机、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动轴。
*   **对比方法**：
    *   **VLA 基准**：OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1, X-VLA 等。
    *   **WAM 基准**：VLA-JEPA, Cosmos-Policy, WorldVLA, GE-Act 等。
*   **诊断测试**：因果槽位干预测试（交换两个物体的地址，观察机器人轨迹是否随之偏转）。

#### 4. 资源与算力
*   **阶段 0（预训练）**：使用 384 张 NVIDIA A100-80GB 显卡，训练约 18 天（总计约 16.6 万 A100 小时）。
*   **阶段 I/II（微调）**：在 8 张 A100-80GB 上进行，每个阶段耗时 3-4 天。
*   **推理延迟**：单张 A100 上感知耗时约 138ms，骨干网络+动作头耗时约 90ms，总闭环频率约 4.3Hz。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了三大主流基准测试，并在 LIBERO-Plus 的 7 个维度上进行了详尽的零样本迁移评估。
*   **消融实验**：设计了 A1（可寻址约束隔离）、A2（因果干预）、A3（世界头作用）、A4（一致性损失）等多组实验。
*   **充分性评价**：实验设计非常充分且具有针对性。特别是通过“地址交换”这种因果干预手段，客观地证明了模型确实是在利用地址空间进行目标锁定，而非简单的模式记忆。

#### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超过了现有 SOTA 水平。
*   **几何鲁棒性**：在 LIBERO-Plus 的几何扰动轴（相机、机器人状态、布局）上表现尤为突出，几何平均分比 $\pi_{0.5}$ 高出 4.8%。
*   **强绑定能力**：在地址交换干预中，OA-WAM 的绑定余弦值达到 0.87，而所有整体性基准模型均低于 0.09，证明了其架构确实实现了对象寻址。
*   **解耦价值**：消融实验显示，关闭地址约束会导致几何鲁棒性大幅下降，而对分布内任务影响较小，说明该设计是 OOD（分布外）泛化的关键。

#### 7. 优点
*   **架构创新**：通过简单的张量掩码和重置操作实现了对象寻址，无需引入复杂的额外标记或端到端语义保证。
*   **可验证性**：提供了一种量化评估模型是否真正“理解”指令目标的方法（Swap-binding test）。
*   **鲁棒性强**：在不牺牲标准任务性能的前提下，显著提升了对空间几何变化的抵抗力。

#### 8. 不足与局限
*   **感知瓶颈**：高度依赖上游感知系统（SAM3, DINOv3）。在传感器噪声轴上表现较差（-17.1%），主要是因为图像损坏导致槽位提取失败，而非策略本身问题。
*   **计算开销**：感知阶段（尤其是多物体分割和特征提取）耗时较长（~138ms），限制了更高频率的实时控制。
*   **假设限制**：目前的干扰项一致性损失假设目标与干扰项之间是弱耦合的，在极度拥挤或物体间有复杂物理依赖的场景下可能受限。
*   **环境局限**：目前仅在模拟器中验证，尚未展示在真实机器人上的部署效果。
