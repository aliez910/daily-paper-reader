## 摘要
视觉-语言-动作模型在通用机器人操作方面展现出巨大潜力，但受限于空间推理不足，尤其是在复杂视觉场景中确定交互位置方面。虽然近期工作引入了多种形式的视觉规划来解决此问题，但现有方法要么依赖全局几何线索、符号中间表征，要么依赖外部生成的视觉信号，这些信号通常与下游动作预测耦合较弱。在这项工作中，我们重新审视VLA系统中的视觉规划，并认为有效的规划应当是局部的、基于视觉的、内部生成的，并与动作直接对齐。基于此洞察，我们提出Afford-VLA，这是一个统一框架，将任务条件化的可供性内化为VLA模型内的显式视觉规划接口。具体而言，我们引入可学习的<AFF>令牌来查询任务相关的交互区域，从多模态特征中解码可供性掩码，并将其转换为紧凑嵌入，直接条件化动作生成。这种设计使得可供性能够在VLA内部生成并利用，形成紧密耦合的感知-动作通路。为进一步支持这种集成，我们采用了训练策略，使得可供性通路能够与动作预测联合优化，提高其对下游控制的有效性。我们在多个仿真基准上评估了方法，包括LIBERO、LIBERO-Plus和SimplerEnv，取得了持续的最先进性能，并在真实世界中取得了强劲结果。这些发现表明，将可供性内化为与动作对齐的视觉规划为改进VLA系统提供了强大范式。

## Abstract
Vision-language-action (VLA) models have shown strong potential for generalist robot manipulation, yet they remain limited by insufficient spatial reasoning, particularly in determining where to interact in complex visual scenes. While recent efforts introduce various forms of visual planning to address this issue, existing approaches either rely on global geometric cues, symbolic intermediate representations, or externally generated visual signals, which are often weakly coupled with downstream action prediction. In this work, we revisit visual planning in VLA systems and argue that effective planning should be local, visually grounded, internally generated, and directly aligned with action. Based on this insight, we propose Afford-VLA, a unified framework that internalizes task-conditioned affordance as an explicit visual planning interface within VLA models. Concretely, we introduce learnable <AFF> tokens to query task-relevant interaction regions, decode affordance masks from multimodal features, and convert them into compact embeddings that directly condition action generation. This design enables affordance to be both generated and utilized within the VLA, forming a tightly coupled perception-action pathway. To further support this integration, we adopt a training strategy that allows the affordance pathway to be jointly optimized with action prediction, improving its effectiveness for downstream control. We evaluate our method on multiple simulation benchmarks, including LIBERO, LIBERO-Plus, and SimplerEnv, achieving consistent state-of-the-art performance, along with strong real-world results. These findings demonstrate that internalizing affordance as action-aligned visual planning provides a powerful paradigm for improving VLA systems.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有视觉-语言-动作（VLA）模型虽在通用机器人操作上取得进展，但空间推理能力不足——无法在复杂视觉场景中精准确定“在哪里交互”。这限制了精细操作任务的成功率。
- **现有方法局限**：当前视觉规划方法可大致分为三类：
  - **基于几何**：依赖3D线索提供全局场景级信息，缺乏局部任务导向的交互区域。
  - **基于符号**：使用文本或令牌抽象表达空间引导，与视觉空间间接关联。
  - **基于视觉**：明确生成点、框、掩码等视觉线索，但通常依赖外部感知模块或作为独立监督目标，与动作预测耦合弱。
- **论文主张**：有效的视觉规划应同时满足四个性质：**局部性**（聚焦任务相关区域）、**视觉基础性**（直接锚定在视觉证据上）、**内部生成性**（由VLA模型自身学习而非外部模块）、**动作对齐性**（可直接用于动作模型的下游决策）。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将**任务条件化的可供性（affordance）** 内化为VLA模型内部的显式视觉规划接口，形成紧密耦合的感知-动作通路。
- **关键技术细节**：
  1. **内部可供性掩码生成**：
     - 在VLM输入序列中增加可学习 `<AFF>` 查询令牌（每视图4个）。
     - `<AFF>` 令牌与图像、语言令牌一起通过VLM自注意力层，最终状态 `A_t` 蕴含视觉-语言上下文。
     - 通过轻量级**可供性头（affordance head）**（双向注意力解码器，2层8头，隐藏维度256）将 `A_t` 与视觉补丁特征 `P_t` 结合，生成补丁级可供性logits `G_t ∈ R^{H_p×W_p}`。
  2. **可供性条件化的动作预测**：
     - 对 `G_t` 执行**硬Top-K补丁池化**（k=16），选择最高logits的补丁，将其视觉特征平均后投影到VLM隐藏维度，得到**可供性嵌入（affordance embedding）`r_t`**。
     - 将 `r_t` 拼接到原始VLM隐藏状态 `H_t` 后，输入**流匹配动作头（flow-matching action head，DiT-B架构）** 预测动作块（8步7自由度关节位置增量）。
  3. **动作对齐训练策略**：
     - **两阶段训练**：
       - 阶段一（预热4K步）：冻结VLM和动作头，仅用密集掩码监督训练可供性头。
       - 阶段二（140K步）：使用预测的可供性logits进行Top-K池化，联合优化动作损失和可供性损失。
     - **直通梯度估计器（Straight-Through Estimator）**：在前向传播中保持硬Top-K选择，反向传播时用softmax代理梯度，使动作损失可以反向更新可供性logits和可供性头。
     - 总损失：`L_joint = L_act + L_aff`，其中 `L_aff` 为二值交叉熵，`L_act` 为流匹配目标。
  4. **多视图扩展**：对每个视图独立生成可供性和嵌入，将所有视图嵌入与隐藏状态拼接后送入动作头。

### 7. 优点

- **方法创新性**：系统提出了视觉规划的四个必要性质，并设计了内部化可供性的完整框架，首次将可供性作为VLA内部的、动作对齐的显式规划接口。
- **端到端梯度流动**：直通估计器使动作信号直接反馈到可供性预测，实现感知-动作紧密耦合，而非仅靠独立监督。
- **简洁高效**：仅增加少量<AFF>令牌和轻量解码器，计算开销较小（总训练时间约20~48小时）。
- **泛化能力强**：零样本鲁棒性测试表现突出，尤其是布局和光照变化。
- **实验充分**：覆盖多个仿真基准、消融实验、真实世界测试，且公开代码和模型（承诺），可复现性强。

### 8. 不足与局限

- **依赖外部可供性标签**：训练所需的可供性掩码由外部模型（RAGNet）生成，其质量可能影响性能，且引入数据偏差风险。
- **仅2D视觉可供性**：未利用3D几何信息，在需要深度推理的任务（如叠方块）中改进有限（SimplerEnv叠方块仅14.6%）。
- **场景与物体多样性有限**：实世界实验仅4种任务，机器人型号单一（ARX X5），环境受控；结论向更复杂、开放场景的外推需谨慎。
- **动作头设计依赖流匹配**：虽然与GR00T一致，但不同动作头架构可能影响对可供性嵌入的利用效率，未做交叉验证。
- **安全风险**：如论文所述，部署在非结构化环境时可能因可供性预测错误导致不安全行为，当前未在真实动态场景中测试，需进一步考虑安全约束。
