## 摘要
机器人模仿学习通常被视为复现演示动作，但动作本质上具有本体特异性。当演示来自形态、运动学或动作空间不同的机器人或人类时，这种以动作为中心的观点需要共享动作空间、启发式重定向或大规模多本体联合训练。相反，我们将演示视为未来目标的隐式规范：目标智能体应推断演示者试图实现的状态，而非演示者的执行方式。我们提出Demo-JEPA，一种解耦演示意图与本体特定执行过程的跨本体模仿框架。基于JEPA世界模型，Demo-JEPA将源视觉演示转换到共享预测表示空间中的目标兼容未来潜在轨迹。然后目标智能体将这些潜在轨迹作为子目标，通过在其自身学习的前向动力学下规划来实现。由于Demo-JEPA避免了动作级对应，且仅需视觉演示加目标智能体自身的交互经验，因此支持异构本体间的灵活模仿。在RLBench和真实世界操作任务上的实验表明，Demo-JEPA匹配专用领域规划器，并能泛化到先前方法失败的未见任务和本体配置。

## Abstract
Robotic imitation learning is often treated as reproducing demonstrated actions, but actions are inherently embodiment-specific. When demonstrations come from humans or robots with different morphology, kinematics, or action spaces, this action-centric view requires shared action spaces, heuristic retargeting, or large-scale multi-embodiment co-training. We instead view demonstrations as implicit specifications of future goals: the target agent should infer what state the demonstrator is trying to realize, rather than how the demonstrator executes it. We propose Demo-JEPA, a cross-embodiment imitation framework that decouples demonstration intent from embodiment-specific execution. Built on a JEPA-based world model, Demo-JEPA translates source visual demonstrations into target-compatible future latent trajectories in a shared predictive representation space. The target agent then uses these latent trajectories as subgoals and realizes them through planning under its own learned forward dynamics. Because Demo-JEPA avoids action-level correspondence and requires only visual demonstrations plus the target agent's own interaction experience, it supports flexible imitation across heterogeneous embodiments. Experiments on RLBench and real-world manipulation tasks show that Demo-JEPA matches specialized in-domain planners and generalizes to unseen tasks and embodiment configurations where prior methods fail.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：机器人模仿学习中存在严重的“本体鸿沟”——演示者（如人类、不同型号的机器人）与学习者之间的形态、运动学、动作空间存在差异。传统的以动作为中心的模仿方法（如动作重定向、大规模多本体联合训练）需要明确的动作对应或共享动作空间，难以泛化到异构本体。
- **整体含义**：作者提出将演示视为对未来目标的隐式规范，而非动作序列的复制。目标智能体应推断演示者的意图（即期望达到的未来状态），而非复现其执行方式。通过解耦“意图”与“执行”，实现无需动作级对应的跨本体模仿。
- **背景**：现有方法如视觉-语言-动作模型（VLA）依赖大规模多本体数据；跨本体模仿学习（如 XSkill、VPP）在分布偏移较大时性能下降明显。Demo-JEPA 旨在解决强分布偏移下的泛化问题。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
### 核心思想
- 将跨本体模仿转化为**潜在目标条件规划**：在共享的 JEPA 潜在预测空间中，从源演示中推断与目标本体兼容的未来潜在状态（子目标），然后利用目标本体自身学习的动作条件世界模型和交叉熵方法（CEM）进行规划，生成动作序列。
- **关键设计**：使用 JEPA 潜在空间而非像素空间，因为 JEPA 优化了抽象的预测性世界结构，能抑制本体特定的外观噪声，保留高层的任务意图。

### 关键技术细节
1. **动作条件世界模型**（V-JEPA 2.1）：编码器 $E$ 将观测 $o$ 映射到潜在状态 $z$；动力学预测器 $F_{wm}$ 根据当前潜在状态、机器人状态和动作预测下一步潜在状态：$\hat{z}_{k+1} = F_{wm}(z_k, s_k, a_k)$。
2. **Dreamer Predictor（梦想预测器）**：核心模块，用于跨本体未来状态翻译。输入当前目标观测 $o_t^k$ 和源演示帧对 $(o_s^k, o_s^{k+n})$，通过共享编码器得到潜在表示。设计两个交叉注意力模块：
   - $f_{emb} = Attn(Q=z_t^k, K=z_s^k, V=z_s^k)$：捕获源-目标之间的本体对应关系。
   - $f_{mot} = Attn(Q=z_s^{k+n}, K=z_s^k, V=z_s^k)$：捕获演示中的时间演变。
   - 然后通过 3D 卷积融合 $f_{fused} = \phi(z_t^k \oplus f_{emb} \oplus f_{mot})$，最后由 Transformer 解码器输出潜在目标 $\hat{z}_{goal}$。
3. **训练流程三阶段**：
   - **Stage 0**：预训练目标本体的动作条件世界模型（使用目标本体交互数据）。
   - **Stage 1（潜在目标预测器训练）**：固定世界模型，训练 Dreamer Predictor。损失函数为预测潜在目标与真实未来潜在状态之间的 L2 距离：$\mathcal{L}_{pred} = \|\hat{z}_{goal} - z_{k+n}^T\|^2$。引入时间扰动正则化（随机偏移源帧时间戳）增强鲁棒性。
   - **Stage 2（动作联合训练）**：冻结 Dreamer Predictor，微调世界模型的动力学预测器 $F_{wm}$。损失函数为世界模型根据动作序列预测的潜在状态与 Dreamer Predictor 输出的潜在目标之间的 L2 距离：$\mathcal{L}_{plan} = \|F_{wm}(z_k^T, s_k, a) - \hat{z}_{goal}\|^2$。
4. **推理流程**：采用自适应目标更新机制。每一步由 Dreamer Predictor 根据当前目标观测和源参考段推断潜在目标，然后通过 CEM 优化动作序列使得世界模型滚动输出匹配该目标。执行动作后，计算当前观测潜在状态与目标的差异，若小于阈值则前进到下一个源参考段，否则维持当前目标重新规划。

### 公式与算法（文字说明）
- 目标函数：动作序列通过最小化世界模型预测潜在状态与潜在目标之间的 L1/L2 距离得到，采用 CEM 迭代优化。
- 推理算法（Algorithm 3）涉及循环：源参考索引 i 从1开始，每步计算潜在目标 → CEM 优化动作 → 执行一步 → 检查差异 → 决定是否前进 i。

## 6. 论文主要结论与发现
- Demo-JEPA 在**分布偏移较大时表现显著优于** VPP 和 XSkill。仿真中 Cross-Embodiment Bridging 平均成功率达 0.45（VPP 0.28, XSkill 0.17），Zero-Shot Generalization 平均 0.36（VPP 0.04, XSkill 0.03）。
- 真实世界中同理，Demo-JEPA 在 Zero-Shot 任务上达到 0.25 成功率，而 VPP 和 XSkill 接近 0。
- **关键发现**：将跨本体演示解释为潜在目标是更有效的策略，而不是固定动作序列或技能原型。
- **缩放实验**：任务多样性比每任务样本量更重要。数据缩放（增加每任务样本）在较低比例时优于任务缩放，但两者均受益于更多数据。
- **Demo-DP 扩展**：将 Dreamer Predictor 与扩散策略结合，在 Behavior Grounding 上优于原始 Demo-JEPA，但在零样本泛化上不如基于规划的执行，说明在强分布偏移下规划更具鲁棒性。
- Conv3D 融合在复杂运动（如扭转、铰接）中显著优于简单的平均池化，在简易任务上差异不大。
