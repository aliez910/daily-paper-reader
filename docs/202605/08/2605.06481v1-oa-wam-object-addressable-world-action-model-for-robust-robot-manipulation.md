## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记（tokens）或全局潜变量。当指令涉及特定对象时，动作解码器很难寻址这些表示，特别是在场景偏移的情况下，对象的身份往往与上下文纠缠在一起。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并与文本、图像、本体感受和过去动作标记在块因果序列中融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码一个 16 步的连续动作块。通过仅通过地址键路由跨槽注意力，并在每个 Transformer 层重置地址切片，强制实现可寻址性，从而在不添加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试显示交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

#### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）在标准基准测试中表现优异，但在面对场景扰动（如相机视角切换、物体布局改变、背景变化）时表现出极差的鲁棒性。
*   **核心痛点**：大多数模型将场景表示为整体图像标记或全局潜在变量，导致目标物体的身份（Identity）与周围上下文（Context）发生纠缠。当指令要求操作特定物体时，模型缺乏一种稳定的“寻址”机制来定位目标，导致动作生成随环境变化而漂移。
*   **研究目标**：提出一种“对象可寻址”（Object-Addressable）的架构，通过在张量层面分离“操作哪个物体”和“物体当前状态”，提升机器人操纵的鲁棒性。

#### 2. 方法论：核心思想与关键技术
OA-WAM 的核心是将世界建模任务分解为独立且可寻址的物体槽位（Slots）。
*   **物体槽位分解**：每一帧被分解为 $N+1$ 个槽位（1个机器人槽位，$N$个物体槽位）。每个槽位向量由两部分组成：
    *   **地址向量 ($addr_k$)**：基于语言标签和初始特征计算，在整个任务过程中保持冻结，代表物体身份。
    *   **内容向量 ($content_{tk}$)**：随时间变化，捕捉物体的姿态、外观等动态状态。
*   **对象可寻址（OA）约束**：
    *   **地址专用键投影**：在 Transformer 的注意力机制中，跨槽位注意力的“键”（Key）投影仅读取 $addr$ 部分，确保路由逻辑仅取决于物体身份。
    *   **残差重置**：在每一层 Transformer 之后，强制将残差流中的地址部分重置为初始值，防止身份信息在深层网络中被内容信息污染。
*   **模型架构**：采用 7B 参数的 Chameleon 风格骨干网络，输入包含文本、图像 VQ、物体槽位、本体感受和历史动作。
*   **双头输出**：
    *   **世界头（World Head）**：回归预测下一帧各槽位的内容和姿态。
    *   **动作头（Action Head）**：利用流匹配（Flow-matching）在单次前向传播中解码 16 步连续动作块。

#### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：包含 Spatial、Object、Goal、Long 四个子集，测试分布内性能。
    *   **SimplerEnv (WidowX/Bridge)**：测试模拟到真实的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性基准，包含相机、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动维度。
*   **对比方法**：包括 SOTA VLA 模型（OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1）和 WAM 模型（VLA-JEPA, Cosmos-Policy, WorldVLA 等）。
*   **特殊测试**：**因果槽位干预测试**。通过在推理时交换两个物体的地址向量，观察机器人是否会转向错误的物体，以此验证模型是否真正实现了对象绑定。

#### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 达到 SOTA 水平。
*   **几何鲁棒性极强**：在 LIBERO-Plus 的几何轴（相机、机器人状态、布局）上，OA-WAM 显著超过 $\pi_{0.5}$ 等强基准，几何平均成功率提升了 4.8%。

#### 7. 优点
*   **架构创新**：通过简单的张量掩码和重置机制，在不增加额外 token 的情况下实现了物体层面的解耦。
*   **可解释性与可验证性**：通过地址交换实验，提供了一种量化评估 VLA 模型“是否听懂指令并找对物体”的有效手段。
*   **鲁棒性提升**：成功解决了 VLA 模型在场景几何变换下容易崩溃的顽疾。

*   **仿真局限**：所有实验均在仿真环境中完成，尚未在真实机器人上进行大规模部署验证。
