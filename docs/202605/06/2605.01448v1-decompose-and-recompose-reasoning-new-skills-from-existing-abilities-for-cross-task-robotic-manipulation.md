## 摘要
跨任务泛化是开放世界机器人操纵中的核心挑战，其关键在于从已知任务中提取可迁移的操纵知识。最近的上下文学习（in-context learning）方法利用已知任务的演示，在不更新参数的情况下为未知任务生成动作。然而，现有方法仅提供低级连续动作序列作为上下文，未能捕获可组合的技能知识，导致模型退化为表层的轨迹模仿。我们提出了“分解与重组”（Decompose and Recompose），这是一个使用原子技能-动作对作为中间表示的技能推理框架。我们的方法将已知演示分解为可解释的技能-动作对齐，使模型能够通过组合推理为未知任务重组这些技能。具体而言，我们通过视觉语义检索结合规划智能体的技能序列构建了一个任务自适应的动态演示库，并辅以覆盖感知的静态库以填补缺失的技能模式。这些共同产生了技能全面的演示，显式地激发了关于技能组合和执行顺序的组合推理。在 AGNOSTOS 基准测试和真实环境中的实验验证了我们方法在零样本跨任务泛化方面的能力。

## Abstract
Cross-task generalization is a core challenge in open-world robotic manipulation, and the key lies in extracting transferable manipulation knowledge from seen tasks. Recent in-context learning approaches leverage seen task demonstrations to generate actions for unseen tasks without parameter updates. However, existing methods provide only low-level continuous action sequences as context, failing to capture composable skill knowledge and causing models to degenerate into superficial trajectory imitation. We propose Decompose and Recompose, a skill reasoning framework using atomic skill-action pairs as intermediate representations. Our approach decomposes seen demonstrations into interpretable skill--action alignments, enabling the model to recompose these skills for unseen tasks through compositional reasoning. Specifically, we construct a task-adaptive dynamic demonstration library via visual-semantic retrieval combined with skill sequences from a planning agent, complemented by a coverage-aware static library to fill missing skill patterns. Together, these yield skill-comprehensive demonstrations that explicitly elicit compositional reasoning for skill composition and execution ordering. Experiments on the AGNOSTOS benchmark and real-world environments validate our method's zero-shot cross-task generalization capability.

#### 1. 论文的核心问题与整体含义
论文聚焦于机器人操纵中的**跨任务泛化（Cross-Task Generalization）**难题。在开放世界中，机器人经常面临从未见过的物体、新目标及动作组合。
*   **研究动机**：现有的视觉-语言-动作（VLA）模型或上下文学习（ICL）方法（如 X-ICM）通常直接将低级数值动作序列作为上下文。这种方式缺乏显式的技能语义，导致大语言模型（LLM）倾向于进行表面的轨迹形状匹配，而非真正的“技能推理”，在面对新技能组合时泛化能力较弱。
*   **核心目标**：通过将复杂的演示分解为可解释、可组合的“原子技能-动作对”，引导 LLM 理解动作背后的逻辑，从而实现从已知能力中推理出完成新任务所需的技能序列。

#### 2. 方法论
论文提出了 **Decompose and Recompose (D&R)** 框架，主要包含以下四个关键步骤：
*   **原子技能集合（Atomic Skill Collection）**：
    *   利用夹爪状态变化和运动停止准则提取演示中的关键帧。
    *   使用视觉语言模型（VLM）结合夹爪约束，将动作段标注为“动词[物体]”格式（如 `Grasp[cube]`），形成技能-动作对齐数据。
*   **动态演示库（Dynamic Demonstrations Library）**：
    *   基于视觉相似度（使用 DINOv3 提取特征）和规划相似度（由规划智能体预测技能序列，并计算 Jaccard 相似度）检索与当前任务最相关的演示。
*   **覆盖感知静态库（Coverage-aware Static Library）**：
    *   为了防止动态检索遗漏关键技能，构建了一个离线静态库。
    *   使用 **IDF（逆文档频率）权重**来衡量技能的稀有度，并结合**长度正则化**，通过贪心算法补充动态库中缺失的技能模式，确保演示集的“技能完备性”。
*   **技能增强上下文学习（Skill-Augmented ICL）**：
    *   将检索到的[指令, 观察, 技能标签, 动作序列]格式化为 Prompt。LLM 在推理时参考技能标签的逻辑，但最终只输出可执行的低级动作。

*   **技能推理的有效性**：显式的技能标签对齐比纯数值动作序列更能激发 LLM 的组合推理能力，尤其在处理复杂技能组合的任务时表现更优。
*   **双库互补性**：动态检索保证了任务相关性，而覆盖感知静态库有效填补了罕见技能的空白，两者结合是提升泛化的关键。

#### 7. 优点
*   **可解释性强**：引入原子技能作为中间表示，使机器人的决策过程（做什么、为什么做）变得透明。
*   **架构灵活**：无需重新训练，能够直接利用现有的预训练视觉模型和 LLM 的强大推理能力。
*   **覆盖全面**：通过 IDF 权重和覆盖感知机制，解决了 ICL 中常见的演示样本分布不均问题。

#### 8. 不足与局限
*   **依赖预定义词汇**：原子技能的提取依赖于预定义的动词集合和 VLM 的标注质量，可能无法覆盖所有极端的操纵动作。

*   **推理延迟**：依赖大型 LLM 进行多轮推理和检索，可能难以满足对实时性要求极高的动态操纵场景。
