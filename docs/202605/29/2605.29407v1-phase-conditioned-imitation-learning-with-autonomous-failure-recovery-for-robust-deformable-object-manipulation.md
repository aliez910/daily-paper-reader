## 摘要
本文提出了一种相位条件、力感知的框架，用于鲁棒的可变形物体操作。标准的模仿学习策略，如基于变换器的动作分块（ACT），在推理时依赖马尔可夫假设，当视觉相似的观测需要矛盾的动作时会导致状态混淆，并阻止从执行故障中自主恢复。我们通过一种闭环分层架构来解决这一问题。一种FiLM条件的ACT编码器根据当前任务阶段调制特征提取，使得单一统一策略能够产生特定阶段的行为，同时在阶段之间共享动作动态。一种融合视觉、力和姿态反馈的多模态相位预测器实时估计相位，检测视觉单独无法看到的接触故障，并自主触发恢复轨迹。该系统由一个用于柔顺执行的混合阻抗控制器和一个用于力感知数据收集的触觉遥操作接口完成。消融研究表明，基于FiLM的调制显著优于无条件和令牌级条件基线，t-SNE分析证实FiLM诱导了分离良好的、特定阶段的特征表示。通过在双臂上挂起和取下T恤进行验证，闭环系统通过自主错误恢复将挂起成功率从56%提高到87%。代码和视频：https://leledeyuan00.github.io/phaser/

## Abstract
This paper presents a phase-conditioned, force-aware framework for robust deformable object manipulation. Standard imitation learning policies such as Action Chunking with Transformers (ACT) rely on a Markovian assumption at inference, causing state aliasing when visually similar observations require contradictory actions and preventing autonomous recovery from execution failures. We address this with a closed-loop hierarchical architecture. A FiLM-conditioned ACT encoder modulates feature extraction based on the current task phase, enabling a single unified policy to produce phase-specific behaviors while sharing action dynamics across phases. A multi-modal phase predictor fusing visual, force, and pose feedback estimates the phase in real time, detecting contact failures that are invisible to vision alone and autonomously triggering recovery trajectories. The system is completed by a hybrid impedance controller for compliant execution and a haptic teleoperation interface for force-aware data collection. Ablation studies show that FiLM-based modulation significantly outperforms both unconditioned and token-level conditioned baselines, and t-SNE analysis confirms that FiLM induces well-separated, phase-specific feature representations. Validated on hanging and removing a T-shirt with dual arms, the closed-loop system improves the hanging success rate from 56\% to 87\% through autonomous error recovery. ## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：在可变形物体操作（如服装操控）中，标准模仿学习策略（如 Action Chunking with Transformers, ACT）在推理时依赖马尔可夫假设（仅根据当前观测预测动作），导致**状态混叠**——视觉上相似的观测可能对应矛盾的动作（例如，成功插入和失败卡住时视觉相似，但需要相反的动作），且无法从执行失败中**自主恢复**。
- **背景**：可变形物体具有无限自由度、严重自遮挡和复杂接触动力学，传统控制方法难以建模。模仿学习虽高效，但在长时程任务中容易因状态混叠产生平均或振荡行为；现有方案如语言条件化（RT-1, DiT）需要大量数据且存在模糊性，阶段条件方法（SCIL）为每个阶段训练独立策略导致扩展性差且无法跨阶段共享。
- **动机**：需要一种能够显式解耦不同阶段、并融合力觉信息实现自主失败恢复的框架，提升在接触丰富任务中的鲁棒性。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将策略分解为**相位条件策略** `πθ(at|ot, p)` 和**多模态相位预测器** `Pϕ(p|ot)`，构成闭环系统。相位作为显式动态先验注入编码器，使同一观测在不同相位下产生不同特征表示；预测器实时监测任务进度并触发恢复相位。
- **关键技术细节**：
  - **中间层：FiLM条件化ACT策略**
    - 采用**特征线性调制（FiLM）**，在每个Transformer编码器层的层归一化和MLP输出上应用仿射变换（缩放和平移），直接调制所有中间表示，避免相位令牌被大量视觉令牌稀释。
    - 输入：4个RGB图像（经共享ResNet-18提取特征）、双臂末端位姿（20维）、力矩（12维）和相位标签嵌入。
    - 编码器输出经CVAE风格变量处理，解码器生成未来50步动作块（每步含双臂位置、6D旋转和夹爪指令）。
    - FiLM仅应用于编码器，解码器保持相位不可知以共享跨阶段低层动态。
  - **顶层：相位预测与失败恢复**
    - 共享已训练策略的冻结图像骨干和位姿/力矩投影层，对各相机特征图进行全局平均池化，加上任务ID嵌入，经MLP输出相位logits（softmax后得到相位概率）。
    - 融合力/力矩反馈以检测视觉不可见的接触故障（如卡住、滑脱），触发对应恢复相位（共设计3种：Phase 8插入卡住→摇动调整、Phase 9严重错位→全局重置、Phase 10半空滑脱→动态抓取）。
    - 相位切换时重置ACT时间集成器，应用1秒低通滤波抑制误判。
  - **底层：混合阻抗控制器**
    - 将策略输出（15 Hz）插值到高频轨迹，并与增量式阻抗控制命令（`xpred - xobs`）融合，根据夹爪状态切换（夹持时偏向力控，张开时保持纯阻抗），实现柔顺执行。
  - **触觉遥操作数据收集系统**
    - 利用Meta Quest Pro控制器映射力传感器数据为振动，使操作者感知接触状态，采集力感知演示（含相位标签）。

- **闭环系统显著提升鲁棒性**：挂起成功率从开环的56%提高到87%（通过自主恢复31/44次接触失败），取下从88%提高到92%（恢复4/8次）。

  - **闭环力感知自主恢复**，无需人类干预即可检测视觉模糊故障并切换恢复策略，显著提升执行时鲁棒性。
