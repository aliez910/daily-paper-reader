## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作策略，但现有方法通常将预测的世界表示为整体图像、视频标记或全局潜变量。当指令涉及特定对象时，动作解码器很难对这些表示进行寻址，特别是在对象身份与上下文纠缠的场景偏移情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并以块因果序列的形式与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码 16 步连续动作块。通过仅地址键路由跨槽位注意力并在每个 Transformer 层重置地址切片，强制实现了可寻址性，从而在不增加额外标记的情况下，将“操作哪个对象”与“该对象当前的状态”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试得出的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

### 1. 核心问题与整体含义（研究动机和背景）
现有的机器人视觉-语言-动作（VLA）策略和世界动作模型（WAM）通常将场景表示为整体图像、视频标记或全局潜在变量。这种“整体性（Holistic）”表示存在一个关键缺陷：**缺乏对象可寻址性（Object Addressability）**。当指令要求操作特定物体时，如果场景发生几何偏移（如相机视角、物体布局或机器人初始位置改变），目标物体的特征往往会与背景或周围上下文纠缠在一起，导致模型无法准确锁定目标，从而造成操作失败。论文旨在通过构建一个在架构上强制实现“对象寻址”的模型，提升机器人在复杂扰动下的鲁棒性。

### 2. 方法论：核心思想与关键技术
OA-WAM 的核心思想是将世界建模为一组**可寻址的对象槽位（Object Slots）**，并在 Transformer 架构层面解耦“操作哪个对象”和“该对象当前状态”。

*   **槽位分解（Slot Decomposition）**：每一帧被分解为 $N+1$ 个槽位（1个机器人槽位，$N$个物体槽位）。每个槽位向量由两部分组成：
    *   `addr_k`：**身份地址**，在回合开始时根据语言标签和初始特征计算，并在整个过程中保持冻结，代表“它是哪个物体”。
    *   `content_tk`：**时变内容**，每帧更新，捕捉物体的姿态、外观和局部上下文。
*   **对象可寻址注意力机制**：
    *   **仅地址键投影（Addr-only Key Projection）**：在 Transformer 的交叉槽位注意力中，Key 投影矩阵仅读取 `addr` 部分，确保注意力路由仅由物体身份决定。
    *   **残差流重置（Residual Reset）**：在每一层 Transformer 之后，强制将残差流中的地址切片重置为初始值，防止身份信息在深层网络中发生漂移。
*   **模型架构**：采用 7B 参数的 Chameleon 风格多模态骨干网络。
    *   **世界头（World Head）**：回归预测下一帧各槽位的内容和姿态。
    *   **动作头（Action Head）**：基于流匹配（Flow-matching）在单次前向传播中解码 16 步连续动作块。
*   **感知前端**：利用 SAM 3 进行分割，DINOv3 提取特征，Qwen3-VL 进行名词短语解析。

### 3. 实验设计
论文在三个主要仿真基准上进行了评估：
*   **LIBERO**：评估分布内（In-distribution）的成功率，包括 Spatial、Object、Goal 和 Long 四个任务套件。
*   **SimplerEnv (WidowX/Bridge)**：评估模拟到真实的视觉匹配能力。
*   **LIBERO-Plus**：核心鲁棒性测试，包含相机视角、机器人初始状态、布局、光照、背景、语言描述和传感器噪声 7 个扰动维度。
*   **对比方法**：包括 SOTA VLA 模型（OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1）和 WAM 模型（VLA-JEPA, Cosmos-Policy, WorldVLA 等）。

论文详细记录了三个阶段的训练过程：

实验设计非常充分且具有高度的客观性：

### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超过了现有最强基准。
*   **几何鲁棒性极强**：在 LIBERO-Plus 的几何扰动轴（相机、机器人初始位姿、布局）上显著优于 $\pi_{0.5}$ 等强力基准，几何平均成功率提升了 4.8%。
*   **机制验证**：在因果槽位干预测试中，OA-WAM 的交换绑定余弦值达到 0.87，而所有整体性模型均低于 0.09，证明了其真正实现了对象级的解耦控制。

### 7. 优点
*   **架构创新**：通过简单的张量级操作（掩码和重置）在 Transformer 内部实现了硬性的归纳偏置，解决了 VLA 模型长期存在的身份纠缠问题。
*   **可验证性**：不同于黑盒模型，OA-WAM 的决策逻辑（即“它在看哪个物体”）可以通过地址空间进行直接干预和验证。
*   **鲁棒性与性能兼顾**：在不牺牲分布内性能的前提下，大幅提升了对环境几何变化的适应力。

### 8. 不足与局限
*   **感知瓶颈**：在“传感器噪声”维度表现较差（-17.1%），原因是光电噪声在感知阶段就破坏了槽位提取，而非策略本身的问题。
*   **计算开销**：虽然骨干网络推理较快，但前端感知（SAM 3 + DINOv3 + Qwen3-VL）引入了约 138ms 的延迟，可能限制极高频的操作。
*   **假设限制**：模型假设物体间是弱耦合的，在处理物体间有复杂物理遮挡或重叠的场景时，性能可能会受限。
*   **环境局限**：目前仅在仿真环境中验证，尚未展示真实机器人的部署结果。
