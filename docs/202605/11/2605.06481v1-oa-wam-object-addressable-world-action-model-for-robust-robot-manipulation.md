## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记（tokens）或全局潜变量。当指令涉及特定对象时，动作解码器很难寻址这些表示，特别是在场景偏移的情况下，此时对象身份与上下文纠缠在一起。我们提出了 OA-WAM，一种用于鲁棒机器人操作的可对象寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并与文本、图像、本体感受和过去动作标记在块因果序列中融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码 16 步连续动作块。通过仅地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，强制实现可寻址性，从而在不添加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试产生的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

### OA-WAM: 用于鲁棒机器人操作的可对象寻址世界动作模型 论文总结

#### 1. 核心问题与整体含义
现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）在面对场景扰动（如相机视角切换、物体布局改变、背景变化）时表现出极差的鲁棒性。其根本原因在于这些模型通常将场景表示为整体图像标记（tokens）或全局潜变量，导致目标物体的“身份信息”与周围的“上下文环境”严重纠缠。当环境发生微小偏移时，模型往往无法准确锁定指令中提到的目标物体。论文提出了**对象寻址（Object Addressability）**的概念，旨在通过架构设计让模型能够稳定地“寻址”并操作特定物体。

#### 2. 方法论
OA-WAM 的核心思想是将每一帧分解为独立的物体槽位（Slots），并在架构层面强制实现身份与状态的分离。

*   **物体槽位分解**：每一帧被分解为 $N+1$ 个槽位（1个机器人槽位，$N$个物体槽位）。每个槽位向量由四部分组成：
    *   `addr_k`：固定地址向量，代表物体身份（由初始帧特征和语言标签决定，全剧不变）。
    *   `cnt_t^k`：时变内容向量，代表物体当前的状态、姿态等。
    *   `π_t` 和 `ρ_k`：时间和角色（机器人/物体）的编码。
*   **可寻址注意力机制（关键技术）**：
    *   **仅地址键投影（Addr-only Key Projection）**：在 Transformer 的交叉槽位注意力计算中，Key 投影矩阵只读取 `addr` 部分。这意味着“动作头关注哪个槽位”仅取决于物体身份，而不受其当前位置或外观变化的影响。
    *   **残差重置（Residual Reset）**：在每一层 Transformer 处理后，强制将残差流中的地址部分重置为初始值，防止身份信息在深度网络中发生漂移。
*   **模型架构**：采用 7B 参数的 Chameleon 风格多模态 Transformer 作为骨干，连接一个预测下一帧物体状态的“世界头”和一个基于流匹配（Flow-matching）的 16 步动作解码头。
*   **算法流程**：感知层（SAM3 + DINOv3）提取物体掩码和特征 -> 槽位适配器映射 -> 块因果序列组装 -> 槽位感知骨干网络处理 -> 动作与世界状态并行输出。

#### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准基准，包含 Spatial、Object、Goal、Long 四个子集，用于评估分布内性能。
    *   **SimplerEnv (WidowX/Bridge)**：用于评估模拟到真实的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性测试，包含相机视角、机器人初始状态、布局、光照、背景、语言描述、传感器噪声 7 个扰动维度。
*   **对比方法**：包括顶级 VLA 模型（OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1）和 WAM 模型（VLA-JEPA, Cosmos-Policy, WorldVLA 等）。
*   **特殊测试**：**因果槽位干预测试**。通过在推理时交换两个物体的地址向量，观察机器人是否会随之改变操作目标，以此验证物体绑定的真实性。

#### 4. 资源与算力
论文详细记录了三个阶段的训练开销：
*   **阶段 0（预训练）**：使用 384 台 A100-80GB GPU，训练约 18 天（总计约 16.6 万 A100 小时），处理了 2.5T tokens 的混合数据。
*   **阶段 I & II（微调）**：在 8 台 A100-80GB 上进行，每个阶段耗时约 3-4 天。
*   **推理延迟**：在单张 A100 上，感知层耗时约 138ms，骨干网络与动作头耗时约 90ms，总闭环频率约为 4.3Hz。

#### 5. 实验数量与充分性
实验设计非常充分且具有高度的客观性：
*   **多维度覆盖**：不仅在标准任务上刷榜，更重点针对 7 种不同的 OOD（分布外）扰动进行了详尽测试。
*   **消融实验**：通过 2x2 因子实验（是否有地址掩码、是否有重置钩子）严谨地证明了鲁棒性增益来源于“可寻址架构”而非单纯的感知增强。
*   **干预实验**：引入“交换绑定余弦值”这一量化指标，从因果层面验证了模型决策逻辑的透明度，这在同类论文中较为罕见。

#### 6. 主要结论与发现
*   **鲁棒性飞跃**：在 LIBERO-Plus 的几何扰动轴（相机/机器人状态/布局）上，OA-WAM 显著超过了 $\pi_{0.5}$ 等强基准，几何平均成功率提升了 4.8%。
*   **物体绑定验证**：OA-WAM 的交换绑定余弦值达到 0.87，而所有整体式（Holistic）基准模型均低于 0.09，证明了其真正实现了语言指令与物理实体的解耦绑定。
*   **性能保持**：在提升鲁棒性的同时，其在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上的绝对性能也达到了 SOTA 水平。

#### 7. 优点
*   **架构创新**：通过简单的张量切片和重置操作，在不增加额外参数的情况下解决了复杂的物体纠缠问题。
*   **可解释性强**：物体槽位的引入使得模型的注意力分配具有物理意义，便于调试和干预。
*   **验证严谨**：通过因果干预实验有力地反驳了“模型只是靠记忆训练集布局”的质疑。

#### 8. 不足与局限
*   **感知依赖**：模型的表现高度依赖于上游的 SAM3 和 DINOv3。在传感器噪声（Sensor Noise）测试中表现不佳（下降 17.1%），因为图像模糊会导致物体提取失败。
*   **计算开销**：感知层（约 138ms）占据了推理时间的大部分，对于高频实时控制可能存在瓶颈。
*   **场景假设**：目前的物体发现机制在处理极小、透明、反射或严重遮挡的物体时仍面临挑战。
*   **模拟局限**：所有实验均在仿真环境中完成，尚未在真实机器人上进行大规模部署验证。
