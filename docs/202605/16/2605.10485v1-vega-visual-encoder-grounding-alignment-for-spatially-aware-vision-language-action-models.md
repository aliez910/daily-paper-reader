## 摘要
精确的空间推理是机器人操作的基础，然而当前视觉-语言-动作（VLA）模型的视觉主干网络主要在2D图像数据上进行预训练，缺乏显式的3D几何监督，导致其表征缺乏准确的空间感知。现有的隐式空间定位方法通过将VLA特征与具有3D感知能力的基础模型的特征进行对齐，在一定程度上解决了这一问题，但它们依赖于经验性的层搜索，并在LLM层级的视觉标记上进行对齐，而此时空间结构已经与语言语义纠缠在一起，限制了泛化能力和几何可解释性。我们提出了VEGA（视觉编码器定位对齐），这是一个简单而有效的框架，它直接将VLA视觉编码器的输出与来自DINOv2-FiT3D的空间感知特征进行对齐，后者是一个通过多视图一致的3D高斯泼溅监督进行微调的DINOv2模型。通过在视觉编码器输出层级进行对齐，VEGA在发生任何语言纠缠之前就建立了空间感知定位，提供了一个更具可解释性且更有原则的对齐目标。该对齐通过一个轻量级投影器实现，该投影器与标准动作预测目标一起使用余弦相似度损失进行训练，并在推理阶段被舍弃，不引入额外的计算开销。在仿真基准和真实世界操作任务上的大量实验表明，VEGA一贯优于现有的隐式空间定位基准，在VLA模型的隐式空间定位方法中达到了新的最先进水平。

## Abstract
Precise spatial reasoning is fundamental to robotic manipulation, yet the visual backbones of current vision-language-action (VLA) models are predominantly pretrained on 2D image data without explicit 3D geometric supervision, resulting in representations that lack accurate spatial awareness. Existing implicit spatial grounding methods partially address this by aligning VLA features with those of 3D-aware foundation models, but they rely on empirical layer search and perform alignment on LLM-level visual tokens where spatial structure has already been entangled with linguistic semantics, limiting both generalizability and geometric interpretability. We propose VEGA (Visual Encoder Grounding Alignment), a simple yet effective framework that directly aligns the output of the VLA's visual encoder with spatially-aware features from DINOv2-FiT3D, a DINOv2 model fine-tuned with multi-view consistent 3D Gaussian Splatting supervision. By performing alignment at the visual encoder output level, VEGA grounds spatial awareness before any linguistic entanglement occurs, offering a more interpretable and principled alignment target. The alignment is implemented via a lightweight projector trained with a cosine similarity loss alongside the standard action prediction objective, and is discarded at inference time, introducing no additional computational overhead. Extensive experiments on simulation benchmark and real-world manipulation tasks demonstrate that VEGA consistently outperforms existing implicit spatial grounding baselines, establishing a new state-of-the-art among implicit spatial grounding methods for VLA models.

### 1. 核心问题与研究动机
*   **核心问题**：当前的视觉-语言-动作（VLA）模型（如 OpenVLA, RT-2）虽然具备强大的指令遵循能力，但其视觉主干网络多在 2D 图像数据上预训练，缺乏显式的 3D 几何监督。这导致模型在需要精确空间推理的任务（如估计物体高度、深度或相对位置）中表现不佳。
*   **研究背景**：
    *   **显式方法**：引入深度图或点云，但会增加推理开销且易受传感器噪声影响。
    *   **隐式方法**：将 VLA 特征与 3D 基础模型对齐。但现有方法通常在 LLM 层的视觉 Token 级别进行对齐，此时视觉特征已与语言语义纠缠，且依赖繁琐的层搜索，缺乏几何可解释性。

### 2. 方法论
*   **核心思想**：提出 **VEGA** 框架，在视觉编码器的**输出层级**直接进行空间对齐。在视觉特征进入 LLM 与语言指令融合之前，通过知识蒸馏的方式注入 3D 空间感知能力。
*   **关键技术细节**：
    *   **空间导师（Teacher）**：选用 **DINOv2-FiT3D**。该模型通过 3D 高斯泼溅（3DGS）监督进行微调，具有极强的多视图几何一致性和密集的 patch 级几何表征。
    *   **对齐机制**：在 VLA 的 DINOv2 分支输出端添加一个轻量级的 **MLP 投影器**。
    *   **损失函数**：总损失 $L_{VEGA} = L_{action} + \lambda L_{align}$。其中 $L_{align}$ 是投影后的视觉特征与导师特征之间的**余弦距离**。
    *   **推理优化**：训练完成后，投影器和导师模型均被舍弃，推理过程与原始 VLA 完全一致，**零额外计算开销**。

    *   **消融实验**：针对对齐系数 $\lambda$、导师模型选择（VGGT vs. FiT3D）、训练效率和数据效率进行了详细分析。
*   **评价**：实验设计较为充分且客观。通过 Hard 设置验证了模型的鲁棒性，通过不同数据比例（25%-100%）验证了数据效率，对比了多种 SOTA 隐式对齐方法，具有较强的说服力。

### 6. 主要结论与发现
*   **性能领先**：VEGA 在 RoboTwin 2.0 的 Easy 和 Hard 设置下均刷新了隐式空间定位方法的 SOTA 记录。
*   **效率提升**：VEGA 表现出极高的训练效率，在 10k 步时的表现即可比拟基准模型 60k 步的表现；在数据稀缺（仅 25% 数据）时，性能提升尤为显著（绝对值提升 10%）。
*   **早期对齐的必要性**：实验证明在视觉编码器层级（语义纠缠前）进行对齐，比在 LLM Token 层级对齐更有效且更稳定。

### 7. 优点（亮点）
*   **零推理成本**：通过训练时的特征对齐，在不增加任何推理延迟的情况下提升了 3D 感知能力。
*   **架构简洁**：避免了复杂的跨层搜索，直接在编码器输出端对齐，逻辑清晰且易于迁移至其他 VLA 架构。
*   **几何一致性**：利用 FiT3D 这种基于 3DGS 的模型作为导师，比传统的几何模型（如 VGGT）在 patch 级别具有更好的特征兼容性。

### 8. 不足与局限
*   **导师依赖性**：VEGA 的效果高度依赖于 DINOv2-FiT3D 的表征质量。如果任务场景与 FiT3D 的预训练场景（室内场景）差异极大，空间感知的增益可能会减弱。
*   **泛化边界**：虽然在 Hard 设置下表现良好，但在极端无结构或完全未见过的分布外（OOD）环境中的表现仍有待进一步验证。
*   **任务类型限制**：目前的实验主要集中在近距离操纵任务，对于需要超远距离空间推理的任务覆盖较少。
