## 摘要
视觉-语言-动作（VLA）模型在通用机器人操作中展现出巨大潜力，但仍受限于空间推理不足，尤其是在复杂视觉场景中确定交互位置方面。尽管近期工作引入多种形式的视觉规划来解决此问题，现有方法要么依赖全局几何线索、符号中间表示，要么依赖外部生成的视觉信号，这些通常与下游动作预测的耦合较弱。在本工作中，我们重新审视VLA系统中的视觉规划，并提出有效规划应是局部的、视觉上 grounded 的、内部生成的，且与动作直接对齐。基于此见解，我们提出Afford-VLA，一个统一框架，将任务条件化的可供性内化为VLA模型内的显式视觉规划接口。具体而言，我们引入可学习的<AFF>标记来查询任务相关的交互区域，从多模态特征中解码可供性掩码，并将其转换为紧凑嵌入，直接条件化动作生成。该设计使可供性在VLA内部既能生成又能利用，形成紧密耦合的感知-动作通路。为进一步支持这一集成，我们采用训练策略，使可供性通路与动作预测联合优化，提高其对下游控制的有效性。我们在多个模拟基准（包括LIBERO、LIBERO-Plus和SimplerEnv）上评估方法，实现了持续的最优性能，并在真实世界中取得强劲结果。这些发现表明，将可供性内化为与动作对齐的视觉规划为改进VLA系统提供了强大范式。

## Abstract
Vision-language-action (VLA) models have shown strong potential for generalist robot manipulation, yet they remain limited by insufficient spatial reasoning, particularly in determining where to interact in complex visual scenes. While recent efforts introduce various forms of visual planning to address this issue, existing approaches either rely on global geometric cues, symbolic intermediate representations, or externally generated visual signals, which are often weakly coupled with downstream action prediction. In this work, we revisit visual planning in VLA systems and argue that effective planning should be local, visually grounded, internally generated, and directly aligned with action. Based on this insight, we propose Afford-VLA, a unified framework that internalizes task-conditioned affordance as an explicit visual planning interface within VLA models. Concretely, we introduce learnable <AFF> tokens to query task-relevant interaction regions, decode affordance masks from multimodal features, and convert them into compact embeddings that directly condition action generation. This design enables affordance to be both generated and utilized within the VLA, forming a tightly coupled perception-action pathway. To further support this integration, we adopt a training strategy that allows the affordance pathway to be jointly optimized with action prediction, improving its effectiveness for downstream control. We evaluate our method on multiple simulation benchmarks, including LIBERO, LIBERO-Plus, and SimplerEnv, achieving consistent state-of-the-art performance, along with strong real-world results. These findings demonstrate that internalizing affordance as action-aligned visual planning provides a powerful paradigm for improving VLA systems.

  - 视觉接地方法（如掩码、热图）虽直接定位区域，但多依赖外部感知模块或作为独立辅助目标训练，未与动作预测紧密集成。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：将可供性内化为VLA模型中的中间表征，同时进行生成和利用，使交互区域直接指导动作生成。
- **关键技术细节**：
  1. **可学习<AFF>查询令牌**：在VLM输入中增加少量可学习的<AFF>令牌（每视角K=4个），与图像令牌、文本令牌一起通过Transformer处理，其隐状态包含任务条件的可供性信息。
  2. **内部可供性掩码生成**：利用轻量级解码器（2层双路注意力，隐藏维度256）将<AFF>隐状态与图像补丁特征结合，输出补丁级可供性logits G_t ∈ R^{H_p×W_p}。
  3. **动作对齐的可供性嵌入**：对G_t进行硬Top-K（K=16）选择，得到二值选择掩码m_t，然后对选中的视觉补丁特征进行平均池化，并通过线性投影W_aff得到紧凑的可供性嵌入r_t，追加到VLM隐状态后输入动作头。
  4. **直通梯度估计（ST）**：在训练时，硬Top-K不可微，使用softmax替代梯度实现动作损失对可供性logits的反传，使可供性通路被动作目标优化。
  5. **两阶段训练**：
     - 第一阶段：仅使用密集掩码监督（BCE）预热可供性头，冻结VLM和动作头（4K步）。
     - 第二阶段：使用模型预测的可供性logits构建嵌入（同测试时），联合优化动作损失（流匹配）和可供性损失，动作梯度通过ST更新可供性头。
- **公式说明**：最终策略为p(a_{t:t+H} | I_t, x, s_t, M_t)，其中M_t表示预测的可供性区域。动作头采用GR00T风格的流匹配（DiT-B），预测8步7DoF关节位置delta动作。

## 6. 论文的主要结论与发现

- Afford-VLA在LIBERO上平均成功率达97.4%，SimplerEnv达58.1%，LIBERO-Plus达78.1%，均显著优于所有对比方法，包括通用VLA和视觉规划方法。
- 内化可供性作为动作对齐的视觉规划能有效提升空间推理和交互定位能力，尤其在需要精确区域定位的任务（如Put Eggplant）上表现突出。
- 零样本鲁棒性验证了可供性引导的池化使策略对视觉分布变化（视角、背景、噪声、光照等）具有更强的抗干扰能力。
- 消融实验表明：显式、内部、可微的动作对齐设计缺一不可，且稀疏硬Top-K + ST梯度是实现紧耦合的最佳池化策略。

## 7. 优点

- **方法论创新**：首次提出满足四个属性（局部、视觉接地、内部生成、动作对齐）的视觉规划范式，将可供性作为内部接口直接条件化动作，形成紧耦合通路。
- **技术设计巧妙**：可学习<AFF>令牌与直通梯度结合，使动作损失反传优化可供性头，既是感知输出也是动作条件。
- **实验严谨**：多基准、多扰动、多消融，且对比了当前最强的开源VLA策略，结果可信。
- **可复现性**：承诺开源代码和模型，超参数详细公开。
- **真实世界验证**：不仅在模拟中SOTA，真实机器人任务也展示出明显提升。

## 8. 不足与局限

- **2D affordance限制**：仅使用2D图像补丁级掩码，未利用3D几何信息（深度、点云），可能限制对细粒度空间关系的理解（如堆叠任务中表现弱于某些方法）。
- **依赖外部监督**：可供性掩码标注来自离线预训练模型RAGNet，其质量影响最终性能，且无法完全避免标注偏差。
- **具身多样性和场景受限**：实验仅在单一机械臂（ARX X5）和有限物体集合上验证，未跨不同机器人形态或大规模杂乱环境评估。
- **训练复杂度**：两阶段训练需调整多个超参数（K、温度、阶段步数等），且对计算资源有一定要求。
- **长期任务和动态场景**：当前主要针对单步或短时任务，长时协作和动态环境尚未充分测试。
