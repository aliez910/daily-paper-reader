## 摘要
机器人模仿学习通常被视为复现演示动作，但动作天生具有具身特异性。当演示来自人类或具有不同形态、运动学或动作空间的机器人时，这种以动作为中心的视角需要共享动作空间、启发式重定向或大规模多具身协同训练。我们转而将演示视为未来目标的隐式规范：目标智能体应推断演示者试图实现的状态，而非演示者如何执行它。我们提出Demo-JEPA，一种跨具身模仿框架，将演示意图与具身特定执行解耦。基于JEPA世界模型，Demo-JEPA将源视觉演示转化为共享预测表示空间中与目标兼容的未来潜在轨迹。然后目标智能体将这些潜在轨迹作为子目标，并通过在其自身学习的前向动力学下进行规划来实现它们。由于Demo-JEPA避免了动作级对应，仅需视觉演示及目标智能体自身的交互经验，它支持跨异质具身的灵活模仿。在RLBench和真实世界操作任务上的实验表明，Demo-JEPA匹配了专门的领域内规划器，并能泛化到先前方法失败的新任务和具身配置上。

## Abstract
Robotic imitation learning is often treated as reproducing demonstrated actions, but actions are inherently embodiment-specific. When demonstrations come from humans or robots with different morphology, kinematics, or action spaces, this action-centric view requires shared action spaces, heuristic retargeting, or large-scale multi-embodiment co-training. We instead view demonstrations as implicit specifications of future goals: the target agent should infer what state the demonstrator is trying to realize, rather than how the demonstrator executes it. We propose Demo-JEPA, a cross-embodiment imitation framework that decouples demonstration intent from embodiment-specific execution. Built on a JEPA-based world model, Demo-JEPA translates source visual demonstrations into target-compatible future latent trajectories in a shared predictive representation space. The target agent then uses these latent trajectories as subgoals and realizes them through planning under its own learned forward dynamics. Because Demo-JEPA avoids action-level correspondence and requires only visual demonstrations plus the target agent's own interaction experience, it supports flexible imitation across heterogeneous embodiments. Experiments on RLBench and real-world manipulation tasks show that Demo-JEPA matches specialized in-domain planners and generalizes to unseen tasks and embodiment configurations where prior methods fail.

- **整体贡献**：提出 **Demo-JEPA**，一种基于 JEPA 世界模型的跨具身模仿框架，通过将源演示映射到目标兼容的潜在轨迹，无需动作级对应，仅依赖视觉演示和目标智能体自身的交互经验。

## 2. 方法论

### 核心思想
将跨具身模仿形式化为 **潜在目标条件规划**（latent goal-conditioned planning）。给定源演示视频，通过 **Dreamer Predictor**（梦想预测器）推断目标智能体可执行的未来潜在轨迹作为子目标，再通过目标智能体自身学习的前向动力学模型（Action-conditioned World Model）进行规划实现。

### 关键技术细节
1. **动作条件世界模型**（基于 V-JEPA 2.1）：
   - JEPA编码器 \(E\) 将观察映射为潜在状态 \(z_k = E(o_k)\)。
   - 动力学预测器 \(F_{wm}\) 建模状态转移：\(\hat{z}_{k+1} = F_{wm}(z_k, s_k, a_k)\)。
   - 规划时采用交叉熵方法（CEM）优化动作序列，使预测潜在轨迹匹配目标。

2. **Dreamer Predictor**：
   - 输入：当前目标观察 \(o_t^T\) 和源演示帧对 \((o_s^k, o_s^{k+n})\)。
   - 处理流程：
     1. 共享编码器提取潜在表示 \(z_t^T, z_s^k, z_s^{k+n}\)。
     2. 两个交叉注意力模块分别建模 **跨具身对应**（\(f_{emb} = Attn(Q=z_t^T, K=z_s^k, V=z_s^k)\)）和 **演示运动**（\(f_{mot} = Attn(Q=z_s^{k+n}, K, V)\)）。
     3. 3D卷积融合 \(f_{fused} = \phi([z_t^T \oplus f_{emb} \oplus f_{mot}])\)。
     4. Transformer预测器解码得到潜在目标 \(\hat{z}_{goal}\)。

3. **三阶段训练过程**：
   - Stage 0：预训练目标智能体的动作条件世界模型（冻结编码器）。
   - Stage 1：训练 Dreamer Predictor（冻结世界模型），使用潜在重建损失 \(L_{pred} = \|\hat{z}_{goal} - z_{k+n}^T\|_2^2\)，并加入时间扰动增强鲁棒性。
   - Stage 2：动作协同训练（冻结 Predictor，微调世界模型），使用规划损失 \(L_{plan} = \|F_{wm}(z_k,...) - \hat{z}_{goal}\|_2^2\)，使世界模型的潜在 rollout 适应 Predictor 输出的目标分布。

4. **推理阶段自适应目标更新**：
   - 计算真实观测潜在与目标潜在的差异 \(D_k = d(z_{k+1}, \hat{z}_{goal})\)，仅当 \(D_k < \epsilon\) 时才推进到下一个源参考帧，否则保持当前目标重新规划。此机制防止因跨具身动力学差异导致的累积误差。

## 3. 实验设计

### 数据集/场景
- **模拟**：RLBench，源具身用 **Sawyer** 臂，目标具身用 **Franka** 臂，共9个任务（如投篮、变道、关盒等）。
- **真实世界**：源具身 **UR5e** 臂，目标具身 **Franka** 臂，6个任务（提杯、提单方块、移盘、按按钮、揭锅盖、移胡椒瓶）。使用单固定RGB相机（未精确标定）。

### Benchmark 与对比方法
- **VPP**（Video Prediction Policy）：通过视频预测学习未来感知表示，再学习动作。
- **XSkill**：通过匹配异质视频到共享技能原型来执行技能序列。
- **消融基线**：V-JEPA 2.1 Naive（直接使用源演示的未来潜在作为目标）、Oracle（使用目标具身真实未来轨迹）、无Conv3D变体。
- **扩展实验**：Demo-DP（将 Dreamer Predictor 的输出作为扩散策略的条件）。

### 评估协议
三个递进套件，分布偏移逐渐增大：
1. **Behavior Grounding**：训练阶段见过的任务（完全监督）。
2. **Cross-Embodiment Bridging**：仅在 Stage I 见过的任务（无动作监督）。
3. **Zero-Shot Generalization**：完全未见过的任务和配置。

每个场景报告30次（模拟）或20次（真实）rollout的成功率。

## 6. 主要结论与发现

1. **提出目标想象优于动作复现**：将跨具身模仿视为潜在目标规划，有效避免动作对齐瓶颈。
2. **Demo-JEPA 在分布偏移大时表现更优**：在零样本泛化套件上显著优于 VPP 和 XSkill（模拟36% vs 4%和3%；真实25% vs 0%和5%）。
3. **Latent Goal 的可迁移性**：V-JEPA 2.1 的潜在特征本身不具备跨具身可迁移性（Naive 方案失败），但 Dreamer Predictor 能将其转化为目标兼容的目标。
4. **Conv3D 对复杂运动至关重要**：在需要旋转、闭合等结构化运动的任务上，无Conv3D变体性能显著下降。
5. **任务多样性比单任务数据量更重要**：缩放研究中，任务缩放对泛化提升显著大于数据缩放。
6. **结合扩散策略可增强接地能力**：Demo-DP 在行为接地套件上表现优于原版 Demo-JEPA，但在零样本泛化上不及规划器版本，表明规划器对分布偏移更鲁棒。
7. **自适应目标更新稳定长周期任务**：通过子目标到达阈值判断，防止因动力学不匹配导致的提前推进错误。

## 7. 优点

- **创新性**：首次将 JEPA 用于跨具身模仿，通过隐式意图推理避免动作级对齐，思路新颖。
- **实用性**：仅需视觉演示和目标智能体自身交互数据，无需昂贵的人机对应标注或重定向规则。
- **泛化能力**：在零样本情况下仍能有效工作，远超现有方法。
- **设计优雅**：Dreamer Predictor 利用交叉注意力分别建模跨具身对应和运动信息，并通过 3D 卷积融合，结构合理。
- **消融与扩展丰富**：不仅拆解核心组件，还将预测器推广到扩散策略场景，显示广泛适用性。
- **真实世界验证**：在物理机器人上实现了跨UR5e → Franka的零样本迁移，证明了实际部署可行性。

## 8. 不足与局限

- **世界模型瓶颈**：性能受限于当前动作条件世界模型（V-JEPA 2.1）的建模精度，在复杂高精度任务上（如精细抓取）效果仍有提升空间。
- **仍需时间或进度对齐**：训练阶段要求源与目标演示之间存在帧级对应（模拟中通过重放制造对应；真实中通过 GTCC 进度对齐），未能实现完全无对齐学习。
- **真实世界场景有限**：仅6个任务，且所有任务为单一机械臂的桌子操作，未涉及移动平台、双机械臂协作、动态环境等复杂场景。
- **计算成本较高**：8×A100 总计约 10.5 天训练，微调也需要 1 天，难以在计算受限条件下复现。
- **相机标定缺失**：真实世界实验中源和目标相机未做精确几何标定，虽然表明方法对视角变化鲁棒，但可能引入额外的不确定性。
