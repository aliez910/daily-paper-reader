## 摘要
视觉-语言-动作（VLA）模型已成为通用机器人操作的一种有前景的范式。当前架构中的常见设计是将语言指令和视觉观察映射为单次前向传递中的动作。虽然概念上简单，但这种表述将指令理解、空间场景感知和运动控制纠缠在同一个学习目标中。因此，动作专家必须隐式地重新学习预训练VLM中已存在的认知和感知能力，这可能会限制学习效率和泛化能力。我们提出了AVP（基于视觉基元的动作），这是一种端到端架构，实现了这种以视觉基元为中心的接口：VLM推断下一阶段目标并生成视觉基元标记，这些标记条件化一个流匹配动作专家，其监督来自末端执行器运动学。在通用拾取和放置任务上的真实机器人实验表明，AVP相比pi_0.5将成功率提高了27.61%，并优于其他近期方法，在数据效率、空间组合泛化和对象级迁移方面均有持续提升。

## Abstract
Vision-Language-Action (VLA) models have emerged as a promising paradigm for generalist robotic manipulation. A common design in current architectures maps language instructions and visual observations to actions in a single forward pass. While conceptually simple, this formulation entangles instruction comprehension, spatial scene understanding, and motor control within a single learning objective. As a result, the action expert must implicitly relearn cognitive and perceptual capabilities already present in the pretrained VLM, which can limit both learning efficiency and generalization. We introduce AVP (Action with Visual Primitives), an end-to-end architecture that implements this visual-primitive-centric interface: the VLM infers the next-stage target and emits visual-primitive tokens that condition a flow-matching action expert, with supervision derived from end-effector kinematics. Real-robot experiments on general pick-and-place tasks show that AVP improves the success rate by 27.61% over pi_0.5 and outperforms other recent methods, with consistent gains in data efficiency, spatial-compositional generalization, and object-level transfer.

## 1. 核心问题与研究动机
**当前VLA模型的局限**：现有Vision-Language-Action（VLA）模型（如RT-2、OpenVLA、π0.5）将指令理解、场景感知和动作控制耦合在同一个学习目标中。动作专家必须隐式地重新学习VLM已具备的认知和感知能力，导致学习效率低下、泛化能力弱（尤其是对象/布局变化时）。

**研究目标**：提出一种更清晰的职责划分——VLM负责“做什么、在哪里做”，动作专家只关注“如何做”。通过引入显式的**视觉基元（Visual Primitives）** 作为VLM与动作专家之间的通信接口，实现解耦。

## 2. 方法论：AVP架构

### 核心思想
构建一个端到端的VLA架构，其中VLM在推理下一阶段目标时，内部生成视觉基元（如点、边界框、遮罩等），这些基元被投影到视觉token空间，作为条件输入到流匹配（flow-matching）动作专家，最终生成机器人动作。视觉基元充当VLM和动作专家之间的显式、可解释的中间表示。

### 关键技术细节
- **组成部分**：预训练VLM + 自回归解码器 + 动作专家。
- **流程**：
  1. VLM接收语言指令 `l` 和多视角观测 `o_t`，理解任务并预测下一阶段的目标。
  2. 视觉基元解码器 `D_ψ` 从VLM的上下文token中自回归地预测离散化的视觉基元 `p_t`（表示下一阶段子任务及其空间目标）。
  3. 将预测的基元 `p_t` 投影到视觉token空间得到 `zvp_t`，与原始多模态token融合形成增强表示 `zaug_t`。
  4. 动作专家 `π_θ` 基于 `zaug_t` 和机器人状态 `s_t` 预测动作序列 `a_{t:t+h}`。
- **监督**：
  - 视觉基元预测损失 `L_vp`：交叉熵，基元标签从末端执行器运动学投影得到（无需外部标注）。
  - 动作预测损失 `L_act`：标准动作预测。
  - 总损失：`L = L_act + λ L_vp`。

### 创新点
- 将视觉基元作为VLM内部产生的token，而非依赖外部模型（如SAM、Grounding DINO）生成，实现端到端联合优化。
- 基元监督完全基于机器人自身运动学（手眼标定 + 投影），自动获取，零额外人工成本。

## 6. 主要结论与发现
- AVP在所有三个任务上全面优于π0.5基线，整体成功率提升**27.61%**（中国象棋：90.28% vs 62.67%；多米诺：88.19% vs 81.94%；通用物体：86.18% vs 64.96%）。
- 空间组合泛化：AVP能零样本执行未见的直接过渡动作（π0.5完全失败），表明视觉基元促进了子技能组合。
- 跨域泛化：仅用中国象棋数据训练的AVP能零样本迁移到45个未见物体（不同背景和指令设置），成功率85%以上；π0.5严重退化。
- 消融表明：引入视觉基元（尤其是框+遮罩+记忆）显著提升性能，验证了解耦的有效性。

## 7. 优点
- **方法创新**：首倡以VLM内部产生的视觉基元作为VLA中VLM与动作专家的显式接口，明确职责划分。
- **自动监督**：基元标签直接来自机器人运动学投影，无需人工标注或外部感知模型，易于扩展。
- **强泛化**：在空间组合和跨域任务上表现突出，优于耦合型VLA。
- **端到端**：基元生成与动作预测联合优化，无级联误差。
- **轻量可扩展**：基元形式紧凑（点、框等），支持记忆扩展、顺序索引等，不影响架构。
