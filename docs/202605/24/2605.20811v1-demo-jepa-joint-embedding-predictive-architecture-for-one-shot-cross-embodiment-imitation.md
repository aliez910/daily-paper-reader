## 摘要
机器人模仿学习通常被视作复现演示动作，但动作本质上是具身特有的。当演示来自形态、运动学或动作空间不同的人类或机器人时，这种以动作中心的视角要求共享动作空间、启发式重定向或大规模多具身协同训练。相反，我们将演示视为未来目标的隐式规范：目标智能体应推断演示者试图实现的状态，而非演示者如何执行。我们提出Demo-JEPA，一个跨具身模仿框架，将演示意图与具身特定执行解耦。基于JEPA世界模型，Demo-JEPA将源视觉演示转化为共享预测表示空间中与目标兼容的未来潜在轨迹。目标智能体随后将这些潜在轨迹用作子目标，并通过在其自身学习的前向动力学下的规划来实现。由于Demo-JEPA避免了动作级对应，且仅需视觉演示加上目标智能体自身的交互经验，它支持异构具身间的灵活模仿。在RLBench和真实世界操作任务上的实验表明，Demo-JEPA匹配了专门领域内规划器，并泛化到先前方法失败的未见任务和具身配置。

## Abstract
Robotic imitation learning is often treated as reproducing demonstrated actions, but actions are inherently embodiment-specific. When demonstrations come from humans or robots with different morphology, kinematics, or action spaces, this action-centric view requires shared action spaces, heuristic retargeting, or large-scale multi-embodiment co-training. We instead view demonstrations as implicit specifications of future goals: the target agent should infer what state the demonstrator is trying to realize, rather than how the demonstrator executes it. We propose Demo-JEPA, a cross-embodiment imitation framework that decouples demonstration intent from embodiment-specific execution. Built on a JEPA-based world model, Demo-JEPA translates source visual demonstrations into target-compatible future latent trajectories in a shared predictive representation space. The target agent then uses these latent trajectories as subgoals and realizes them through planning under its own learned forward dynamics. Because Demo-JEPA avoids action-level correspondence and requires only visual demonstrations plus the target agent's own interaction experience, it supports flexible imitation across heterogeneous embodiments. Experiments on RLBench and real-world manipulation tasks show that Demo-JEPA matches specialized in-domain planners and generalizes to unseen tasks and embodiment configurations where prior methods fail.

跨具身模仿学习面临的核心瓶颈是**具身鸿沟**：演示者（如人类或不同机器人）与学习机器人在形态、运动学、动作空间上存在根本差异。现有方法多依赖动作级对齐——需要共享动作空间、启发式重定向或大规模多具身协同训练，这不仅数据成本高昂，且难以推广到高度异构的具身。作者提出一种全新的视角：**将视觉演示视为未来目标的隐式规范**，学习智能体应推断演示者意图实现的未来状态（“做什么”），而非复现其具体的执行方式（“怎么做”）。Demo-JEPA 正是围绕这一思想构建的跨具身模仿框架。

### 论文提出的方法论

**核心思想**：将跨具身模仿转化为**潜在目标条件规划**。通过联合嵌入预测架构（JEPA）建立一个共享预测表示空间，在此空间中，利用一个具身感知的“梦想家预测器”（Dreamer Predictor）将源视觉演示翻译成与目标机器人兼容的未来潜在轨迹（子目标），然后目标机器人利用自身学习的前向动力学模型进行规划（通过交叉熵方法CEM）来达成这些子目标。

**关键技术细节**：

- **动作条件世界模型**：采用 V-JEPA 2.1 作为基础，包含编码器 \(E\) 将观测映射为潜在状态 \(z_k = E(o_k)\)，以及动力学预测器 \(F_{wm}\) 建模条件于机器人状态和动作的潜在转移：\(\hat{z}_{k+1} = F_{wm}(z_k, s_k, a_k)\)。
- **Dreamer Predictor**：输入当前目标观测 \(o_k^T\) 和源演示的一对帧 \((o_s^i, o_s^{i+n})\)，通过共享 JEPA 编码器得到各自潜在表示。采用两个交叉注意力模块分别建模**跨具身对应**（目标与源当前帧）和**动作演化**（源未来帧与源当前帧），然后将三者融合（使用 3D卷积 \(\phi\)），最后经由一个 Transformer 预测器解码得到潜在目标 \(\hat{z}_{goal}\)。
- **三阶段训练**：
  - Stage 0：预训练目标机器人的动作条件世界模型⁠🌲（冻结编码器，仅训练动力学预测器）。
  - Stage 1：训练 Dreamer Predictor，以潜在重建损失 \(\mathcal{L}_{pred} = \|\hat{z}_{goal} - z_{k+n}\|^2\) 优化，并引入**时间扰动**（随机偏移源帧索引）增强鲁棒性。
  - Stage 2：冻结 Dreamer Predictor，微调世界模型的动力学预测器，使其 rollout 与预测器输出的目标分布对齐，损失 \(\mathcal{L}_{plan} = \|F_{wm}(z_k, s_k, a_{k:k+n-1}) - \hat{z}_{goal}\|^2\)。
- **推理时自适应目标更新机制**：根据当前观测与目标之间的潜在距离 \(D_k\) 决定是否前进到下一个源参考段，避免因跨具身动力学差异导致的时序累积误差。

### 优点

- **创新性思想**：将跨具身模仿重新定义为潜在目标规划，避免动作级对应，从根本上简化了跨具身迁移。
- **框架简洁实用**：仅需视觉演示 + 目标机器人自身交互经验，无需昂贵的人工动作注释或重定向规则。
- **JEPA 的选择合理**：JEPA 潜在空间抽象了预测性世界结构，抑制了背景、光照、具身外观等无关变量，非常适合跨具身意图保持。
- **自适应目标更新机制**：解决了长时序任务中因跨具身动力学差异导致的目标错位问题，提升稳定性和成功率。
- **实验论证充分**：在仿真和真实环境、不同分布偏移强度下均进行验证，并包含丰富的消融与扩展实验。

### 不足与局限

- **世界模型瓶颈**：当前动作条件世界模型（V-JEPA 2.1）的能力限制了复杂、高精度任务（如精密装配）的表现，框架性能上限受限于世界模型质量。
- **训练仍需对齐**：虽然避免了动作级对应，但在 Stage I 训练 Dreamer Predictor 时仍需要源/目标具身之间的时间或进度对齐（仿真中使用重放对齐，真实中使用 GTCC 特征对齐），尚未实现完全无监督对齐。
- **任务与本体覆盖有限**：仿真和真实各仅一种源/目标具身配对（Sawyer→Franka / UR5e→Franka），任务类型也未覆盖灵巧操作、动态交互等更复杂场景。
- **计算资源较大**：8 卡 A100 训练数天，对于资源有限的团队门槛较高。
- **未与最新 VLA 方法对比**：论文发表时未与 π0、OpenVLA 等大规模视觉-语言-动作模型进行直接比较，虽然这些方法也涉及跨具身训练，但数据和计算需求不同。

### 总结与展望

Demo-JEPA 为跨具身模仿学习开辟了一条新路径：从动作级对齐转向目标级意图理解，通过 JEPA 的预测表示空间实现异构本体之间的语义桥接。该方法在零样本泛化场景下展现出显著优势，证明了“理解意图”比“复现动作”更适用于跨具身迁移。未来可向以下方向演进：

- **世界模型增强**：引入更强大的生成式世界模型（如扩散 Transformer 或基于物理的模拟器），以支撑高精度操作和复杂接触任务。
- **无监督对齐**：探索无需时间或进度对齐的预训练策略（例如利用视频对比学习或循环一致性损失），实现完全的跨具身无监督迁移。
- **多本体扩展**：在更多异构机器人（如移动操作平台、人形机器人）及灵巧手任务上验证框架的通用性。
- **与 VLA 融合**：将 Demo-JEPA 的潜在目标预测能力与大规模视觉-语言-动作模型结合，利用语言指令引导意图推理，进一步降低数据需求。

总体而言，该工作为具身智能中的“鸿沟跨越”提供了理论基础与实用框架，其将模仿学习从“教动作”转向“教意图”的设计理念值得后续研究深入挖掘。
