## 摘要
我们介绍SOMA，一个用于视觉-语言-动作（VLA）模型中视觉外操作的空间记忆框架。大多数现有的VLA隐式假设任务相关对象始终可见，导致当目标超出相机视野时，行为变得脆弱且反应式。SOMA通过为VLA配备一个由可移动头部相机获取的多视角观测构建的持久空间记忆来解决这一限制，使得推理能够超越当前视觉锥体。该框架包含三个组件：空间记忆构建（通过扫描将角度观测聚合为统一的空间语义表示）、动态记忆精炼（随时间保持全局一致性）以及上下文记忆检索（在操作过程中激活与指令相关的空间线索）。我们在五个具有挑战性的真实世界视觉外操作任务上评估SOMA，包括目标物体初始不可见的多步骤和双臂场景。实验结果表明，SOMA不仅提高了任务成功率，还引发了质上不同的操作行为：更快的目标定位、减少的视点搜索，以及在部分可观测性下近乎一次性的抓取。在RoboCasa GR1和SimplerEnv上的额外实验进一步验证了SOMA记忆设计在常规完全可观测设置下的有效性。代码即将发布。

## Abstract
We introduce SOMA, the Spatial Memory framework for Out-of-Vision Manipulation in Vision-Language-Action (VLA) models. Most existing VLAs implicitly assume that task-relevant objects are always visible, leading to brittle and reactive behaviors when targets fall outside the camera's field of view. SOMA addresses this limitation by equipping VLAs with a persistent spatial memory constructed from multi-view observations acquired via a movable head camera, enabling reasoning beyond the current visual frustum. The framework consists of three components: Spatial Memory Construction, which aggregates angular-wise observations into a unified spatial-semantic representation through scanning; Dynamic Memory Refinement, which maintains global consistency over time; and Contextual Memory Retrieval, which activates instruction-relevant spatial cues during manipulation. We evaluate SOMA on five challenging real-world out-of-vision manipulation tasks, including multi-step and dual-arm scenarios where target objects are initially invisible. Experimental results show that SOMA not only improves task success rates, but also induces qualitatively different manipulation behaviors, with faster target localization, reduced viewpoint search, and near one-shot grasping under partial observability. Additional experiments on RoboCasa GR1 and SimplerEnv further validate the effectiveness of SOMA's memory design under conventional fully observable settings. Code will be released soon.

- **整体含义**：论文指出，单纯增加模型容量或推理深度，无法弥补因缺乏感知依据（特别是从未观察到或未保留的过去观测）而产生的不足。作者提出，要使机器人在复杂、动态环境中可靠操作，必须建立主动获取并持久保留空间证据的机制。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：为VLA模型配备一个由可移动头部摄像头多视角观测构建的**持久性空间记忆**。该记忆不仅存储语义信息，还存储几何（3D）信息，从而使模型能够推理当前视野之外的物体，并支持记忆引导下的操作。

- **技术架构：SOMA框架由三个核心组件构成**

    1.  **空间记忆构建 (Spatial Memory Construction)**：
        - **触发机制**：当任务指令中的目标在当前视野中未被检测到时，系统触发主动的头部扫描过程。
        - **流程**：
            - 可移动头部摄像头沿预定轨迹扫描场景，获取多视角视频序列。
            - 对采样帧进行统一处理：
                - **几何感知**：使用VGGT网络估计每帧的相机姿态和粗略场景几何。
                - **语义感知**：使用YOLO进行目标检测与分类，使用DINOv3提取高维视觉特征。
                - **3D提升**：将2D检测框映射到全局3D场景坐标系中，得到3D边界框。
            - 通过外观-几何联合相似度进行跨视角的实例关联与融合，形成全局对象集合（每个对象由一个三元组表示：外观嵌入、3D中心、3D边界框）。
            - 将外观特征和3D位置编码融合为统一的“空间-语义”嵌入`m0_k`，集合形成初始全局场景记忆`M0`。

    2.  **动态记忆精炼 (Dynamic Memory Refinement)**：
        - **目的**：在操作过程中，根据头部相机的最新观测，持续更新和精炼记忆，以适应环境动态变化（如物体移动、遮挡等）。
        - **流程**：
            - 将当前观测`o_t`通过相同管道处理，得到当前记忆令牌`M_t`。
            - 对每个观测实例，与上一时间步记忆`M_{t-1}`中同类的实例进行关联。
            - **自适应相似度感知融合**：计算语义相似度`s`和动态融合分数`g`，二者乘积得到自适应更新系数`α`。
            - 使用时间指数移动平均（EMA）更新匹配的记忆令牌：`m_t^k = α * m_t_new + (1-α) * m_{t-1}^k`。
            - 未匹配的观测添加为新条目，未匹配的旧记忆保留以应对临时遮挡。

    3.  **上下文记忆检索 (Contextual Memory Retrieval)**：
        - **目的**：在生成动作时，从精炼后的记忆`M_hat_t`中检索与当前指令和感知相关的空间上下文信息，并将其注入到VLA模型中。
        - **流程**：
            - VLM生成的视觉-语言令牌`X_vl`作为查询（Query），记忆`M_hat_t`作为键-值对（Key, Value）。
            - 通过一个交叉注意力模块，将记忆令牌投影到VLM特征空间，让`X_vl`中的每个令牌选择性地关注与其相关的空间区域。
            - 记忆增强的特征`X_boost`作为空间-语义先验，被注入到后续的DiT（Diffusion Transformer）块中，与当前观测、机器人状态和带噪动作嵌入共同处理，最终解码出动作块`{a_hat_t, a_hat_{t+1}, ...}`。

### 6. 论文的主要结论与发现

1.  **核心结论**：SOMA框架通过提供一个持久的、几何感知的空间记忆，有效地解决了VLA模型在“视觉外操作”（OOV）场景中的核心局限。这不仅仅是成功率的提升，更带来了**质上不同的操作行为**。
2.  **关键发现**：

    - **泛化能力**：在**常规完全可见**的场景（RoboCasa GR1, SimplerEnv）下，SOMA同样表现优异，甚至优于一些专门为全观测设计的模型。这说明空间记忆机制不仅对OOV有效，作为一种结构化归纳偏置，也能提升在标准场景下的学习效率和泛化能力。
    - **数据效率**：在RoboCasa GR1上，SOMA仅用30个演示就能达到甚至超过GR00T-N1.5在全量数据上的平均表现，展现了更强的数据效率。

    - **真实世界与仿真结合**：既在挑战性的真实OOV场景中证明了有效性，又在标准仿真benchmark上验证了泛化性。
    - **行为层面分析**：不只报告成功率，还深入分析了行为指标（首次注视时间、搜索长度等），揭示了SOMA带来的“质变”，说服力强。
    - **详细的消融实验**：几乎对每个设计选择都进行了消融，结果清晰有力地支持了论文的论点。
    - **公平对比**：与当前SOTA方法进行了一致和全面的对比。

### 8. 不足与局限

1.  **场景假设与局限性**：
    - **静态初始环境**：空间记忆构建依赖于静态场景下的短期扫描。在高度动态、密集杂乱或机器人大范围移动的复杂环境中，该方法可能不够鲁棒。
    - **预处理依赖性**：空间记忆构建（特别是几何和语义提取）在训练时是离线预处理的，虽然加速了训练，但实际部署中仍需要实时计算，增加了推理延迟（相比其他方法增加约0.2-0.3秒/动作块，文中表12）。
2.  **对感知模块的强依赖**：系统高度依赖VGGT（几何）、YOLO（检测）、DINOv3（特征提取）等外部预训练模型。这些模型本身的误差（如对细小或纹理不清晰物体的几何估计偏差）会直接传播到空间记忆中，导致失败（参考附录Table 15）。
3.  **记忆表征的粒度**：记忆操作在对象级别，缺乏“任务阶段感知”。例如，无法区分“抽屉打开”和“抽屉关闭”这种有意义的状态变化，在处理多阶段任务时可能受限。
4.  **通用性与可扩展性**：论文主要聚焦于桌面操作场景。扩展到移动操作或房间级规模的环境，需要引入分层记忆结构或SLAM等对齐技术，以控制长程漂移。作者已将其作为未来工作方向提及。
5.  **泛化到新物体/情境的验证不足**：虽然在仿真中展示了不同物体/布局的泛化性，但真实世界测试的物体种类相对固定（虽然有多种物体，但任务结构相似）。对全新物体或未见过任务布局的零样本泛化能力尚待进一步验证。
