## 摘要
现实世界中的精细操作，特别是双臂操作，通常需要低延迟控制和稳定的视觉定位，而收集大规模数据成本高昂，且有限的演示可能导致定位漂移。现有方法在不同方面进行了权衡：诸如 ACT 之类的动作分块（action-chunking）策略实现了低延迟执行和数据效率，但依赖于缺乏显式空间一致性的密集视觉特征；诸如 Diffusion Policy 之类的生成式方法提高了表达能力，但会产生迭代采样延迟；视觉-语言-动作（VLA）和基于体素的方法增强了泛化能力和几何基础，但需要更高的计算成本和系统复杂性。我们引入了一个多阶段空间注意力模块，该模块提取稳定的 2D 注意力点，并结合时间对齐损失联合预测未来的注意力序列。该多阶段注意力模块基于带有预训练 ResNet 视觉先验的 ACT 构建，提取与任务相关的 2D 注意力点作为动作预测的局部空间模态。为了保持一致的物体追踪，我们引入了一种自监督目标，将预测的注意力序列与未来帧的视觉特征相对齐，在无需关键点标注的情况下抑制漂移，并提高有限数据下视觉到动作映射的稳定性。在 ALOHA 双臂平台上进行的模拟和真实世界精细操作任务实验，评估了任务成功率、注意力漂移、推理延迟以及对视觉干扰的鲁棒性。结果表明，在测试条件下，该方法在保持低延迟推理的同时，提高了定位稳定性和任务性能。

## Abstract
Real-world fine manipulation, particularly in bimanual manipulation, typically requires low-latency control and stable visual localization, while collecting large-scale data is costly and limited demonstrations may lead to localization drift. Existing approaches make different trade-offs: action-chunking policies such as ACT enable low-latency execution and data efficiency but rely on dense visual features without explicit spatial consistency, generative methods such as Diffusion Policy improve expressiveness but can incur iterative sampling latency, vision-language-action and voxel-based methods enhance generalization and geometric grounding but require higher computational cost and system complexity. We introduce a multistage spatial attention module that extracts stable 2D attention points and jointly predicts future attention sequences with a temporal alignment loss. Built upon ACT with a pretrained ResNet visual prior, a multistage attention module extracts task-relevant 2D attention points as a local spatial modality for action prediction. To maintain consistent object tracking, we introduce a self-supervised objective that aligns predicted attention sequences with visual features from future frames, suppressing drift without keypoint annotations and improving stability of the vision-to-action mapping under limited data. Experiments on simulated and real-world fine manipulation tasks, conducted on the ALOHA bimanual platform, evaluate task success, attention drift, inference latency, and robustness to visual disturbances. Results indicate improvements in localization stability and task performance while maintaining low-latency inference under the tested conditions.

### 1. 核心问题与整体含义（研究动机和背景）
在机器人精细操作（尤其是双臂协作）中，**低延迟控制**和**稳定的空间定位**是成功的关键。
*   **研究背景**：现有的模仿学习方法存在权衡。例如，ACT（Action Chunking with Transformers）虽然延迟低且数据效率高，但依赖于高维视觉特征，缺乏显式的空间一致性约束，在视觉干扰下容易产生定位漂移；Diffusion Policy 表达能力强但推理延迟高；VLA（视觉-语言-动作）模型计算开销巨大。
*   **核心问题**：如何在有限的演示数据下，既保持极低的推理延迟，又能实现鲁棒的空间对齐，防止视觉定位漂移导致的精细操作失败。

### 2. 方法论：核心思想与关键技术
MSACT 在 ACT 框架的基础上，引入了显式的空间结构表征和自监督的时间对齐机制。
*   **核心思想**：通过一个轻量级的“多阶段空间注意力（MSA）”模块提取 2D 关键点，并要求模型预测未来的关键点序列，从而在时间维度上强制约束视觉特征的稳定性。
*   **关键技术细节**：
    *   **多阶段空间注意力 (MSA)**：采用三层 CNN 层次结构提取多尺度特征。通过计算 Query（各阶段特征）与 Key（输入图像的线性投影）的点积生成注意力图，再利用 **Soft-argmax** 操作提取 $N=6$ 个 2D 坐标点（注意力点）。
    *   **结构化输入**：将提取的 2D 坐标线性投影为 Token，与 ResNet 提取的全局视觉特征、机器人关节状态一同输入 Transformer 编码器。
    *   **自监督时间对齐损失 ($L_{att}$)**：模型不仅预测未来 $K$ 步的动作序列，还预测未来 $K$ 步的注意力点序列。训练时，将预测的序列与从未来真实帧中重新提取的注意力点进行 $L_1$ 损失计算。
    *   **总损失函数**：$L = L_{rec} + \beta D_{KL} + L_{att}$。其中 $L_{att}$ 在无需人工标注关键点的情况下，抑制了注意力漂移。

### 3. 实验设计
*   **实验平台**：ALOHA 双臂机器人平台（50Hz 控制频率）。
*   **数据集/场景**：
    *   **仿真环境**：方块传输（Cube Transfer）、双臂插入（Bimanual Insertion）。
    *   **现实世界**：拔网线（Detach Network Cable）、穿魔术贴（Thread Velcro）、插茶包（Insert Tea Bag）、开火柴盒（Open Match Box）。每个任务仅使用 50 份人类演示数据。
*   **Benchmark 与对比方法**：
    *   **ACT**：基准动作分块模型。
    *   **Diffusion Policy (DP)**：生成式动作预测模型。
    *   **SmolVLA (0.45B) & $\pi_{0.5}$ (3.6B)**：轻量级视觉-语言-动作大模型。
    *   **MSACT (SA)**：消融实验，使用传统的单阶段空间注意力。

### 4. 资源与算力
*   **硬件**：所有方法均在单张 **NVIDIA RTX 4090 GPU** 上部署。
*   **软件框架**：使用 Hugging Face 的 **LeRobot** 机器人学习框架。
*   **训练时长**：
    *   MSACT/ACT：训练 100K 步，Batch Size 为 8。
    *   Diffusion Policy：训练 200K 步，Batch Size 为 32。
    *   VLA 模型（SmolVLA/$\pi_{0.5}$）：微调 30K 步，Batch Size 为 32。

### 5. 实验数量与充分性
*   **实验规模**：现实世界实验非常充分，每个任务独立进行 100 次测试，总计 **400 次真实实验**。仿真实验每个任务运行 50 次，并使用 3 个随机种子取平均值。
*   **客观性与公平性**：所有模型在相同的数据集上训练，共享相同的动作分块长度（Chunk Size）、观察历史和控制频率。针对 VLA 和 DP 模型，应用了标准的推理优化（如 DDIM 步数缩减、缓存开启）以确保实时性对比的公平。
*   **消融与鲁棒性**：专门设计了针对视觉干扰（遮挡、光照变化、人为干扰）的鲁棒性测试，以及针对注意力模态贡献度的分析。

### 6. 主要结论与发现
*   **性能提升**：MSACT 在所有现实任务中均取得最高成功率。总成功率从 ACT 的 23.25% 提升至 **53.00%**，提升显著（$p < 0.001$）。
*   **低延迟保持**：MSACT 的推理延迟为 **45.40ms**，与 ACT（45.34ms）几乎持平，远低于 DP（158ms）和 VLA 模型（91-112ms）。
*   **稳定性**：自监督的时间对齐损失有效防止了注意力点在操作过程中的漂移，使其能始终锁定在任务相关的物体和夹爪上。
*   **鲁棒性**：在面对未见过的干扰物、光照变化和动态人为干扰时，MSACT 的成功率下降幅度远小于 ACT。

### 7. 优点（亮点）
*   **轻量且高效**：在不增加计算负担的前提下，通过引入显式的 2D 空间先验显著提升了性能。
*   **无需额外标注**：利用时间一致性实现自监督学习，避免了繁琐的人工关键点标注。
*   **解释性强**：通过可视化 2D 注意力点，开发者可以直观地观察模型在操作过程中“看”哪里，增加了端到端模型的透明度。
*   **硬件友好**：不需要深度传感器或复杂的相机标定，仅依靠 RGB 图像即可实现精细对齐。

### 8. 不足与局限
*   **几何感知局限**：在仿真实验中发现，若仅有单视角（Top View），模型在处理需要深度信息的空中对齐任务时提升有限，说明其仍依赖多视角提供的互补信息。
*   **2D 局限性**：虽然引入了空间点，但本质上仍是 2D 空间表征，对于极度复杂的 3D 几何关系理解可能不如基于体素（Voxel）或点云的方法。
*   **任务复杂度**：目前的实验主要集中在短时精细操作，对于需要长程逻辑推理的任务，尚未验证其与大语言模型结合的效果。
