## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署的机器人会遇到分布偏移、长尾故障、任务变化以及人类纠错机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略开始，通过利用在机器人车队中收集的自主展开和人类干预，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，用于基于流的 VLA 动作生成器中的策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和 3-5 分钟的长跨度任务。随着车队经验的积累，单一通用策略不断改进，平均成功率达到 95%，其中在长跨度任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

### 1. 核心问题与研究动机
*   **核心问题**：现有的通用机器人策略（如 VLA 模型）虽然通过大规模离线预训练获得了广泛的能力，但在实际部署中仍面临**分布偏移（Distribution Shift）**、**长尾失败**和**任务变体**等挑战。
*   **研究动机**：离线数据无法涵盖所有现实场景。作者认为“部署”不应是训练的终点，而应成为持续改进的源泉。论文旨在构建一个“数据飞轮”，通过机器人车队（Fleet）在部署过程中收集的自主尝试和人类干预数据，实现策略的持续在线进化。

### 2. 方法论：LWD 框架
LWD 采用“离线到在线”的强化学习架构，包含两个关键技术组件：
*   **分布隐式价值学习（DIVL, Distributional Implicit Value Learning）**：
    *   **核心思想**：针对车队数据异构、奖励稀疏且包含多种结果（成功、失败、恢复）的特点，DIVL 不使用标量价值估计，而是学习回报的**分类分布（Categorical Distribution）**。
    *   **技术细节**：借鉴 IQL（隐式 Q 学习）的非对称更新思想，通过提取分布的特定分位数（Quantile）作为 TD 目标，既保留了高回报模式，又避免了查询分布外动作导致的过估计。
    *   **自适应策略**：引入基于分布熵的自适应 $\tau$ 参数，在不确定性高时保持保守，在置信度高时追求乐观。
*   **伴随匹配 Q 学习（QAM, Q-learning with Adjoint Matching）**：
    *   **核心思想**：解决如何从学习到的价值函数中提取策略。
    *   **技术细节**：针对基于流匹配（Flow Matching）的 VLA 动作生成器，QAM 将价值函数的动作梯度转化为流模型的步进式监督信号。这避免了直接通过复杂的 ODE 求解过程进行反向传播，保证了训练的数值稳定性。
*   **训练流程**：分为离线预训练（在演示、历史尝试和人类引导数据上）和在线持续后训练（在车队实时回传的混合数据上）两个阶段。

### 3. 实验设计

    2.  **长程操纵任务**（4项）：包括泡功夫茶、制作鸡尾酒、制作果汁、鞋盒打包。每项任务持续 3-5 分钟，包含 5-8 个子步骤。
*   **对比基准（Baselines）**：

    *   **RECAP**：一种基于优势加权的离线强化学习后训练方法。
    *   **HG-DAgger**：交互式模仿学习，利用人类干预数据。
    *   **LWD (Offline)**：仅使用离线数据的 LWD 版本。

### 4. 资源与算力

*   **实验规模**：在 8 个复杂任务上进行了评估，涵盖了从短程语义任务到极长程的接触密集型任务，实验设置具有较高的挑战性。

### 6. 主要结论与发现
*   **性能卓越**：LWD 在所有任务上的平均成功率达到 **95%**，显著优于 SFT (76%)、RECAP (85%) 和 HG-DAgger (85%)。
*   **长程任务优势**：在泡茶、调酒等长程任务中，LWD 的提升最为巨大，证明了强化学习在处理稀疏奖励和错误恢复方面的能力。
*   **效率提升**：相比 SFT，LWD 训练出的策略在完成任务时更加果断，平均循环时间（Cycle Time）缩短了约 23.75 秒。
*   **价值函数有效性**：可视化证明了学习到的分布价值函数能够准确追踪任务进度，并在失败发生前给出预警信号。

### 7. 亮点
*   **车队级规模化**：展示了如何通过分布式机器人集群构建闭环学习系统，是目前真实世界机器人 RL 领域规模较大的工作。
*   **算法稳定性**：DIVL 和 QAM 的结合解决了 VLA 模型在强化学习中难以收敛和不稳定的痛点。
*   **数据利用率**：能够同时利用成功演示、失败尝试、人类干预和自主探索数据。

### 8. 不足与局限
*   **安全建模缺失**：目前的框架未显式建模执行安全性，在处理更危险的工具或环境时可能存在风险。
*   **推理能力限制**：长程任务仍依赖简单的语言指令，对于需要复杂逻辑推理或多轮对话的任务，当前的 VLA 架构可能不足。
*   **更新频率**：在线学习采用固定的实时调度，对于更大规模（如成千上万台）的部署，如何更高效地同步策略仍需研究。
*   **算力成本**：虽然交互时间短，但后台运行大型 VLM 的强化学习训练对计算资源的需求极高。
