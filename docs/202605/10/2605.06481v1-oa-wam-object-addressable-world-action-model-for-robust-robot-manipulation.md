## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记（tokens）或全局潜变量。当指令涉及特定对象时，动作解码器很难对这些表示进行寻址，特别是在场景偏移导致对象身份与上下文纠缠的情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并以块因果序列的形式与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码一个 16 步的连续动作块。通过仅通过地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，从而强制实现可寻址性，在不添加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了与强 VLA 和 WAM 基准相当的水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试得出的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

#### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）在标准基准测试中表现优异，但在面对场景扰动（如摄像头视角切换、物体布局改变、背景变化）时表现出极差的鲁棒性。
*   **核心痛点**：大多数模型将场景表示为整体图像标记或全局潜在变量，导致目标物体的“身份”与周围的“上下文”纠缠在一起。当场景发生几何偏移时，动作解码器难以稳定地定位指令所指的对象。
*   **研究动机**：提出“对象可寻址性”（Object Addressability）概念，即模型应能通过稳定的接口直接查询任务相关的物体，而不受环境噪声或布局变化的影响。

#### 2. 核心方法论
OA-WAM 提出了一种将世界建模与动作生成解耦的对象级架构：
*   **对象槽位分解（Object-Slot Decomposition）**：将每一帧分解为 $N+1$ 个槽位（1个机器人槽位，$N$个物体槽位）。每个槽位向量由两部分组成：
    *   **地址向量 ($addr_k$)**：基于语言标签和初始特征计算，在整个任务过程中保持**冻结**，代表物体身份。
    *   **内容向量 ($cnt_k^t$)**：每帧更新，代表物体的实时状态（姿态、外观）。
*   **地址路由注意力机制（Address-only Key Projection）**：在 Transformer 的每一层中，跨槽位注意力的“键”（Key）投影仅读取 $addr$ 部分。这意味着“动作头关注哪个物体”是由物体身份决定的，而非其当前位置或背景。
*   **残差流重置（Residual Reset）**：在每层 Transformer 后，强制将残差流中的地址切片重置为初始值，防止身份信息在深度网络中发生漂移。
*   **统一序列建模**：使用 7B 参数的 Chameleon 风格骨干网络，融合文本、图像 VQ、槽位、本体感受和过去动作标记。
*   **双头输出**：
    *   **世界头**：回归预测下一帧的槽位内容和姿态。
    *   **动作头**：采用流匹配（Flow-matching）解码 16 步连续动作块。

#### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准操纵基准（Spatial, Object, Goal, Long 四个子集）。
    *   **SimplerEnv (WidowX/Bridge)**：评估模拟到现实的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性测试，包含摄像头、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动轴。
*   **对比方法**：包括 SOTA VLA 模型（OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1）和 WAM 模型（VLA-JEPA, Cosmos-Policy, WorldVLA 等）。
*   **因果干预测试**：通过交换两个物体的地址向量，观察机器人轨迹是否发生相应偏移，以验证模型是否真正实现了对象绑定。

#### 5. 实验数量与充分性
*   **实验规模**：所有报告数据均为 3 个随机种子的平均值。涵盖了从标准性能到极端 OOD（分布外）扰动的全面评估。

#### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超越了现有 SOTA。
*   **几何鲁棒性极强**：在 LIBERO-Plus 的几何扰动轴（摄像头、机器人状态、布局）上，OA-WAM 显著优于 $\pi_{0.5}$ 等强基准，几何平均成功率提升了 4.8%。
*   **可验证的绑定**：在地址交换干预中，OA-WAM 的绑定余弦值达到 0.87，而所有整体式（Holistic）基准模型均低于 0.09，证明了其决策过程确实是基于对象的。

#### 7. 优点
*   **架构创新**：通过简单的张量级操作（掩码和重置）实现了对象寻址，无需引入复杂的额外标记或模块。
*   **鲁棒性解耦**：成功将“物体识别”与“动作生成”在特征空间分离，解决了 VLA 模型常见的“因背景改变而动作漂移”的问题。
*   **世界模型集成**：将对象级预测作为辅助任务，增强了模型对物理演变的理解。

#### 8. 不足与局限
*   **感知瓶颈**：模型高度依赖上游的感知组件（SAM3, DINOv3）。在“传感器噪声”测试中表现不佳（-17.1%），原因是图像噪声导致槽位提取失败，而非策略本身失效。
*   **推理延迟**：虽然动作生成很快，但前端感知（SAM3+Qwen3-VL）耗时较长（~138ms），限制了更高频率的实时控制。
*   **环境假设**：目前的实验主要在模拟器中完成，尚未在真实机器人上进行大规模部署验证。
*   **复杂场景限制**：对于极小、透明、反光或严重遮挡的物体，现有的槽位提取机制可能失效。
