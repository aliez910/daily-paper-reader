## 摘要
视觉-语言-动作（VLA）模型在训练中见过的机器人任务上表现良好，但在泛化到未见场景和物体时面临挑战。一个关键限制在于其隐式视觉表示，这种表示将物体外观、背景和场景布局纠缠在一起，导致策略对视觉变化非常敏感。先前的工作通过将视觉内容对象化的结构化中间表示来提高可迁移性。然而，这些表示主要捕捉场景语义，而非与动作相关的关系。因此，动作预测仍然受限于外观统计数据。我们观察到，操纵动作取决于“物体-手-任务”三元关系结构，该结构支配着任务需求、机器人状态和物体属性之间的交互。基于这一观察，我们提出了 TriRelVLA，一个用于可泛化具身操纵的三元关系 VLA 框架。我们的方法包含三个组件：1）我们从多模态输入中构建显式的物体-手-任务三元表示作为关系原语。2）我们构建了一个基于任务的关系图。任务引导的交叉注意力形成节点，而关系感知图变换器（Graph Transformer）对它们之间的交互进行建模。3）我们执行基于关系条件的动作生成。关系结构被压缩到一个瓶颈空间中，并投影到大语言模型（LLM）中进行动作预测。这种三元关系瓶颈减少了对外观统计数据的依赖，并实现了跨场景、跨物体和跨任务组合的迁移。我们还进一步引入了一个用于微调的真实世界机器人数据集。实验表明，该方法在微调任务上表现强劲，并在跨场景、跨物体和跨任务泛化方面取得了显著提升。

## Abstract
Vision-language-action (VLA) models perform well on training-seen robotic tasks but struggle to generalize to unseen scenes and objects. A key limitation lies in their implicit visual representations, which entangle object appearance, background, and scene layout. This makes policies sensitive to visual variations. Prior work improves transferability through structured intermediate representations that objectify visual content. However, these representations mainly capture scene semantics instead of action-relevant relations. As a result, action prediction remains tied to appearance statistics. We observe that manipulation actions depend on the object-hand-task relational structure, which governs interactions among task requirements, robot states, and object properties. Based on this observation, we propose TriRelVLA, a triadic relational VLA framework for generalizable embodied manipulation. Our approach consists of three components: 1) We construct explicit object-hand-task triadic representations from multimodal inputs as relational primitives. 2) We build a task-grounded relational graph. Task-guided cross-attention forms nodes, and a relation-aware graph transformer models interactions among them. 3) We perform relation-conditioned action generation. The relational structure is compressed into a bottleneck space and projected into the LLM for action prediction. This triadic relational bottleneck reduces reliance on appearance statistics and enables transfer across scenes, objects, and task compositions. We further introduce a real-world robotic dataset for fine-tuning. Experiments show strong performance on fine-tuned tasks and clear gains in cross-scene, cross-object, and cross-task generalization.

现有的视觉-语言-动作（VLA）模型在处理训练集内的任务时表现出色，但在面对未见过的场景（背景变化）、新物体（外观变化）或任务组合时，泛化能力显著下降。其根本原因在于：
*   **隐式表示的局限性**：模型通常将物体外观、背景纹理和场景布局纠缠在稠密的视觉 Token 中，导致策略对视觉细节过于敏感。
*   **缺乏动作相关逻辑**：现有的结构化方法多侧重于场景语义（如关键点、语义图），而忽略了操纵任务中核心的“物体-机械臂-任务”之间的交互逻辑。
论文旨在通过构建显式的**三元关系结构（Triadic Relational Structure）**，使模型从关注“物体长什么样”转向关注“物体、手与任务之间的交互关系”，从而提升泛化性。

#### 2. 方法论
TriRelVLA 框架分为三个核心阶段：
*   **三元表示构建（Triadic Representation）**：
    *   从多模态输入中提取三种原语：**物体（Object）**（通过学习到的查询从 3D 视觉潜空间中提取）、**手（Hand）**（利用机器人本体感受状态作为锚点提取）、**任务（Task）**（将指令分解为动作、角色、约束和阶段四个维度）。
    *   视觉编码结合了 SigLIP（语义特征）和 VGGT（几何特征），形成统一的 3D 视觉潜空间。
*   **任务引导的关系图（Task-Grounded Relational Graph）**：
    *   **节点构建**：利用任务 Token 通过交叉注意力引导物体和手 Token，形成任务相关的图节点。
    *   **关系建模**：定义四种边（任务-物体、任务-手、物体-手、物体-物体），捕捉几何位置、姿态、距离和接触得分。
    *   **图变换器**：使用关系感知图变换器（Relation-aware Graph Transformer）更新节点特征，使交互信息在结构中传播。
*   **关系条件动作生成（Action Generation）**：
    *   **关系瓶颈（Relational Bottleneck）**：将增强后的节点特征压缩为紧凑的动作相关 Token，过滤冗余信息。
    *   **LLM 预测**：将压缩后的关系 Token 与语言 Token 拼接，输入大语言模型（如 Qwen3-4B）预测 7 自由度的动作参数。
*   **优化策略**：采用动作损失（L1）以及物体/手掩码的辅助监督损失（BCE），确保 Token 能够准确对齐物理实体。

*   **实验规模**：在 LIBERO 的四个子集和 CSOT-Bench 的三个维度（场景、物体、任务）上进行了详尽测试。

#### 6. 主要结论与发现
*   **泛化性显著提升**：TriRelVLA 在跨任务泛化上的表现远超隐式 VLA 模型，证明了显式建模交互关系比单纯增加视觉 Token 密度更有效。
*   **关系瓶颈的作用**：引入瓶颈空间不仅降低了计算开销（GFLOPs 下降），还通过过滤任务无关的物体干扰提升了成功率。
*   **本体感受的重要性**：将机械臂状态（Proprioception）引入“手”的表示中，能显著增强模型在遮挡或视角变化下的鲁棒性。

#### 7. 优点
*   **架构创新**：提出了“物体-手-任务”的三元结构，精准捕捉了操纵任务的本质。
*   **抗干扰能力强**：通过结构化表示和瓶颈空间，有效解耦了物体外观与操纵逻辑，减少了对背景和纹理的依赖。
*   **数据集贡献**：CSOT-Bench 提供了丰富的多模态标注（尤其是掩码），对后续研究具有参考价值。

#### 8. 不足与局限
*   **长程任务挑战**：在处理极长时序的组合任务或完全未见过的复杂任务时仍有困难。
*   **缺乏长期记忆**：目前模型主要基于当前观测，缺乏对历史状态和多阶段子目标的显式时空记忆建模。
*   **应用限制**：虽然引入了 3D 几何特征，但在极端精细的力控操纵或高速动态交互中的表现尚未深入探讨。
