## Abstract
This paper presents a phase-conditioned, force-aware framework for robust deformable object manipulation. Standard imitation learning policies such as Action Chunking with Transformers (ACT) rely on a Markovian assumption at inference, causing state aliasing when visually similar observations require contradictory actions and preventing autonomous recovery from execution failures. We address this with a closed-loop hierarchical architecture. A FiLM-conditioned ACT encoder modulates feature extraction based on the current task phase, enabling a single unified policy to produce phase-specific behaviors while sharing action dynamics across phases. A multi-modal phase predictor fusing visual, force, and pose feedback estimates the phase in real time, detecting contact failures that are invisible to vision alone and autonomously triggering recovery trajectories. The system is completed by a hybrid impedance controller for compliant execution and a haptic teleoperation interface for force-aware data collection. Ablation studies show that FiLM-based modulation significantly outperforms both unconditioned and token-level conditioned baselines, and t-SNE analysis confirms that FiLM induces well-separated, phase-specific feature representations. Validated on hanging and removing a T-shirt with dual arms, the closed-loop system improves the hanging success rate from 56\% to 87\% through autonomous error recovery. # 基于阶段条件模仿学习与自主故障恢复的鲁棒可变形物体操作（论文总结）

## 1. 核心问题与整体含义（研究动机和背景）
- **背景**：可变形物体操作（DOM，如挂取衣物）因高自由度、严重自遮挡和复杂接触动力学而极具挑战性。现有模仿学习方法（如基于Transformer的动作分块ACT、扩散策略DP）在推理时基于马尔可夫假设，将当前观测直接映射到未来动作。
- **问题**：在长时程任务中，不同任务阶段可能呈现视觉上相似的观测，但需要完全相反的动作（例如，正常插入阶段 vs. 故障恢复阶段都需要“向前推进”vs.“后退”）。这导致**状态混叠**：策略学到混合的、矛盾的动作，产生平均或振荡行为。此外，标准方法无法从执行故障中自主恢复。
- **意义**：提出一种闭环分层架构，引入**阶段条件化**（phase conditioning）和**自主故障恢复**（failure recovery），显著提升DOM任务的鲁棒性。

## 2. 方法论
### 核心思想
- 将任务分解为多个阶段（phase），每个阶段对应一个子任务或状态（正常阶段+恢复阶段）。策略**条件于**当前阶段，从而在视觉相似的观测下产生阶段特定的、确定性的行为。
- 在ACT编码器中通过**FiLM（Feature-wise Linear Modulation）**注入阶段信息，强制调制每一层的特征提取，避免阶段信号被大量视觉token稀释。
- 设计一个**多模态阶段预测器**（融合视觉、力和位姿），实时检测当前阶段和故障（如卡住、滑落），并自动切换至相应的恢复阶段，形成闭环。
- 底层使用**混合阻抗控制器**实现柔顺执行，并配备**触觉遥操作接口**收集力感知演示。

### 关键技术细节
- **FiLM调制的ACT**：
  - 四张RGB图像经共享ResNet-18提取特征，加上位姿（6D旋转）和力/力矩信息，与阶段嵌入一起输入Transformer编码器。
  - FiLM对编码器每一层的LayerNorm和MLP输出施加仿射变换（尺度+偏移），使同一观测在不同阶段下产生不同的特征表示。
  - 解码器保持阶段无关，学习跨阶段共享的低级动作动力学（如到达、插入、后退）。
  - 训练时使用CVAE编码器学习动作序列的潜在分布，推理时使用先验均值确定性地解码动作块。
- **阶段预测器**：
  - 共享策略网络的冻结视觉骨干和位姿/力投影层。
  - 对每张图像特征图做全局平均池化，加上任务ID嵌入（限制预测空间），拼接后经MLP输出各阶段的softmax概率。
  - 在阶段切换时重置ACT时序集成器，并应用1秒低通滤波抑制假阳性切换。
- **混合阻抗控制器**：
  - 策略以15Hz输出目标末端位姿，控制器内插生成高频平滑轨迹，同时叠加增量命令（目标-当前位姿）。
  - 根据夹具状态切换控制模式：夹爪闭合时启用力导向模式（柔顺接触），张开时用纯阻抗模式（防止漂移）。
  - 底层通过重力补偿阻抗控制器实现力位混合控制。
- **触觉遥操作**：利用Meta Quest Pro的振动反馈实时映射力传感器读数，使操作者感知视觉不可见的接触状态。

## 3. 实验设计

  - **Phase 9**：完全错过衣架（gross misalignment），执行全局复位（retreat）并重试。
  - **Phase 10**：取下时衣服滑落（mid-air sliding），执行动态空中重新抓取。
- **数据集**：

  - 恢复任务：每种恢复场景30个演示，分别约2.3、1.4、5.2分钟。

## 6. 主要结论与发现
- **FiLM调制**是实现阶段条件化的关键：相比阶段作为输入token，FiLM强制调制每一层，编码器输出特征形成良好分离的阶段簇，消融实验在所有扰动条件下获得最高成功率（正常90%，错位80%，卡住100%）。
- **自主故障恢复**显著提升鲁棒性：挂取闭环系统最终成功率达87%（无恢复仅56%）；取下任务达92%（无恢复88%）。恢复了90.91%的检测到故障（挂取）和66.67%（取下）。
- **合理范围**：恢复策略有效，但取下任务的恢复率较低（50%），受限于空间感知能力（无法精确判定抓取点）。
- **高速执行**：直接跳帧动作块会降低平滑度和精度，总执行时间反而更长，表明当前系统更适合15Hz的平稳慢速操作。

## 7. 优点
- 创新性地将**阶段条件化**与**力反馈闭环恢复**结合，解决了状态混叠和自恢复这一对DOM中的关键瓶颈。
- FiLM调制机制设计精巧：在编码器层调制而非简单加token，确保阶段信号不会被大量视觉特征淹没；解码器保持无阶段共享，提升数据效率。

## 8. 不足与局限
