## 摘要
我们提出了SOMA，即面向视觉-语言-动作（VLA）模型中视野外操作的空间记忆框架。现有的大多数VLA模型隐式假设任务相关物体始终可见，导致当目标物体位于相机视野之外时产生脆弱且反应式的行为。SOMA通过为VLA配备一个基于可移动头部相机获取的多视角观测构建的持久空间记忆，从而实现对当前视锥之外的推理。该框架由三个组件组成：空间记忆构建，通过扫描将角度观测聚合为统一的空间语义表示；动态记忆精化，保持全局一致性随时间推移；以及上下文记忆检索，在操作过程中激活与指令相关的空间线索。我们在五个具有挑战性的真实世界视野外操作任务上评估了SOMA，包括目标物体初始不可见的多步骤和双臂场景。实验结果表明，SOMA不仅提高了任务成功率，还引发了质上不同的操作行为，包括更快的目标定位、减少的视角搜索以及在部分可观测性下的近一次抓取。在RoboCasa GR1和SimplerEnv上的额外实验进一步验证了SOMA记忆设计在传统完全可观测设置下的有效性。代码即将发布。

## Abstract
We introduce SOMA, the Spatial Memory framework for Out-of-Vision Manipulation in Vision-Language-Action (VLA) models. Most existing VLAs implicitly assume that task-relevant objects are always visible, leading to brittle and reactive behaviors when targets fall outside the camera's field of view. SOMA addresses this limitation by equipping VLAs with a persistent spatial memory constructed from multi-view observations acquired via a movable head camera, enabling reasoning beyond the current visual frustum. The framework consists of three components: Spatial Memory Construction, which aggregates angular-wise observations into a unified spatial-semantic representation through scanning; Dynamic Memory Refinement, which maintains global consistency over time; and Contextual Memory Retrieval, which activates instruction-relevant spatial cues during manipulation. We evaluate SOMA on five challenging real-world out-of-vision manipulation tasks, including multi-step and dual-arm scenarios where target objects are initially invisible. Experimental results show that SOMA not only improves task success rates, but also induces qualitatively different manipulation behaviors, with faster target localization, reduced viewpoint search, and near one-shot grasping under partial observability. Additional experiments on RoboCasa GR1 and SimplerEnv further validate the effectiveness of SOMA's memory design under conventional fully observable settings. Code will be released soon.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
SOMA为VLA模型配备持久空间记忆，该记忆由可移动头部摄像头获取的多视角观测构建，支持对视野外物体的推理和操作。框架包含三个关键阶段：空间记忆构建、动态记忆精化、上下文记忆检索。

### 关键技术细节

#### (A) 空间记忆构建（Spatial Memory Construction）
- **输入**：通过可移动头部摄像头沿预定扫描轨迹获取的视频序列 \( V = \{f_i\}_{i=1}^{N_v} \)。
- **采样**：每隔 \( N \) 帧均匀采样得到子集 \( \tilde{V} \)。
- **统一感知管道**：
  - 几何先验网络 VGGT：预测相机姿势和粗略场景几何。
  - 语义感知模块 YOLO：目标定位与分类。
  - 语义编码器 DINOv3：提取高维视觉特征。
- **处理流程**：
  - 每帧检测到的2D边界框通过空间平均池化得到实例外观嵌入 \( f_j^{(i)} \)。
  - 结合VGGT的几何先验，将2D检测提升到全局3D场景坐标系，得到3D边界框 \( b_j^{(i)} \)。
  - 跨视图实例关联与融合：基于DINOv3余弦相似度和3D空间一致性，合并同类实例，平均外观和几何特征。
  - 最终每个全局实例表示为三元组 \( (f_k, c_k, b_k) \)。
- **记忆令牌生成**：3D边界框通过嵌入模块 \( \Phi_{pos} \) 得到空间位置向量 \( p_k \)；外观特征通过 \( \Phi_{mem} \) 投影；组合得到 \( m_k^0 = \Phi_{mem}(f_k) + p_k \)；所有令牌构成初始场景记忆 \( M_0 \)。

#### (B) 动态记忆精化（Dynamic Memory Refinement）
- **目标**：在操作过程中，环境变化（物体移动、遮挡等）时保持记忆的全局一致性。
- **过程**：当前头部相机观测 \( o_t^h \) 经相同管道得到当前记忆令牌 \( M_t \)。
- **实例关联**：对每个新观测令牌 \( m_t^j \)，在上一记忆 \( M_{t-1} \) 中寻找同类的匹配实例。
- **自适应融合**：
  - 计算语义相似度 \( s_{tkj} = \sigma(\Phi_{sim}([m_{t-1}^k - m_t^j])) \)。
  - 计算动态融合分数 \( g_{tkj} = \sigma(\Phi_{fuse}([m_{t-1}^k, m_t^j])) \)。
  - 自适应融合系数 \( \alpha_{tkj} = g_{tkj} \cdot s_{tkj} \)。
  - 更新记忆令牌：\( m_t^k = \alpha_{tkj} m_t^j + (1 - \alpha_{tkj}) m_{t-1}^k \)。
- **未匹配与保留**：新观测未匹配则追加为新条目；过去未匹配条目保留以应对临时遮挡。

#### (C) 上下文记忆检索（Contextual Memory Retrieval）
- **目的**：将全局空间语义知识注入多模态表示，以支持指令引导的推理。
- **过程**：
  - VLM的视觉-语言令牌 \( X_{vl} \) 作为查询，场景记忆 \( \hat{M}_t \) 作为键-值存储。
  - 记忆令牌通过对齐函数 \( \tilde{m}_t^k = \Phi_{align}(\hat{m}_t^k) \) 投影到VLM特征空间。
  - 通过交叉注意力实现检索：\( X_{boost} = \text{softmax}(QK^\top / \sqrt{C})V \)。
- **输出**：增强后的特征 \( X_{boost} \) 注入DiT块，与当前观测、机器人状态、带噪声动作一起处理，最终由动作解码器生成下一动作块。

## 7. 优点：方法或实验设计上的亮点

### 方法亮点

- **多视角一致性处理**：通过实例关联融合多视角信息，支持全局语义-几何联合表示。
- **支持可移动头部相机**：主动扫描+记忆结合，与固定相机设置根本不同。

## 8. 不足与局限

### 实验覆盖方面
- **真实世界任务规模较小**：每任务仅20个episodes，可能不足以完全反映实际分布的方差；物体数量和分布相对简单。
- **动态环境鲁棒性有限**：空间记忆构建假设准静态场景，动态场景下（如快速移动物体、频繁遮挡）未充分测试。
- **语义粒度有限**：记忆仅保持对象级位置和外观，无法区分语义状态变化（如抽屉打开vs关闭），多阶段任务表现受限。
- **故障分析显示主要失败原因**：真实场景中48%失败来自动态视角下的抓取姿势估计误差；仿真中44%来自不完美3D估计导致的噪声空间令牌。

### 方法局限
- **依赖短时扫描**：构建记忆需额外扫描时间，在大范围或复杂场景中可能耗时。
- **头-臂协调未显式建模**：头部移动时执行抓取存在运动耦合问题，导致精度下降。
- **阈值依赖**：实例关联和融合使用固定相似度阈值，在密集场景或类别相似时可能误关联。
- **未考虑尺度扩展**：当前为桌面级，向移动操作或房间级扩展需层次化记忆和闭环对齐。

### 偏差风险
- **仅使用特定硬件平台**（Realman ZM73双臂 + Intel Realsense D435），可能引入平台偏差。
- **VR遥操作数据**可能隐含操作者习惯偏好，且数据量有限（每任务400个演示）。
- **对比方法未全覆盖**：真实OOV任务中仅对比了固定头部和少数活跃头部VLA，未与所有SOTA（如π0等）在同等条件下对比。

### 应用限制
- **安全性**：持久记忆可能存储错误信息，在共享空间需置信度评估和刷新机制。
- **隐私与伦理**：空间记忆本质是视觉场景记忆，需考虑数据存储和潜在滥用风险。
- **推理延迟**：虽然论文报告延迟约1.58s/动作块（与其他方法相当），但实时性仍受限于感知管线和记忆检索开销。
