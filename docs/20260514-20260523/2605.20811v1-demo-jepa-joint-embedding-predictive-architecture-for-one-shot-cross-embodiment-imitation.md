## 摘要
机器人模仿学习通常被视为复现演示动作，但动作本质上是特定于本体的。当演示来自人类或具有不同形态、运动学或动作空间的机器人时，这种以动作为中心的视角需要共享动作空间、启发式重定向或大规模多体协同训练。我们转而将演示视为未来目标的隐式规范：目标智能体应推断演示者试图实现的状态，而非演示者如何执行。我们提出Demo-JEPA，一种跨体模仿框架，将演示意图与本体特定执行解耦。基于JEPA世界模型，Demo-JEPA将源视觉演示转换为共享预测表示空间中的目标兼容未来潜轨迹。然后，目标智能体将这些潜轨迹作为子目标，通过在其自身学习的前向动力学下进行规划来实现。由于Demo-JEPA避免了动作级对应，仅需视觉演示和目标智能体的自身交互经验，它支持跨异构本体的灵活模仿。在RLBench和真实世界操作任务上的实验表明，Demo-JEPA匹配了专门领域规划器，并泛化至先前方法失败的新任务和本体配置。

## Abstract
Robotic imitation learning is often treated as reproducing demonstrated actions, but actions are inherently embodiment-specific. When demonstrations come from humans or robots with different morphology, kinematics, or action spaces, this action-centric view requires shared action spaces, heuristic retargeting, or large-scale multi-embodiment co-training. We instead view demonstrations as implicit specifications of future goals: the target agent should infer what state the demonstrator is trying to realize, rather than how the demonstrator executes it. We propose Demo-JEPA, a cross-embodiment imitation framework that decouples demonstration intent from embodiment-specific execution. Built on a JEPA-based world model, Demo-JEPA translates source visual demonstrations into target-compatible future latent trajectories in a shared predictive representation space. The target agent then uses these latent trajectories as subgoals and realizes them through planning under its own learned forward dynamics. Because Demo-JEPA avoids action-level correspondence and requires only visual demonstrations plus the target agent's own interaction experience, it supports flexible imitation across heterogeneous embodiments. Experiments on RLBench and real-world manipulation tasks show that Demo-JEPA matches specialized in-domain planners and generalizes to unseen tasks and embodiment configurations where prior methods fail.

### 1. 核心问题与整体含义（研究动机和背景）

- **核心问题：** 机器人模仿学习中普遍存在的 **“体态差距”（Embodiment Gap）**。即，演示者（如人类、其他机器人）与学习者的物理形态、运动学和动作空间存在根本性差异，导致直接模仿动作（Action-level Imitation）在跨异构体态时效率低下且泛化能力差。
- **研究动机：** 传统方法依赖动作级对齐（如共享动作空间、启发式重定向、大规模多体态协同训练），成本高昂且对异构体态泛化困难。人类可以理解演示者的意图并根据自身身体条件执行，而机器人难以做到。作者受此启发，希望将演示的 **“意图”（Intent）** 与 **“执行”（Execution）** 分离。
- **整体含义：** 论文提出一种新的跨体态模仿范式，不再将演示视为需复现的动作序列，而是将其视为对未来状态的隐式规范（Implicit Goal Specification）。目标机器人应推断演示者试图达到的 *状态*，而非 *如何到达该状态*。

### 2. 方法论：核心思想、关键技术细节

- **核心思想：** 将跨体态模仿问题转化为 **基于共享预测表示空间中的潜目标条件规划（Latent Goal-Conditioned Planning）**。即，从源演示中推断出与目标体态兼容的潜目标轨迹，然后目标体态利用自身学习的前向动力学模型，通过规划来达成这些子目标。
- **关键技术：Demo-JEPA框架**
    1.  **基础架构（JEPA世界模型）：** 采用V-JEPA 2.1作为动作条件世界模型。一个JEPA编码器 `E()` 将观测 `o` 映射为潜状态 `z`；一个动力学预测器 `F_wm()` 基于潜状态、机器人状态和动作预测下一时刻潜状态：`ˆz_{k+1} = F_wm(z_k, s_k, a_k)`。
    2.  **Dreamer Predictor（核心组件）：** 一个体态感知的潜预测器，负责跨体态的未来状态转换。
        - **输入：** 当前目标体态观测 `o_t` 以及从源演示中采样的一对帧 `(o_s, o_{s+n})`（n为时间偏移）。
        - **处理流程：** 所有观测经共享编码器得到潜表示 `(z_t, z_s, z_{s+n})`。
            - **跨体态对应模块（Cross-Embodiment Correspondence）：** 使用交叉注意力`f_emb = Attn(Q=z_t, K=z_s, V=z_s)`建立源和目标体态间的对应。
            - **时间演化模块（Temporal Evolution）：** 使用交叉注意力`f_mot = Attn(Q=z_{s+n}, K=z_s, V=z_s)`捕捉演示中的动态趋势。
            - **特征融合：** 将 `z_t`、`f_emb` 和 `f_mot` 拼接，通过一个3D卷积 `ϕ` 进行融合，得到融合特征 `f_fused`。
            - **解码：** 一个Transformer预测器 `T` 将 `f_fused` 解码为最终的潜目标 `ˆz_goal`。
    3.  **训练流程（三阶段）：**
        - **阶段 0：** 预训练目标体态的动作条件世界模型（使用目标体态自身的交互经验）。
        - **阶段 1：潜目标预测器训练：** 冻结世界模型编码器和动力学预测器，仅训练Dreamer Predictor。优化目标是最大化预测潜目标 `ˆz_goal` 与真实目标未来状态 `z_{k+n}` 的相似度（L2损失）。引入**时间扰动**（Temporal Perturbation）增强泛化性。
        - **阶段 2：动作协同训练：** 冻结Dreamer Predictor，微调世界模型的动力学预测器 `F_wm`。目标是让世界模型的轨迹预测分布与Dreamer Predictor的输出分布对齐，提升下游规划稳定性。优化目标是最小化规划损失 `L_plan`。
    4.  **推理流程（自适应目标更新）：**
        - 在每个时间步，Dreamer Predictor根据当前观测和源演示的参考帧对推断出潜目标 `ˆz_goal`。
        - 目标体态使用交叉熵方法（CEM）在世界模型中规划最优动作序列，以最小化潜状态与目标的距离。
        - **自适应更新机制：** 规划并执行动作后，计算到达的新潜状态与当前潜目标的差异 `D`。如果 `D < 阈值`，则认为子目标达成，前进到下一个源参考帧对；否则，保持当前目标不变，继续使用CEM规划。此机制有效避免了因体态差异导致的时序累积误差。

### 3. 实验设计

- **数据集/场景：**
    1.  **模拟环境：** RLBench。源体态为Sawyer机械臂，目标体态为Franka机械臂。
    2.  **真实世界：** UR5e机械臂（源）与Franka机械臂（目标）的物理操作任务。
- **Benchmark：** 论文将评估分为三个难度递增的套件（Suites），用于衡量不同程度的分布漂移（Distribution Shift）：
    1.  **行为基础（Behavior Grounding）：** 评估训练阶段已见过的任务。
    2.  **跨体态桥接（Cross-Embodiment Bridging）：** 使用仅在阶段一（潜目标预测器训练）中出现，但阶段二（动作协同训练）中未见过的任务，测试无直接动作监督下的执行能力。
    3.  **零样本泛化（Zero-Shot Generalization）：** 评估在训练阶段完全未见的新任务和体态配置上的泛化能力。
- **对比方法：**
    - **VPP (Video Prediction Policy)：** 一种基于视频预测的策略学习方法。
    - **XSkill：** 一种通过匹配异构视频到共享技能原型来执行技能序列的方法。
    - **V-JEPA 2.1 (Naive / Oracle)：** 作为消融基线，对比直接使用源演示的未来潜状态（Naive，失败）和理想的目标体态真实未来潜状态（Oracle，上界）。
    - **Demo-DP：** 将Dreamer Predictor预测的潜目标作为扩散策略（Diffusion Policy, DP）的条件输入，用于生成动作。

### 6. 主要结论与发现

1.  **性能表现：** 在**行为基础**（标准设定）上，Demo-JEPA稍逊于专门优化的VPP。但在更具挑战性的**跨体态桥接**和**零样本泛化**设定下，Demo-JEPA显著优于VPP和XSkill，随着分布漂移增大，优势更加明显。
2.  **核心优势：** 将演示视为“潜目标”而非“动作序列”，并结合基于规划的执行方式，是实现强泛化能力的关键。Dreamer Predictor能有效提取演示中的抽象意图并转换为目标体态兼容的潜目标。
3.  **组件价值：**
    - Dreamer Predictor结合3D卷积融合模块对于处理复杂、结构化的运动至关重要。
    - CEM规划器在零样本泛化场景下优于受分布限制的Diffusion Policy（Demo-DP）。
4.  **数据效率：** 任务多样性是训练有效跨体态对应（Source-to-Target mapping）的主要驱动力，比单纯增加单个任务的数据量更重要。

### 7. 优点（方法或实验设计亮点）

- **范式创新：** 提出“意图-执行”解耦的跨体态模仿新范式，从根本上绕开了动作级对齐的瓶颈，思路清晰且有效。
- **方法优雅：** Demo-JEPA框架设计巧妙，Dreamer Predictor架构（双交叉注意力 + 3D融合）合理建模了跨体态对应和运动演化。
- **实验完善：** 三套件评估体系设计出色，精准衡量了方法的泛化性能。消融实验覆盖全面，证据链完整，有力地支撑了论文的核心论点。缩放研究提供了有价值的工程见解。
- **物理世界验证：** 不仅在模拟器中表现优异，还在真实机器人上进行了验证，增强了结论的可信度和实用价值。

### 8. 不足与局限

1.  **世界模型瓶颈：** 论文明确指出当前方法的性能受限于动作条件世界模型本身的建模能力。在处理复杂、高精度任务时，世界模型的预测误差会累积并影响规划效果。
2.  **对齐需求：** 尽管避免了动作级对齐，但在训练Dreamer Predictor时，仍需要“进展感知”或“时间对齐”（Progress-aware Alignment/Frame-level Correspondence）来建立源和目标演示之间的帧对应关系，这在无控制、完全异步的数据采集中可能构成障碍。未来工作提到需要开发完全无对齐的学习策略。
3.  **计算开销：** 在推理时使用CEM进行在线规划带来了额外的计算成本，可能限制其在需要高实时性的场景中的应用。
4.  **实验规模：** 尽管真实世界任务涉及6个，每个任务20次rollout，但与一些大规模机器人学习基准相比，实验的绝对规模（尤其是真实世界部分）仍相对有限。Robustness的评估可以进一步拓展到更多样化的体态和环境。
5.  **配置的局限：** 方法未在动态环境或需要与人类紧密交互的场景下进行测试，其适应性和稳定性在这些情况下尚待验证。
