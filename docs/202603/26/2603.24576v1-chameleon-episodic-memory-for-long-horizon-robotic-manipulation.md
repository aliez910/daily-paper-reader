## 摘要
机器人操作通常需要记忆：遮挡和状态变化可能导致决策时的观测产生感知混淆，使得动作选择在观测层面呈现非马尔可夫特性，因为相同的观测可能源于不同的交互历史。大多数具身智能体通过语义压缩的轨迹和基于相似性的检索来实现记忆，这丢弃了用于消除歧义的细粒度感知线索，并可能返回感知上相似但与决策无关的情节。受人类情景记忆的启发，我们提出了 Chameleon，它通过写入基于几何定位的多模态 Token 来保留消除歧义的上下文，并通过可微记忆栈实现目标导向的回忆。我们还推出了 Camo-Dataset，这是一个涵盖感知混淆下的情景回忆、空间追踪和序列化操作的真实 UR5e 机器人数据集。在各项任务中，Chameleon 在感知易混淆的场景下，相较于强基线方法，一致地提高了决策可靠性和长程控制能力。

## Abstract
Robotic manipulation often requires memory: occlusion and state changes can make decision-time observations perceptually aliased, making action selection non-Markovian at the observation level because the same observation may arise from different interaction histories. Most embodied agents implement memory via semantically compressed traces and similarity-based retrieval, which discards disambiguating fine-grained perceptual cues and can return perceptually similar but decision-irrelevant episodes. Inspired by human episodic memory, we propose Chameleon, which writes geometry-grounded multimodal tokens to preserve disambiguating context and produces goal-directed recall through a differentiable memory stack. We also introduce Camo-Dataset, a real-robot UR5e dataset spanning episodic recall, spatial tracking, and sequential manipulation under perceptual aliasing. Across tasks, Chameleon consistently improves decision reliability and long-horizon control over strong baselines in perceptually confusable settings.

### 论文结构化深度总结：Chameleon

#### 1. 核心问题与整体含义（研究动机和背景）
论文聚焦于机器人长程操纵中的**感知歧义（Perceptual Aliasing）**问题。在复杂的现实任务中，机器人当前的观测往往具有非马尔可夫特性——即由于遮挡、物体移动或状态改变，不同历史背景下的当前视觉表现可能完全相同（例如在“壳牌游戏”中，交换后的杯子外观一致，仅凭当前视觉无法判断球在哪个杯下）。
*   **研究动机**：现有的具身智能体记忆机制通常依赖语义压缩（丢失细粒度几何信息）或基于相似性的检索（容易引入感知相似但决策无关的干扰）。
*   **核心目标**：受人类“情景记忆”启发，构建一种能够保留细粒度感知线索、实现目标导向检索的可微分记忆架构，以解决长程任务中的决策可靠性问题。

#### 2. 论文提出的方法论
Chameleon 采用了一种“感知 $\rightarrow$ 记忆 $\rightarrow$ 策略”的闭环架构，模拟大脑的内嗅皮层（EC）、海马体（HC）和前额叶皮层（PFC）功能。

*   **核心思想**：通过几何关联的多模态 Token 写入记忆，并利用可微分的层级记忆栈进行任务相关的上下文回溯。
*   **关键技术细节**：
    *   **感知层（双流架构）**：分为**腹侧流（Ventral）**提取 DINOv2 视觉特征，和**背侧流（Dorsal）**提取以末端执行器（EE）为锚点的几何特征。利用**对极几何偏置（Epipolar Bias）**实现多视角（前视与手眼相机）的特征融合。
    *   **记忆层（层级记忆栈）**：
        *   **情景记忆（Episodic Memory）**：使用槽位化（Slot-based）的 Mamba 选择性状态空间模型（SSM），按空间锚点和多时间尺度存储历史。
        *   **工作记忆（Working Memory）**：整合情景回溯内容，输出单一的决策状态 $h_t$。
    *   **HoloHead（潜在想象）**：通过一个辅助头预测未来的 2D/3D 航点，强制记忆状态 $h_t$ 具备“模式补全”能力，即从部分线索中恢复任务目标。
    *   **策略层**：基于**条件整流流匹配（Conditional Rectified Flow Matching）**，根据记忆状态 $h_t$ 一次性生成未来一段时界的末端位姿轨迹。

#### 3. 实验设计
*   **数据集与场景**：推出了 **Camo-Dataset**，包含在 UR5e 物理机器人上采集的 120 个演示视频。
*   **任务设置**：
    1.  **清理指定盘子**（情景回忆）：识别并记住最初被污染的盘子。
    2.  **壳牌游戏**（空间追踪）：在杯子多次交换位置后追踪隐藏物体。
    3.  **添加调料**（序列操作）：按特定顺序使用勺子，需记住已完成的步骤。
*   **对比基准（Baselines）**：
    *   **Diffusion Policy**（扩散策略）
    *   **Flow Matching**（流匹配）
    *   **ACT**（动作分块 Transformer）
*   **消融实验**：对比了无记忆、基于检索的记忆库（Memory Bank）、原生 Mamba、无背侧几何流、无 HoloHead 等变体。

#### 4. 资源与算力
*   **硬件**：单张 **NVIDIA RTX 4090 GPU**。
*   **训练时长**：每个任务训练 20,000 步（20K steps）。
*   **推理性能**：端到端推理延迟平均为 **82ms**，满足实时操纵需求。

#### 5. 实验数量与充分性
*   **实验规模**：每个任务类别进行了 36 次独立的真实机器人测试，总计超过 100 次物理实验。
*   **充分性评价**：实验设计较为充分且客观。作者不仅报告了总成功率（SR），还细分了**决策成功率（DSR）**和**操纵成功率（MSR）**，并引入了 Cohen’s kappa 系数来排除随机猜测的影响。消融实验覆盖了架构的各个核心组件，验证了每一部分的必要性。

#### 6. 主要结论与发现
*   **决策可靠性显著提升**：在感知歧义严重的任务中，Chameleon 的决策成功率远超基线（如在情景回忆任务中达到 100% DSR，而基线仅约 30%）。
*   **记忆结构的必要性**：简单的循环神经网络（如原生 Mamba）在长程序列中容易产生干扰，而 Chameleon 的槽位化结构能有效分离干扰。
*   **几何引导的作用**：背侧流提供的几何锚点对于空间追踪任务至关重要，移除后性能大幅下降。
*   **表征分离**：通过 UMAP 可视化发现，Chameleon 学习到的记忆状态在感知相似但决策不同的时刻能清晰地聚类分离。

#### 7. 优点（亮点）
*   **生物启发性强**：将神经科学中的情景记忆机制（模式分离与补全）成功转化为可工程化的深度学习模块。
*   **解决非马尔可夫难题**：针对机器人领域长期存在的感知歧义问题提供了系统性的解决方案。
*   **端到端可微**：不同于传统的 RAG 或显式数据库检索，其记忆读写完全集成在模型动力学中。
*   **高质量数据集**：Camo-Dataset 为评估机器人记忆能力提供了极具挑战性的 Benchmark。

#### 8. 不足与局限
*   **缺乏语义泛化**：目前主要依赖视觉和几何特征，未与大语言模型（LLM）或视觉语言模型（VLM）深度结合，在处理抽象指令或零样本迁移方面可能受限。
*   **单一平台验证**：实验仅在 UR5e 机器人上完成，尚未验证在不同构型机器人或移动操纵平台上的跨具身泛化性。
*   **操作精度波动**：虽然决策极准，但在某些精细操作（如抓取瞬间）的成功率（MSR）有时略低于专门优化动作分布的扩散策略。
