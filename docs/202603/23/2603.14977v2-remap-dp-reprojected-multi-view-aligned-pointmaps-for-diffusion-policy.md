## 摘要
基于二维视觉表示的通用机器人策略在语义推理方面表现出色，但本质上缺乏高精度任务所需的显式三维空间感知。由于稀疏点云的结构不规则性以及多视图正交渲染引入的几何畸变，现有的三维集成方法难以弥合这一差距。为了克服这些障碍，我们提出了 ReMAP-DP，这是一个将标准化透视重投影与结构感知双流扩散策略相结合的新型框架。通过将重投影视图与像素对齐的点图（PointMaps）耦合，我们的双流架构利用可学习的模态嵌入来融合冻结的语义特征和显式几何描述符，确保了精确的隐式补丁级（patch-level）对齐。在仿真和真实环境中的广泛实验证明了 ReMAP-DP 在多种操作任务中的卓越性能。在 RoboTwin 2.0 上，它达到了 59.3% 的平均成功率，比 DP3 基准提高了 6.6%。在 ManiSkill 3 上，我们的方法在具有几何挑战性的堆叠方块（Stack Cube）任务中比 DP3 提高了 28%。此外，ReMAP-DP 表现出卓越的现实世界鲁棒性，仅需少量演示即可实现高精度和动态操作，具有极高的数据效率。项目页面见：https://icr-lab.github.io/ReMAP-DP/

## Abstract
Generalist robot policies built upon 2D visual representations excel at semantic reasoning but inherently lack the explicit 3D spatial awareness required for high-precision tasks. Existing 3D integration methods struggle to bridge this gap due to the structural irregularity of sparse point clouds and the geometric distortion introduced by multi-view orthographic rendering. To overcome these barriers, we present ReMAP-DP, a novel framework synergizing standardized perspective reprojection with a structure-aware dual-stream diffusion policy. By coupling the re-projected views with pixel-aligned PointMaps, our dual-stream architecture leverages learnable modality embeddings to fuse frozen semantic features and explicit geometric descriptors, ensuring precise implicit patch-level alignment. Extensive experiments across simulation and real-world environments demonstrate ReMAP-DP's superior performance in diverse manipulation tasks. On RoboTwin 2.0, it attains a 59.3% average success rate, outperforming the DP3 baseline by +6.6%. On ManiSkill 3, our method yields a 28% improvement over DP3 on the geometrically challenging Stack Cube task. Furthermore, ReMAP-DP exhibits remarkable real-world robustness, executing high-precision and dynamic manipulations with superior data efficiency from only a handful of demonstrations. Project page is available at: https://icr-lab.github.io/ReMAP-DP/

#### 1. 核心问题与整体含义（研究动机和背景）
当前的机器人通用策略（如 VLA 模型）主要依赖 2D 视觉表示，虽然语义推理能力强，但缺乏高精度操作所需的 **3D 空间感知**。现有的 3D 集成方案存在明显缺陷：
*   **稀疏点云（Sparse Point Clouds）**：结构不规则，难以直接适配标准的视觉架构（如 ViT）。
*   **正交渲染（Orthographic Rendering）**：虽然能提供多视图，但会产生几何畸变，破坏了预训练 2D 视觉大模型（如 DINOv2）的透视先验。
*   **模态融合困难**：简单的通道拼接无法在 Token 级别实现语义（“是什么”）与几何（“在哪里”）的精确对齐。

**ReMAP-DP** 旨在通过“透视重投影”和“双流对齐架构”，在保留 2D 语义先验的同时，引入精确的 3D 几何结构。

#### 2. 方法论：核心思想与关键技术
ReMAP-DP 提出了一种结合重投影多视图与扩散策略的框架，核心流程如下：
*   **工作空间多视图投影（Workspace Multi-View Projection）**：
    *   将原始 RGB-D 输入提升为全局点云，再通过**透视投影**（而非正交投影）渲染出虚拟相机的 RGB 图像和**点图（PointMap）**。
    *   **点图**：一种 2D 矩阵，每个像素存储对应的 (x, y, z) 绝对度量坐标。这确保了 RGB 图像与几何坐标在像素级完全对齐。
*   **双流观测编码（Dual-Stream Encoding）**：
    *   **语义流**：使用冻结的 **DINOv2 (ViT-S/14)** 提取 Patch 级特征，保留空间拓扑。
    *   **几何流**：使用可训练的 **Projective PointMap ViT** 处理点图，其 Patch 大小与语义流一致，实现隐式空间对齐。
*   **多模态 Transformer 融合**：
    *   引入**可学习模态嵌入（Modality Embeddings）**，通过 Cross-Modal Transformer 融合语义 Token 和几何 Token。
*   **条件扩散动作生成**：
    *   使用 **1D Temporal U-Net** 作为去噪网络，通过 **FiLM（特征线性调制）** 层将融合后的观测特征注入，迭代生成机器人动作轨迹。

#### 7. 优点（亮点）
*   **架构优雅**：通过 Patch 级对齐巧妙解决了 2D 语义与 3D 几何的融合难题，无需复杂的坐标变换学习。
*   **视角不变性**：通过虚拟相机重投影，使策略对物理相机的安装位置不敏感，增强了系统的通用性。

*   **高精度瓶颈**：在极高精度的约束任务（如 Peg Insertion 销钉插入）中，所有方法成功率均极低（<1%），说明仅靠视觉反馈仍难以处理此类任务。
