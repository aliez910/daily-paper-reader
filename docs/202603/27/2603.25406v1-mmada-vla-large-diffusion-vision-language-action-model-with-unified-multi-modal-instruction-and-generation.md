## 摘要
视觉-语言-动作（VLA）模型旨在根据视觉观测和自然语言指令控制机器人进行操作。然而，现有的分层和自回归范式通常会引入架构开销，面临时间不一致性和长时程误差累积的问题，且缺乏在不增加额外模块的情况下捕获环境动力学的机制。为此，我们提出了 MMaDA-VLA，这是一个完全原生的预训练扩散 VLA 大模型，在单一框架中统一了多模态理解与生成。我们的核心思想是一种原生的离散扩散公式，将语言、图像和连续机器人控制嵌入到统一的离散 token 空间中，并通过掩码 token 去噪训练单个骨干网络，从而并行地联合生成未来目标观测和动作块。迭代去噪实现了全局、无序的细化，在无需辅助世界模型的情况下，将动作锚定在预测的未来视觉结果中，提升了长时程一致性。在仿真基准和真实世界任务中的实验表明，该模型达到了最先进的性能，在 LIBERO 上实现了 98.0% 的平均成功率，在 CALVIN 上达到了 4.78 的平均长度。

## Abstract
Vision-Language-Action (VLA) models aim to control robots for manipulation from visual observations and natural-language instructions. However, existing hierarchical and autoregressive paradigms often introduce architectural overhead, suffer from temporal inconsistency and long-horizon error accumulation, and lack a mechanism to capture environment dynamics without extra modules. To this end, we present MMaDA-VLA, a fully native pre-trained large diffusion VLA model that unifies multi-modal understanding and generation in a single framework. Our key idea is a native discrete diffusion formulation that embeds language, images, and continuous robot controls into one discrete token space and trains a single backbone with masked token denoising to jointly generate a future goal observation and an action chunk in parallel. Iterative denoising enables global, order-free refinement, improving long-horizon consistency while grounding actions in predicted future visual outcomes without auxiliary world models. Experiments across simulation benchmarks and real-world tasks show state-of-the-art performance, achieving 98.0% average success on LIBERO and 4.78 average length on CALVIN.

*   **核心挑战**：现有模型大多缺乏显式的环境动力学建模（即预测未来视觉状态的能力），且在处理多步动作序列时，自回归的顺序依赖限制了全局轨迹规划。
*   **论文目标**：提出 **MMaDA-VLA**，一个原生的、预训练的大型离散扩散模型，旨在单一框架内统一多模态理解与生成，通过并行预测未来观测和动作块来提升长程任务的连贯性。

### 2. 方法论
*   **核心思想**：将语言指令、图像观测和连续机器人动作全部映射到同一个**离散 Token 空间**，并使用**掩码 Token 去噪（Masked Token Denoising）**目标训练一个统一的骨干网络。
*   **关键技术细节**：
    *   **统一 Token 化**：文本使用 LLaDA 分词器，图像使用 MAGVIT-v2 量化器，动作则将每个维度离散化为 256 个 Bin。
    *   **并行生成架构**：模型同时输入当前观测和指令，并行生成“未来目标观测（Goal Obs）”和“动作块（Action Chunk）”。这种设计让动作生成能够隐式地参考预测的未来视觉演变。
    *   **混合注意力机制（Hybrid Attention）**：模态内部采用双向全注意力（允许全局信息交换），模态之间采用因果注意力（确保信息流向生成目标）。
    *   **迭代去噪推理**：在推理阶段，通过多次迭代（如 24 步）从掩码状态恢复 Token。为了提高实时性，引入了**训练无关的缓存框架（KV Cache）**，仅在去噪步骤间更新变化显著的 Token。

### 6. 主要结论与发现
*   **性能领先**：MMaDA-VLA 在 LIBERO 上达到 **98.0%** 的平均成功率，在 CALVIN 上达到 **4.78** 的平均完成长度（满分 5），均刷新了纪录。

### 7. 优点
*   **架构简洁统一**：无需额外的策略头或独立的世界模型模块，降低了系统复杂性。
*   **无序性优势**：扩散模型避免了为本质上无序的动作维度强加自回归顺序，减少了归纳偏置带来的错误。
*   **高效推理**：通过自适应缓存机制解决了扩散模型推理慢的顽疾，使其能满足机器人实时控制的需求。

### 8. 不足与局限
*   **视觉生成质量**：生成的未来观测图像在精细细节（如夹爪闭合状态、微小物体形状）上存在模糊，可能限制极高精度任务的表现。
*   **离散化损失**：虽然使用了 256 个 Bin，但离散化动作在处理极度平滑的连续轨迹时可能仍不如连续回归方法。
*   **推理开销**：尽管有缓存优化，但迭代去噪的计算成本依然高于单次前向传播的自回归模型。
