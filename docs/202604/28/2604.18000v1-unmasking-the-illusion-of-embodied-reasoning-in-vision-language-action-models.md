## Abstract
Recent Vision-Language-Action (VLA) models report impressive success rates on standard robotic benchmarks, fueling optimism about general-purpose physical intelligence. However, recent evidence suggests a systematic misalignment between standard benchmark success and true embodied reasoning, raising the question of whether these high scores reflect genuine cognitive capability. To address this gap, we introduce BeTTER, a diagnostic Benchmark for Testing True Embodied Reasoning in robotic policies. BeTTER applies targeted causal interventions (e.g., spatial layout shifts, temporal extrapolation) while enforcing kinematic isolation to explicitly decouple high-level reasoning failures from low-level execution limits. Through systematic evaluation, we reveal that state-of-the-art VLAs catastrophically fail in dynamic scenarios, exhibiting severe lexical-kinematic shortcuts, behavioral inertia, and semantic feature collapse. Crucially, our mechanistic analysis traces these symptoms to fundamental architectural bottlenecks - such as capacity compression and myopic downsampling - which systematically degrade the model's foundational semantic representation. We demonstrate that highly static evaluation protocols effectively mask this degradation by allowing optimization to overfit to sensorimotor priors. Supported by real-world robotic validation, our findings confirm that this representational breakdown is not a simulation artifact, highlighting the critical need for future VLA paradigms to resolve the structural tension between high-frequency control and high-level reasoning.

### 1. 核心问题与整体含义（研究动机和背景）
*   **研究动机**：尽管当前的 VLA 模型（如 RT-2, $\pi_0$ 等）在标准机器人基准测试中取得了极高的成功率，但它们在面对环境微调或动态变化时往往表现脆弱。
*   **核心问题**：高成功率是否代表模型具备了真正的“具身推理”（即理解复杂指令、进行物理环境推理并跨任务适应技能的能力）？作者怀疑这些模型实际上是利用了训练数据中的统计捷径（如视觉-运动模式的过拟合），而非真正的语义理解。
*   **整体含义**：论文旨在通过构建更严苛的诊断性基准，拆解模型在感知、规划和执行层面的失败模式，并从架构设计角度解释推理能力退化的根源。

### 2. 论文提出的方法论
作者提出了 **BeTTER** (Benchmark for Testing True Embodied Reasoning) 诊断基准，其核心思想是通过**因果干预**和**运动隔离**来解耦高层推理失败与底层执行限制。

*   **核心思想**：如果一个动作在物理上是可执行的（运动隔离），但模型失败了，那么失败必然源于高层推理（如语义理解或状态跟踪）。
*   **关键技术细节（四类干预）**：
    1.  **空间布局偏移 (Spatial Layout Shift)**：改变物体位置或引入干扰项，测试模型对分布外配置的适应性。
    2.  **原语重组 (Primitive Recomposition)**：将已学动作原语以新顺序组合（如从 A→B, A→C 泛化到 B→C），测试零样本组合能力。
    3.  **对抗性物体扰动 (Adversarial Object Perturbation)**：用视觉相似但语义不同的物体替换目标，测试语义对齐。
    4.  **时间外推 (Temporal Extrapolation)**：延长任务链或引入重复动作，测试因果状态跟踪。
*   **自动化流水线**：利用 VLM（如 Gemini）根据模板生成任务描述，从 Objaverse 检索 3D 资产，并利用特权仿真状态生成无幻觉的 VQA 数据。

### 3. 实验设计
*   **实验场景**：包含 10 个基础操纵任务（如准备咖啡、整理桌面、组装汉堡等），扩展为 60 个受控变体。
*   **对比方法**：评估了三个代表性 SOTA VLA 模型：**$\pi_{0.5}$**、**GR00T-N1.6** 和 **Being-H0.5**。
*   **Benchmark 对比**：除了 BeTTER，还在 LIBERO（静态基准）和 CALVIN（长程任务基准）上进行了消融实验，以验证模型在分布内外的表现差异。
*   **真实世界验证**：在 SO101 机器人平台上进行物理压力测试，验证仿真发现的普遍性。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了仿真环境下的 60 种变体测试，以及真实世界中针对短程和长程任务的因果干预矩阵测试。
*   **消融实验**：非常充分。作者从模型容量（8B vs 2B）、预训练数据配比（VLA vs VLM）、感知约束（多 Patch vs 单 224px 输入）三个维度系统分析了推理能力如何一步步退化。
*   **客观性与公平性**：通过“运动隔离”确保了对比的公平性，避免了因机器人硬件执行能力不同而导致的干扰；引入 LLM 作为公正的答案提取器来评估 VLM 基准。

### 6. 主要结论与发现
1.  **推理幻觉**：SOTA 模型在动态场景下表现崩溃，存在**词汇-运动捷径**（看到特定词汇直接触发动作而不看环境）和**行为惯性**（倾向于重复训练中最常见的起始动作）。
2.  **因果状态跟踪失败**：模型往往依赖视觉表面特征而非因果逻辑，容易在初始状态改变时产生“幻觉”，跳过必要步骤。
3.  **表征退化根源**：
    *   **容量压缩**：为了实时控制将模型缩小（如 8B→2B）会严重损害推理带宽。
    *   **“近视”感知**：为了高频控制而采用低分辨率单图输入，导致细粒度语义特征崩溃。
4.  **基准测试悖论**：静态基准（如 LIBERO）的高分掩盖了表征退化，因为模型可以通过过拟合传感器运动先验（$z_a$）来弥补认知表征（$z$）的缺失。

### 7. 优点
*   **诊断精准**：通过“运动隔离”和“因果干预”精准定位了 VLA 模型的认知缺陷，而非笼统地归咎于“泛化性差”。
*   **机制分析深入**：不仅指出了问题，还通过消融实验从架构和数据流角度给出了“为什么推理能力会消失”的解释。
*   **真机验证**：实验不仅停留在仿真，真机上的“空气抓取”等失败案例极具说服力。

### 8. 不足与局限
*   **任务范畴**：主要集中在桌面操纵任务，对于移动导航或更复杂的全身协调任务涉及较少。
*   **实时性权衡**：虽然指出了高频控制与高层推理的矛盾，但尚未提出一个完美的架构方案来同时兼顾两者（尽管提出了未来方向）。
*   **样本量**：真机实验的轨迹数量（100 条）相对较小，虽然足以诊断模式，但在统计严谨性上弱于仿真。
