## 摘要
具身视觉追踪（EVT）是具身智能中的核心动态任务，要求智能体精确跟随语言指定的任务目标。然而，现有大多数方法依赖于单智能体模仿学习，面临专家数据成本高昂以及由于静态训练环境导致的泛化能力有限等问题。受竞争驱动的能力演化启发，我们提出了 CoMaTrack，这是一个竞争性博弈论多智能体强化学习框架，在具有竞争性子任务的动态对抗环境中训练智能体，从而产生更强的自适应规划和抗干扰策略。我们进一步推出了 CoMaTrack-Bench，这是首个基于 Habitat 的开源基准协议和剧集集，用于具有动态对决特征的语言条件竞争性 EVT，涵盖了追踪者与自适应对手在多样化环境和指令下的博弈场景，实现了主动对抗交互下的标准化鲁棒性评估。实验表明，CoMaTrack 在标准基准测试和 CoMaTrack-Bench 上均取得了当前最先进的结果。值得注意的是，使用我们框架训练的 3B VLM 在具有挑战性的 EVT-Bench 上超越了此前基于 7B 模型的单智能体模仿学习方法，在 STT、DT 和 AT 指标上分别达到了 92.1%、74.2% 和 57.5%。基准测试代码将在 https://github.com/wlqcode/CoMaTrack-Bench 提供。

## Abstract
Embodied Visual Tracking (EVT), a core dynamic task in embodied intelligence, requires an agent to precisely follow a language-specified target. Yet most existing methods rely on single-agent imitation learning, suffering from costly expert data and limited generalization due to static training environments. Inspired by competition-driven capability evolution, we propose CoMaTrack, a competitive game-theoretic multi-agent reinforcement learning framework that trains agents in a dynamic adversarial setting with competitive subtasks, yielding stronger adaptive planning and interference-resilient strategies. We further introduce CoMaTrack-Bench, the first open-source Habitat-based benchmark protocol and episode set for language-conditioned competitive EVT featuring dynamic dueling, featuring game scenarios between a tracker and adaptive opponents across diverse environments and instructions, enabling standardized robustness evaluation under active adversarial interactions. Experiments show that CoMaTrack achieves state-of-the-art results on both standard benchmarks and CoMaTrack-Bench. Notably, a 3B VLM trained with our framework surpasses previous single-agent imitation learning methods based on 7B models on the challenging EVT-Bench, achieving 92.1% in STT, 74.2% in DT, and 57.5% in AT. The benchmark code will be available at https://github.com/wlqcode/CoMaTrack-Bench.

*   **核心问题**：具身视觉追踪（EVT）要求机器人根据语言指令在复杂动态环境中持续跟随目标。现有的主流方法主要依赖**单智能体模仿学习（IL）**，这导致了两个瓶颈：

    2.  **泛化性差**：训练环境往往是静态的，智能体难以应对分布外（OOD）场景、主动避让的目标或复杂的动态干扰。
*   **研究动机**：受生物进化中“竞争驱动能力演化”的启发，作者认为通过引入对抗性的竞争对手，可以迫使追踪智能体在不断升级的难度中学习到更具鲁棒性的预测、规划和抗干扰策略。

### 2. 方法论
*   **核心思想**：将 EVT 任务建模为**竞争性多智能体博弈**。通过让追踪者（Tracker）与一个自适应的对手（Opponent）在同一环境中博弈，构建一个自动升级的课程学习（Curriculum Learning）过程。
*   **关键技术细节**：
    *   **架构**：基于 **Qwen2.5VL-3B** 视觉语言模型（VLM），并在其基础上增加了一个**流匹配（Flow-matching）动作头**，用于生成连续的多模态未来轨迹。
    *   **两阶段训练**：
        1.  **监督微调（SFT）**：在混合了导航、追踪和 VQA 的大规模数据集上进行预训练，建立基础的感知和指令遵循能力。
        2.  **多智能体强化学习（MARL）**：使用 **GRPO（群体相对策略优化）** 算法。追踪者和对手共享模型架构，但奖励函数不对称。
    *   **不对称奖励设计**：
        *   **追踪者奖励**：鼓励保持安全距离（1-3m）、目标在视野内、避免碰撞。
        *   **对手奖励**：鼓励对手更激进地靠近目标（目标距离设为更近的 1.25m），从而通过物理遮挡或路径交叉来干扰追踪者。
    *   **输入表征**：融合了多视角当前观测（精细 Token）和历史视觉序列（粗糙 Token），以平衡空间精度和长程上下文。

### 3. 实验设计
*   **数据集与场景**：
    *   **EVT-Bench**：包含单目标追踪（STT）、干扰追踪（DT）和歧义追踪（AT）三个子任务。
    *   **CoMaTrack-Bench（新提出）**：基于 Habitat 3.0 构建，引入了三种对手行为：静态障碍、随机干扰、竞争性追踪。
*   **对比方法**：对比了包括 IBVS、PoliFormer、Uni-NaVid、TrackVLA、TrackVLA++ 等在内的十余种 SOTA 方法。
*   **评估指标**：成功率（SR）、追踪率（TR）、碰撞率（CR）。

### 4. 资源与算力
*   **训练算力**：
    *   **SFT 阶段**：使用 **48 台 NVIDIA H20 GPU** 集群进行 1 个 epoch 的训练。
    *   **RL 阶段**：使用 **4 台 NVIDIA L20 GPU** 进行 1 个 epoch 的微调（仅训练 LoRA 参数）。
*   **部署算力**：实机部署在 Unitree Go2 X 四足机器人上，远程推理服务器配备 **NVIDIA RTX 4090**。

### 5. 实验数量与充分性
*   **实验规模**：
    *   在标准 EVT-Bench 的三个任务上进行了全面测试。
    *   在自建的 CoMaTrack-Bench 上进行了零样本（Zero-shot）性能评估。
    *   **消融实验**：对比了 SFT 模型、单智能体 RL 和多智能体 RL 的效果，并分析了不同强度对手对训练结果的影响。
*   **充分性评价**：实验设计较为充分，涵盖了从仿真到实机的迁移验证。通过对比不同参数规模（3B vs 7B）的模型，证明了算法框架（竞争博弈）比单纯堆砌模型规模更有效，实验结果具有较强的说服力。

### 6. 主要结论与发现
*   **性能突破**：CoMaTrack 训练的 **3B 模型** 在 EVT-Bench 上的表现全面超越了此前基于 **7B 模型** 的方法（如 TrackVLA++），SR 分别达到 92.1% (STT)、74.2% (DT) 和 57.5% (AT)。
*   **竞争的价值**：多智能体对抗训练显著降低了碰撞率（从 3.1% 降至 0.9%），证明智能体学会了更高级的空间推理和避障策略。
*   **对手强度的影响**：实验发现，对手越强（从静态到竞争性追踪），训练出的追踪者鲁棒性越高，验证了“对手即环境”的逻辑。

### 7. 优点与亮点
*   **范式创新**：首次将竞争性多智能体博弈论 RL 与现代 VLA 模型结合用于具身追踪，摆脱了对昂贵专家数据的过度依赖。
*   **高效性**：证明了较小规模的 VLM（3B）通过策略进化可以战胜大规模模型（7B），降低了部署门槛。
*   **开源贡献**：发布了首个标准化的竞争性 EVT 基准测试集 CoMaTrack-Bench，填补了行业空白。
*   **实机验证**：完成了从仿真到真实四足机器人的零样本迁移，展示了极高的实用价值。

### 8. 不足与局限
*   **任务覆盖面**：虽然在追踪任务上表现优异，但尚未在更广泛的具身任务（如复杂物体搬运、多层级指令导航）上进行大规模验证。
