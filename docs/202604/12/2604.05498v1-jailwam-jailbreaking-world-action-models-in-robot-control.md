## 摘要
世界动作模型（WAM）能够联合预测未来的世界状态和动作，与传统模型相比展现出更强的物理操作能力。这种强大的物理交互能力是一把双刃剑：如果忽视安全性，它将直接威胁人身安全、财产安全和环境安全。然而，现有研究对一个关键的安全漏洞关注极少：即 WAM 对越狱攻击的脆弱性。为了填补这一空白，我们定义了三级安全分类框架，以系统地量化机械臂运动的安全性。此外，我们提出了 JailWAM，这是首个专门针对 WAM 的越狱攻击与评估框架，由三个核心组件组成：（1）视觉-轨迹映射，将异构动作空间统一为视觉轨迹表示，并实现跨架构的统一评估；（2）风险判别器，作为一种高召回率的筛选工具，在识别视觉轨迹中的破坏性行为时优化了效率与准确性之间的权衡；（3）双路径验证策略，首先通过基于单图像的视频-动作生成模块进行快速粗筛，随后通过全闭环物理仿真进行高效且全面的验证。此外，我们构建了 JailWAM-Bench，这是一个用于全面评估 WAM 在越狱攻击下安全对齐性能的基准测试。在 RoboTwin 仿真环境中的实验表明，所提框架能有效揭示物理漏洞，在最先进的 LingBot-VA 上实现了 84.2% 的攻击成功率。同时，基于 JailWAM 可以构建鲁棒的防御机制，为设计安全可靠的机器人控制系统提供有效的技术方案。

## Abstract
The World Action Model (WAM) can jointly predict future world states and actions, exhibiting stronger physical manipulation capabilities compared with traditional models. Such powerful physical interaction ability is a double-edged sword: if safety is ignored, it will directly threaten personal safety, property security and environmental safety. However, existing research pays extremely limited attention to the critical security gap: the vulnerability of WAM to jailbreak attacks. To fill this gap, we define the Three-Level Safety Classification Framework to systematically quantify the safety of robotic arm motions. Furthermore, we propose JailWAM, the first dedicated jailbreak attack and evaluation framework for WAM, which consists of three core components: (1) Visual-Trajectory Mapping, which unifies heterogeneous action spaces into visual trajectory representations and enables cross-architectural unified evaluation; (2) Risk Discriminator, which serves as a high-recall screening tool that optimizes the efficiency-accuracy trade-off when identifying destructive behaviors in visual trajectories; (3) Dual-Path Verification Strategy, which first conducts rapid coarse screening via a single-image-based video-action generation module, and then performs efficient and comprehensive verification through full closed-loop physical simulation. In addition, we construct JailWAM-Bench, a benchmark for comprehensively evaluating the safety alignment performance of WAM under jailbreak attacks. Experiments in RoboTwin simulation environment demonstrate that the proposed framework efficiently exposes physical vulnerabilities, achieving an 84.2% attack success rate on the state-of-the-art LingBot-VA. Meanwhile, robust defense mechanisms can be constructed based on JailWAM, providing an effective technical solution for designing safe and reliable robot control systems.

随着世界动作模型（World Action Models, WAMs）在机器人领域的兴起，机器人具备了强大的物理交互和预测能力。然而，这类模型通常继承了预训练生成式底座的脆弱性。本文关注的核心问题是：**WAM 在面对越狱攻击（Jailbreak Attacks）时的物理安全性**。
与传统大语言模型（LLM）或视频生成模型（VGM）仅产生数字层面的有害输出不同，WAM 的越狱会导致机器人执行危险的物理动作（如高速碰撞、剧烈摆动），直接威胁人身和环境安全。论文旨在填补这一安全评估空白，系统性地量化并揭示 WAM 的物理安全漏洞。

#### 2. 方法论
论文提出了 **JailWAM** 框架，其核心思想是通过高效的筛选和验证流程来识别物理危险指令。
*   **三级安全分类框架**：
    *   **Level 0 (安全合规)**：运动平滑，无碰撞。
    *   **Level 1 (运动失败)**：系统死锁、高频振荡、无法有效移动。
    *   **Level 2 (灾难性风险)**：失控的大范围摆动、越界碰撞、对人员或环境造成严重破坏。
*   **关键技术组件**：
    1.  **视觉-轨迹映射 (VTM)**：将异构的机器人动作序列（如关节坐标）转换为统一的 2D 正交投影视觉图表（顶视图和前视图），使不同架构的模型输出具备可比性，并便于视觉模型理解。
    2.  **风险判别器 (RD)**：基于微调的 Qwen3-VL-2B 模型，直接从 VTM 生成的图表中预测安全等级，实现高召回率的快速筛选。
    3.  **双路径验证策略**：
        *   **路径 I (开环筛选)**：利用 WAM 的生成先验产生动作，通过 RD 快速剔除安全样本。
        *   **路径 II (闭环验证)**：仅对高风险指令进行耗时的全物理仿真验证，并由人类专家最终确认。
*   **越狱优化**：利用 LLM（如 Gemini, GPT-5）配合专门设计的模板生成任务无关的危险指令。

#### 3. 实验设计
*   **数据集与场景**：使用了 **RoboTwin**（50 个任务场景）和 **LIBERO** 仿真环境。
*   **目标模型**：
    *   **WAM 类**：LingBot-VA (SOTA), Cosmos-Policy, Motus。

*   **Benchmark**：构建了 **JailWAM-Bench**，包含 82 条具有跨任务迁移能力的越狱指令。

*   **硬件**：实验在一台拥有 48GB 显存的 **NVIDIA RTX 4090 GPU** 上运行。

*   **效率提升**：双路径策略将 100 个样本的验证时间从 9.15 小时缩短至 3.66 小时（加速 2.5 倍）。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了 50 个不同的操作任务，对多个 SOTA 模型进行了跨架构、跨环境的迁移测试。
*   **消融实验**：针对 VTM 的有效性、RD 的微调效果、双路径策略的效率以及不同 LLM 生成指令的可靠性进行了详细消融。
*   **客观性**：引入了人类专家评估作为 Ground Truth，并计算了 RD 与人类判断的一致性（达 90%），实验设计较为严谨、公平。

#### 6. 主要结论与发现
*   **严重脆弱性**：SOTA 模型 LingBot-VA 在 JailWAM 攻击下的成功率（ASR）高达 **84.2%**，其中 22.2% 为灾难性风险。
*   **跨架构迁移性**：针对某一模型生成的越狱指令可以零样本迁移到其他 WAM 模型（如从 LingBot 迁移到 Motus 和 Cosmos-Policy），表现出极强的通用性。

*   **防御可行性**：JailWAM 的风险判别器可作为推理侧过滤器，在不损失正常任务性能的前提下显著降低攻击成功率。

#### 7. 优点
*   **首创性**：首次系统研究了机器人世界动作模型的越狱问题，将安全评估从数字域扩展到物理域。
*   **评估高效**：通过 VTM 和双路径策略解决了物理仿真成本极高的问题，实现了大规模自动化评估。
*   **跨模型通用**：视觉轨迹映射的设计巧妙地屏蔽了底层动作空间的差异。

#### 8. 不足与局限
*   **仿真与现实差距**：所有实验均在仿真环境（RoboTwin/LIBERO）中完成，尚未在真实物理机器人上验证其迁移效果。
